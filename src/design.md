# SSourcing China - Design System

## Visual Style
Professional, clean, trustworthy B2B design for international sourcing services.

### Color Palette
- Primary: #0F172A (Slate 900) - Deep navy for headers and CTAs
- Secondary: #1E40AF (Blue 800) - Trust blue for accents and links
- Accent: #0EA5E9 (Sky 500) - Light blue for highlights
- Neutral: #F8FAFC (Slate 50) - Light background
- Text: #1E293B (Slate 800) - Primary text
- Muted: #64748B (Slate 500) - Secondary text
- Success: #059669 (Emerald 600) - Trust indicators
- Border: #E2E8F0 (Slate 200) - Subtle dividers

### Typography
- Font: Inter (system-ui fallback)
- Headings: 700 weight, tight tracking
- Body: 400 weight, 1.6 line-height
- Small text: 500 weight for labels

### Spacing
- Section padding: 80px vertical (desktop), 48px (mobile)
- Container max-width: 1280px
- Card gap: 24px
- Consistent 8px grid system

### Components
- Buttons: Rounded corners (6px), 48px height, clear hover states
- Cards: White background, subtle shadow, 1px border
- Forms: Clean inputs with focus states, proper labels
- Navigation: Fixed header, clear active states

### Visual Elements
- Factory/QC/shipping imagery via stock image system
- Icons: Lucide React (consistent 24px stroke)
- Trust badges: Clean, minimal
- Process steps: Numbered with connecting lines

### Responsive
- Mobile-first approach
- Breakpoints: 640px, 1024px
- Touch-friendly interactive elements (min 44px)
