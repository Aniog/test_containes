# ARTITECT MACHINERY Design Guidelines

## Brand
- Name: ARTITECT MACHINERY
- Industry: Industrial machinery (sheet metal folding machines)
- Tone: Elegant, sophisticated, professional, user-friendly

## Colors
- Primary: Deep navy blue (#0a1628) - used for headers, footers, main backgrounds
- Primary Light: (#1a2a4a) - used for hover states, secondary elements
- Accent: Warm gold (#c8a45c) - used for CTAs, highlights, decorative elements
- Accent Light: (#e0c882) - used for hover states on accent elements
- Background: Off-white (#f8f7f4) - main page background
- Card Background: White (#ffffff) - for product cards, sections
- Text Primary: #1a1a2e - main body text
- Text Secondary: #5a6378 - secondary text
- Border: #e2e0dc - light borders

## Typography
- Headings: 'Inter', sans-serif (weights: 600, 700, 800)
- Body: 'Inter', sans-serif (weights: 300, 400, 500)
- Font Sizes: h1: 3.5rem, h2: 2.5rem, h3: 1.5rem, body: 1rem

## Spacing
- Section padding: py-20 md:py-28
- Container max-width: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8
- Card gap: gap-8
- Element spacing: space-y-6, space-y-4

## Visual Styles
- Rounded corners: rounded-lg (8px) for cards, rounded-full for buttons
- Shadows: subtle shadow-md for cards, shadow-sm for buttons
- Transitions: all transitions 300ms ease
- Borders: thin, subtle borders (border border-gray-100)
- Cards: white background, subtle shadow, hover lift effect

## Do's
- Use plenty of white space
- Use consistent gold accent on CTAs
- Use high-quality product imagery
- Keep navigation simple and clear
- Use professional, confident language
- Showcase machinery with detailed specs
- Use subtle animations on scroll

## Don'ts
- Don't use bright, flashy colors
- Don't overcrowd sections
- Don't use overly technical jargon without explanation
- Don't use low-contrast text
- Don't use cluttered layouts
- Don't mix multiple accent colors
