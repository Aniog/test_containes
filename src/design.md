# SSourcing China - Design Guide

## Typography
- Font family: Inter (sans-serif)
- Headings: Bold, tight leading
- Body: Regular weight, comfortable line-height
- Use font-sans utility class

## Color Palette
- Brand primary: brand-600 (#146eff) - used for CTAs, links, accents
- Brand hover: brand-700 (#0d59f0)
- Brand light: brand-50 (#eef7ff) - used for section backgrounds
- Neutral text: neutral-900 (#111827) - primary text
- Neutral muted: neutral-500 (#6b7280) - secondary/subtitle text
- Background: white (#ffffff) - main bg
- Section alt bg: gray-50 (#f9fafb) - alternating section backgrounds
- Success/trust: green-600 - for checkmarks, trust indicators
- Border: gray-200 (#e5e7eb) - card borders, dividers

## Spacing
- Section padding: py-16 md:py-24
- Container max-width: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8
- Card padding: p-6 md:p-8
- Grid gap: gap-8 md:gap-10

## Components
### Cards
- White bg, rounded-xl, shadow-sm border border-gray-100
- Hover: shadow-md transition

### Buttons
- Primary: bg-brand-600 text-white, rounded-lg, font-semibold, px-6 py-3
- Secondary: border-2 border-brand-600 text-brand-600, rounded-lg, font-semibold, px-6 py-3

### Sections
- Alternating: white background, then gray-50
- Section headings: centered, text-3xl/4xl, font-bold
- Section subheadings: centered, text-lg, text-neutral-500

### Visual style
- Clean, professional, minimal
- Plenty of whitespace
- No exaggerated gradients or decorative elements
- Sharp, clear content hierarchy
- Use data-strk-img for realistic factory/QC/shipping photography
