# Velmora Fine Jewelry - Quick Start Guide

## Project Overview
Building a high-end DTC jewelry e-commerce storefront with:
- Quiet luxury aesthetic
- Warm color palette (cream, gold, charcoal)
- Elegant typography (Cormorant Garamond + Inter)
- Fully responsive design
- Working cart functionality

✓ Design system configured (tailwind.config.js, index.css)
✓ Directory structure created
✓ Basic App.jsx created

## Design System
- Colors: velmora-cream, velmora-warm, velmora-taupe, velmora-gold, velmora-charcoal
- Fonts: Cormorant Garamond (serif), Inter (sans-serif)
- Product names: UPPERCASE with wide letter-spacing
- Buttons: Premium feel with solid or outlined accent
- Whitespace: Generous
- Dividers: Thin hairline
