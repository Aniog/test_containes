# SSourcing China - Website Specification

## Project Overview
- **Project Name**: SSourcing China Website
- **Type**: B2B Corporate Website
- **Core Functionality**: Professional sourcing agent website to generate qualified inquiries from overseas buyers
- **Target Users**: Overseas buyers seeking reliable Chinese suppliers, procurement managers, business owners

## SEO Specification
- **Title**: China Sourcing Agent | Supplier Verification, QC & Shipping | SSourcing China
- **Description**: SSourcing China is a professional China-based sourcing agent helping overseas buyers find reliable suppliers, verify factories, inspect quality, follow production, and coordinate shipping.
- **Keywords**: China sourcing agent, supplier verification, QC inspection, manufacturing agent, shipping coordination, factory verification

## Visual & Design Specification

### Color Palette
- **Primary**: #1A365D (Deep Navy Blue - trust, professionalism)
- **Secondary**: #2D5A87 (Medium Blue - corporate)
- **Accent**: #E67E22 (Warm Orange - CTA buttons, energy)
- **Accent Hover**: #D35400 (Darker Orange)
- **Background Light**: #F8FAFC (Off-white)
- **Background Alt**: #EDF2F7 (Light gray)
- **Text Primary**: #1A202C (Near black)
- **Text Secondary**: #4A5568 (Dark gray)
- **Text Muted**: #718096 (Medium gray)
- **Border**: #E2E8F0 (Light border)
- **Success**: #38A169 (Green)
- **White**: #FFFFFF

### Typography
- **Headings**: "DM Sans", sans-serif (Bold, professional, modern)
- **Body**: "DM Sans", sans-serif (Regular, clean)
- **Font Sizes**:
  - H1: 56px (hero), 48px (page titles)
  - H2: 40px (section titles)
  - H3: 28px (subsection titles)
  - H4: 22px (card titles)
  - Body Large: 18px
  - Body: 16px
  - Small: 14px
  - Caption: 12px

### Spacing System
- Section padding: 100px vertical, responsive
- Container max-width: 1200px
- Grid gap: 32px
- Card padding: 32px
- Button padding: 16px 32px

### Visual Effects
- Card shadows: 0 4px 20px rgba(0,0,0,0.08)
- Hover shadows: 0 8px 30px rgba(0,0,0,0.12)
- Border radius: 8px (cards), 6px (buttons), 4px (inputs)
- Transitions: 0.3s ease

## Components

### Navigation
- Logo (left)
- Nav links: Home, Services, How It Works, Products, Case Studies, Blog, Contact
- CTA Button: "Get a Free Quote"
- Mobile: Hamburger menu

### Buttons
- Primary: Orange background, white text
- Secondary: Navy outline, navy text
- Hover: Darker shade, slight lift

### Cards
- Service cards: Icon, title, description, "Learn more" link
- Case study cards: Image, title, category, brief result
- Product cards: Image, category name

### Form Fields
- Name, Email, Company, Phone, Product Details
- Inquiry type dropdown
- Message textarea

### Trust Indicators
- Years in business
- Suppliers verified
- Inspections conducted
- Client satisfaction rate

- Desktop: 1200px+
