# ARTITECT MACHINERY Website

## Overview
Professional website for ARTITECT MACHINERY - a precision sheet metal folding machine manufacturer.

## Brand Identity
- **Brand Name**: ARTITECT MACHINERY
- **Style**: Elegant, professional, industrial precision
- **Color Scheme**: 
  - Primary: Blue 900 (#1e3a8a)
  - Secondary: Blue 800 (#1e40af)
  - Accent: Blue 400 (#60a5fa)
  - Background: White and Gray 50/900

## Products Showcased
1. Double Folding Machine
2. Double Folder
3. Sheet Metal Folder
4. Sheet Metal Folding Machine
5. Metal Folder
6. Metal Folder Machine
7. Metal Folding Machine

## Website Sections

### 1. Header (Navigation)
- Fixed position with shadow
- Logo with AM monogram
- Navigation links: Home, Products, Features, About, Contact
- Mobile-responsive hamburger menu
- "Get Quote" CTA button

### 2. Hero Section
- Full-screen gradient background (gray 900 to blue 900)
- Compelling headline: "Precision in Every Fold"
- Key statistics: 25+ years, 1000+ machines, 50+ countries
- Two CTA buttons: "Explore Our Machines" and "Get a Quote"

### 3. Products Section
- Grid layout (3 columns on desktop, 2 on tablet, 1 on mobile)
- 7 product cards with:
  - Product name
  - Description
  - Key features (4 per product)
  - "Learn More" button
- Hover effects with shadow transitions

### 4. Features Section
- 6 feature cards highlighting:
  - Precision Engineering
  - Built to Last
  - Energy Efficient
  - Easy Maintenance
  - Fast Setup
  - Global Support
- Certification badges: ISO 9001, CE, 5 Year Warranty

### 5. About Section
- Company history since 1999
- Statistics display: 25+ years, 500+ employees, 50+ awards, 1000+ machines
- Mission statement
- Key differentiators list

### 6. Contact Section
- Two-column layout:
  - Left: Contact information with icons
  - Right: Contact form with fields:
    - Full Name*
    - Email Address*
    - Company Name
    - Phone Number
    - Product Interest (dropdown)
    - Message*
- "Why Choose Us?" benefits list

### 7. Footer
- 4-column layout:
  - Company info with social links
  - Products list
  - Quick links
  - Contact information
- Copyright notice
- "Scroll to top" button

### Key Features
- Fully responsive design

- Form validation
- Mobile-friendly navigation
- SEO-optimized title and structure

## Design Principles
1. **Elegant**: Clean lines, ample white space, sophisticated color palette
2. **User-Friendly**: Intuitive navigation, clear CTAs, accessible forms
3. **Professional**: High-quality content, trust signals, certification badges
4. **Industrial**: Reflects precision manufacturing and engineering excellence
