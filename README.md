# Hello World Webpage

A simple "Hello World" webpage demonstration.

## Files
- `index.html`: The main HTML file containing the Hello World webpage
- `server.py`: A simple Python HTTP server to serve the webpage

## How to Run

1. Navigate to the `test` directory:
   ```
   cd /workspace/test
   ```

2. Start the server:
   ```
   python3 server.py
   ```

3. Open your web browser and go to:
   ```
   http://localhost:50344
   ```

4. To stop the server, press `Ctrl+C` in the terminal where the server is running.
