# Hello World Web Application

A simple "Hello World" web application created with HTML and Express.js.

## How to Run

1. Make sure you have Node.js installed
2. Navigate to the project directory:
   ```
   cd todo-appdemo
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Start the server:
   ```
   npm start
   ```
5. Open your browser and visit: http://localhost:50611

## Files

- `index.html`: The main HTML file that displays "Hello World"
- `server.js`: A simple Express.js server to serve the HTML file
- `package.json`: Contains project metadata and dependencies
