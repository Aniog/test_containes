# SSourcing China - Design Guidelines

## Visual Style
- **Aesthetic**: Clean, professional, trustworthy B2B design
- **Typography**: Inter font family with clear hierarchy
- **Colors**: 
  - Primary: #1E3A5F (deep navy blue - trust, professionalism)
  - Secondary: #2E7D32 (forest green - growth, success)
  - Accent: #F57C00 (warm orange - CTAs, energy)
  - Background: #FFFFFF (white) and #F8FAFC (light gray)
  - Text: #1F2937 (dark gray) and #6B7280 (muted gray)
- **Spacing**: Generous whitespace, 8px grid system
- **Borders**: Subtle, rounded corners (8px-16px)
- **Shadows**: Soft, subtle shadows for depth

## Color Palette
- Primary Navy: #1E3A5F
- Primary Navy Light: #2C5282
- Success Green: #2E7D32
- Accent Orange: #F57C00
- Background White: #FFFFFF
- Background Light: #F8FAFC
- Background Navy Light: #EBF4FF
- Text Dark: #1F2937
- Text Muted: #6B7280
- Border: #E5E7EB

## Typography Scale
- Hero: 48px/56px, 800 weight
- H1: 36px/44px, 700 weight
- H2: 30px/38px, 700 weight
- H3: 24px/32px, 600 weight
- H4: 20px/28px, 600 weight
- Body Large: 18px/28px, 400 weight
- Body: 16px/24px, 400 weight
- Small: 14px/20px, 400 weight

## Layout
- Max content width: 1280px
- Section padding: 80px vertical (desktop), 48px (mobile)
- Card padding: 24px-32px
- Grid: 12-column on desktop, responsive on mobile

## Component Guidelines
- Buttons: Rounded (8px), solid primary or outline style
- Cards: White background, subtle shadow, 16px border-radius
- Icons: Lucide React icons, 24px default size
- Images: Realistic factory/QC/shipping visuals
- Forms: Clean inputs with clear labels

## Do's
- Use professional photography style with factory/QC imagery
- Clear, actionable CTAs in accent orange
- Trust signals (statistics, testimonials, certifications)
- Consistent spacing and alignment
- Mobile-responsive design

## Don'ts
- Avoid cluttered layouts
- No exaggerated claims or unrealistic imagery
- No low-contrast text
- No excessive decoration
- No magic values - use Tailwind tokens
