# Velmora Fine Jewelry — Design System

## Brand Voice
Quiet luxury, warm, editorial. Premium demi-fine jewelry. NOT loud, NOT discount, NOT generic.

## Color Palette
- Background: #FBF9F6 (warm off-white/cream)
- Surface: #FFFFFF (white cards, overlays)
- Primary text: #1A1A1A (near-black)
- Secondary text: #6B6258 (warm taupe)
- Accent / Gold: #C9A96E (soft gold)
- Accent hover: #B8944F (darker gold)
- Border / Hairline: #E8E2D8 (warm beige)
- Muted text: #9C9286 (taupish gray)
- Dark surface: #2C2823 (deep espresso — for hero overlays, dark sections)
- Trust / Badge: #4A7C59 (muted sage green for trust elements)

## Typography
- Headings: Cormorant Garamond (Google Font). Use serif for hero titles, section headers, product names.
- Product names: UPPERCASE, tracking-widest (letter-spacing 0.15em+)
- Body / UI: Inter. Use font-light (300) for body, font-medium (500) for UI text.
- All headings: tracking-wide by default.

## Spacing
- Section padding: py-20 md:py-28
- Content max-width: max-w-7xl mx-auto px-6
- Card gap grids: gap-6 md:gap-8

## Borders & Dividers
- Thin hairline: border border-[#E8E2D8]
- Dividers: hr with height 0.5px, bg-[#E8E2D8]

## Shadows
- Card hover: shadow-lg with subtle warm tone
- Nav scroll: shadow-sm

## Transitions
- All interactive: transition-all duration-300 ease-out
- Card hover: translateY(-2px) with shadow increase
- Button hover: bg-accent/90 with subtle transform

## Buttons
- Primary (accent): bg-[#C9A96E] text-white, hover:bg-[#B8944F], uppercase, tracking-widest, text-sm, px-8 py-3
- Outline: border border-[#1A1A1A] text-[#1A1A1A] hover:bg-[#1A1A1A] hover:text-white, uppercase, tracking-widest, text-sm, px-8 py-3
- All buttons: transition-all duration-300

## Do's
- Use generous whitespace
- Use large editorial imagery
- Use thin hairline dividers
- Use warm gold accents sparingly
- Keep UI clean and uncluttered
- Use uppercase with wide letter-spacing for product names

## Don'ts
- No bright/flashy colors
- No heavy borders
- No discount-store aesthetics
- No cluttered layouts
- No generic e-commerce feel
