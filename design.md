# Velmora Fine Jewelry - Design System

## Visual Identity

### Mood
- Quiet luxury, warm, editorial
- Premium demi-fine jewelry aesthetic
- NOT loud, NOT discount-looking, NOT generic e-commerce

### Color Palette
- **Primary Background**: `#FAF9F7` (warm off-white/cream)
- **Secondary Background**: `#F5F3EF` (warm light beige)
- **Dark Background**: `#1A1918` (warm charcoal/black)
- **Primary Text**: `#1A1918` (warm charcoal)
- **Secondary Text**: `#6B6560` (warm gray)
- **Accent Gold**: `#C9A962` (warm gold)
- **Accent Gold Hover**: `#B8954F` (deeper gold)
- **Border/Divider**: `#E5E2DC` (warm light gray)
- **Success**: `#4A7C59` (muted green)
- **Error**: `#A65D57` (muted red)

### Typography
- **Headings**: Cormorant Garamond (serif) - elegant, editorial
  - H1: 48px/56px desktop, 32px mobile
  - H2: 36px/44px desktop, 28px mobile
  - H3: 24px/32px desktop, 20px mobile
- **Body/UI**: Manrope (sans-serif) - clean, modern
  - Body: 16px/24px
  - Small: 14px/20px
  - Caption: 12px/16px
- **Product Names**: UPPERCASE, letter-spacing: 0.15em (wide tracking)

### Spacing System
- Base unit: 4px
- Section padding: 80px vertical (desktop), 48px (mobile)
- Container max-width: 1280px
- Grid gap: 24px (desktop), 16px (mobile)

### Visual Effects
- Hairline dividers: 1px solid #E5E2DC
- Box shadows: `0 4px 20px rgba(26, 25, 24, 0.08)` (subtle)
- Hover transitions: 300ms ease
- Button hover: slight lift + shadow increase

## Components

### Buttons
- **Primary**: Gold background (#C9A962), dark text (#1A1918), rounded-sm
- **Secondary**: Transparent, gold border, gold text
- **States**: Hover darkens gold, subtle lift shadow

### Cards
- White background
- Subtle shadow on hover
- Image with second image reveal on hover
- Quick add to cart on hover

### Forms
- Clean input fields with subtle border
- Focus state: gold border
- Placeholder: secondary text color

## Responsive Breakpoints
- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px

## Animations
- Fade-in on scroll for sections
- Image scale on hover (1.02)
- Button hover lift
- Cart drawer slide-in
- Smooth page transitions
