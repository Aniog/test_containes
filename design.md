# Velmora Fine Jewelry Design System

## Direction
Velmora uses a quiet-luxury editorial aesthetic: warm neutrals, rich ink contrast, brushed gold accents, generous spacing, and refined serif typography.

## Palette
Add these colors to Tailwind as named tokens and use the tokens consistently.

- `brand.ink` — deep espresso text and hero overlays
- `brand.cocoa` — softer body copy
- `brand.parchment` — primary page background
- `brand.mist` — elevated soft panels
- `brand.sand` — secondary warm surfaces
- `brand.line` — hairline borders and dividers
- `brand.gold` — restrained metallic accent
- `brand.goldDeep` — darker gold for hover states
- `brand.forest` — optional deep supporting accent for subtle contrast

## Typography
- Headings and product names: `font-serif` using Cormorant Garamond
- Body, UI, forms: `font-sans` using Inter
- Product names: uppercase with wide spacing using `tracking-product`
- Eyebrows and overlines: `tracking-overline uppercase`

## Surfaces and spacing
- Main page background: `bg-brand-parchment`
- Cards: `bg-brand-mist` or `bg-white`
- Dividers: `border-brand-line`
- Shadows: `shadow-soft` or `shadow-panel`
- Use large vertical rhythm: `py-16 md:py-24`, `space-y-6`, `gap-8 md:gap-12`

## Buttons
- Primary: `bg-brand-gold text-brand-ink hover:bg-brand-goldDeep hover:text-brand-parchment`
- Secondary: `border border-brand-line bg-transparent text-brand-ink hover:bg-brand-ink hover:text-brand-parchment`

## Imagery
- Warm-lit jewelry editorial imagery only
- Prefer model close-ups, velvet trays, gift-box styling, and soft natural light
- Avoid busy, colorful, discount-style product photography

## Do
- Use thin borders, soft shadows, and muted supporting text
- Keep content airy and premium
- Maintain strong contrast on every surface
- Use subtle transitions only

## Don't
- Do not use loud sale colors or bright gradients
- Do not use dense card layouts with cramped spacing
- Do not use low-contrast beige-on-beige text
- Do not mix multiple visual directions
