# Design System — Horses Site

## Color Palette
- **Primary (Deep Brown):** `#3B1F0A` — headings, nav background
- **Accent (Warm Amber):** `#C8873A` — CTA buttons, highlights
- **Light Cream:** `#F5EFE6` — page background, card backgrounds
- **Muted Tan:** `#D4B896` — borders, dividers, subtle backgrounds
- **Forest Green:** `#3A5A40` — secondary accent, badges
- **White:** `#FFFFFF` — text on dark backgrounds

## Typography
- **Font:** Playfair Display (headings) + Inter (body) via Google Fonts
- **Hero Heading:** `text-5xl md:text-7xl font-bold` (Playfair Display)
- **Section Heading:** `text-3xl md:text-4xl font-bold` (Playfair Display)
- **Body:** `text-base md:text-lg` (Inter), color `text-stone-700`
- **Caption / Label:** `text-sm uppercase tracking-widest` (Inter)

## Spacing
- Section padding: `py-20 px-6 md:px-16`
- Card padding: `p-6 md:p-8`
- Gap between cards: `gap-6 md:gap-8`

## Borders & Shadows
- Cards: `rounded-2xl shadow-md border border-stone-200`
- Buttons: `rounded-full`
- Hero overlay: semi-transparent dark gradient

## Buttons
- Primary CTA: `bg-amber-600 hover:bg-amber-700 text-white rounded-full px-8 py-3 font-semibold`
- Secondary: `border-2 border-white text-white hover:bg-white hover:text-stone-900 rounded-full px-8 py-3 font-semibold`

## Do's
- Use warm earthy tones throughout
- Pair Playfair Display headings with Inter body text
- Use full-bleed hero images with dark gradient overlays for text legibility
- Keep layouts spacious and elegant

## Don'ts
- Don't use cold blues or grays as primary colors
- Don't use tight, cramped layouts
- Don't use light text on light backgrounds
