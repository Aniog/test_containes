# SSourcing China Visual Design Guide

## Brand feel
Professional, trustworthy, practical, and international. The design should feel like a reliable B2B sourcing partner for overseas buyers working with factories in China.

## Color system
- Primary navy: use `brand-navy` for headers, navigation, dark sections, and important text.
- Primary blue: use `brand-blue` for main CTAs, links, icons, and active states.
- Accent amber: use `brand-amber` sparingly for process numbers, highlighted badges, and inquiry prompts.
- Neutral surfaces: use `brand-cloud`, `brand-mist`, and white for clean B2B layouts.
- Text: use `brand-ink` for primary copy and `brand-slate` for secondary copy.

## Typography
- Use Inter from Google Fonts.
- Headings should be clear, compact, and confident with `font-semibold` or `font-bold`.
- Body copy should be practical and readable with generous line height.

## Layout and spacing
- Use wide, centered containers with `max-w-7xl mx-auto px-6 lg:px-8`.
- Sections should have generous vertical spacing: `py-16`, `py-20`, or `py-24`.
- Cards should use white backgrounds, subtle borders, and soft shadows.
- Desktop layouts should use multi-column grids. Mobile layouts should stack cleanly.

## Components
- Buttons: rounded, confident, and high contrast. Primary buttons use blue background with white text.
- Cards: `rounded-2xl border border-slate-200 bg-white shadow-sm`.
- Trust badges: compact, readable, and subtle.
- Forms: clear labels, visible borders, strong focus states, and concise helper text.

## Imagery
- Use realistic factory, quality inspection, production, packaging, and freight visuals.
- Avoid exaggerated luxury visuals or unrelated stock photos.
- Images should be cropped cleanly with rounded corners and subtle shadows.

## Do
- Keep claims measured and specific.
- Make inquiry paths obvious.
- Use practical service language for sourcing, factory verification, QC, production follow-up, and shipping coordination.

## Don't
- Do not use hype-heavy language like “guaranteed best supplier”.
- Do not use low contrast text.
- Do not use arbitrary hex colors directly in JSX class strings; add colors to Tailwind config first.
