# Design System - Velmora Fine Jewelry

## Visual Identity
- **Mood**: Quiet luxury, warm, editorial, premium demi-fine jewelry.
- **Palette**:
  - **Base**: Off-white / Cream (#FDFCFB)
  - **Text (Primary)**: Deep Charcoal / Onyx (#1A1A1A)
  - **Accent/Secondary**: Muted Gold / Brass (#B08D57)
  - **Secondary Neutral**: Soft Stone (#E5E1DE)
- **Typography**:
  - **Headings**: Cormorant Garamond (Serif), Uppercase for accents, regular for body headings.
  - **Body**: Manrope (Sans-serif), clean, optimized for readability.
  - **Product Names**: Uppercase with wide letter-spacing (`tracking-[0.2em]`).
- **UI Elements**:
  - **Dividers**: Thin hairline borders (Slate 100/200).
  - **Transitions**: Subtle ease-in-out hover effects.
  - **Imagery**: Large, warm-lit editorial photography.

## Tailwind Classes (Examples)
- **Heading 1**: `font-serif text-4xl uppercase tracking-widest`
- **Product Title**: `font-serif text-lg uppercase tracking-[0.2em]`
- **Cta Button**: `bg-[#B08D57] text-white px-8 py-3 uppercase tracking-wider text-sm hover:bg-[#967648] transition-colors`
- **Link Underline**: `after:block after:h-[1px] after:bg-current after:scale-x-0 group-hover:after:scale-x-100 after:transition-transform`

## Do's and Don'ts
- **Do**: Use generous whitespace. Use high-quality editorial images. Keep elements minimal.
- **Don't**: Use loud colors (neon, bright red). Use heavy shadows. Overcrowd sections. Use generic e-commerce badges.
