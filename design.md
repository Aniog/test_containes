# SSourcing China - Design System

## Overview
Professional B2B website for a China-based sourcing agent. Clean, trustworthy, international feel.

## Typography
- Font family: Inter (Google Fonts)
- Weights: 300, 400, 500, 600, 700, 800
- Base size: 16px
- Line height: 1.5

## Colors
- Primary background: white (#ffffff)
- Secondary background: slate-50 (#f8fafc)
- Dark background: slate-900 (#0f172a)
- Primary text: slate-900 (#0f172a)
- Secondary text: slate-600 (#475569)
- Muted text: slate-500 (#64748b)
- Accent/CTA: slate-900 (#0f172a)
- Success: emerald-500 (#10b981)
- Border: slate-200 (#e2e8f0)

## Spacing
- Section padding: py-20 (5rem)
- Container max-width: max-w-7xl
- Grid gap: gap-8 to gap-12
- Card padding: p-6 to p-8

## Components
- Buttons: rounded-lg, px-6 py-3, text-sm font-semibold
- Cards: rounded-2xl, border border-slate-200, shadow-sm
- Inputs: rounded-lg, border border-slate-300, focus:ring-1 focus:ring-slate-900

## Visual Style
- Minimal, clean design
- Subtle shadows and borders
- Professional, trustworthy appearance
- No exaggerated colors or effects
- Consistent spacing and alignment
