# Velmora Fine Jewelry — Design System

## Typography
- **Headings / Logo / Product names**: Playfair Display (serif, elegant, editorial). Product names in UPPERCASE with tracking-widest (0.1–0.15em).
- **Body / UI / Nav / Buttons**: Inter (clean sans-serif).

## Color Palette
| Token | Hex | Usage |
|---|---|---|
| cream (bg) | #FAF7F2 | Page background, light sections |
| white | #FFFFFF | Cards, surfaces |
| espresso | #1C1612 | Primary text, headings |
| taupe | #8C7B6E | Secondary text, muted labels |
| gold | #B88A4E | Accent: buttons, links, highlights, dividers |
| gold-hover | #9C7340 | Button hover, active states |
| gold-light | #D4B87A | Lighter gold for subtle accents |
| hairline | #E8E0D5 | Thin borders, dividers |
| warm-dark | #2C2416 | Dark footer, deep sections |
| star | #C5903A | Rating star fill |

## Spacing & Layout
- Generous whitespace: section padding y-24–y-32 on desktop, y-16 on mobile
- Container max-width: 1280px (max-w-7xl)
- Product grid gaps: 6–8 (gap-6 to gap-8)

## Buttons
- Primary (accent): solid bg-gold text-white, hover:bg-gold-hover, rounded-none or rounded-sm, px-8 py-3, uppercase wide tracking, subtle transition
- Secondary: outlined border-gold text-gold, hover:bg-gold hover:text-white
- Ghost: text-espresseo hover:text-gold

## Visual Style
- Thin hairline dividers (border-t border-hairline)
- Generous whitespace, editorial layout
- Subtle hover transitions (transition-all duration-300)
- Soft shadows: shadow-sm on cards, shadow-md on hover
- No bright/neon colors, no discount badges, no flashy animations
- Restrained, quiet luxury aesthetic
