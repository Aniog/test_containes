# ARTITECT MACHINERY Design System

## Typography
- Primary font: Inter (Google Fonts)
- Headings: font-weight 700-800, tight letter-spacing
- Body text: font-weight 400-500, relaxed line-height
- Hero text: text-4xl md:text-5xl lg:text-6xl font-bold
- Section titles: text-3xl md:text-4xl font-bold
- Subtitles: text-lg md:text-xl text-muted-foreground

## Colors
- Primary: slate-900 (dark backgrounds, headings)
- Secondary: slate-700 (secondary text)
- Accent: blue-600 (buttons, links, highlights)
- Accent hover: blue-700
- Background: white, slate-50 (alternating sections)
- Card background: white with subtle shadow
- Border: slate-200
- Muted text: slate-500

## Spacing
- Section padding: py-16 md:py-24
- Container max-width: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8
- Component gaps: gap-6, gap-8, gap-12
- Card padding: p-6

## Borders & Shadows
- Cards: rounded-xl shadow-sm hover:shadow-md transition
- Buttons: rounded-lg
- Inputs: rounded-lg border-slate-300
- Section dividers: border-t border-slate-200

## Layout
- Grid layouts: grid-cols-1 md:grid-cols-2 lg:grid-cols-3
- Flex layouts: flex flex-col md:flex-row
- Responsive breakpoints: sm (640px), md (768px), lg (1024px), xl (1280px)

## Do's
- Use high contrast for readability
- Maintain consistent spacing scale
- Use subtle animations and transitions
- Keep navigation clean and minimal
- Use professional imagery for machinery

## Don'ts
- Don't use bright/neon colors
- Don't overcrowd sections with text
- Don't use more than 2-3 font weights
- Don't use harsh shadows or borders
- Don't use inconsistent spacing
