# Velmora Fine Jewelry — Design System

## Visual Identity
- **Mood**: Quiet luxury, warm, editorial. Premium demi-fine jewelry — NOT loud, NOT discount-looking.
- **Direction**: Warm neutral base with gold metallic accents. Deep refined backgrounds for contrast sections.

## Color Palette
| Token | Hex | Usage |
|---|---|---|
| brand-cream | #FAF7F2 | Primary background |
| brand-ivory | #F5F0E8 | Alternate section background |

## Typography
- **Headings**: Cormorant Garamond (serif), light-to-regular weight
- **Body/UI**: Inter (sans-serif), regular weight
- **Product names**: UPPERCASE, letter-spacing 0.25em (ultra-wide)
- **Section labels**: UPPERCASE, letter-spacing 0.2em (super-wide)
- **CTA buttons**: UPPERCASE, letter-spacing 0.2em, font-weight 500

## Spacing & Layout
- Generous whitespace: sections py-16 md:py-24
- Max content width: max-w-7xl (1280px)
- Hairline dividers: h-px bg-brand-pale-gray or bg-brand-gold (accent)
- Grid gaps: gap-4 md:gap-6

## Buttons
- Primary: bg-brand-charcoal text-brand-cream, hover:bg-brand-gold
- Accent: bg-brand-gold text-white, hover:bg-brand-gold-dark
- Outlined: border border-brand-charcoal, hover:bg-brand-charcoal hover:text-brand-cream
- All buttons: text-xs tracking-super-wide uppercase font-sans font-medium, py-3-4

## Images
- Product cards: aspect-[3/4], object-cover
- Category tiles: aspect-[3/4] or aspect-[4/5]
- UGC reel: aspect-[9/16]
- Hover: scale-105 transition-transform duration-500

## Animations
- Subtle and smooth: duration-300 to duration-500
- Hover transitions on buttons, links, images
- Cart drawer: slide-in-right
- No bouncy or flashy animations

## Do's
- Use serif for emotional/editorial content
- Use generous whitespace
- Use gold as accent sparingly
- Ensure strong text contrast on all backgrounds
- Use thin hairline dividers

## Don'ts
- Don't use bright/saturated colors
- Don't use heavy shadows or thick borders
- Don't use sans-serif for headings
- Don't crowd sections with too much content
- Don't use discount/sale visual language
