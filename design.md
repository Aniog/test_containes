# SSourcing China - Visual Style Guide

## Brand Colors
- Primary: Blue (#1e40af) - trust, professionalism
- Primary Light: (#3b82f6)
- Primary Dark: (#1e3a8a)
- Accent: Orange (#ea580c) - CTAs, highlights
- Accent Light: (#f97316)
- Background: White (#ffffff) for main content
- Surface: Gray (#f8fafc) for alternate sections
- Text: Dark (#0f172a) for headings, (#475569) for body
- Muted: (#94a3b8) for secondary text
- Border: (#e2e8f0)
- Success: Green (#16a34a) for trust indicators

## Typography
- Font Family: Inter (headings), system-ui (body)
- Headings: font-bold, tracking-tight
- Body: font-normal, leading-relaxed
- Sizes: h1: text-4xl/tracking-tight, h2: text-3xl, h3: text-xl, body: text-base

## Spacing
- Section padding: py-16 md:py-24
- Container max-width: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8
- Card padding: p-6
- Gap between cards: gap-6 md:gap-8
- Section gap: space-y-4 (within), space-y-16 (between sections)

## Borders & Shadows
- Cards: rounded-xl border border-gray-200 shadow-sm
- Hover: shadow-md transition-shadow
- Buttons: rounded-lg
- Input: rounded-lg border border-gray-300

## Visual Style
- Clean, minimal, professional B2B aesthetic
- Use blue as primary trustworthy color
- Orange for CTAs and accent elements
- White cards on light gray alternate backgrounds
- Subtle shadows for depth without being flashy
- Icons in Lucide React for service/feature illustrations
- Stock photos for factory/QC/shipping/team visuals using data-strk-img attributes
- Responsive: mobile-first with md: breakpoints

## Do's
- Use consistent spacing and alignment
- Keep CTAs prominent with orange/blue
- Use icons for service features
- Show trust signals (badges, certifications)
- Professional photography style
- Clean typography with good hierarchy

## Don'ts
- No flashy animations or gradients
- No exaggerated claims or hype language
- No dark backgrounds for main content
- No crowded layouts
- No inconsistent spacing
