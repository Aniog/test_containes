# ARTITECT MACHINERY - Website Design Specification

## Project Overview
- **Brand Name**: ARTITECT MACHINERY
- **Business Type**: Industrial machinery manufacturer specializing in sheet metal folding machines
- **Products**: Double folding machine, double folder, sheet metal folder, sheet metal folding machine, metal folder, metal folder machine, metal folding machine
- **Target Audience**: Manufacturing companies, metal fabrication shops, industrial equipment buyers

## Visual Style
- **Overall Aesthetic**: Elegant industrial - clean, professional, sophisticated yet approachable
- **Design Philosophy**: Minimalist with strong visual hierarchy, emphasizing product quality and engineering excellence

## Color Palette
- **Primary**: `#1a1a2e` (Deep navy/charcoal - sophistication)
- **Secondary**: `#16213e` (Dark blue - trust, reliability)
- **Accent**: `#e94560` (Coral red - energy, action)
- **Light**: `#f8f9fa` (Off-white - clean, modern)
- **Text Primary**: `#1a1a2e`
- **Text Secondary**: `#6c757d`
- **Border/Divider**: `#dee2e6`

## Typography
- **Headings**: "Playfair Display" - elegant serif for brand sophistication
- **Body**: "Inter" - clean, readable sans-serif
- **Font Sizes**:
  - H1: 3.5rem (56px)
  - H2: 2.5rem (40px)
  - H3: 1.75rem (28px)
  - Body: 1rem (16px)
  - Small: 0.875rem (14px)

## Layout Structure

### Header/Navigation
- Fixed top navigation bar
- Logo on left (brand name in elegant typography)
- Navigation links: Home, Products, About, Contact
- Transparent on hero, solid on scroll

### Hero Section
- Full viewport height
- Large headline with brand positioning
- Subheadline describing product range
- Primary CTA button
- Subtle industrial background pattern or gradient

### Products Section
- Section title with decorative underline
- Grid layout (3 columns desktop, 2 tablet, 1 mobile)
- Product cards with:
  - Icon/illustration area
  - Product name
  - Brief description
  - "Learn More" link

### About Section
- Two-column layout (text + visual)
- Company story and value proposition
- Key statistics or highlights

### Contact Section
- Contact form (name, email, message)
- Company contact information
- Map or visual element

### Footer
- Brand logo and tagline
- Quick links
- Contact information
- Copyright

## Components

### Navigation
- Smooth scroll to sections
- Active state indication
- Mobile hamburger menu

### Buttons
- Primary: Accent color background, white text
- Secondary: Transparent with border
- Hover: Slight scale and shadow

### Cards
- White background
- Subtle shadow
- Rounded corners (8px)
- Hover: Lift effect with enhanced shadow

### Form Inputs
- Clean border style
- Focus state with accent color
- Clear labels and placeholders

## Responsive Breakpoints
- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px

## Animations & Effects
- Smooth scroll behavior
- Fade-in on scroll for sections
- Button hover transitions (0.3s ease)
- Card hover lift effect
- Navigation background transition on scroll

## Spacing System
- Section padding: 80px vertical (desktop), 48px (mobile)
- Container max-width: 1200px
- Grid gap: 32px
- Component padding: 24px
