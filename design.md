# SSourcing China Design System

## Typography
- Font family: Inter, system-ui, sans-serif
- Headings: font-weight 700-800, tight letter-spacing
- Body: font-weight 400-500, relaxed line-height
- Small text: font-weight 400, muted color

## Colors
- Primary: blue-700 (#1d4ed8) - trust, professionalism
- Primary dark: blue-800 (#1e40af) - hover states
- Primary light: blue-50 (#eff6ff) - backgrounds
- Accent: slate-900 (#0f172a) - headings, strong text
- Body text: slate-600 (#475569)
- Muted text: slate-500 (#64748b)
- Background: white, slate-50 (#f8fafc)
- Borders: slate-200 (#e2e8f0)
- Success: green-600
- Warning: amber-500

## Spacing
- Section padding: py-16 md:py-24
- Container: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8
- Card gap: gap-6 lg:gap-8
- Element spacing: space-y-4, space-y-6

## Components
- Cards: rounded-xl border border-slate-200 bg-white shadow-sm
- Buttons: rounded-lg px-6 py-3 font-medium
- Primary button: bg-blue-700 text-white hover:bg-blue-800
- Secondary button: border border-slate-300 text-slate-700 hover:bg-slate-50
- Badges: rounded-full px-3 py-1 text-sm font-medium

## Do's
- Use plenty of white space
- Keep sections clean and focused
- Use icons from Lucide React consistently
- Maintain strong contrast for readability
- Use stock images for factory/QC/shipping visuals

## Don'ts
- No exaggerated claims or hype language
- No cluttered layouts
- No bright/neon colors
- No overly decorative elements
- No low-contrast text
