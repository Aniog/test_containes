# Velmora Fine Jewelry Design System

## Visual Identity
- **Mood**: Quiet luxury, warm, editorial, premium.
- **Palette**:
  - Primary: `#1A1A1A` (Deep Charcoal/Base)
  - Background: `#FDFCF9` (Soft Warm Cream)
  - Accent: `#C5A059` (Muted Gold)
  - Text: `#1A1A1A` (Rich Black)
  - Secondary Text: `#666666` (Soft Gray)
  - Divider: `#E5E5E5` (Thin Hairline)

## Typography
- **Headings**: `Cormorant Garamond`, serif. Elegant, wide letter-spacing for uppercase.
  - Example: `font-serif tracking-[0.2em] uppercase`
- **Body & UI**: `Inter` or `Manrope`, sans-serif. Clean, readable.
  - Example: `font-sans text-sm tracking-normal`

## UI Elements
- **Buttons**:
  - Primary: Solid `#1A1A1A` with `#FDFCF9` text.
  - Secondary: Outlined `#1A1A1A` with transparent background.
  - Accent: Solid `#C5A059` with white text.
- **Form Inputs**: Minimal, single line or subtle border.
- **Cards**: Generous whitespace, large product images, hover effects.
- **Transitions**: Soft fade-in, subtle hover scale/opacity.

## Implementation Details
- Standard spacing: `rem` based, but prefer Tailwind's `p-` and `m-` classes.
- Responsive breakpoints: Standard Tailwind (`sm`, `md`, `lg`, `xl`).
- Mobile First: Always prioritize mobile readability and layout.
