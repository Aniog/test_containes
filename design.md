# ARTITECT MACHINERY Website Design Guidelines

## Brand Identity
- **Brand Name**: ARTITECT MACHINERY
- **Products**: Double folding machine, double folder, sheet metal folder, sheet metal folding machine, metal folder, metal folder machine, metal folding machine
- **Style**: Elegant but user-friendly — clean industrial aesthetic with warm accessibility

## Visual Style

### Color Palette
- **Primary**: Deep Slate (#1E293B) — professional, industrial
- **Secondary**: Steel Blue (#475569) — complementary depth
- **Accent**: Amber Gold (#F59E0B) — premium, attention-grabbing, warm
- **Background Light**: Slate 50 (#F8FAFC) — clean, airy
- **Background Dark**: Slate 900 (#0F172A) — dramatic sections
- **Text Primary**: Slate 800 (#1E293B)
- **Text Light**: White (#FFFFFF)

### Typography
- **Headings**: Inter (700, 600) — clean, modern, authoritative
- **Body**: Inter (400, 500) — readable, friendly
- **Accent/Labels**: Inter (600) uppercase with letter-spacing

### Layout & Spacing
- **Max Content Width**: 1280px
- **Section Padding**: 80px vertical (desktop), 48px (mobile)
- **Component Gap**: 24-32px
- **Border Radius**: 12px for cards, 8px for buttons

### Visual Elements
- Subtle shadows with layered depth (shadow-xl, shadow-2xl)
- Gradient overlays for hero sections
- Clean geometric lines suggesting precision engineering
- Professional machinery imagery with industrial aesthetic

## Do's
- Use amber accent sparingly but strategically (CTAs, highlights)
- Large typography for impact with clear hierarchy
- Generous whitespace for readability
- Smooth transitions and hover effects
- Professional product imagery placeholders

## Don'ts
- Avoid cluttered layouts — keep elements breathing
- Don't overuse the accent color
- Avoid overly decorative elements that distract from content
- Don't use low-contrast text

## Section Structure
1. **Navigation**: Fixed header with logo, nav links, CTA button
2. **Hero**: Full-width with strong headline, subtext, CTA buttons, hero image
3. **Products Grid**: 3-column product cards with features
4. **Features**: Split layout with icon features and benefits
5. **Stats/Trust**: Numbers and trust indicators
6. **Contact/CTA**: Call-to-action section
7. **Footer**: Links, contact info, copyright
