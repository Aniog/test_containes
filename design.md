# Velmora Fine Jewelry - Design System

## Visual Identity
**Mood**: Quiet luxury, editorial, warm, premium.
**Core Principle**: Minimalist but warm, high-end demi-fine jewelry aesthetic.

## Color Palette
- **Primary (Background/Text)**: `#000000` (Pure Black)
- **Secondary (Accents)**: `#D4AF37` (Classic Gold - `bg-[#D4AF37]`)
- **Muted Gold**: `#C5A059` (`text-[#C5A059]`)
- **Neutral Soft**: `#FDFCFB` (Linen/Silk - `bg-[#FDFCFB]`)
- **Warm Gray**: `#F4F4F4` (`bg-[#F4F4F4]`)
- **Border/Divider**: `#E5E5E5` (Hairline - `border-[#E5E5E5]`)

## Typography
- **Headings (Serif)**: Cormorant Garamond
- **Body & UI (Sans)**: Inter / Manrope
- **Product Names**: Uppercase with tracking-widest (`uppercase tracking-[0.2em] font-serif`)

## UI Elements
- **Buttons**:
  - Primary: Solid black, white text, uppercase, small text, generous padding.
  - Secondary: Outlined black or gold, tracking-widest.
- **Dividers**: 1px hairline, often with generous vertical spacing.
- **Cards**: Minimal, no borders or shadows unless very subtle (`shadow-sm`).
- **Transitions**: Smooth, subtle opacity and transform changes (`duration-500 ease-in-out`).

## Dos & Don'ts
- **Do**: Use generous whitespace.
- **Do**: Use high-quality editorial imagery.
- **Do**: Keep UI elements thin and refined.
- **Don't**: Use rounded corners excessively (keep them sharp or slightly softened).
- **Don't**: Use loud shadows or bright non-metallic colors.
