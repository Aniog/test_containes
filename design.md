# SSourcing China - B2B Website Design Guidelines

## Brand Overview
- **Company Name**: SSourcing China
- **Industry**: China sourcing agent / B2B services
- **Target Audience**: Overseas buyers, e-commerce sellers, procurement managers
- **Primary Goal**: Generate qualified sourcing inquiries

## Visual Style
Professional, trustworthy, clean, international B2B aesthetic.

## Color Palette

### Primary Colors
- **Navy Blue**: `#1e3a5f` - Primary brand color, headers, CTAs
- **Dark Slate**: `#2d3748` - Body text, secondary elements
- **Steel Blue**: `#4a6fa5` - Accents, links, hover states

### Secondary Colors
- **Light Gray**: `#f7fafc` - Background sections
- **Off White**: `#fafbfc` - Page background
- **White**: `#ffffff` - Cards, content areas

### Accent Colors
- **Gold/Amber**: `#d69e2e` - Trust indicators, highlights, badges
- **Success Green**: `#38a169` - Success states, checkmarks
- **Warning Orange**: `#dd6b20` - Important callouts

## Typography

### Font Stack
- **Primary**: Inter (Google Fonts) - Clean, professional, highly readable
- **Fallback**: system-ui, -apple-system, sans-serif
- **Weights**: 400 (body), 500 (medium), 600 (semibold), 700 (bold), 800 (extra bold)

### Headings
- **H1**: 3rem (48px) / 800 weight / Navy Blue
- **H2**: 2.25rem (36px) / 700 weight / Navy Blue
- **H3**: 1.5rem (24px) / 600 weight / Dark Slate
- **H4**: 1.25rem (20px) / 600 weight / Dark Slate

### Body Text
- **Size**: 1rem (16px) / 400 weight / Dark Slate
- **Line Height**: 1.6 - 1.75 for readability
- **Small**: 0.875rem (14px) for captions, labels

## Spacing System
- **Base unit**: 4px
- **Section padding**: 80px - 120px vertical
- **Container max-width**: 1280px
- **Card padding**: 24px - 32px
- **Gap between elements**: 16px - 24px

## Border Radius
- **Small** (buttons, inputs): 6px - 8px
- **Medium** (cards): 12px - 16px
- **Large** (sections): 0px (keep clean)

## Shadows
- **Card shadow**: `0 4px 6px -1px rgba(0, 0, 0, 0.1), 0 2px 4px -1px rgba(0, 0, 0, 0.06)`
- **Hover shadow**: `0 10px 15px -3px rgba(0, 0, 0, 0.1), 0 4px 6px -2px rgba(0, 0, 0, 0.05)`
- **Header shadow**: `0 1px 3px 0 rgba(0, 0, 0, 0.1)`

## Component Guidelines

### Buttons
- **Primary CTA**: Navy Blue background, white text, rounded-lg
- **Secondary**: White background, Navy Blue border and text
- **Hover**: Slight darken/lift effect
- **Size**: Padding 12px 24px, text 16px weight 500

### Cards
- White background
- Border radius 12px
- Light shadow
- Hover lift effect

### Icons
- Lucide React icons
- Size: 20px - 48px depending on context
- Color: Inherit from parent or Steel Blue

### Images
- Realistic factory/QC/shipping visuals
- Professional photography style
- 16:9 ratio for hero sections
- 4:3 ratio for cards

## Layout Principles
- Clean, spacious design with ample white space
- Grid-based layouts (12-column system)
- Consistent vertical rhythm
- Mobile-first responsive design
- No clutter - every element has purpose

## Trust Elements
- Checkmark lists
- Star ratings
- Statistics/counters
- Client logos
- Certification badges
- Testimonials with photos

## Animations
- Subtle fade-in on scroll
- Hover transitions: 200ms ease
- Button press: scale(0.98)
- No excessive animations - keep professional

## Responsive Breakpoints
- **Mobile**: < 640px
- **Tablet**: 640px - 1024px
- **Desktop**: > 1024px
- **Large**: > 1280px

## Do's and Don'ts

### Do
- Use clean, professional imagery
- Include clear CTAs on every page
- Show real statistics and metrics
- Use white space generously
- Maintain consistent spacing
- Include trust indicators

### Don't
- Use low-quality stock photos
- Add excessive animations
- Use more than 2-3 font colors
- Cram too much content
- Use flashy gradients
- Make exaggerated claims
