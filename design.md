# SSourcing China - Design System

## Overview
Professional B2B sourcing agent website. Clean, trustworthy, international aesthetic.

## Color Palette
- **Primary**: #1a4b8c (deep trust blue)
- **Primary Dark**: #143d73
- **Secondary**: #f97316 (warm orange for CTAs)
- **Secondary Dark**: #ea580c
- **Background**: #ffffff
- **Surface**: #f8fafc (light gray for sections)
- **Surface Alt**: #f1f5f9
- **Text Primary**: #0f172a (slate-900)
- **Text Secondary**: #475569 (slate-600)
- **Text Muted**: #94a3b8 (slate-400)
- **Border**: #e2e8f0 (slate-200)
- **Success**: #10b981
- **Warning**: #f59e0b

## Typography
- **Font**: Inter (Google Fonts)
- **Headings**: font-weight 700-800, tight line-height
- **Body**: font-weight 400, line-height 1.6
- **Hero H1**: 3rem (48px) on mobile, 3.75rem (60px) on desktop
- **H2**: 2rem (32px)
- **H3**: 1.25rem (20px)
- **Body**: 1rem (16px)
- **Small**: 0.875rem (14px)

## Spacing
- Section padding: py-16 (64px) mobile, py-24 (96px) desktop
- Container max-width: max-w-7xl (1280px)
- Card padding: p-6 (24px)
- Gap between grid items: gap-6 to gap-8

## Components
- **Buttons**: rounded-lg, font-semibold
  - Primary: bg-secondary text-white hover:bg-secondary-dark
  - Secondary/Outline: border-2 border-primary text-primary hover:bg-primary hover:text-white
- **Cards**: rounded-xl, bg-white, shadow-sm, border border-border
- **Icons**: Lucide React, 24px default
- **Forms**: rounded-lg border-border focus:border-primary focus:ring-primary

## Layout
- Sticky top navigation, white bg with subtle shadow
- Hero with background image, overlay, centered or left-aligned content
- Alternating white/surface section backgrounds
- Footer with dark background

## Do's and Don'ts
- DO use plenty of whitespace
- DO use real factory/QC/shipping imagery
- DO keep language professional and practical
- DON'T use flashy animations
- DON'T use exaggerated marketing language
- DON'T use dark mode by default
