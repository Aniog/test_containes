# ARTITECT MACHINERY - Design System

## Brand Essence
Elegant industrial - sophisticated, professional, and user-friendly. Trustworthy and precise, like the machinery we sell.

## Colors

### Primary Palette
- **Primary (Deep Teal)**: `#1a365d` - primary brand color, used for headers, main CTAs, nav backgrounds
- **Primary Light**: `#2b6cb0` - hover states, secondary accents
- **Accent (Warm Gold)**: `#c9a84c` - accent color for highlights, badges, decorative elements
- **Accent Light**: `#e2c76c` - lighter accent hover

### Neutral Palette
- **Background**: `#f8f9fb` - page background (very light cool gray)
- **Surface**: `#ffffff` - card backgrounds, sections
- **Border**: `#e2e8f0` - subtle borders
- **Text Primary**: `#1a202c` - main body text
- **Text Secondary**: `#4a5568` - secondary text
- **Text Muted**: `#a0aec0` - muted/placeholder text

### Semantic
- **Success**: `#38a169`
- **Error**: `#e53e3e`
- **Info**: `#3182ce`

## Typography
- **Font Family**: Inter (Google Font) - primary typeface throughout
- **Headings**: Font weight 600-700, letter-spacing -0.02em
- **Body**: Font weight 400, letter-spacing normal
- **Small/Caption**: Font weight 400, size small

### Font Sizes (Tailwind scale)
- `h1`: text-5xl md:text-6xl lg:text-7xl
- `h2`: text-3xl md:text-4xl
- `h3`: text-xl md:text-2xl
- `body`: text-base
- `small`: text-sm

## Spacing
- Section padding: py-16 md:py-24 lg:py-32
- Card padding: p-6 md:p-8
- Gap between cards: gap-6 md:gap-8
- Container max-width: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8

## Borders & Shadows
- Cards: rounded-xl with subtle shadow (shadow-sm hover:shadow-md)
- Buttons: rounded-lg
- Input fields: rounded-lg, border border-gray-200
- Section dividers: use subtle background color changes instead of visible borders
- No sharp corners for interactive elements

## Visual Style
- Clean, minimal layouts with generous whitespace
- Subtle hover transitions on interactive elements (transition-all duration-300)
- Industrial feel through dark header/nav, gold accents
- Cards with white backgrounds and subtle shadows
- Images should show industrial machinery in clean, well-lit settings
- CTAs: Primary filled buttons (deep teal), secondary outlined buttons
- Links in nav: medium weight, uppercase letter-spacing for nav items
- Feature icons: use Lucide icons in gold accent color

## Do's
- Use consistent spacing throughout
- Keep copy concise and professional
- Use high-quality imagery of machinery
- Make CTAs prominent and clear
- Use gold accent sparingly for emphasis

## Don'ts
- Don't use overly bright or playful colors
- Don't crowd elements together
- Don't use multiple font families
- Don't use heavy industrial grunge aesthetic - keep it clean and elegant
