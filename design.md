# Velmora Fine Jewelry - Design System

## Visual Identity
- **Mood**: Quiet luxury, warm, editorial. Premium demi-fine jewelry aesthetic.
- **Feel**: NOT loud, NOT discount-looking, NOT generic e-commerce.

## Color Palette

### Primary Colors
- **Ivory Cream**: `#FAF8F5` - Primary background, warm off-white
- **Champagne Gold**: `#C9A86C` - Primary accent, warm metallic gold
- **Deep Espresso**: `#2C2420` - Primary text, warm dark brown

### Secondary Colors
- **Warm Stone**: `#E8E4DF` - Secondary backgrounds, subtle cards
- **Rose Gold**: `#D4A373` - Secondary accent, soft metallic
- **Muted Bronze**: `#8B7355` - Tertiary accent, subtle warmth

### Neutrals
- **Charcoal**: `#3D3835` - Secondary text, subtle contrast
- **Soft Gray**: `#9A958F` - Placeholder text, dividers
- **Light Warm Gray**: `#D9D4CD` - Borders, subtle dividers

### Functional Colors
- **Success**: `#6B8E6B` - Muted sage green
- **Error**: `#C17B7B` - Muted rose
- **Star Rating**: `#C9A86C` - Gold stars

## Typography

### Font Families
- **Headings & Product Names**: Cormorant Garamond (serif), fallback: Georgia, Times New Roman
- **Body & UI**: Inter (sans-serif), fallback: system-ui, sans-serif

### Type Scale
- **Display**: 72px / 1.0 line-height / Cormorant Garamond
- **H1**: 48px / 1.1 line-height / Cormorant Garamond
- **H2**: 36px / 1.2 line-height / Cormorant Garamond
- **H3**: 28px / 1.3 line-height / Cormorant Garamond
- **H4**: 22px / 1.4 line-height / Cormorant Garamond
- **Body Large**: 18px / 1.6 line-height / Inter
- **Body**: 16px / 1.6 line-height / Inter
- **Small**: 14px / 1.5 line-height / Inter
- **Caption**: 12px / 1.4 line-height / Inter

### Product Names
- Font: Cormorant Garamond
- Transform: UPPERCASE
- Letter-spacing: 0.15em
- Weight: 500-600

## Spacing System
- **xs**: 4px
- **sm**: 8px
- **md**: 16px
- **lg**: 24px
- **xl**: 32px
- **2xl**: 48px
- **3xl**: 64px
- **4xl**: 96px

## Component Guidelines

### Buttons
- **Primary**: Solid champagne gold background, deep espresso text
- **Secondary**: Outlined with champagne gold border, transparent background
- **Hover**: Subtle scale (1.02) + soft shadow
- **Transitions**: 200-300ms ease

### Cards
- Background: Ivory Cream
- Border: 1px solid Light Warm Gray (optional)
- Shadow: Subtle warm shadow on hover
- Border-radius: 4px (subtle, not rounded)

### Dividers
- Hairline style: 1px
- Color: Light Warm Gray
- Optional decorative diamond center element

### Form Elements
- Border: 1px solid Light Warm Gray
- Focus: Champagne Gold border
- Border-radius: 4px

## Animation Guidelines
- **Hover transitions**: 200-300ms ease
- **Page transitions**: Fade in/out 300ms
- **Scroll reveals**: Fade up with 400ms ease
- **Image hovers**: Subtle scale 1.02-1.05
- **Button hovers**: Scale 1.02 + shadow lift

## Shadows
- **Subtle**: 0 2px 8px rgba(44, 36, 32, 0.06)
- **Medium**: 0 4px 16px rgba(44, 36, 32, 0.08)
- **Elevated**: 0 8px 24px rgba(44, 36, 32, 0.12)

## Breakpoints
- **Mobile**: < 640px
- **Tablet**: 640px - 1024px
- **Desktop**: > 1024px
