# Velmora Fine Jewelry — Design System

## Brand
Premium DTC demi-fine jewelry. Quiet luxury, warm, editorial.

## Color Palette
- **Base**: #1C1917 (stone-950) — primary dark background
- **Surface**: #FAF8F5 (warm ivory) — light backgrounds
- **Surface Alt**: #F3EDE6 (warm linen) — cards, subtle sections
- **Accent**: #C6A664 (antique gold) — buttons, highlights, metallic
- **Accent Dark**: #A88B4A — hover states
- **Accent Light**: #F5EEDC — subtle tint backgrounds
- **Text Primary**: #1C1917 (stone-950) — headings, body on light
- **Text Secondary**: #78716C (stone-500) — muted text
- **Text Light**: #FAF8F5 — text on dark backgrounds
- **Border**: #E7E0D6 — dividers, subtle borders
- **Highlight**: #E8DFCF — hover backgrounds

## Typography
- **Headings**: Playfair Display (serif) — elegant, editorial
- **Body**: Inter (sans-serif) — clean, readable
- **Product Names**: UPPERCASE, letter-spacing: 0.12em
- **Nav Links**: Inter, medium weight, 0.05em letter-spacing

## Spacing
- Section padding: 80px desktop / 48px mobile
- Container max-width: 1280px
- Grid gap: 24px desktop / 16px mobile
- Generous whitespace between elements

## Visual Style
- Thin hairline dividers (#E7E0D6)
- Soft shadows on cards (0 2px 16px rgba(0,0,0,0.06))
- Smooth transitions (300ms ease)
- Rounded corners on cards/buttons (8px default)
- Gold accents on interactive elements

## Buttons
- Primary: Gold bg (#C6A664), dark text, 8px radius
- Primary Hover: Darker gold (#A88B4A)
- Outlined: Gold border, transparent bg, gold text
- All buttons: medium font weight, slight letter-spacing

## Do's
- Use generous whitespace
- Keep imagery warm-toned
- Maintain consistent gold accent usage
- Use thin borders and subtle shadows

## Don'ts
- No bright colors or gradients
- No heavy shadows or 3D effects
- No discount/sale badges
- No cluttered layouts
