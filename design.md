# SSourcing China — Design System

## Overview
A professional B2B website for a China-based sourcing agent. Clean, trustworthy, international aesthetic. Goal: generate qualified sourcing inquiries.

## Colors
- Primary: `#0B3D4C` — deep teal/navy (trust, professionalism)
- Primary Accent: `#1A5A6E` — lighter teal for hover states
- Secondary: `#E87722` — warm orange (CTAs, energy)
- Secondary Hover: `#D06A1E` — darker orange
- Background: `#FFFFFF` — white for main content
- Background Alt: `#F5F7FA` — very light gray for alternating sections
- Text Primary: `#1A1A2E` — near-black for headings
- Text Secondary: `#4A5568` — slate gray for body text
- Text Muted: `#718096` — lighter gray for captions
- Border: `#E2E8F0` — light border color
- Success: `#22C55E` — green for trust indicators

## Typography
- Font: Inter (Google Fonts), weights 300, 400, 500, 600, 700, 800
- Hero heading: 48px / 700 / line-height 1.15
- H1: 40px / 700 / line-height 1.2
- H2: 32px / 600 / line-height 1.25
- H3: 24px / 600 / line-height 1.3
- Body: 16px / 400 / line-height 1.6
- Small: 14px / 400 / line-height 1.5
- Caption: 12px / 500 / uppercase / letter-spacing 0.1em

## Spacing
- Section padding: py-20 (80px) desktop, py-12 (48px) mobile
- Content max-width: max-w-7xl (1280px) centered
- Grid gap: 6 (24px) standard, 8 (32px) for cards
- Card padding: p-6 (24px) or p-8 (32px)

## Components
- Buttons: rounded-lg, px-6 py-3, font-semibold. Primary = bg-secondary text-white. Secondary = bg-primary text-white.
- Cards: rounded-xl, bg-white, border border-border, shadow-sm, hover:shadow-md transition.
- Section headers: centered or left-aligned with accent line (w-16 h-1 bg-secondary).
- Icons: Lucide icons, size 24-48px, color primary or secondary.

## Do's and Don'ts
- DO use plenty of whitespace — B2B buyers value clarity
- DO use the teal + orange palette consistently
- DO keep all text readable with strong contrast
- DON'T use gimmicky animations or excessive effects
- DON'T use low-contrast text on any background
- DON'T crowd elements — keep layouts breathable
