# Velmora Fine Jewelry — Design System

## Visual Direction
Quiet luxury, warm, editorial. Premium demi-fine jewelry — NOT loud, NOT discount-looking, NOT generic e-commerce.

## Color Palette

### Base Colors
- **Charcoal** (#1C1917): Primary background for dark sections, nav solid state. Tailwind: `stone-900`
- **Warm Cream** (#FAF7F2): Primary page background. Tailwind: custom `cream`
- **Ivory** (#F5F0E8): Card backgrounds, subtle sections. Tailwind: custom `ivory`
- **Soft Taupe** (#A89F91): Muted text, borders, dividers. Tailwind: custom `taupe`

### Accent Colors
- **Antique Gold** (#B8924A): Primary accent — buttons, links, highlights. Tailwind: custom `gold`
- **Light Gold** (#D4B978): Hover state for gold. Tailwind: custom `gold-light`
- **Dark Gold** (#8B6D2F): Active/pressed state for gold. Tailwind: custom `gold-dark`

### Text Colors
- **Rich Black** (#1C1917): Headings, primary text on light backgrounds. Tailwind: `stone-900`
- **Warm Gray** (#57534E): Body text on light backgrounds. Tailwind: `stone-600`
- **Light Text** (#FAF7F2): Text on dark backgrounds. Tailwind: custom `cream`

## Typography

### Headings & Product Names
- Font: Cormorant Garamond (serif), loaded via Google Fonts
- Weight: 400 (regular), 500 (medium), 600 (semibold)
- Product names: UPPERCASE with `tracking-[0.2em]` letter-spacing
- H1: `text-4xl md:text-5xl lg:text-6xl font-medium`
- H2: `text-3xl md:text-4xl font-medium`
- H3: `text-2xl md:text-3xl font-medium`

### Body Text
- Font: Inter (sans-serif), loaded via Google Fonts
- Weight: 400 (regular), 500 (medium)
- Body: `text-base text-stone-600`
- Small: `text-sm text-stone-500`

## Spacing & Layout
- Section padding: `py-16 md:py-24`
- Container: `max-w-6xl mx-auto px-4`
- Card padding: `p-6`
- Gap: `gap-6` or `gap-8`

## Colors (Tailwind)
- Primary background: white or `stone-50`
- Card background: white with `shadow-sm`
- Accent: `stone-900` (dark)
- Border: `stone-200`
- Muted: `stone-100`

## Visual Style
- Clean, minimal design with warm tones
- Subtle shadows and rounded corners
- Q版可爱风格: round shapes, big heads, small bodies, expressive eyes
- Soft color palette with warm undertones

## Do's
- Use Cormorant Garamond for all headings and product names
- Use Inter for body text and UI elements
- Maintain consistent spacing using Tailwind utilities
- Use subtle shadows and borders for depth
- Keep designs clean and uncluttered

## Don'ts
- Don't use bright neon colors
- Don't mix too many font families
- Don't use harsh shadows or heavy borders
- Don't use thin/light weight text for important content
- Don't use dark backgrounds with dark text
