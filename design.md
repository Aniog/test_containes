# ARTITECT MACHINERY - Design Guidelines

## Brand Identity
- Brand: ARTITECT MACHINERY
- Tone: Precision, craftsmanship, reliability, innovation
- Style: Elegant, sophisticated, user-friendly, modern industrial

## Color Palette
- Primary: Deep navy blue (#0f172a) - represents precision and professionalism
- Secondary: Warm bronze/copper (#b8860b / #cd7f32 tones) - represents metal craftsmanship
- Accent: Metallic gold (#d4a843) - for highlights and CTAs
- Background: Warm off-white (#f8f6f3) - elegant, warm backdrop
- Surface: White (#ffffff) - cards, dialogs
- Text primary: Near-black (#1a1a2e)
- Text muted: (#64748b)
- Borders: (#e2e8f0)

## Typography
- Headings: 'Playfair Display', serif - for elegance and sophistication
- Body: 'Inter', sans-serif - for readability and modern feel
- Scale: h1: text-5xl, h2: text-4xl, h3: text-2xl, h4: text-xl

## Spacing
- Section padding: py-20 md:py-28
- Container max-width: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8
- Card gap: gap-8

## Visual Style
- Cards with subtle shadow (shadow-sm hover:shadow-md transition)
- Smooth transitions (transition-all duration-300)
- Subtle border-radius on cards (rounded-lg)
- Clean, minimalist design
- Ample whitespace
- Navigation: fixed top, transparent/scrolled state
- Buttons: rounded-md, with hover effects
- Use of accent borders and subtle gradients

## Do's
- Keep lots of whitespace for elegance
- Use the bronze/gold accents sparingly for impact
- Professional photography/imagery of machinery
- Clean typography hierarchy
- Subtle hover animations
- Grid layouts for product showcases

## Don'ts
- Don't use bright/neon colors
- Don't overcrowd sections
- Don't use overly technical jargon without context
- Don't use heavy drop shadows
- Don't use flashy animations
