# ARTITECT MACHINERY - Design System

## Overview
An elegant, industrial-styled website for a sheet metal folding machinery manufacturer.
Style: Elegant but user-friendly. Dark industrial luxury meets clean modern usability.

## Colors
- Primary: `#0f172a` — Deep slate/navy (headers, dark sections)
- Secondary: `#1e293b` — Dark slate (cards, secondary backgrounds)
- Accent: `#c9a227` — Warm gold/bronze (CTAs, highlights, borders)
- Accent Hover: `#b8941f` — Darker gold for hover states
- Background Light: `#f8f9fa` — Off-white for light sections
- Background Dark: `#0f172a` — Deep navy for dark sections
- Text Primary: `#1e293b` — Dark text on light backgrounds
- Text Light: `#f8f9fa` — Light text on dark backgrounds
- Text Muted: `#94a3b8` — Muted text for secondary info
- Border: `#e2e8f0` — Subtle borders

## Typography
- Font: Inter (Google Fonts)
- Headings: 700-800 weight, tight letter-spacing (-0.02em)
- Body: 400 weight, line-height 1.6
- Section titles: 2.5rem (40px), bold
- Subsection: 1.5rem (24px), semibold
- Body: 1rem (16px)
- Small: 0.875rem (14px)

## Spacing
- Section padding: py-20 (80px vertical)
- Container max-width: 1280px
- Card gap: 24px
- Element spacing: 16px small, 24px medium, 32px large

## Components
- Buttons: 
  - Primary: bg-gold text-navy rounded-md px-6 py-3, hover:bg-gold-hover
  - Secondary: border border-gold text-gold rounded-md px-6 py-3, hover:bg-gold/10
- Cards: bg-white rounded-lg shadow-md border border-gray-100, hover:shadow-lg transition
- Navigation: fixed top, bg-navy/95 backdrop-blur, text-light
- Hero: full viewport height, dark overlay on background image

## Shadows
- Card: `0 4px 6px -1px rgba(0,0,0,0.1), 0 2px 4px -2px rgba(0,0,0,0.1)`
- Card hover: `0 10px 15px -3px rgba(0,0,0,0.1), 0 4px 6px -4px rgba(0,0,0,0.1)`

## Do's
- Use plenty of whitespace
- Use gold accent sparingly for emphasis
- Keep text readable with strong contrast
- Use responsive grid layouts
- Use smooth transitions (300ms ease)

## Don'ts
- Don't use too many colors — stick to navy, gold, white, gray
- Don't use cluttered layouts
- Don't make text too small on mobile
- Don't use harsh shadows
