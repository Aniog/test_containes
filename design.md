# Visual Style Guide - ARTITECT MACHINERY

## Typography
- **Font Family**: 'Inter', sans-serif (Standard via Google Fonts)
- **Headings**: Semi-bold to Bold, tracking-tight (e.g., `font-bold tracking-tight`)
- **Body**: Regular, leading-relaxed (e.g., `text-slate-600 leading-relaxed`)

## Colors
- **Primary**: Slate 900 (`#0f172a`) - Used for text, backgrounds, and main branding
- **Secondary**: Blue 600 (`#2563eb`) - Used for highlights, buttons, and call-to-actions
- **Background**: White (`#ffffff`) or Slate 50 (`#f8fafc`)
- **Accents**: Slate 400-500 for secondary text and borders

## Components
- **Buttons**: Rounded-full, high contrast (e.g., `rounded-full px-8 py-4 px-6 py-2`)
- **Cards**: Large border radius, subtle shadows, clean borders (e.g., `rounded-3xl shadow-sm border border-slate-100`)
- **Sections**: Generous padding (e.g., `py-24`)

## Layout
- **Max Width**: 7xl (`max-w-7xl mx-auto px-4`)
- **Grid**: Standard responsive grids (e.g., `grid-cols-1 md:grid-cols-3`)

## Design Principles
- **Elegant**: Minimalist use of color, high-quality typography, professional stock images.
- **User-Friendly**: Clear navigation, intuitive button placement, descriptive text.
- **Industrial**: Subtle gear or metal textures in backgrounds (via stock system), strong bold uppercase headers.
