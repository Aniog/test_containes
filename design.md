# MicroCosmos Design System

## Theme
A dark, immersive, science-inspired design that evokes the mystery and beauty of the microscopic world. Deep dark backgrounds with vibrant accent colors reminiscent of fluorescence microscopy.

## Colors
- **Background (dark):** `midnight` — #0a0e1a (deep navy-black)
- **Surface:** `surface` — #111827 (dark card backgrounds)
- **Surface Light:** `surface-light` — #1f2937 (lighter card/section backgrounds)
- **Primary:** `primary` — #06b6d4 (cyan, like bioluminescence)
- **Secondary:** `secondary` — #8b5cf6 (violet, like UV fluorescence)
- **Accent:** `accent` — #10b981 (emerald green, like chlorophyll)
- **Text Primary:** `text-primary` — #f1f5f9 (near-white)
- **Text Secondary:** `text-secondary` — #94a3b8 (muted slate)
- **Glow:** `glow` — #22d3ee (bright cyan for glowing effects)

## Typography
- **Font Family:** "Space Grotesk" for headings, "Inter" for body text
- **Headings:** font-bold, tracking-tight
- **Body:** font-normal, leading-relaxed
- **Hero Title:** text-5xl md:text-7xl font-bold
- **Section Title:** text-3xl md:text-4xl font-bold
- **Card Title:** text-xl font-semibold
- **Body Text:** text-base md:text-lg

## Spacing
- **Section padding:** py-20 md:py-28 px-4 md:px-8
- **Card padding:** p-6
- **Grid gap:** gap-6 md:gap-8
- **Container max-width:** max-w-7xl mx-auto

## Borders & Shadows
- **Card border:** border border-surface-light/50 rounded-2xl
- **Glow effect:** shadow-lg shadow-primary/10
- **Image border-radius:** rounded-xl or rounded-2xl

## Visual Style
- Dark immersive backgrounds
- Subtle gradient overlays on hero sections
- Rounded corners on all cards and images
- Glowing accent effects on hover states
- Smooth transitions: transition-all duration-300

## Do's
- Use dark backgrounds with high-contrast text
- Apply subtle glow effects to interactive elements
- Use generous whitespace between sections
- Make images large and prominent
- Use gradient text for hero headings

## Don'ts
- Don't use light/white backgrounds
- Don't use small, cramped image sizes
- Don't use sharp corners (always round)
- Don't use low-contrast text combinations
- Don't overcrowd sections with too much text
