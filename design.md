# Velmora Fine Jewelry — Design System

## Brand Direction
Quiet luxury. Warm editorial. Premium demi-fine jewelry. NOT loud, NOT discount-looking.

## Color Palette

### Base
- `obsidian`: #1A1714 — primary dark background, nav solid state
- `charcoal`: #2C2825 — secondary dark surface
- `stone`: #4A4540 — muted text, borders

### Warm Neutrals
- `parchment`: #F5F0E8 — primary light background
- `cream`: #FAF7F2 — card backgrounds, sections
- `linen`: #EDE8DF — subtle dividers, hover states

### Gold Accents
- `gold`: #C9A96E — primary accent, CTAs, highlights
- `gold-light`: #DFC08A — hover states, lighter gold
- `gold-dark`: #A8854A — pressed states, deep gold

### Text
- `ink`: #1A1714 — primary body text on light
- `mist`: #8C8480 — secondary/muted text
- `ivory`: #FAF7F2 — text on dark backgrounds

## Typography

### Fonts
- Headings: "Cormorant Garamond" (serif) — elegant, editorial
- Body/UI: "Inter" (sans-serif) — clean, readable

### Scale
- Display: 72px / Cormorant Garamond / font-light / tracking-wide
- H1: 56px / Cormorant Garamond / font-light
- H2: 40px / Cormorant Garamond / font-light
- H3: 28px / Cormorant Garamond / font-normal
- Product Name: 20px / Cormorant Garamond / UPPERCASE / tracking-widest
- Body: 15px / Inter / font-normal / leading-relaxed
- Caption: 12px / Inter / font-normal / tracking-wide / uppercase

## Spacing
- Section padding: py-20 to py-28
- Container max-width: max-w-7xl mx-auto px-6 lg:px-12
- Card gap: gap-6 to gap-8
- Generous whitespace between sections

## Components

### Buttons
- Primary: bg-gold text-obsidian px-8 py-3 text-sm tracking-widest uppercase font-medium hover:bg-gold-light transition-all
- Outlined: border border-gold text-gold px-8 py-3 text-sm tracking-widest uppercase hover:bg-gold hover:text-obsidian
- Ghost: text-ink underline-offset-4 hover:underline text-sm tracking-wide

### Cards
- Product card: bg-cream, no border, soft shadow on hover, overflow-hidden
- Hover: scale-[1.02] transition-transform duration-500

### Dividers
- Hairline: border-t border-linen (1px, very subtle)
- Accent: border-t border-gold/30

### Shadows
- Soft: shadow-[0_4px_24px_rgba(0,0,0,0.06)]
- Hover: shadow-[0_8px_40px_rgba(0,0,0,0.12)]

## Do's
- Use Cormorant Garamond for all headings and product names
- UPPERCASE + wide letter-spacing for product names and labels
- Warm gold (#C9A96E) as the single accent color
- Generous whitespace — let the jewelry breathe
- Thin hairline dividers, never thick borders
- Subtle hover transitions (300–500ms ease)

## Don'ts
- No bright/saturated colors
- No thick borders or heavy shadows
- No generic e-commerce blue/green CTAs
- No crowded layouts
- No bold/heavy font weights for headings
