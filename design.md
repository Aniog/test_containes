# SSourcing China - Design System

## Brand Identity
- Professional B2B sourcing agency
- Trustworthy, clean, international feel
- No exaggerated claims - practical and clear

## Colors
- Primary: Deep navy blue `#1e3a5f` - trust, professionalism
- Primary hover: `#16304d`
- Accent: Warm amber `#d97706` - CTA, highlights
- Accent hover: `#b45309`
- Background: White `#ffffff` and light gray `#f8fafc`
- Text primary: `#1e293b` (slate-800)
- Text secondary: `#475569` (slate-600)
- Text muted: `#94a3b8` (slate-400)
- Border: `#e2e8f0` (slate-200)
- Success: `#16a34a`

## Typography
- Font: Inter (Google Fonts)
- Headings: font-weight 700-800, tracking tight
- Body: font-weight 400, line-height 1.6-1.75
- H1: text-4xl md:text-5xl lg:text-6xl
- H2: text-3xl md:text-4xl
- H3: text-xl md:text-2xl
- Body: text-base (16px)

## Spacing
- Section padding: py-16 md:py-24
- Container: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8
- Card padding: p-6 md:p-8
- Gap between grid items: gap-6 md:gap-8

## Components
- Buttons: rounded-lg, font-medium, transition colors
  - Primary: bg-[#1e3a5f] text-white hover:bg-[#16304d]
  - Accent/CTA: bg-[#d97706] text-white hover:bg-[#b45309]
  - Outline: border-2 border-[#1e3a5f] text-[#1e3a5f]
- Cards: bg-white rounded-xl shadow-sm border border-slate-200 hover:shadow-md transition
- Badges: rounded-full px-3 py-1 text-sm font-medium

## Visual Style
- Clean white backgrounds with subtle gray sections for contrast
- Professional stock imagery of factories, QC, shipping
- Icons from Lucide React
- Subtle shadows, no heavy gradients
- Generous whitespace

## Do's
- Use consistent spacing and alignment
- Ensure high contrast for readability
- Use navy + amber color pairing consistently
- Keep layouts clean and uncluttered
- Use responsive grid layouts

## Don'ts
- Don't use dark backgrounds with dark text
- Don't use more than 2 brand colors prominently
- Don't use exaggerated marketing language
- Don't clutter sections with too many elements
- Don't use low-contrast text
