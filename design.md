# Velmora Fine Jewelry - Design System

## Visual Identity

### Mood
- **Quiet luxury, warm, editorial**
- Premium demi-fine jewelry
- NOT loud, NOT discount-looking, NOT generic e-commerce

### Color Palette

**Primary Colors:**
- `cream` - #FAF7F2 (warm off-white background)
- `charcoal` - #1A1A1A (primary text, dark backgrounds)
- `warm-black` - #0D0D0D (deepest backgrounds)

**Accent Colors:**
- `gold` - #C9A962 (primary accent - warm gold)
- `gold-light` - #E5D4A1 (gold highlights)
- `champagne` - #F5E6C8 (subtle gold tints)

**Neutral Colors:**
- `stone-50` - #FAFAF9
- `stone-100` - #F5F5F4
- `stone-200` - #E7E5E4
- `stone-300` - #D6D3D1
- `stone-400` - #A8A29E
- `stone-500` - #78716C
- `stone-600` - #57534E

**Semantic Colors:**
- `success` - #4A7C59 (muted green)
- `error` - #9F4A4A (muted red)

### Typography

**Headings & Product Names:**
- Font: **Cormorant Garamond** (serif)
- Weights: 400, 500, 600
- Product names: UPPERCASE with letter-spacing: 0.15em

**Body & UI:**
- Font: **Inter** (sans-serif)
- Weights: 300, 400, 500, 600
- Clean, readable, modern

**Font Sizes:**
- Display: 56px / 3.5rem
- H1: 40px / 2.5rem
- H2: 32px / 2rem
- H3: 24px / 1.5rem
- H4: 20px / 1.25rem
- Body: 16px / 1rem
- Small: 14px / 0.875rem
- Caption: 12px / 0.75rem

### Spacing System
- Base unit: 4px
- Spacing scale: 4, 8, 12, 16, 24, 32, 48, 64, 96, 128px

### Visual Effects

**Shadows:**
- `shadow-sm`: 0 1px 2px rgba(0,0,0,0.05)
- `shadow-md`: 0 4px 6px rgba(0,0,0,0.07)
- `shadow-lg`: 0 10px 15px rgba(0,0,0,0.1)
- `shadow-soft`: 0 2px 20px rgba(0,0,0,0.08)

**Borders:**
- Hairline dividers: 1px solid #E7E5E4
- Border radius: 0px (sharp) for premium feel, or 2px for buttons

**Animations:**
- Transitions: 300ms ease-out default
- Hover transitions: subtle, 200-300ms
- Fade-in on scroll: opacity + translateY

### Components

**Buttons:**
- Primary: Gold background (#C9A962), charcoal text, uppercase, letter-spacing
- Secondary: Transparent with gold border, gold text
- Hover: Slight brightness increase, subtle shadow

**Cards:**
- Clean white/cream background
- Minimal borders
- Subtle shadow on hover
- Product image with second image on hover

**Forms:**
- Clean input fields with subtle borders
- Focus state: gold border
- Placeholder: stone-400

### Layout

**Container:**
- Max-width: 1280px
- Padding: 24px (mobile), 48px (desktop)

**Grid:**
- Product grid: 2 columns (mobile), 3-4 columns (desktop)
- Generous whitespace between elements

### Responsive Breakpoints
- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px

### Do's and Don'ts

**DO:**
- Use generous whitespace
- Keep imagery large and editorial
- Use thin hairline dividers
- Apply subtle hover transitions
- Use gold sparingly as accent
- Keep UI elements clean and minimal

**DON'T:**
- Use loud, bright colors
- Create cluttered layouts
- Use generic e-commerce styling
- Apply heavy shadows or borders
- Overuse the gold accent color
- Mix serif fonts inconsistently
