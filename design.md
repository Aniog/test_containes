# SSourcing China - B2B Website Design Guidelines

## Project Overview
- **Project Name**: SSourcing China Website
- **Type**: B2B corporate website for China sourcing agent services
- **Goal**: Generate qualified sourcing inquiries from overseas buyers
- **Target Audience**: International B2B buyers, procurement managers, business owners looking to source products from China

## Visual Style

### Overall Aesthetic
- Clean, professional, trustworthy
- International B2B feel
- No exaggerated claims - practical and results-oriented
- Realistic factory/QC/shipping visuals

### Color Palette
- **Primary**: `#1E3A5F` (Deep Navy Blue - trust, professionalism)
- **Secondary**: `#2D5A87` (Medium Blue - reliability)
- **Accent**: `#E67E22` (Warm Orange - CTA buttons, energy)
- **Success**: `#27AE60` (Green - trust indicators)
- **Background Light**: `#F8FAFC` (Off-white)
- **Background Dark**: `#0F172A` (Dark navy for footer)
- **Text Primary**: `#1E293B` (Dark slate)
- **Text Secondary**: `#64748B` (Muted gray)
- **Border**: `#E2E8F0` (Light gray)

### Typography
- **Headings**: "DM Sans" - modern, professional, excellent readability
- **Body**: "DM Sans" - clean and consistent
- **Font Sizes**:
  - H1: 48px (hero), 36px (page titles)
  - H2: 32px (section titles)
  - H3: 24px (subsection titles)
  - Body: 16px
  - Small: 14px

### Spacing System
- Section padding: 80px vertical (desktop), 48px (mobile)
- Container max-width: 1200px
- Grid gap: 32px
- Card padding: 24px
- Button padding: 12px 24px

### Visual Effects
- Subtle box shadows on cards: `0 4px 6px -1px rgba(0, 0, 0, 0.1)`
- Hover transitions: 0.3s ease
- Border radius: 8px (cards), 6px (buttons)
- No excessive animations - professional restraint

## Layout Structure

### Header (Sticky)
- Logo (left)
- Navigation menu (center): Home, Services, How It Works, Products, Case Studies, Blog, Contact
- CTA button (right): "Get a Free Quote"

### Footer
- Company info and logo
- Quick links
- Contact information
- Social media links
- Copyright

### Responsive Breakpoints
- Mobile: < 768px (single column)
- Tablet: 768px - 1024px (2 columns)
- Desktop: > 1024px (full layout)

## Component Guidelines

### Buttons
- Primary: Orange background (#E67E22), white text, hover darken
- Secondary: Navy outline, navy text, hover fill
- Size: Medium (12px 24px padding)

### Cards
- White background
- Light border
- Subtle shadow
- 8px border radius

### Form Inputs
- Light background
- Border on focus
- Clear labels
- Placeholder text in muted color

### Icons
- Lucide React icons
- Consistent sizing (24px default)
- Muted colors for secondary icons

## Page-Specific Guidelines

### Homepage
- Hero with headline, subtext, CTA, and background image
- Services overview (4-6 cards)
- Sourcing process (numbered steps)
- Products we source (category grid)
- Problems we solve (pain points)
- Trust points (stats, certifications)
- Case studies preview
- FAQ accordion
- Inquiry form

### Services Page
- Service cards with detailed descriptions
- Icons for each service
- Call-to-action after each section

### How It Works Page
- Step-by-step visual process
- Timeline or numbered steps
- Contact CTA at bottom

### Products We Source Page
- Category grid with images
- Brief descriptions
- Link to inquiry form

### Case Studies Page
- Success story cards
- Results and metrics
- Client testimonials

### Blog Page
- Article grid
- Categories
- Recent posts

### Contact Page
- Contact form
- Company information
- Map or office image
- Direct contact details

## Content Tone
- Professional and clear
- Practical, not promotional
- Results-focused
- No exaggerated claims
- Trust-building language

## Accessibility
- Semantic HTML
- Proper heading hierarchy
- Alt text for images
- Keyboard navigation
- Sufficient color contrast
- Form labels and error messages
