# SSourcing China - Design Guidelines

## Project Overview
- **Project Name**: SSourcing China Website
- **Type**: B2B Corporate Website
- **Goal**: Generate qualified sourcing inquiries from overseas buyers
- **Tone**: Professional, clear, practical, trustworthy

## Visual Style

### Color Palette
- **Primary**: #1a365d (Deep Navy Blue) - Trust, professionalism
- **Secondary**: #2d5a87 (Medium Blue) - Supporting blue
- **Accent**: #e67e22 (Warm Orange) - CTAs, highlights
- **Success**: #27ae60 (Green) - Trust indicators
- **Background**: #f8fafc (Light Gray) - Clean, modern
- **Surface**: #ffffff (White) - Cards, sections
- **Text Primary**: #1e293b (Dark Slate) - Main text
- **Text Secondary**: #64748b (Slate Gray) - Secondary text
- **Border**: #e2e8f0 (Light Border)

### Typography
- **Headings**: "DM Sans", sans-serif - Modern, professional
- **Body**: "DM Sans", sans-serif - Clean readability
- **Font Sizes**:
  - H1: 48px (desktop), 36px (mobile)
  - H2: 36px (desktop), 28px (mobile)
  - H3: 24px (desktop), 20px (mobile)
  - Body: 16px
  - Small: 14px

### Spacing System
- Section padding: 80px vertical (desktop), 48px (mobile)
- Container max-width: 1200px
- Grid gap: 32px
- Card padding: 24px

### Visual Effects
- Card shadows: 0 4px 20px rgba(0,0,0,0.08)
- Hover transitions: 0.3s ease
- Border radius: 8px (cards), 4px (buttons)

## Layout Structure

### Header
- Fixed navigation
- Logo (left)
- Navigation links (center)
- CTA button (right)
- Mobile: hamburger menu

### Footer
- 4-column layout (desktop)
- Company info, Quick links, Services, Contact
- Social links
- Copyright

## Component Guidelines

### Buttons
- Primary: Navy background, white text, orange hover
- Secondary: White background, navy border
- Padding: 12px 24px
- Border radius: 4px

### Cards
- White background
- Subtle shadow
- 8px border radius
- Hover: slight lift effect

### Icons
- Lucide React icons
- Consistent sizing (24px default)
- Navy or accent color

### Forms
- Clean input fields with borders
- Clear labels
- Validation states
- Submit button prominent

## Page-Specific Guidelines

### Home Page
- Hero with background image/gradient
- Services grid (3 columns)
- Process timeline (horizontal)
- Products grid with images
- Trust badges/logos
- Case study cards
- FAQ accordion
- Inquiry form section

### Services Page
- Service cards with icons
- Detailed descriptions
- Process flow diagrams

### How It Works
- Step-by-step visual guide
- Timeline or numbered steps
- Contact CTA

### Products Page
- Category grid
- Product images
- Brief descriptions

### Case Studies
- Story format
- Challenge/Solution/Result
- Testimonial quotes

### Blog
- Article cards
- Category tags
- Read more links

### Contact
- Inquiry form
- Contact information
- Map or office image

## Responsive Breakpoints
- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px

## Do's and Don'ts
- DO: Use professional imagery (factory, QC, shipping)
- DO: Keep text concise and scannable
- DO: Use clear CTAs
- DO: Include trust signals
- DON'T: Use exaggerated claims
- DON'T: Clutter with too many elements
- DON'T: Use casual language
- DON'T: Overuse stock photos
