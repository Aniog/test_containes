# Strikingly AI Generators Hub Visual Style

This page follows Strikingly's marketing-page style: clean white surfaces, geometric uppercase headings, compact body copy, and restrained AI accents.

## Typography
- Headings and buttons use Brandon Grotesque when available, with Josefin Sans as the Google Fonts fallback. Use uppercase, weight 700, line-height 1.2.
- Body text uses Open Sans, weight 400, 14px base, line-height 1.5.
- Keep headings clear and concise; do not use system fonts for headlines.

## Colors
- Headings: `#4B5056`, hero first line `#2E2E2F`.
- Body text: `#636972`.
- Brand purple: `#8159BB` for accents, tags, borders, and ghost buttons.
- AI gradient: `linear-gradient(90deg, #7671FF, #CB0C9F)` only for primary CTAs and the H1 gradient text.
- Borders: `#C6C9CD`; dividers: `#E2E4E7`; light background: `#F4F6F8`; surfaces: `#FFFFFF`.

## Components
- Cards are white with `1px #C6C9CD` borders, `3px` radius, and `20px` padding. Hover uses a subtle lift and brand-purple border, with no scale/rotation.
- Filled CTA buttons use white text on the AI gradient. Ghost buttons are transparent with purple border and text.
- Tags are uppercase, 11px heading font, ghost style.

## Layout
- Max content width is 1200px, centered.
- Use 40px between major sections, 20px between blocks, and 10px between adjacent buttons.
- Hero spacing is 60–80px. Use responsive stacking on mobile and avoid horizontal scroll.

## Do / Don't
- Do keep directory entries in real cards with useful names and descriptions.
- Do use inline SVG for illustrations and icons.
- Do maintain readable contrast on every surface.
- Don't add fake social proof, extra schema types, embedded builder forms, or external image dependencies.
