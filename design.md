# ARTITECT MACHINERY Design System

## Brand Identity
- **Brand**: ARTITECT MACHINERY
- **Industry**: Industrial Sheet Metal Folding Equipment
- **Tone**: Professional, precise, trustworthy, elegant

## Color Palette
- **Primary**: Deep navy blue (#1e3a5f) - conveys trust and industrial strength
- **Secondary**: Warm steel gray (#6b7280) - metallic, professional
- **Accent**: Amber/gold (#d97706) - highlights and CTAs
- **Background**: Clean white (#ffffff) and light gray (#f8fafc)
- **Text**: Dark charcoal (#111827) for headings, medium gray (#4b5563) for body

## Typography
- **Headings**: Inter, bold weights (700-800), tight letter spacing
- **Body**: Inter, regular weight (400-500), comfortable line height (1.6-1.75)
- **Accent text**: Inter, semibold (600)

## Spacing & Layout
- Generous whitespace to convey elegance and precision
- Consistent 8px grid system
- Max content width: 1200px
- Section padding: 80-120px vertical

## Visual Elements
- Subtle shadows for depth (not heavy)
- Rounded corners (8-12px) for modern feel
- Thin borders (1px) in light gray
- Gradient accents sparingly used
- Icons: Lucide React, clean line style

## Components
- Buttons: Solid fills, rounded, clear hover states
- Cards: White background, subtle shadow, rounded corners
- Navigation: Clean, minimal, sticky header
- Forms: Clean inputs with focus states

## Responsive Behavior
- Mobile-first approach
- Breakpoints: sm (640px), md (768px), lg (1024px), xl (1280px)
- Touch-friendly targets (min 44px)
- Readable font sizes on all devices

## Do's
- Use ample whitespace
- Keep typography clean and readable
- Use consistent spacing
- Show product images prominently
- Use subtle animations

## Don'ts
- Don't use cluttered layouts
- Don't use low-contrast text
- Don't overuse colors
- Don't use decorative elements that distract from products
- Don't use tiny text on mobile
