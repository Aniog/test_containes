# MicroCosmos Design System

## Visual Style Guide

### Color Palette
- **Primary Green**: `bg-emerald-600` - Deep forest green for primary elements
- **Secondary Green**: `bg-emerald-400` - Lighter green for accents
- **Earth Tones**: `bg-amber-100`, `bg-amber-200`, `bg-amber-300` - Warm earth backgrounds
- **Dark Forest**: `bg-slate-800`, `bg-slate-900` - Dark sections and text
- **Natural White**: `bg-stone-50`, `bg-white` - Clean backgrounds
- **Accent Brown**: `bg-amber-600`, `bg-amber-700` - Tree bark and wood tones

### Typography
- **Headings**: `font-bold`, `font-extrabold` with `text-slate-800` or `text-white`
- **Body Text**: `text-slate-700`, `text-slate-600` for readable content
- **Captions**: `text-sm`, `text-slate-500` for image descriptions
- **Hero Text**: Large, bold typography with `text-4xl`, `text-5xl`, `text-6xl`

### Spacing & Layout
- **Sections**: `py-16`, `py-20`, `py-24` for vertical spacing
- **Containers**: `max-w-7xl`, `max-w-6xl` with `mx-auto` and `px-4`, `px-6`
- **Cards**: `p-6`, `p-8` with `rounded-xl`, `rounded-2xl`
- **Grid Gaps**: `gap-6`, `gap-8`, `gap-12` for consistent spacing

### Visual Elements
- **Shadows**: `shadow-lg`, `shadow-xl`, `shadow-2xl` for depth
- **Borders**: `border-emerald-200`, `border-amber-200` for subtle divisions
- **Rounded Corners**: `rounded-xl`, `rounded-2xl`, `rounded-3xl` for modern feel
- **Gradients**: `bg-gradient-to-br from-emerald-50 to-amber-50` for backgrounds

### Image Styling
- **Aspect Ratios**: `aspect-video` (16:9), `aspect-square` (1:1), `aspect-[4/3]`
- **Object Fit**: `object-cover` for consistent image display
- **Hover Effects**: `hover:scale-105`, `transition-transform duration-300`
- **Overlays**: Semi-transparent overlays with `bg-black/20`, `bg-emerald-900/30`

### Interactive Elements
- **Buttons**: `bg-emerald-600 hover:bg-emerald-700` with `px-6 py-3 rounded-lg`
- **Cards**: `hover:shadow-xl transition-shadow duration-300`
- **Links**: `text-emerald-600 hover:text-emerald-700`

### Responsive Design
- **Mobile First**: Start with mobile styles, then add `md:`, `lg:`, `xl:` breakpoints
- **Grid Systems**: `grid-cols-1 md:grid-cols-2 lg:grid-cols-3 xl:grid-cols-4`
- **Text Scaling**: `text-2xl md:text-3xl lg:text-4xl xl:text-5xl`

### Do's and Don'ts
**Do:**
- Use natural, organic shapes and rounded corners
- Maintain high contrast for text readability
- Use earth tones and forest greens consistently
- Include plenty of whitespace for breathing room
- Make images the hero of each section

**Don't:**
- Use harsh, artificial colors
- Overcrowd sections with too much content
- Use small, hard-to-read text
- Forget mobile responsiveness
- Use low-contrast color combinations
