# Velmora Fine Jewelry — Design System

## Mood
Quiet luxury, warm, editorial. Premium demi-fine jewelry — NOT loud, NOT discount-looking, NOT generic e-commerce.

## Color Palette
- **Base (deep warm charcoal):** `#1C1917` (stone-900) — primary background for hero, nav, footer
- **Surface (warm off-white):** `#FAFAF9` (stone-50) — page background, cards
- **Accent (warm gold):** `#B8860B` — CTAs, highlights, decorative elements
- **Accent hover:** `#9A7209` — darker gold for hover states
- **Text primary:** `#1C1917` — on light surfaces
- **Text secondary:** `#78716C` (stone-500) — muted text, descriptions
- **Text on dark:** `#FAFAF9` — on dark surfaces
- **Border/divider:** `#E7E5E4` (stone-200) — hairline dividers
- **Card shadow:** `0 1px 3px rgba(0,0,0,0.06)` — subtle, restrained

## Typography
- **Headings:** Cormorant Garamond (serif) — elegant, editorial
- **Body/UI:** Inter (sans-serif) — clean, modern
- **Product names:** UPPERCASE, letter-spacing 0.15em, Cormorant Garamond
- **Nav links:** UPPERCASE, letter-spacing 0.1em, Inter, text-sm

## Spacing
- Generous whitespace throughout
- Section padding: 80px–120px vertical on desktop, 48px–64px on mobile
- Container max-width: 1280px, centered
- Grid gaps: 24px–32px

## Buttons
- Primary: solid accent gold bg, white text, uppercase, letter-spacing 0.1em, rounded-none, px-8 py-3
- Secondary: outlined (1px accent border), accent text, transparent bg
- Hover: subtle darkening, smooth 200ms transition

## Images
- Large, editorial, warm-lit
- Product cards: square or 4:5 ratio
- Hero: full-bleed, 16:9 or wider
- UGC reels: 9:16 vertical

## Animations
- Subtle hover transitions (200ms ease)
- Image crossfade on product hover
- Smooth scroll
- No bouncy or flashy animations

## Do's
- Use generous whitespace
- Use serif for all headings and product names
- Use hairline dividers (1px, stone-200)
- Keep accent color restrained — only for CTAs and key highlights
- Ensure strong contrast for accessibility

## Don'ts
- Don't use bright/saturated colors
- Don't use rounded buttons (use sharp/slightly rounded)
- Don't use heavy shadows
- Don't mix too many font styles
- Don't use generic e-commerce patterns (badges, countdown timers, etc.)
