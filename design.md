# Velmora Fine Jewelry — Design System

## Brand Direction
Quiet luxury, warm, editorial. Premium demi-fine jewelry. Not loud, not discount-looking, not generic e-commerce.

## Color Palette
- Background: `#ffffff`
- Foreground / text: `#0f0f0f`
- Muted surface: `#f5f5f0`
- Muted text: `#6b6b6b`
- Accent / gold: `#b08d57`
- Accent hover: `#9a7a4a`
- Border: `rgba(15, 15, 15, 0.08)`

Use the gold accent sparingly for CTAs, active states, and key highlights. Keep the rest of the UI neutral so jewelry imagery remains the focal point.

## Typography
- Serif headings: `Cormorant Garamond`, `Playfair Display`, Georgia
- Sans body/UI: `Inter`, `Manrope`, system-ui
- Product names: uppercase with wide letter-spacing
- Base size: 16px
- Line height: 1.5–1.6

## Spacing & Layout
- Generous whitespace
- Max content width: `max-w-7xl`
- Section padding: `py-16` to `py-24`
- Card padding: consistent, restrained

## Components
- Buttons: solid gold for primary actions, outlined for secondary
- Cards: subtle shadow, thin borders, soft hover transitions
- Dividers: thin hairline (`border-black/5` to `border-black/10`)
- Images: warm, editorial, large format

## Responsive
- Mobile-first
- Breakpoints: `sm:`, `md:`, `lg:`
- Ensure readability and touch targets on mobile

## Do
- Use Tailwind utility classes
- Keep contrast strong and accessible
- Use the gold accent intentionally

## Don't
- Use low-contrast text
- Over-decorate; let imagery lead
- Use generic e-commerce patterns that feel discount-oriented
