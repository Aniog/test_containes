# Velmora Fine Jewelry - Design System

## Visual Identity
- **Mood**: Quiet luxury, warm, editorial. Premium demi-fine jewelry.
- **Color Palette**:
  - Primary: Stone-900 (#1c1917) - Deep warm black
  - Secondary: Amber-700 (#b45309) - Warm metallic accent
  - Background: White, Stone-50 (#fafaf9)
  - Text: Stone-900 (headings), Stone-600 (body), Stone-400 (muted)
- **Typography**:
  - Headings: Cormorant Garamond / Playfair Display (serif)
  - Body: Inter / Manrope (sans-serif)
  - Product names: UPPERCASE with wide letter-spacing
- **Spacing**: Generous whitespace, large editorial imagery
- **Dividers**: Thin hairline borders (border-stone-200)
- **Buttons**: Solid or outlined accent, premium feel
- **Animations**: Subtle hover transitions, soft shadows

## Tailwind Classes
- Use `font-serif` for headings and product names
- Use `tracking-wider` or `tracking-widest` for uppercase text
- Use `text-stone-900` for primary text, `text-stone-600` for body
- Use `bg-amber-700` for accent buttons and highlights
- Use `border-stone-200` for subtle dividers
- Use `hover:text-amber-700` for interactive elements

## Do's and Don'ts
- DO use generous padding and margins
- DO use thin borders and subtle shadows
- DO maintain strong contrast for readability
- DON'T use loud colors or discount-style badges
- DON'T crowd elements on mobile
- DON'T use generic e-commerce styling
