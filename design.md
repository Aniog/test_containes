# Strikingly AI Generators Hub Design

Use a clean Strikingly marketing-page style with a white page background, thin dividers, compact cards, and geometric uppercase headings.

## Typography
- Headings and buttons use Brandon Grotesque where available, with Josefin Sans as the Google Fonts fallback. Use uppercase headings with 700 weight and 1.2 line-height.
- Body text uses Open Sans at a 14px base size and 1.5 line-height.

## Colors
- Heading text: `#4B5056`; hero first line: `#2E2E2F`.
- Body text: `#636972`.
- Brand purple accents: `#8159BB`.
- AI gradient for primary CTAs and H1 second line only: `#7671FF` to `#CB0C9F`.
- Card borders: `#C6C9CD`; subtle dividers: `#E2E4E7`; light background: `#F4F6F8`; page/card background: `#FFFFFF`.

## Layout and components
- Max content width is 1200px, centered with responsive inline padding.
- Section spacing uses 40px rhythm, with hero top/bottom spacing between 60px and 80px.
- Cards use white backgrounds, 1px borders, 3px radius, and 20px padding. Hover states lift subtly with a purple border.
- Buttons use 4px radius, uppercase heading font, visible focus rings, and white text on filled gradients.

## Do's and don'ts
- Do keep all text readable with explicit foreground colors.
- Do use inline SVG or CSS illustrations only.
- Do preserve the static HTML directory so links are available without JavaScript.
- Do not use fake social proof, image URLs, dark text on gradient fills, or placeholder `href="#"` links.
