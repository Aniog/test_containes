# Velmora Fine Jewelry — Design System

## Brand Mood
Quiet luxury, warm, editorial. Premium demi-fine jewelry. NOT loud, NOT discount-looking, NOT generic e-commerce.

## Color Palette
| Token | Hex | Usage |
|---|---|---|
| cream | #F8F5F0 | Page backgrounds, section alternation |
| warm-white | #FAF8F5 | Cards, modals, light surfaces |
| charcoal | #1C1917 | Primary text, headings |
| deep-brown | #292524 | Nav bar solid state, dark sections |
| warm-gray | #78716C | Secondary text, captions |
| light-gray | #D6D3D1 | Borders, dividers, muted elements |
| gold | #B8860B | Primary accent, buttons, links |
| gold-light | #D4A843 | Hover states, highlights |
| gold-muted | #C9A96E | Tags, subtle accents |
| gold-pale | #E8D5A3 | Background tints, skeleton loaders |

## Typography
- **Headings**: Cormorant Garamond, serif. Product names UPPERCASE with tracking-[0.12em].
- **Body/UI**: Inter, sans-serif. 15px body size.
- **Overlines**: Inter uppercase, tracking-widest, text-overline size.

## Spacing
- Section padding: py-20 md:py-28
- Card gap: gap-6 md:gap-8
- Content max-width: max-w-7xl mx-auto

## Borders & Dividers
- Hairline: border border-border (light) or border-b border-light-gray
- Card: rounded-lg, shadow-card

## Buttons
- Primary: bg-gold text-white hover:bg-gold-light, py-3.5 px-8, uppercase tracking-widest text-sm font-sans
- Secondary/Outline: border border-gold text-gold hover:bg-gold hover:text-white

## Animations
- Hover transitions: duration-300 ease-luxury
- Cards: hover:shadow-card-hover hover:-translate-y-1
- Images: hover:scale-105 transition-transform duration-500

## Do's
- Use generous whitespace
- Keep imagery large and editorial
- Use thin hairline dividers between sections
- Maintain warm, golden tones throughout
- Ensure strong text contrast (charcoal on cream)

## Don'ts
- Don't use discount/sale banners
- Don't use bright saturated colors
- Don't use rounded/cartoon styling
- Don't use generic e-commerce layouts
- Don't let text contrast fail accessibility
