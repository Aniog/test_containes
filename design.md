# Velmora Fine Jewelry - Design System

## Visual Identity
Quiet luxury, warm editorial aesthetic. Premium demi-fine jewelry brand targeting women 25-45.

## Color Palette
- **Background (ivory):** `#FAF8F5` — warm cream, primary page background
- **Foreground (near-black):** `#1C1917` — primary text color
- **Card:** `#FFFFFF` — card surfaces
- **Card Foreground:** `#1C1917`
- **Primary (warm gold):** `#B8964E` — CTAs, accents, gold metallic
- **Primary Foreground:** `#FFFFFF` — text on primary buttons
- **Secondary (bronze):** `#8B7355` — secondary accents, muted gold
- **Muted:** `#F5F0EB` — subtle background sections
- **Muted Foreground:** `#78716C` — secondary text, descriptions
- **Border:** `#E7DFD5` — dividers, card borders, hairlines
- **Dark Section:** `#1C1917` — dark footer, hero overlay
- **Dark Foreground:** `#FAF8F5` — text on dark backgrounds
- **Accent Hover:** `#A6843E` — gold hover state (darker)

## Typography
- **Headings & Product Names:** Cormorant Garamond (serif) — elegant, editorial
- **Body & UI:** Inter (sans-serif) — clean, readable
- **Product Names:** UPPERCASE, `letter-spacing: 0.15em`
- **Headings:** Title case, semi-bold, generous line-height

## Spacing & Layout
- Generous whitespace — sections use `py-20 md:py-28`
- Max content width: `max-w-7xl mx-auto`
- Card padding: `p-6`
- Section gap: `gap-8 md:gap-12`

## Visual Elements
- Thin hairline dividers: `border border-[#E7DFD5]`
- Subtle shadows: `shadow-sm` for cards, `shadow-lg` for dropdowns
- Hover transitions: `transition-all duration-300`
- Rounded corners: `rounded-sm` for buttons, `rounded-md` for cards

## Buttons
- **Primary:** Solid gold background (#B8964E), white text, uppercase letter-spacing
- **Secondary:** Outlined with gold border, gold text
- **Ghost:** Transparent, muted text, subtle hover

## Do's
- Use generous whitespace
- Keep gold tones warm and consistent
- Use serif for all headings and product names
- Maintain high contrast for accessibility

## Don'ts
- Use bright/loud colors
- Use decorative or script fonts
- Overcrowd layouts
- Use discount/sale visual language
