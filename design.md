# ARTITECT MACHINERY - Design Guidelines

## Brand Identity
- **Brand Name**: ARTITECT MACHINERY
- **Industry**: Industrial sheet metal folding machines
- **Products**: Double folding machine, double folder, sheet metal folder, sheet metal folding machine, metal folder, metal folder machine, metal folding machine

## Visual Style
Elegant but user-friendly - Professional industrial aesthetic with clean, modern design that feels approachable.

## Color Palette
- **Primary**: `#1a1a2e` (Deep navy/charcoal - professional, industrial)
- **Secondary**: `#16213e` (Darker blue for depth)
- **Accent**: `#e94560` (Vibrant red-orange for CTAs and highlights)
- **Light**: `#f8f9fa` (Off-white for backgrounds)
- **Text Primary**: `#1a1a2e` (Dark for readability)
- **Text Secondary**: `#6c757d` (Muted gray)
- **Border**: `#dee2e6` (Subtle borders)

## Typography
- **Headings**: "Outfit" - Modern, geometric sans-serif with industrial feel
- **Body**: "DM Sans" - Clean, readable, friendly
- **Font Sizes**:
  - Hero: 56px (desktop), 36px (mobile)
  - H1: 42px
  - H2: 32px
  - H3: 24px
  - Body: 16px
  - Small: 14px

## Layout & Spacing
- **Max Width**: 1280px
- **Section Padding**: 80px vertical (desktop), 48px (mobile)
- **Container Padding**: 24px horizontal
- **Grid Gap**: 32px
- **Border Radius**: 12px (cards), 8px (buttons), 4px (small elements)

## Components

### Header/Navigation
- Fixed position, white background with subtle shadow on scroll
- Logo on left, navigation links center, CTA button right
- Mobile: Hamburger menu with slide-out drawer

### Hero Section
- Full-width with subtle geometric background pattern
- Large headline with brand name emphasis
- Subheadline describing the products
- Two CTAs: "View Products" (primary), "Contact Us" (secondary)
- Background: Subtle gradient with industrial pattern overlay

### Product Cards
- White background with subtle shadow
- Product image area (placeholder with gradient)
- Product name
- Brief description
- "Learn More" link
- Hover: Lift effect with enhanced shadow

### Features Section
- Icon + text layout
- 3-4 key selling points
- Clean grid layout

### About Section
- Split layout: Image left, text right
- Company description
- Key statistics

### Contact Section
- Clean form layout
- Fields: Name, Email, Phone, Message
- Submit button with loading state

### Footer
- Dark background
- Logo, navigation links, contact info
- Copyright

## Animations & Effects
- Smooth scroll behavior
- Fade-in on scroll for sections
- Button hover: Scale and color transition
- Card hover: Lift and shadow enhancement
- Header shadow on scroll

## Responsive Breakpoints
- **Mobile**: < 640px
- **Tablet**: 640px - 1024px
- **Desktop**: > 1024px

## Do's
- Use generous whitespace for elegance
- Keep text concise and scannable
- Use consistent spacing throughout
- Make CTAs prominent but not aggressive
- Use subtle shadows for depth
- Maintain visual hierarchy

## Don'ts
- Don't use too many colors (keep it minimal)
- Don't clutter with too many elements
- Don't use aggressive or bright colors for CTAs (keep accent subtle)
- Don't use decorative fonts for body text
- Don't make it look too corporate or cold
