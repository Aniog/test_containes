# Strikingly Generators Hub Design System

## Typography
- Headings and buttons use Brandon Grotesque when available, with Josefin Sans as the primary web-safe fallback and Poppins as a secondary fallback.
- Headings are uppercase, weight 700, and line-height 1.2.
- Body copy uses Open Sans, weight 400, 14px base size, and line-height 1.5.

## Colors
- Brand purple: `#8159BB` for accents, borders, and ghost buttons.
- AI gradient: `linear-gradient(90deg, #7671FF 0%, #CB0C9F 100%)` only for primary CTAs and the second hero headline line.
- Body text: `#636972`.
- Section headings: `#4B5056`.
- Hero headline dark: `#2E2E2F`.
- Card border: `#C6C9CD`.
- Dividers: `#E2E4E7`.
- Light background: `#F4F6F8`.
- Default surface: `#FFFFFF`.

## Components
- Buttons are uppercase, 4px radius, 36px standard height or 44px large height, with 9px 15px padding.
- Filled gradient buttons always use white text.
- Ghost buttons use transparent background, a 1px brand-purple border, and brand-purple text.
- Cards use white background, a 1px `#C6C9CD` border, 3px radius, and 20px padding.
- Card hover state uses subtle shadow lift and brand-purple border with no scale or rotation.
- Category tags use uppercase heading font, 11px size, 2px 6px padding, 3px radius, brand-purple text, and a brand-purple border.

## Layout
- Max content width is 1200px centered.
- Spacing uses 10px multiples, with 5px only for tight pairs.
- Major sections are spaced by 40px.
- Hero uses 60px to 80px vertical padding.
- Desktop grids are 3 columns where specified, tablet grids are 2 columns, and mobile grids are 1 column.

## Do's and Don'ts
- Do keep the directory useful and organized, with every generator inside a real card.
- Do use inline SVG for decorative illustrations.
- Do keep all links and controls keyboard accessible with visible focus states.
- Do not use fake ratings, testimonials, social proof, or placeholder `href="#"` links.
- Do not use the AI gradient as a large background treatment outside primary CTAs and the hero headline accent.
