# 秋易广告 Design System

## Brand Identity
- Company: 秋易广告 (Qiuyi Advertising)
- Tagline: 创意无界，广告有道
- Industry: Advertising / Creative Agency

## Color Palette
- Primary: Deep orange-red `#E8431A` (brand accent)
- Secondary: Dark charcoal `#1A1A2E` (backgrounds, headers)
- Accent Gold: `#F5A623` (highlights, CTAs)
- Light Cream: `#FFF8F0` (section backgrounds)
- White: `#FFFFFF`
- Text Dark: `#1A1A2E`
- Text Muted: `#6B7280`
- Border: `#E5E7EB`

## Typography
- Heading Font: "Noto Serif SC" (Chinese serif for elegance)
- Body Font: "Noto Sans SC" (clean Chinese sans-serif)
- Fallback: system-ui, sans-serif

### Scale
- Hero Title: text-5xl md:text-7xl font-bold
- Section Title: text-3xl md:text-4xl font-bold
- Card Title: text-xl font-semibold
- Body: text-base leading-relaxed
- Caption: text-sm text-gray-500

## Spacing
- Section padding: py-20 px-4 md:px-8
- Container max-width: max-w-7xl mx-auto
- Card gap: gap-6 md:gap-8
- Component padding: p-6 md:p-8

## Borders & Shadows
- Card: rounded-2xl shadow-lg
- Button: rounded-full
- Image: rounded-xl overflow-hidden
- Hover shadow: hover:shadow-2xl transition-shadow duration-300

## Visual Style
- Modern Chinese advertising agency aesthetic
- Bold typography with large hero images
- Grid-based portfolio/case study layouts
- Warm orange-red brand color as accent
- Dark header/footer with light content sections
- Generous whitespace between sections
- Subtle gradient overlays on hero images

## Do's
- Use large, full-width hero images with text overlays
- Use grid layouts for portfolio/case studies (2-3 columns on desktop)
- Use the brand orange-red for CTAs and highlights
- Use rounded corners on cards and images
- Use smooth hover transitions on interactive elements
- Keep navigation sticky and minimal

## Don'ts
- Don't use low-contrast text on colored backgrounds
- Don't use more than 3 font weights per section
- Don't stack single-column layouts on desktop
- Don't use harsh drop shadows; prefer soft shadows
