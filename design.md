# ARTITECT MACHINERY - Design System

## Brand Overview
- **Brand Name**: ARTITECT MACHINERY
- **Industry**: Industrial machinery, sheet metal fabrication equipment
- **Tone**: Elegant, professional, trustworthy, approachable

## Visual Style

### Color Palette
- **Primary**: Deep charcoal #1C2526 (headers, navigation, accents)
- **Secondary**: Warm steel gray #4A5568 (body text, secondary elements)
- **Accent**: Industrial bronze #B8860B (CTAs, highlights, hover states)
- **Light Background**: Off-white #F8F7F4 (main background)
- **Card/Surface**: Pure white #FFFFFF (cards, modals, sections)
- **Border/Subtle**: Soft gray #E5E4E0 (dividers, borders)
- **Success/Trust**: Deep forest green #2D5A4A (for trust signals)

### Typography
- **Primary Font**: Inter (system-ui fallback)
- **Headings**: 700 weight, tight letter-spacing (-0.02em)
- **Body**: 400-500 weight, comfortable line-height (1.7)
- **Scale**:
  - H1: 3.5rem (56px) - Hero headlines
  - H2: 2.25rem (36px) - Section titles
  - H3: 1.5rem (24px) - Card titles, subsections
  - Body: 1rem (16px) - Standard text
  - Small: 0.875rem (14px) - Captions, meta

### Spacing & Layout
- **Section Padding**: 5rem top/bottom (80px)
- **Container Max Width**: 1280px (centered)
- **Card Padding**: 2rem (32px)
- **Gap Between Elements**: 1.5rem (24px) standard
- **Consistent 8px grid system**

### Visual Elements
- **Buttons**: 
  - Primary: Solid charcoal background, white text, bronze hover
  - Secondary: White background, charcoal border, subtle hover
  - Rounded corners: 6px
- **Cards**: Clean white with subtle shadow (0 4px 6px -1px rgb(0 0 0 / 0.1)), soft border
- **Images**: High-quality industrial photography, consistent aspect ratios
- **Icons**: Lucide React icons, line weight 2, sized appropriately

## User Experience Principles

### Elegant
- Generous whitespace
- Sophisticated typography hierarchy
- Subtle animations and transitions (150-200ms)
- Professional photography and imagery
- Refined color combinations

### User-Friendly
- Clear navigation with sticky header
- Prominent call-to-action buttons
- Easy-to-scan product cards
- Simple contact form
- Mobile-responsive at all breakpoints
- Fast loading, optimized images
- Clear value propositions

## Page Structure

1. **Navigation** - Sticky, clean, logo + menu + CTA
2. **Hero** - Bold headline, subheadline, primary CTA, hero image
3. **About** - Brief company story and expertise
4. **Products** - Grid of all product variants with descriptions
5. **Why Choose Us** - Key benefits and differentiators
6. **Specifications** - Technical highlights table or cards
7. **Contact** - Lead capture form + contact info

## Product Categories to Feature
- Double folding machine
- Double folder
- Sheet metal folder
- Sheet metal folding machine
- Metal folder
- Metal folder machine
- Metal folding machine

Each product card should highlight:
- Product name
- Short description
- Key capability (e.g., "Precision folding up to 4mm steel")
- "Learn More" or "Request Quote" action

## Responsive Breakpoints
- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px

## Accessibility
- High contrast text
- Proper heading hierarchy
- Alt text for all images
- Keyboard navigable
- Focus states visible

## Do's
- Use ample white space
- Keep copy concise and benefit-focused
- Use professional industrial imagery
- Maintain consistent alignment
- Provide clear next steps (CTAs)

## Don'ts
- Avoid cluttered layouts
- No bright/neon colors
- Don't use cartoonish icons
- Avoid overly technical jargon without explanation
- No low-contrast text
