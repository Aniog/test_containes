# ARTITECT MACHINERY - Design Guidelines

## Brand Identity
- **Brand Name**: ARTITECT MACHINERY
- **Industry**: Industrial sheet metal folding machines
- **Products**: Double folding machine, double folder, sheet metal folder, sheet metal folding machine, metal folder, metal folder machine, metal folding machine

## Visual Style: Elegant but User-Friendly

### Color Palette
- **Primary**: `#1a1a2e` (Deep navy/charcoal - professional, industrial)
- **Secondary**: `#16213e` (Darker navy for depth)
- **Accent**: `#e94560` (Vibrant red-orange - machinery/industrial feel)
- **Accent Secondary**: `#0f3460` (Medium blue for contrast)
- **Text Primary**: `#ffffff` (White)
- **Text Secondary**: `#a0a0a0` (Muted gray)
- **Background**: `#0f0f1a` (Near black with blue undertone)
- **Card Background**: `#1a1a2e` with subtle border `#2a2a4e`
- **Success/Highlight**: `#4ade80` (Green for CTAs)

### Typography
- **Headings**: "Outfit" - Modern, geometric sans-serif with industrial feel
- **Body**: "DM Sans" - Clean, readable, professional
- **Font Weights**: 300 (light), 400 (regular), 500 (medium), 600 (semibold), 700 (bold)
- **Heading Sizes**: 
  - H1: 4rem (64px) - Hero
  - H2: 2.5rem (40px) - Section titles
  - H3: 1.5rem (24px) - Card titles
  - H4: 1.25rem (20px) - Subheadings

### Layout & Spacing
- **Max Content Width**: 1400px
- **Section Padding**: 80px vertical, 40px horizontal
- **Card Padding**: 32px
- **Grid Gap**: 32px
- **Border Radius**: 12px (cards), 8px (buttons), 4px (small elements)

### Visual Effects
- **Card Shadows**: `0 4px 24px rgba(0, 0, 0, 0.3)`
- **Hover Effects**: Scale 1.02, shadow increase, border glow
- **Transitions**: 0.3s ease for all interactive elements
- **Background**: Subtle geometric patterns with gradient overlays

### Components

#### Header/Navigation
- Fixed position, transparent with blur backdrop
- Logo on left, navigation center, CTA button right
- Mobile: Hamburger menu

#### Hero Section
- Full viewport height
- Large headline with gradient text accent
- Subheadline describing products
- Two CTA buttons: Primary (accent color) and Secondary (outline)
- Background: Dark with subtle industrial pattern or gradient

#### Features Section
- 3-column grid of feature cards
- Icon + Title + Description
- Subtle hover animation

#### Products Section
- Product cards with image placeholder, title, description
- "Learn More" link
- Grid layout

#### About Section
- Split layout: Image + Text
- Company story and expertise

#### Contact/CTA Section
- Call to action with background
- Contact information
- Inquiry form or button

#### Footer
- Logo, navigation links, contact info
- Social links
- Copyright

### Do's
- Use ample white space for readability
- Keep text concise and scannable
- Use clear visual hierarchy
- Include clear CTAs
- Make it mobile responsive
- Use subtle animations for engagement

### Don'ts
- Don't use too many colors (keep to palette)
- Don't clutter with too much text
- Don't use generic stock photos that don't match industrial theme
- Don't make it feel cold or impersonal - balance elegance with warmth
