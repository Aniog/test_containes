# Brand Name
ARTITECT MACHINERY

# Style
Elegant but user-friendly. Geared towards industrial machinery with a sleek, professional, modern, yet highly readable presentation.

# Colors
Primary: Dark Slate (`slate-900` / `#0f172a`) - for a solid, industrial feel.
Secondary: Steel Blue (`slate-600` / `#475569`) - for secondary elements.
Accent: Subtle Amber/Gold (`amber-500` / `#f59e0b`) - for call-to-actions, buttons, highlighting the "elegant" part.
Background: Light Gray (`gray-50` / `#f9fafb`) - for a clean, user-friendly space.
Text: Dark Gray (`gray-800` / `#1f2937`) - for high readability.

# Typography
Headings: Inter, bold, tracking tight.
Body: Inter, regular, legible line height.

# Layout
- Strong but elegant geometry. Clean lines and solid colors. Use of whitespace to make content easy to digest.
- Responsive design with stacking layout on mobile, side-by-side or grid on desktop.
- Subtle shadows on cards.

# Common Elements
- Buttons: Rounded corners (md or lg, not full pill shape to keep it industrial but elegant), solid background, hover effect (e.g. `bg-amber-500 hover:bg-amber-600 text-white`).
- Cards: White background, subtle shadow, rounded corners, clean padding.
