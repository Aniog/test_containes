# Design System

## Visual Style
Clean, modern SaaS landing page aesthetic. Light background with indigo as the primary accent color. Professional and trustworthy feel.

## Colors
- Primary: Indigo — `bg-indigo-600`, `text-indigo-600`, `hover:bg-indigo-700`
- Primary light: `bg-indigo-50`, `text-indigo-700`
- Background: White `bg-white` for cards, `bg-slate-50` for page background
- Surface: `bg-white` with `shadow-sm border border-slate-200`
- Text primary: `text-slate-900`
- Text secondary: `text-slate-600`
- Text muted: `text-slate-400`
- Border: `border-slate-200` or `border-slate-300`
- Success: `text-emerald-600`, `bg-emerald-50`
- Error: `text-red-600`, `bg-red-50`

## Typography
- Font: Inter (Google Fonts)
- Hero heading: `text-4xl md:text-5xl lg:text-6xl font-bold text-slate-900 tracking-tight`
- Section heading: `text-3xl font-bold text-slate-900`
- Card heading: `text-xl font-semibold text-slate-900`
- Body: `text-base text-slate-600 leading-relaxed`
- Small/caption: `text-sm text-slate-500`
- Label: `text-sm font-medium text-slate-700`

## Spacing
- Section padding: `py-20 px-4`
- Card padding: `p-6` or `p-8`
- Form field gap: `space-y-4`
- Section gap: `space-y-12`

## Borders & Shadows
- Card: `rounded-2xl border border-slate-200 shadow-sm`
- Input: `rounded-lg border border-slate-300`
- Button: `rounded-lg`
- Large card/hero: `rounded-3xl`

## Components
- Primary button: `bg-indigo-600 text-white hover:bg-indigo-700 rounded-lg px-5 py-2.5 font-medium`
- Outline button: `border border-slate-300 bg-white text-slate-700 hover:bg-slate-50`
- Form inputs: white background, slate border, indigo focus ring
- Feature cards: white bg, subtle border, rounded-2xl, hover shadow

## Do's
- Use generous whitespace between sections
- Keep forms clean with clear labels above inputs
- Use indigo for all primary CTAs
- Use slate grays for text hierarchy
- Responsive: stack columns on mobile, side-by-side on md+

## Don'ts
- Don't use dark backgrounds for the main page
- Don't use more than 2 accent colors
- Don't use tight spacing in forms
- Don't use low-contrast text on colored backgrounds
