# Velmora Fine Jewelry - Design System

## Visual Identity

### Mood
- Quiet luxury, warm, editorial
- Premium demi-fine jewelry aesthetic
- NOT loud, NOT discount-looking, NOT generic e-commerce

### Color Palette
**Primary Colors:**
- Deep Charcoal: `#1A1A1A` (main background, text)
- Warm Cream: `#FAF7F2` (light background sections)
- Rich Black: `#0D0D0D` (dark sections)

**Accent Colors:**
- Warm Gold: `#C9A962` (primary accent, CTAs, highlights)
- Soft Gold: `#E8DCC4` (secondary accent, borders)
- Muted Bronze: `#8B7355` (subtle accents)

**Neutral Colors:**
- Warm Gray: `#6B6560` (secondary text)
- Light Stone: `#E5E0D8` (dividers, subtle backgrounds)
- Off White: `#FDFCFA` (cards, elevated surfaces)

### Typography
**Headings & Product Names:**
- Font: Cormorant Garamond (Google Fonts)
- Weights: 400, 500, 600
- Product names: UPPERCASE with letter-spacing: 0.15em

**Body & UI:**
- Font: Manrope (Google Fonts)
- Weights: 300, 400, 500, 600

**Font Sizes:**
- Hero headline: 56px desktop / 36px mobile
- Section titles: 40px desktop / 28px mobile
- Product names: 14px (uppercase, wide spacing)
- Body: 16px
- Small/UI: 14px
- Caption: 12px

### Spacing System
- Base unit: 4px
- Section padding: 80px vertical desktop / 48px mobile
- Card padding: 24px
- Grid gap: 24px desktop / 16px mobile

### Visual Effects
- Hairline dividers: 1px solid #E5E0D8
- Soft shadows: 0 4px 20px rgba(0,0,0,0.06)
- Hover transitions: 300ms ease
- Image hover: subtle scale (1.02) with opacity change

## Components

### Buttons
**Primary (Accent):**
- Background: #C9A962
- Text: #0D0D0D
- Padding: 16px 32px
- Border-radius: 0 (sharp corners for premium feel)
- Hover: darken background, subtle lift

**Secondary (Outlined):**
- Border: 1px solid #C9A962
- Text: #C9A962
- Background: transparent
- Hover: fill with gold, darken text

### Cards
- Background: #FDFCFA
- Border: 1px solid #E5E0D8
- Padding: 24px
- Hover: subtle shadow increase, image zoom

### Form Inputs
- Border: 1px solid #E5E0D8
- Background: #FDFCFA
- Focus: border-color #C9A962
- Padding: 12px 16px

## Layout

### Container
- Max-width: 1400px
- Padding: 0 24px (desktop), 0 16px (mobile)

### Grid
- Desktop: 5 columns for products
- Tablet: 3 columns
- Mobile: 2 columns

### Responsive Breakpoints
- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px

## Animations

### Transitions
- Default: 300ms ease
- Fast: 150ms ease
- Slow: 500ms ease

### Effects
- Fade in on scroll
- Image zoom on hover
- Button lift on hover
- Smooth page transitions
- Cart drawer slide

## Do's and Don'ts

### Do
- Use generous whitespace
- Keep text minimal and elegant
- Use high-quality imagery
- Maintain consistent gold accent color
- Use serif fonts for headings

### Don't
- Use bright/discount colors
- Overcrowd elements
- Use generic sans-serif for headings
- Add too many CTAs
- Use rounded corners (keep sharp for premium feel)
