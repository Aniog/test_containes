# Design Guidelines - ARTITECT MACHINERY

## Visual Style
Elegant, professional, and industrial. The design should convey precision, reliability, and high-end engineering.

## Typography
- **Primary Font**: 'Inter', sans-serif (Global font)
- **Headings**: Semi-bold to Bold (font-weight: 600 or 700) for a strong, industrial feel.
- **Body**: Regular (font-weight: 400) for readability.

## Color Palette
- **Primary (Industrial Navy)**: `#1a2b3c` - Deep blue/grey for a professional look.
- **Secondary (Steel Silver)**: `#f5f7f9` - Light grey backgrounds.
- **Accent (Precision Orange/Gold)**: `#d4af37` - Gold or muted orange for call-to-actions, highlighting quality.
- **Text (Charcoal)**: `#333333` - Primary text for readability.
- **Muted**: `#666666` - Secondary text.

## Spacing and Layout
- **Container**: Use `max-w-7xl mx-auto px-4 sm:px-6 lg:px-8` for consistent horizontal padding.
- **Sections**: Vertical padding of `py-12 md:py-24`.
- **Gaps**: Standardize gaps using Tailwind's `gap-4`, `gap-8`, etc.

## Components
- **Buttons**: Rounded (but not pill) for a modern, industrial feel. Using `rounded-md`.
- **Cards**: Subtle shadows (`shadow-sm` or `shadow-md`) and white backgrounds to keep it clean and elegant.
- **Navigation**: Sticky header with clear navigation links.

## Icons
- Use **Lucide React** for consistent, clean iconography.

## Images
- Use the stock image system with `data-strk-img` and `data-strk-bg`.
- Focus on close-ups of machinery, architectural metalwork, and precision engineering.
