# Strikingly Generators Hub Design

## Visual direction
- Clean SaaS marketing layout that feels native to Strikingly.
- White page background with light section breaks using `#F4F6F8`.
- Thin dividers with `#E2E4E7` and card borders with `#C6C9CD`.
- Use restrained purple accents with the AI gradient only for primary CTAs and hero highlighted text.

## Typography
- Headings: `"Brandon Grotesque", "Josefin Sans", "Poppins", sans-serif`, weight 700, uppercase, tight line-height.
- Body: `"Open Sans", sans-serif`, 14px base, 1.5 line-height.
- Buttons and tags use the heading font in uppercase.

## Color tokens
- Brand purple: `#8159BB`
- AI gradient: `linear-gradient(135deg, #7671FF 0%, #CB0C9F 100%)`
- Heading gray: `#4B5056`
- Hero dark heading: `#2E2E2F`
- Body text: `#636972`
- Divider: `#E2E4E7`
- Card border: `#C6C9CD`
- Light section background: `#F4F6F8`
- White: `#FFFFFF`

## Components
- Cards: white fill, 1px border, 3px radius, 20px padding, subtle hover shadow, purple hover border.
- Buttons: 4px radius, 36px standard height, 44px large height, uppercase heading font.
- Filled buttons always use white text.
- Ghost buttons use white background or transparent background with purple border and purple text.
- Tags are small ghost badges with purple border and text.

## Spacing
- Use a centered container with max width 1200px.
- Prefer spacing in 10px increments; 20px within cards and between nearby blocks.
- Use 40px or more between major sections.

## Do
- Keep directory pages highly scannable with strong headings and one-line descriptions.
- Preserve clear contrast for all text and buttons.
- Keep interactions subtle and fast.

## Don't
- Do not use fake testimonials or social proof.
- Do not flood the page with gradients or tinted panels.
- Do not rely on JavaScript for core content visibility.
