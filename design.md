# Design Document - MicroCosmos

## Visual Style
The website for "MicroCosmos" should evoke a sense of wonder and discovery of the unseen world. The design uses a dark, sophisticated palette to allow the intricate details of microscopic images to stand out.

### Colors
- **Primary Background**: `bg-[#0a0a0c]` (A very dark, almost black blue-gray)
- **Secondary Background**: `bg-[#121216]` (For cards and sections)
- **Accent Primary**: `text-cyan-400` / `bg-cyan-400` (Bioluminescent cyan)
- **Accent Secondary**: `text-emerald-400` / `bg-emerald-400` (Organic green)
- **Text Primary**: `text-slate-50`
- **Text Secondary**: `text-slate-400`

### Typography
- **Heading**: `font-sans font-bold tracking-tight`
- **Body**: `font-sans leading-relaxed`
- **Global Font**: 'Inter', sans-serif

### Spacing & Layout
- Large vertical sections with generous padding (`py-24` or `py-32`).
- Responsive grid layouts for image galleries.
- Rounded corners for cards and images (`rounded-2xl`).

### Components Style
- **Cards**: Subtle borders (`border-white/5`) and soft shadows.
- **Buttons**: Glow effects using `shadow-[0_0_15px_rgba(34,211,238,0.3)]`.
- **Images**: High-quality macro/micro photography using the `data-strk-img` system.

### Do's and Don'ts
- **Do**: Use high-contrast imagery.
- **Do**: Make text legible against dark backgrounds.
- **Do**: Use immersive scroll sections.
- **Don't**: Use flat, uninspired colors.
- **Don't**: Overcrowd text; let the images speak.
