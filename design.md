# Velmora Fine Jewelry - Design System

## Visual Identity

### Color Palette
- **Primary Background**: `#FAF9F7` (warm off-white/cream)
- **Secondary Background**: `#F5F3EF` (warm light beige)
- **Dark Background**: `#1A1815` (warm charcoal)
- **Primary Text**: `#1A1815` (warm charcoal)
- **Secondary Text**: `#6B6560` (warm gray)
- **Accent Gold**: `#C9A962` (warm gold)
- **Accent Gold Hover**: `#B8954F` (deeper gold)
- **Border Color**: `#E8E4DE` (warm light border)
- **White**: `#FFFFFF`

### Typography
- **Headings**: Cormorant Garamond (serif) - elegant, editorial
  - H1: 56px / 64px line-height / font-weight 500
  - H2: 40px / 48px line-height / font-weight 500
  - H3: 28px / 36px line-height / font-weight 500
  - Product names: 16px / uppercase / 0.15em letter-spacing / font-weight 500
- **Body**: Manrope (sans-serif) - clean, modern
  - Body: 15px / 24px line-height / font-weight 400
  - Small: 13px / 20px line-height
  - Button: 13px / uppercase / 0.1em letter-spacing / font-weight 500

### Spacing System
- Base unit: 4px
- Section padding: 80px vertical (desktop), 48px (mobile)
- Container max-width: 1280px
- Grid gap: 24px (desktop), 16px (mobile)

### Visual Effects
- **Shadows**: 
  - Card: `0 4px 20px rgba(26, 24, 21, 0.06)`
  - Hover: `0 8px 30px rgba(26, 24, 21, 0.1)`
  - Button: `0 4px 12px rgba(201, 169, 98, 0.3)`
- **Border radius**: 0px (sharp, editorial) for buttons, 4px for cards
- **Transitions**: 0.3s ease for all interactive elements

### Components

#### Buttons
- **Primary**: Gold background, dark text, solid
  - Hover: Slightly darker gold, subtle shadow
- **Secondary**: Transparent with gold border
  - Hover: Gold background with dark text

#### Product Cards
- Sharp corners (0px radius)
- Image with hover zoom effect
- Quick add overlay on hover
- Product name in uppercase serif with wide spacing

#### Dividers
- Hairline: 1px solid #E8E4DE
- Used between sections

## Responsive Breakpoints
- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px
