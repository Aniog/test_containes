# Velmora Fine Jewelry — Design System

## Visual Identity
- Mood: Quiet luxury, warm, editorial. Premium demi-fine jewelry.
- NOT loud, NOT discount-looking, NOT generic e-commerce.

## Color Palette
- **Background**: `brand-cream` (#FAF7F2) — warm off-white, primary background
- **Surface**: `brand-warm` (#F5EDE3) — slightly warmer, for cards and sections
- **Accent surface**: `brand-sand` (#E8DDD0) — hover states, subtle backgrounds
- **Primary accent**: `brand-gold` (#C9A96E) — CTAs, links, decorative elements
- **Gold hover**: `brand-gold-light` (#D4BA8A) — hover state for gold elements
- **Gold pressed**: `brand-gold-dark` (#A8864E) — active state for gold elements
- **Text primary**: `brand-charcoal` (#2C2C2C) — headings, body text
- **Text secondary**: `brand-warm-gray` (#8A8279) — captions, secondary text
- **Text muted**: `brand-cool-gray` (#6B6560) — subtle labels
- **Dark surface**: `brand-espresso` (#1A1A1A) — hero overlays, footer
- **Pure surface**: `brand-ivory` (#FFFEF9) — newsletter, special sections

## Typography
- **Headings**: Cormorant Garamond (serif), weights 400-600
- **Body/UI**: Inter (sans-serif), weights 300-500
- **Product names**: UPPERCASE, letter-spacing 0.2em (tracking-super-wide)
- **Section headings**: UPPERCASE, letter-spacing 0.15em (tracking-extra-wide)
- **Nav links**: UPPERCASE, letter-spacing 0.15em, font-size 0.75rem

## Spacing
- Section padding: py-20 md:py-28
- Container max-width: max-w-7xl mx-auto px-5 md:px-8
- Card gaps: gap-6 md:gap-8
- Generous whitespace throughout

## Borders & Dividers
- Hairline dividers: border-t border-brand-sand/60
- Card borders: none (use subtle shadows instead)
- Button borders: 1px solid brand-gold for outlined buttons

## Buttons
- **Primary (solid)**: bg-brand-gold text-white hover:bg-brand-gold-dark, px-8 py-3, tracking-extra-wide uppercase text-xs font-sans font-medium
- **Secondary (outlined)**: border border-brand-gold text-brand-gold hover:bg-brand-gold hover:text-white, same sizing
- **Subtle**: text-brand-charcoal hover:text-brand-gold, underline on hover

## Shadows
- Cards: shadow-sm hover:shadow-md
- Modals/drawers: shadow-xl
- Subtle and warm, never harsh

## Transitions
- Default: transition-all duration-300
- Hover lifts: hover:-translate-y-0.5
- Smooth opacity changes for image reveals

## Do's
- Use generous whitespace between sections
- Use serif for all headings and product names
- Keep product names in UPPERCASE with wide letter-spacing
- Use warm, muted tones throughout
- Ensure strong contrast for readability
- Use thin hairline dividers between sections

## Don'ts
- Never use bright/saturated colors
- Never use default blue/purple schemes
- Never use heavy shadows or thick borders
- Never use sans-serif for headings
- Never crowd elements — always give breathing room
- Never use lowercase for product names
