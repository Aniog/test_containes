# Design Guidelines - SSourcing China

## Visual Style
Clean, professional, and trustworthy B2B aesthetic. Focus on clarity, reliability, and international standards.

## Typography
- **Primary Font**: 'Inter', sans-serif (via Google Fonts)
- **Heading Font**: 'Inter', sans-serif (bold, tight tracking)
- **Body Font**: 'Inter', sans-serif (regular, good line-height for readability)

## Color Palette
- **Primary (Trust Blue)**: `#0F4C81` (Classic Blue/Professional) - `bg-[#0F4C81]`, `text-[#0F4C81]`
- **Secondary (Accent Silver)**: `#E5E7EB` - `bg-gray-200`, `text-gray-600`
- **Action (CTA Orange/Gold)**: `#D97706` (Amber-600) - `bg-[#D97706]`, `hover:bg-[#B45309]`
- **Background**: `#FFFFFF` (White) / `#F9FAFB` (Off-white for sections) - `bg-white`, `bg-gray-50`
- **Text**: `#111827` (Darkest Gray for headers) - `text-gray-900`
- **Muted Text**: `#4B5563` (Medium Gray for descriptions) - `text-gray-600`

## Layout & Spacing
- **Container**: Max width 1280px (`max-w-7xl mx-auto px-4 sm:px-6 lg:px-8`)
- **Spacing**: Generous whitespace between sections (`py-16` to `py-24`)
- **Grid**: Responsive 12-column grid, switching to 1 or 2 columns on mobile.

## Components Style
- **Buttons**: Rounded corners (`rounded-md`), clear shadows, subtle hover transitions.
- **Cards**: Minimalist borders or soft shadows (`shadow-sm`), consistent padding.
- **Forms**: Clean inputs with focus states, prominent labels.
- **Icons**: Lucide React for consistent, thin-stroke UI icons.

## Imagery
- High-quality, realistic visuals of factories, quality control inspections, and container shipping.
- Avoid low-quality stock photos; prefer shots that look like real business operations in China.

## Do's
- Use explicit foreground colors for contrast.
- Ensure all CTA buttons are prominent.
- Maintain a consistent professional tone.

## Don'ts
- Use overly bright or neon colors.
- Use hard-to-read thin fonts on busy backgrounds.
- Overcrowd sections with too much text.
