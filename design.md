# ARTITECT MACHINERY Website Design

## Visual Style
- **Aesthetic**: Industrial elegance meets modern minimalism
- **Feel**: Professional, trustworthy, sophisticated yet approachable

## Color Palette
- **Primary**: #1a1f36 (Deep Navy) - Authority and trust
- **Secondary**: #2d3555 (Steel Blue) - Industrial strength
- **Accent**: #c9a227 (Brushed Gold) - Premium quality
- **Background**: #f8f9fc (Off White) - Clean and professional
- **Text Primary**: #1a1f36 (Deep Navy)
- **Text Secondary**: #6b7280 (Slate Gray)
- **Surface**: #ffffff (White)
- **Border**: #e5e7eb (Light Gray)

## Typography
- **Headings**: Inter (700 weight) - Bold, modern, professional
- **Body**: Inter (400, 500 weight) - Clean readability
- **Accent Text**: Inter (600 weight) - Emphasis

## Spacing System
- Base unit: 4px
- Section padding: 80px vertical (desktop), 48px (mobile)
- Component gaps: 24px-32px
- Card padding: 24px-32px

## Visual Elements
- Subtle shadows for depth (shadow-lg with low opacity)
- Rounded corners (rounded-xl for cards, rounded-full for buttons)
- Thin accent lines using gold color
- Clean grid layouts with generous whitespace

## Do's
- Use deep navy backgrounds for hero sections
- Add gold accent lines under headings
- Use clean white cards with subtle shadows
- Include professional product imagery
- Maintain consistent spacing throughout

## Don'ts
- Don't use bright, flashy colors
- Don't overcrowd sections
- Don't use low contrast text
- Don't use harsh shadows
