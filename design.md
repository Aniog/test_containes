# ARTITECT MACHINERY - Design Specification

## Brand Identity
- **Brand Name**: ARTITECT MACHINERY
- **Industry**: Industrial machinery manufacturing (sheet metal folding machines)
- **Products**: Double folding machine, double folder, sheet metal folder, sheet metal folding machine, metal folder, metal folder machine, metal folding machine

## Visual Style

### Color Palette
- **Primary**: #1a1a2e (Deep Navy) - Trust, professionalism, industrial strength
- **Secondary**: #16213e (Dark Blue) - Depth, reliability
- **Accent**: #e94560 (Coral Red) - Energy, call-to-action, machinery precision
- **Light Background**: #f8f9fa (Off-white) - Clean, modern
- **Text Primary**: #1a1a2e (Deep Navy)
- **Text Secondary**: #6c757d (Muted Gray)
- **White**: #ffffff

### Typography
- **Headings**: Inter (700 weight) - Clean, modern, professional
- **Body**: Inter (400-500 weight) - Excellent readability
- **Accent Text**: Inter (600 weight)
- **Font Scale**: 
  - Hero: 4rem (64px)
  - H1: 3rem (48px)
  - H2: 2.25rem (36px)
  - H3: 1.5rem (24px)
  - Body: 1rem (16px)
  - Small: 0.875rem (14px)

### Layout & Spacing
- **Container Max Width**: 1280px
- **Section Padding**: 80px vertical (desktop), 48px (mobile)
- **Component Gap**: 24px
- **Border Radius**: 8px (cards), 4px (buttons), 50% (badges)

### Visual Effects
- **Shadows**: Subtle box-shadows for cards (0 4px 20px rgba(0,0,0,0.08))
- **Hover Effects**: Smooth transitions (0.3s ease), slight lift on cards
- **Gradients**: Subtle gradient overlays on hero sections
- **Borders**: Light borders (1px solid #e5e7eb)

### Design Principles
1. **Elegant but User-Friendly**: Clean layouts with clear hierarchy
2. **Industrial Professionalism**: Convey precision and reliability
3. **Modern Minimalism**: Ample white space, focused content
4. **Clear CTAs**: Prominent action buttons with accent color
5. **Responsive**: Mobile-first approach with elegant breakpoints

## Component Guidelines

### Navigation
- Sticky header with subtle shadow on scroll
- Logo on left, navigation links center/right
- Clean, minimal design with hover states
- Mobile hamburger menu

### Hero Section
- Full-width with gradient overlay
- Bold headline with brand name
- Clear value proposition
- Primary CTA button

### Product Cards
- Clean white cards with subtle shadow
- Product image/icon area
- Product name and brief description
- Hover lift effect

### Feature Cards
- Icon + title + description format
- Grid layout (3 columns desktop, 1 mobile)
- Consistent spacing

### Contact Section
- Clean form layout
- Company contact information
- Professional styling

### Footer
- Multi-column layout
- Company info, quick links, contact
- Social media icons
- Copyright notice

## Do's and Don'ts

### Do
- Use Inter font for all text
- Apply coral red (#e94560) sparingly for CTAs and accents
- Keep generous white space
- Use high-quality placeholder images for machinery
- Ensure clear visual hierarchy
- Make buttons and CTAs prominent

### Don't
- Use more than 2-3 colors prominently
- Overcrowd sections with too much content
- Use low-contrast text colors
- Make the design feel cluttered or busy
- Use overly decorative fonts
