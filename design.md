# ARTITECT MACHINERY - Design System

## Brand Identity
Industrial elegance meets user-friendly precision. The visual language conveys strength, craftsmanship, and technical expertise while remaining approachable.

## Color Palette

### Primary Colors
- `--charcoal`: #1B1F2E - Deep charcoal for headers and primary text
- `--steel`: #2D3142 - Steel gray for secondary text and UI elements
- `--slate`: #4F5D75 - Mid slate for body text and subtle elements

### Accent Colors
- `--amber`: #C9A84C - Warm amber/gold for CTAs, highlights, and accents
- `--amber-light`: #D4B96A - Lighter amber for hover states
- `--amber-dark`: #B08D3A - Darker amber for active states

### Neutral Colors
- `--white`: #FFFFFF
- `--cream`: #FAFAF8 - Warm cream for section backgrounds
- `--light-gray`: #F0F0ED - Light warm gray for alternating sections
- `--border`: #E2E0DB - Subtle warm border color
- `--muted`: #8A8D9B - Muted text color for secondary information

### Functional
- `--success`: #4A7C59 - Success states
- `--error`: #C44B4B - Error states

## Typography

### Font Family
- Primary: Inter (Google Fonts)
- Weights: 300 (Light), 400 (Regular), 500 (Medium), 600 (SemiBold), 700 (Bold), 800 (ExtraBold)

### Type Scale
- `--text-hero`: 4rem / 3.5rem (desktop/mobile) - ExtraBold
- `--text-h1`: 2.5rem / 2rem - Bold
- `--text-h2`: 2rem / 1.5rem - SemiBold
- `--text-h3`: 1.5rem / 1.25rem - SemiBold
- `--text-body`: 1rem - Regular
- `--text-small`: 0.875rem - Regular
- `--text-caption`: 0.75rem - Medium, uppercase, letter-spacing: 0.1em

## Spacing
- Section padding: 6rem (desktop) / 3rem (mobile)
- Container max-width: 1200px
- Component gap: 1.5rem - 2rem
- Card padding: 2rem

## Layout
- Full-width hero sections with contained content
- 12-column grid system
- Alternating section backgrounds (white / cream)
- Cards with subtle shadow and warm border

## Components Style

### Buttons
- Primary: Amber background, charcoal text, subtle shadow
- Secondary: Transparent with charcoal border, charcoal text
- Hover: Slight lift + enhanced shadow
- Border-radius: 4px (slight, professional)

### Cards
- White background
- Subtle shadow: 0 2px 12px rgba(27,31,46,0.08)
- Border: 1px solid var(--border)
- Border-radius: 8px
- Hover: Slight lift + enhanced shadow

### Navigation
- Fixed top, white background with subtle bottom border
- Logo left, links center, CTA right
- Scroll state: tighter padding, shadow

## Do's
- Use warm tones throughout to soften the industrial feel
- Keep plenty of whitespace for elegance
- Use amber accent sparingly for maximum impact
- Ensure all text has strong contrast (WCAG AA minimum)
- Use subtle shadows for depth, not flat design

## Don'ts
- Don't use pure black (#000) - use charcoal instead
- Don't use cold blue tones - keep warmth
- Don't overcrowd sections - elegance needs breathing room
- Don't use more than 2 accent colors
- Don't use sharp corners everywhere - slight radius is friendlier
