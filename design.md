# ARTITECT MACHINERY Design System

## Visual Style
- **Theme**: Elegant industrial - combining precision engineering aesthetics with modern web design
- **Color Palette**:
  - Primary: Deep navy blue (`#1a2744`) - conveys trust, professionalism, industrial strength
  - Secondary: Steel gray (`#4a5568`) - metallic, industrial feel
  - Accent: Warm amber/gold (`#d4a853`) - premium quality, elegance
  - Background: Light warm gray (`#f7f8fa`) - clean, professional
  - Surface: White (`#ffffff`) - crisp, modern
  - Text: Dark charcoal (`#1a202c`) for headings, medium gray (`#4a5568`) for body
  - Success: Green (`#38a169`)
  - Error: Red (`#e53e3e`)

## Typography
- **Headings**: Inter, weight 600-800, tight letter-spacing for impact
- **Body**: Inter, weight 400-500, comfortable line-height (1.6-1.7)
- **Hero text**: Large, bold, with subtle letter-spacing
- **Navigation**: Medium weight, uppercase for main nav items

## Layout & Spacing
- **Container**: Max-width 1280px, centered with generous padding
- **Section padding**: 80-120px vertical on desktop, 48-64px on mobile
- **Grid gaps**: 24-32px between cards, 16px within components
- **Border radius**: 8px for cards, 4px for buttons, 12px for hero elements

## Components
- **Cards**: White background, subtle shadow, hover lift effect
- **Buttons**: Primary (navy bg, white text), Secondary (outline navy), Accent (amber bg)
- **Navigation**: Sticky header, transparent on hero, solid on scroll
- **Footer**: Dark navy background, white text, organized in columns

## Do's
- Use generous whitespace to convey premium quality
- Include subtle animations (fade-in, slide-up) for elegance
- Use high-quality industrial imagery
- Maintain consistent spacing scale (4, 8, 12, 16, 24, 32, 48, 64, 80, 96, 128)
- Use icons sparingly but effectively

## Don'ts
- Don't use bright/neon colors
- Don't overcrowd layouts
- Don't use more than 3 font weights
- Don't use harsh shadows or gradients
- Don't use arbitrary hex codes - stick to the palette
- Don't use pixel values - use Tailwind spacing scale
