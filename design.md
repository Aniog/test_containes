# ARTITECT MACHINERY Design Guide

## Visual direction
Elegant industrial, precise, and user friendly. The site should feel trustworthy for professional buyers while remaining easy to scan.

## Typography
- Use Inter as the global font.
- Headlines: bold, tight tracking, clear hierarchy. Example Tailwind: `text-4xl md:text-6xl font-semibold tracking-tight`.
- Body text: comfortable line height and readable contrast. Example Tailwind: `text-base leading-7 text-slate-600`.

## Colors
- Primary dark: deep slate / near navy for premium machinery surfaces. Example Tailwind: `bg-slate-950 text-white`.
- Main page background: soft cool gray. Example Tailwind: `bg-slate-50`.
- Card surface: clean white with slate text. Example Tailwind: `bg-white text-slate-950`.
- Accent: amber for calls to action and highlights. Example Tailwind: `bg-amber-500 text-slate-950`, `text-amber-400`.
- Supporting metallic tones: slate and zinc. Example Tailwind: `border-slate-200`, `text-slate-500`.

## Layout and spacing
- Use generous section padding. Example Tailwind: `py-16 md:py-24`.
- Keep content constrained and centered. Example Tailwind: `mx-auto max-w-7xl px-4 sm:px-6 lg:px-8`.
- Desktop layouts should use multi-column grids; mobile layouts should stack naturally.

## Components
- Cards: rounded corners, subtle border, soft shadow. Example Tailwind: `rounded-3xl border border-slate-200 bg-white shadow-sm`.
- Buttons: rounded full, strong contrast, clear hover states. Example Tailwind: `rounded-full bg-amber-500 px-6 py-3 font-semibold text-slate-950 hover:bg-amber-400`.
- Badges: compact and high contrast. Example Tailwind: `rounded-full bg-white/10 px-3 py-1 text-sm text-white`.

## Do's
- Use crisp machinery-inspired lines, grids, stats, and structured product sections.
- Keep every text element clearly readable against its background.
- Use professional language focused on precision, productivity, durability, and support.

## Don'ts
- Do not use playful colors or overly decorative effects.
- Do not crowd product details on mobile.
- Do not use low contrast text on dark or image backgrounds.
