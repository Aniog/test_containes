# ARTITECT MACHINERY Design System

## Brand Identity
ARTITECT MACHINERY is a premium industrial machinery brand specializing in sheet metal folding equipment. The design language conveys engineering precision, reliability, and sophistication.

## Typography
- **Display / Headings**: Playfair Display (serif) - elegant, authoritative
- **Body / UI**: Inter (sans-serif) - clean, highly readable
- Heading sizes: text-4xl to text-6xl (desktop), text-2xl to text-4xl (mobile)
- Body: text-base (16px), leading-relaxed

## Color Palette
- **Primary Dark**: #1A1A2E (deep navy-black) - backgrounds, headers
- **Primary Mid**: #16213E (dark navy) - secondary sections
- **Accent Gold**: #C9A84C - CTAs, highlights, brand accent
- **Accent Gold Hover**: #B8953E
- **Surface Light**: #F5F0E8 (warm cream) - light section backgrounds
- **Surface White**: #FFFFFF - cards, forms
- **Text Primary**: #1A1A2E on light, #F5F0E8 on dark
- **Text Muted**: #6B7280 (gray-500)
- **Border**: #E5E0D6 (warm gray border)
- **Success**: #059669
- **Error**: #DC2626

## Spacing & Layout
- Max content width: max-w-7xl (1280px)
- Section padding: py-20 to py-32 (desktop), py-12 to py-16 (mobile)
- Card padding: p-6 to p-8
- Grid gaps: gap-6 to gap-8

## Visual Style
- Elegant industrial aesthetic: dark backgrounds with gold accents
- Generous whitespace for breathing room
- Subtle shadows and smooth transitions
- Cards with rounded corners (rounded-xl to rounded-2xl)
- Hero sections with background images using dark overlays
- Gold accent lines/dividers for visual separation

## Do's
- Use Playfair Display for all headings and hero text
- Use gold (#C9A84C) sparingly for maximum impact
- Maintain high contrast: light text on dark backgrounds, dark text on light backgrounds
- Use Inter for all body text, buttons, and navigation
- Keep layouts spacious and clean

## Don'ts
- Never use bright/neon colors
- Never use low-contrast text combinations
- Never crowd elements together
- Never use more than 2 font families
- Never use Comic Sans or similar casual fonts
