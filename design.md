# MicroCosmos Design System

## Brand Identity
MicroCosmos is a visually rich, science-meets-art website exploring the hidden microscopic world. The aesthetic is dark, mysterious, and awe-inspiring — like peering through a microscope into another universe.

## Color Palette
- Background (deep dark): `bg-slate-950` (#020617)
- Surface / Card: `bg-slate-900` (#0f172a)
- Surface elevated: `bg-slate-800` (#1e293b)
- Primary accent (teal/cyan): `text-cyan-400`, `bg-cyan-500` (#06b6d4)
- Secondary accent (emerald): `text-emerald-400`, `bg-emerald-500` (#10b981)
- Highlight (violet): `text-violet-400` (#a78bfa)
- Text primary: `text-white`
- Text secondary: `text-slate-300`
- Text muted: `text-slate-500`
- Border: `border-slate-700`

## Typography
- Font: Inter (Google Fonts)
- Hero heading: `text-6xl md:text-8xl font-black tracking-tight`
- Section heading: `text-4xl md:text-5xl font-bold`
- Subheading: `text-xl md:text-2xl font-semibold`
- Body: `text-base md:text-lg text-slate-300`
- Caption: `text-sm text-slate-500`
- Label/tag: `text-xs font-semibold uppercase tracking-widest`

## Spacing
- Section padding: `py-20 md:py-32`
- Container: `max-w-7xl mx-auto px-4 md:px-8`
- Card gap: `gap-6 md:gap-8`

## Components

### Hero
- Full-viewport height with dark overlay on background image
- Large bold headline with gradient text (cyan to violet)
- Subtle animated glow effects

### Image Cards
- Rounded corners: `rounded-2xl`
- Overflow hidden with hover zoom: `overflow-hidden group`
- Image hover: `group-hover:scale-110 transition-transform duration-700`
- Dark overlay on hover with text reveal

### Section Dividers
- Thin cyan line: `border-t border-cyan-500/20`

### Badges / Tags
- `bg-cyan-500/10 text-cyan-400 border border-cyan-500/20 rounded-full px-3 py-1 text-xs font-semibold uppercase tracking-widest`

### Buttons
- Primary: `bg-cyan-500 hover:bg-cyan-400 text-slate-950 font-bold px-6 py-3 rounded-full transition-colors`
- Ghost: `border border-slate-600 hover:border-cyan-500 text-slate-300 hover:text-cyan-400 px-6 py-3 rounded-full transition-colors`

## Do's
- Use dark backgrounds throughout for immersive feel
- Use cyan/teal as the primary accent color
- Use large, full-bleed images wherever possible
- Use gradient text for hero headings
- Use subtle glow/blur effects for depth
- Ensure all text is clearly readable against dark backgrounds

## Don'ts
- No white or light backgrounds
- No flat, plain layouts — always add depth with overlays and gradients
- No small, cramped images — let them breathe
- Never use light text on light backgrounds
