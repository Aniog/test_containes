# Velmora Fine Jewelry — Design System

## Visual Identity
- Mood: Quiet luxury, warm, editorial. Premium demi-fine jewelry.
- Not loud, not discount-looking, not generic e-commerce.

## Color Palette
- **Base (deep warm charcoal):** #1C1917 (stone-900) — primary background for hero/dark sections
- **Cream (warm off-white):** #FAF7F2 — primary light background
- **Gold Accent:** #C9A96E — primary accent (buttons, highlights, links)
- **Gold Hover:** #B8944F — darker gold for hover states
- **Text Primary (dark):** #1C1917 — on light backgrounds
- **Text Secondary:** #78716C (stone-500) — muted text on light backgrounds
- **Text on Dark:** #FAF7F2 — on dark backgrounds
- **Divider:** #E7E5E4 (stone-200) — hairline dividers
- **Card Background:** #FFFFFF — product cards on cream

## Typography
- **Headings:** Cormorant Garamond (serif), weights 400/500/600
- **Body/UI:** Inter (sans-serif), weights 300/400/500/600
- **Product names:** UPPERCASE, letter-spacing 0.15em, Cormorant Garamond
- **Nav links:** UPPERCASE, letter-spacing 0.12em, Inter 500
- **Buttons:** UPPERCASE, letter-spacing 0.1em, Inter 500

## Spacing & Layout
- Generous whitespace. Sections: py-20 md:py-28
- Container max-width: 1280px, centered
- Product grid gap: 24px
- Section headings: large serif, centered

## Components
- **Buttons:** Solid gold (#C9A96E) with cream text, or outlined with gold border + gold text. Rounded-none (sharp corners) for premium feel. Hover: darken gold.
- **Cards:** White bg, subtle shadow on hover, thin border. Image aspect 4x3 or 3x4.
- **Dividers:** 1px stone-200, used between sections
- **Accordions:** Clean, minimal, plus/minus icon

## Do's
- Use warm, editorial imagery
- Keep whitespace generous
- Use serif for emotional/editorial moments
- Use thin hairline dividers
- Subtle hover transitions (200-300ms ease)

## Don'ts
- Don't use bright/saturated colors
- Don't use rounded buttons (no rounded-full or rounded-lg on primary buttons)
- Don't crowd sections with too much content
- Don't use generic e-commerce patterns (no starburst badges, no aggressive sale banners)
- Don't use light text on light backgrounds
