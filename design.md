# Velmora Fine Jewelry — Design System

## Mood
Quiet luxury, warm, editorial. Premium demi-fine jewelry — NOT loud, NOT discount-looking, NOT generic e-commerce.

## Color Palette
- **Base (deep warm charcoal):** `#1C1917` (stone-900) — primary background for hero/dark sections
- **Cream (warm off-white):** `#FAF7F2` — primary light background
- **Ivory:** `#F5F0E8` — secondary light background, card backgrounds
- **Gold Accent:** `#C9A96E` — primary accent, CTAs, highlights
- **Gold Hover:** `#B8944F` — darker gold for hover states
- **Text Primary:** `#1C1917` — on light backgrounds
- **Text Secondary:** `#78716C` (stone-500) — muted text on light
- **Text on Dark:** `#FAF7F2` — on dark backgrounds
- **Text Muted on Dark:** `#A8A29E` (stone-400) — secondary text on dark
- **Divider:** `#E7E5E4` (stone-200) — hairline dividers
- **White:** `#FFFFFF` — cards, modals

## Typography
- **Headings:** Cormorant Garamond, serif. Elegant, refined.
  - H1: 48-64px, weight 400-500, tracking wide
  - H2: 36-48px, weight 400-500
  - H3: 24-30px, weight 500
- **Product Names:** Cormorant Garamond, UPPERCASE, letter-spacing 0.15-0.2em, weight 500
- **Body/UI:** Inter, sans-serif. Clean, modern.
  - Body: 14-16px, weight 400
  - Buttons: 12-14px, weight 500-600, uppercase, letter-spacing 0.1em
  - Labels: 11-12px, weight 500, uppercase, letter-spacing 0.12em

## Spacing & Layout
- Generous whitespace throughout
- Section padding: 80-120px vertical on desktop, 48-64px on mobile
- Max content width: 1280px
- Grid gaps: 16-24px for product cards

## Visual Elements
- Hairline dividers (1px, stone-200)
- Soft shadows: `0 2px 8px rgba(0,0,0,0.06)`
- Border radius: 0 (sharp, editorial) for cards; 2px for buttons
- Subtle hover transitions: 300ms ease
- Buttons: solid gold accent or outlined with gold border

## Do's
- Use generous whitespace
- Use serif for all headings and product names
- Use uppercase with wide letter-spacing for product names
- Keep imagery large and editorial
- Use thin hairline dividers between sections
- Ensure strong contrast for accessibility

## Don'ts
- Don't use bright/saturated colors
- Don't use rounded corners on cards (keep editorial/sharp)
- Don't use heavy shadows
- Don't use generic e-commerce patterns (badges, countdown timers)
- Don't use more than 2 typefaces
- Don't use low-contrast text
