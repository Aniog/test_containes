# Velmora Fine Jewelry — Design System

## Visual Identity
- **Mood**: Quiet luxury, warm, editorial. Premium demi-fine jewelry — NOT loud, NOT discount-looking, NOT generic e-commerce.
- **Target**: Women 25–45, gifting and self-purchase, premium-but-accessible ($30–$120).

## Color Palette
- **Primary Dark** (`charcoal`): #1A1714 — deep warm charcoal for text, nav, dark sections
- **Background** (`cream`): #FAF7F2 — warm cream, main page background
- **Surface** (`white`): #FFFFFF — cards, modals, drawers
- **Accent Gold** (`gold`): #C9A96E — warm metallic gold for CTAs, highlights, accents
- **Accent Gold Dark** (`gold-dark`): #A88B52 — hover state for gold buttons
- **Text Primary** (`charcoal`): #1A1714 — headings, body text
- **Text Secondary** (`warm-gray`): #6B6560 — descriptions, captions
- **Border** (`sand`): #E8E2DA — hairline dividers, card borders
- **Muted Surface** (`linen`): #F0EBE4 — subtle backgrounds, hover states

## Typography
- **Headings**: Cormorant Garamond (serif), weights 400/500/600/700
- **Body/UI**: Inter (sans-serif), weights 300/400/500/600
- **Product names**: UPPERCASE, letter-spacing 0.15em, Cormorant Garamond 500
- **Section headings**: Cormorant Garamond 400/500, large sizes (2rem–3.5rem)
- **Body text**: Inter 400, 0.875rem–1rem, line-height 1.6–1.8

## Spacing & Layout
- Generous whitespace: section padding 4rem–6rem desktop, 2.5rem–3rem mobile
- Max content width: 1280px
- Grid gaps: 1.5rem–2rem
- Hairline dividers: 1px solid #E8E2DA

## Component Styles
- **Buttons**: Solid gold (#C9A96E) with white text, or outlined with gold border. Rounded-none or rounded-sm. Padding: 0.75rem 2rem. Hover: darken gold. Transitions: 300ms ease.
- **Cards**: White background, subtle shadow on hover (shadow-md), border: 1px solid #E8E2DA
- **Inputs**: Border #E8E2DA, focus: ring gold, rounded-none
- **Nav**: Transparent over hero, solid cream on scroll. Hairline bottom border when solid.
- **Cart Drawer**: Slide-in from right, white background, overlay backdrop

## Animations
- Subtle hover transitions (300ms ease)
- Image crossfade on product hover
- Smooth scroll for UGC reel row
- Nav background transition on scroll

## Do's
- Use generous whitespace between sections
- Use Cormorant Garamond for all headings and product names
- Use UPPERCASE + wide letter-spacing for product names
- Use warm gold (#C9A96E) sparingly as accent
- Ensure strong text contrast on all backgrounds
- Use thin hairline dividers between sections

## Don'ts
- Don't use bright or saturated colors
- Don't use rounded-full buttons (not premium feeling)
- Don't use heavy shadows or gradients
- Don't mix too many font weights
- Don't use generic e-commerce patterns (badges, countdown timers, etc.)
- Don't use low-contrast text (muted text must still be readable)
