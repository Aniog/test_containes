# SSourcing China - B2B Website Design Guidelines

## Project Overview
- **Project Name**: SSourcing China Website
- **Type**: B2B Corporate Website
- **Goal**: Generate qualified sourcing inquiries from overseas buyers
- **Tone**: Professional, clear, practical, trustworthy

## Visual Style

### Color Palette
- **Primary**: `#1E3A5F` (Deep Navy Blue - trust, professionalism)
- **Primary Light**: `#2C5282` (Medium Navy)
- **Secondary**: `#C9A227` (Gold/Amber - premium, quality)
- **Accent**: `#38A169` (Green - success, reliability)
- **Background**: `#F8FAFC` (Off-white)
- **Surface**: `#FFFFFF` (White)
- **Text Primary**: `#1A202C` (Near black)
- **Text Secondary**: `#4A5568` (Dark gray)
- **Text Muted**: `#718096` (Medium gray)
- **Border**: `#E2E8F0` (Light gray)
- **Error**: `#E53E3E` (Red)

### Typography
- **Font Family**: "Inter", system-ui, sans-serif
- **Headings**: 
  - H1: 48px/56px, font-weight 700
  - H2: 36px/44px, font-weight 700
  - H3: 24px/32px, font-weight 600
  - H4: 20px/28px, font-weight 600
- **Body**: 16px/24px, font-weight 400
- **Small**: 14px/20px, font-weight 400

### Spacing System
- Base unit: 4px
- Spacing scale: 4, 8, 12, 16, 24, 32, 48, 64, 96, 128px
- Section padding: 96px vertical (desktop), 64px (mobile)
- Container max-width: 1200px

### Visual Effects
- **Shadows**: 
  - Card: `0 4px 6px -1px rgba(0, 0, 0, 0.1), 0 2px 4px -1px rgba(0, 0, 0, 0.06)`
  - Elevated: `0 10px 15px -3px rgba(0, 0, 0, 0.1), 0 4px 6px -2px rgba(0, 0, 0, 0.05)`
- **Border Radius**: 8px (cards), 4px (buttons), 9999px (pills)
- **Transitions**: 200ms ease-in-out

## Layout Guidelines

### Header
- Fixed position, white background with subtle shadow on scroll
- Logo on left, navigation center, CTA button on right
- Mobile: hamburger menu

### Hero Section
- Full-width, split layout (text left, image right) or centered
- Primary headline + subheadline + CTA button
- Background: subtle gradient or solid color with image overlay

### Content Sections
- Alternating background colors (white / off-white)
- Max-width container with consistent padding
- Clear section headings with optional subheadings

### Footer
- Dark background (#1E3A5F)
- Multi-column layout: navigation, contact info, social links
- Copyright and legal links

## Component Guidelines

### Buttons
- **Primary**: Navy background, white text, gold hover
- **Secondary**: White background, navy border, navy text
- **CTA**: Gold background, navy text, prominent size

### Cards
- White background, subtle shadow, 8px border radius
- Consistent padding (24px)
- Optional image on top

### Form Elements
- Input fields: full-width, 48px height, light border
- Labels above inputs
- Clear error states

### Icons
- Lucide React icons
- Consistent sizing (24px default)
- Muted colors for secondary icons

## Responsive Breakpoints
- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px

## Do's and Don'ts
- DO: Use professional imagery showing factories, QC processes, shipping
- DO: Include trust signals (certifications, client logos, statistics)
- DO: Keep copy clear and practical, avoid exaggerated claims
- DON'T: Use overly flashy animations or effects
- DON'T: Use low-quality or stock-photo-heavy imagery
- DON'T: Make claims that seem too good to be true
