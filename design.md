# Velmora Fine Jewelry - Design System

## Visual Identity

### Mood
- Quiet luxury, warm, editorial
- Premium demi-fine jewelry
- NOT loud, NOT discount-looking, NOT generic e-commerce

### Color Palette
**Primary Colors:**
- Background: `#0D0D0D` (rich black)
- Surface: `#1A1A1A` (elevated black)
- Surface Light: `#262626` (card backgrounds)

**Accent Colors:**
- Gold Primary: `#C9A962` (warm gold)
- Gold Light: `#E5D4A1` (light gold for hover)
- Gold Dark: `#A68B4B` (deep gold)

**Neutral Colors:**
- White: `#FFFFFF`
- Off-White: `#F5F5F0` (warm white)
- Muted: `#8A8A8A` (secondary text)
- Border: `#333333` (hairline dividers)

### Typography
**Headings (Serif):**
- Font: "Cormorant Garamond", serif
- Weights: 400, 500, 600
- Product names: UPPERCASE with letter-spacing: 0.15em

**Body (Sans-serif):**
- Font: "Manrope", sans-serif
- Weights: 400, 500, 600
- Base size: 16px

**Font Sizes:**
- Hero: 56px (desktop), 36px (mobile)
- H1: 42px (desktop), 28px (mobile)
- H2: 32px (desktop), 24px (mobile)
- H3: 24px (desktop), 20px (mobile)
- Body: 16px
- Small: 14px
- Caption: 12px

### Spacing System
- Base unit: 4px
- Section padding: 80px vertical (desktop), 48px (mobile)
- Container max-width: 1280px
- Grid gap: 24px (desktop), 16px (mobile)

### Visual Effects
- Hairline dividers: 1px solid #333333
- Card shadows: 0 4px 20px rgba(0,0,0,0.3)
- Hover transitions: 300ms ease
- Button hover: subtle scale (1.02) + glow
- Image hover: slight zoom (1.05)

### Components

#### Buttons
- Primary: Gold background (#C9A962), black text, full-width or large
- Secondary: Transparent with gold border, gold text
- Hover: Lighten background, subtle glow

#### Cards
- Background: #1A1A1A
- Border: 1px solid #333333
- Border-radius: 4px
- Hover: Border color changes to gold

#### Form Inputs
- Background: #1A1A1A
- Border: 1px solid #333333
- Focus: Border color gold
- Border-radius: 4px

#### Dividers
- Hairline style: 1px solid #333333
- Used between sections

### Responsive Breakpoints
- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px

## Design Principles
1. Generous whitespace
2. Large editorial imagery
3. Thin hairline dividers
4. Restrained accent color
5. Subtle hover transitions
6. Soft shadows
7. Premium button feel
