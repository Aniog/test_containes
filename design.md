# Visual Identity - Velmora Fine Jewelry

## Mood
Quiet luxury, warm, editorial. Premium demi-fine jewelry.

## Palette
- **Primary/Background**: #FDFCF8 (Alabaster - a warm, soft off-white)
- **Secondary/Text**: #1A1A1A (Onyx - deep, refined charcoal)
- **Accent/Metallic**: #C5A059 (Golden Ochre - warm, sophisticated gold tone)
- **Muted**: #717171 (Stone - for secondary labels and helpers)
- **Border**: #E5E5E5 (Light Grey - for subtle hairline dividers)

## Typography
- **Headings/Product Names**: Serif - 'Cormorant Garamond' (Google Font)
  - Style: Uppercase for product names with wide letter-spacing (`tracking-[0.2em]`).
- **Body/UI**: Sans-serif - 'Inter' (Google Font)
  - Style: Clean, legible, modern.

## UI Elements
- **Buttons**:
  - Primary: Solid Onyx with Alabaster text, minimal rounding.
  - Secondary: Outlined Onyx.
- **Dividers**: 1px hairline dividers (`border-border`).
- **Transitions**: Subtle hover effects (e.g., opacity, slight scale).
- **Shadows**: Soft, almost imperceptible shadows (`shadow-sm`).
- **Whitespace**: Generous, editorial spacing.

## Tailwind Configuration (suggested)
- font-serif: 'Cormorant Garamond', serif
- font-sans: 'Inter', sans-serif
- color-gold: #C5A059
- color-pearl: #FDFCF8
