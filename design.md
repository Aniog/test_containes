# Visual Style Guide for SSourcing China

## Typography
- Main Font: Inter (Google Fonts)
- Weights: 400 (Regular), 500 (Medium), 600 (Semi-Bold), 700 (Bold), 800 (Extra-Bold)
- Headings: Bold/Extra-Bold, Primary color (#003366)
- Body: 400/500, Slate-600 for readability

## Colors
- **Primary**: #003366 (Navy Blue) - Trust, Professionalism.
- **Secondary**: #0055A4 (Classic Blue) - Stability, International Trade.
- **Accent**: #F97316 (Orange/Accent) - Energy, CTA, Action.
- **Background**: #FFFFFF (White) / #F8FAFC (Slate-50) for contrast sections.
- **Text**: #1E293B (Slate-800) for headers, #475569 (Slate-600) for body.

## Spacing & Layout
- Use a maximum width of 1280px (7xl) for content containers.
- Generous padding (py-24) between sections to maintain a clean, organized feel.
- Grid-based layouts (usually 2 or 3 columns) for services and features.
- Consistent border-radius: rounded-md (6px) for buttons, rounded-2xl (16px) for cards and images.

## UI Elements
- **Buttons**: Shadowed, bold text. Accent color for main CTAs.
- **Cards**: Light borders (#F1F5F9), subtle shadows (shadow-sm), hover effects (shadow-xl).
- **Icons**: Lucide React icons, consistently colored with Secondary or Accent tones.
- **Images**: Realistic industrial/factory visuals using the stock image system.

## Brand Voice
- Professional, clear, and practical.
- Trustworthy and transparent.
- International B2B focus.
