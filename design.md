# Strikingly AI Generators Hub Visual Style

## Typography
- Headings and buttons use Brandon Grotesque when available, with Josefin Sans as the Google Fonts fallback and Poppins as secondary fallback.
- Headings are uppercase, 700 weight, line-height 1.2, and use #4B5056 except the first hero line which uses #2E2E2F.
- Body copy uses Open Sans, 400 weight, 14px base, line-height 1.5, and #636972.

## Colors
- Brand purple: #8159BB for accents, borders, badges, and ghost buttons.
- AI gradient: linear-gradient(90deg, #7671FF 0%, #CB0C9F 100%) only for primary CTA fills and the second H1 line.
- Body text: #636972.
- Section headings: #4B5056.
- Hero dark heading: #2E2E2F.
- Card border: #C6C9CD.
- Divider: #E2E4E7.
- Light background: #F4F6F8.
- Default page background: #FFFFFF.

## Layout and Spacing
- Max content width is 1200px, centered with responsive side padding.
- Use spacing multiples of 10px, with 5px only for tight pairs.
- Sections are separated by 40px; hero uses 60px to 80px vertical padding.
- Desktop layouts use 2 or 3 columns where appropriate, tablet 2 columns, mobile 1 column.

## Buttons
- Standard buttons are 36px tall; large buttons are 44px tall.
- Buttons use 4px radius, uppercase heading font, and 9px by 15px padding.
- Filled/gradient buttons must always use white text.
- Ghost buttons use transparent background, 1px #8159BB border, and #8159BB text.

## Cards and Tags
- Cards use white background, 1px #C6C9CD border, 3px radius, and 20px padding.
- Hover cards lift subtly with a small shadow and brand-purple border; no scale or rotation transforms.
- Tags use 11px uppercase heading font, 2px by 6px padding, 3px radius, brand-purple text, and 1px brand-purple border.

## Do
- Keep the page clean, useful, and directory-like.
- Use semantic HTML and accessible focus states.
- Keep every important text color explicit and readable.
- Use inline SVG illustrations only; no external image dependencies.

## Don't
- Do not use fake social proof, ratings, testimonials, or inflated scale claims.
- Do not embed the AI builder form on the hub page.
- Do not apply the AI gradient to section backgrounds or headers.
- Do not use bare link lists for directory entries.
