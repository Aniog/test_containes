# Velmora Fine Jewelry — Design System

## Visual Identity
- **Mood**: Quiet luxury, warm, editorial. Premium demi-fine jewelry — NOT loud, NOT discount-looking.
- **Target**: Women 25–45, gifting and self-purchase, premium-but-accessible ($30–$120).

## Color Palette
- **Background**: `#FAF7F2` (warm off-white / cream)
- **Foreground / Text**: `#1C1917` (warm near-black, stone-900)
- **Muted**: `#78716C` (stone-500)
- **Muted Foreground**: `#A8A29E` (stone-400)
- **Accent**: `#C9A96E` (warm gold — refined, not brassy)
- **Accent Hover**: `#D4B87A` (lighter gold)
- **Accent Dark**: `#A88B4A` (deeper gold for pressed states)
- **Accent Foreground**: `#FFFFFF` (white on gold buttons)
- **Card**: `#FFFFFF` (white)
- **Card Foreground**: `#1C1917`
- **Border**: `#E7E5E4` (stone-200)
- **Surface Dark**: `#1C1917` (for hero overlays, footer)
- **Surface Dark Foreground**: `#FAF7F2`
- **Warm Ivory**: `#F5F0E8` (alternate light section backgrounds)

## Typography
- **Headings**: Cormorant Garamond (serif), weights 400/500/600/700
- **Body / UI**: Inter (sans-serif), weights 300/400/500/600
- **Product names**: UPPERCASE, letter-spacing 0.15em, serif
- **Nav links**: UPPERCASE, letter-spacing 0.12em, sans-serif, text-xs

## Spacing & Layout
- Generous whitespace. Sections: py-16 md:py-24 lg:py-28
- Container max-width: 1280px, centered, px-4 md:px-6 lg:px-8
- Hairline dividers: border-b border-stone-200
- Card gaps: gap-6 md:gap-8

## Buttons
- **Primary (accent)**: bg-accent text-white, rounded-none, px-8 py-3, uppercase, tracking-widest, text-xs, hover:bg-accent-hover
- **Secondary (outlined)**: border border-accent text-accent, bg-transparent, same sizing, hover:bg-accent hover:text-white
- Transitions: all 300ms ease

## Images
- Editorial, warm-lit, dark/neutral backgrounds for jewelry
- Product cards: 4x3 ratio, hover reveals second image
- Hero: 16x9 full-bleed
- UGC reels: 9x16 vertical

## Shadows
- Subtle only: shadow-sm on cards, shadow-md on hover
- No heavy drop shadows

## Animations
- Subtle hover transitions (300ms ease)
- Smooth scroll
- Cart drawer slides from right
- Nav background transition on scroll

## Do's
- Use serif for all headings and product names
- Use warm gold accent sparingly
- Keep whitespace generous
- Use thin hairline dividers
- Ensure strong contrast for accessibility

## Don'ts
- Don't use bright/saturated colors
- Don't use heavy shadows or thick borders
- Don't use sans-serif for headings
- Don't crowd sections — let them breathe
- Don't use rounded buttons (use sharp/editorial corners)
