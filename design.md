# ARTITECT MACHINERY - Design System

## Brand Identity
Industrial precision meets elegant design. The visual language communicates trust, durability, and engineering excellence.

## Color Palette
- **Charcoal** (primary dark): #1a1d23 — Main dark backgrounds, nav, footer
- **Steel** (secondary dark): #2a2d35 — Cards, panels on dark sections
- **Slate** (muted): #6b7280 — Secondary text, subtle elements
- **Amber** (accent): #c9a962 — CTAs, highlights, accent borders
- **Amber Light**: #d4b87a — Hover states for amber
- **Warm White**: #f8f7f4 — Light section backgrounds
- **Off White**: #edeae3 — Alternate light sections
- **Charcoal Text**: #1a1d23 — Primary text on light backgrounds
- **Light Text**: #e8e6e1 — Primary text on dark backgrounds

## Typography
- Font: Inter (Google Fonts)
- Hero headings: 4xl/5xl/6xl, font-bold, tracking-tight
- Section headings: 3xl/4xl, font-bold
- Subheadings: xl/2xl, font-semibold
- Body: base/lg, font-normal, leading-relaxed
- Small/captions: sm, font-medium

## Spacing
- Section padding: py-20 md:py-28
- Container: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8
- Card padding: p-6 md:p-8
- Element gap: gap-6 to gap-12

## Borders & Shadows
- Cards: border border-stone-200, shadow-sm
- Featured cards: border-2 border-amber-600/30, shadow-lg
- Subtle dividers: border-t border-stone-200
- Hover transitions: transition-all duration-300

## Component Style
- Buttons primary: bg-amber-600 hover:bg-amber-700 text-white font-semibold px-8 py-4
- Buttons secondary: border-2 border-amber-600 text-amber-600 hover:bg-amber-600 hover:text-white
- Cards: bg-white rounded-xl border border-stone-200 overflow-hidden hover:shadow-lg transition-all
- Dark cards: bg-[#2a2d35] rounded-xl border border-stone-700/30

## Visual Rules
- Use data-strk-img tags for all product and hero images
- Gold accent line or border on key sections
- Clean white space between sections
- Professional photography style for machinery images
- No emojis anywhere in the UI
- All text must be clearly readable against backgrounds
