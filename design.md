# Visual Identity - Velmora Fine Jewelry

## Mood
- Quiet luxury, warm, editorial.
- Premium demi-fine jewelry - NOT loud, NOT discount-looking.

## Colors
- Base: deep refined charcoal or off-white/cream for a clean look. Let's go with a warm cream/stone base for that premium editorial feel.
- Accent: Warm gold/brass for metallic highlights.
- Text: Deep charcoal/ebony for readability and premium contrast.
- Secondary: Soft taupe/beige for dividers and subtle backgrounds.

| Name | Tailwind Class | Hex (approx) | Usage |
|------|----------------|--------------|-------|
| Primary Base | `bg-[#FDFCFB]` | #FDFCFB | Page backgrounds |
| Surface | `bg-[#F5F2F0]` | #F5F2F0 | Secondary sections, cards |
| Accent | `text-[#C5A059]` | #C5A059 | Gold highlights, CTAs |
| Deep Text | `text-[#1C1C1C]` | #1C1C1C | Main headings, body |
| Muted Text| `text-[#717171]` | #717171 | Subtext, labels |
| Border | `border-[#E5E5E5]` | #E5E5E5 | Hairline dividers |

## Typography
- Headings: `font-serif` (Playfair Display or Cormorant Garamond).
- Body/UI: `font-sans` (Inter or Manrope).
- Product Names: Uppercase with `tracking-[0.2em]`.

## UI Elements
- Buttons: Solid charcoal or gold with minimal rounded corners (2px or 0px).
- Dividers: Thin 1px hairlines.
- Transitions: Subtle 300ms ease.
- Whitespace: Generous padding, editorial layouts.
