# ARTITECT MACHINERY - Design Guide

## Brand Identity
- Brand: ARTITECT MACHINERY
- Industry: Industrial machinery (metal folding machines)
- Style: Elegant, professional, user-friendly

## Color Palette
- Primary: Deep navy (#0f172a) - header, footer, primary buttons
- Primary light: (#1e293b) - card backgrounds, section backgrounds
- Accent Gold: (#c9a84b) - CTAs, highlights, decorative elements
- Accent Gold Light: (#f0e6c8) - subtle gold backgrounds
- Background: (#f8fafc) - main page background
- Surface: (#ffffff) - cards, sections
- Text Primary: (#0f172a) - headings
- Text Secondary: (#475569) - body text
- Text Muted: (#94a3b8) - subtle text

## Typography
- Headings: Inter, system-ui (bold 600-800 weight)
- Body: Inter, system-ui (400 weight)
- Use Inter from Google Fonts (weights 300-800)

## Spacing
- Section padding: py-20 md:py-28
- Container max-width: max-w-7xl mx-auto
- Card padding: p-6 md:p-8
- Gap between cards: gap-6 md:gap-8

## Visual Style
- Rounded corners: rounded-lg for cards, rounded-full for badges
- Shadows: shadow-sm for cards, shadow-md on hover
- Borders: border border-slate-100 for subtle separation
- Transitions: smooth hover transitions on cards and buttons
- Do NOT use heavy industrial/gritty styles
- Do NOT use bright/neon colors
- Keep layouts clean with ample whitespace
