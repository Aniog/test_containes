# SSourcing China - Visual Style Guide

## Brand Colors
- Primary: Blue (#1e40af / blue-800) - Trust, professionalism
- Primary Light: (#3b82f6 / blue-500)
- Primary Dark: (#1e3a8a / blue-900)
- Accent: (#ea580c / orange-600) - CTA buttons, highlights
- Accent Light: (#f97316 / orange-500)
- Background: (#f8fafc / slate-50) - Page background
- Surface: (#ffffff / white) - Cards, sections
- Border: (#e2e8f0 / slate-200)
- Text Primary: (#0f172a / slate-900)
- Text Secondary: (#475569 / slate-600)
- Text Muted: (#94a3b8 / slate-400)

## Typography
- Font Family: Inter (Google Fonts)
- Headings: font-bold, tracking-tight
- Body: font-normal, leading-relaxed
- Page Title: text-4xl md:text-5xl font-bold
- Section Title: text-3xl font-bold
- Subtitle: text-lg text-slate-600

## Spacing
- Section padding: py-16 md:py-24
- Container max-width: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8
- Card padding: p-6 md:p-8
- Gap between cards: gap-6 md:gap-8

## Borders & Shadows
- Cards: rounded-xl border border-slate-200 bg-white shadow-sm
- Hover: shadow-md transition-shadow
- Buttons: rounded-lg
- Inputs: rounded-lg border border-slate-300

## Buttons
- Primary CTA: bg-orange-600 hover:bg-orange-500 text-white font-semibold px-6 py-3 rounded-lg
- Secondary: bg-blue-800 hover:bg-blue-700 text-white font-semibold px-6 py-3 rounded-lg
- Outline: border border-blue-800 text-blue-800 hover:bg-blue-50 font-medium px-6 py-3 rounded-lg

## Navigation
- Sticky header with white background and shadow-sm
- Logo on left, nav links in center, CTA button on right
- Active link: text-blue-800 font-semibold
- Mobile: hamburger menu with slide-out panel

## Visual Style
- Clean, spacious layout with plenty of white space
- Professional imagery: factories, QC inspections, shipping containers, products
- Consistent use of icons with Lucide React
- Subtle gradient accents on hero and section headers
- Numbered steps for process sections
- Client logos or flags for trust signals
