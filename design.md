# ARTITECT MACHINERY - Design System

## Brand
- Name: ARTITECT MACHINERY
- Industry: Industrial machinery / Sheet metal folding equipment
- Tone: Professional, precise, trustworthy, industrial elegance

## Color Palette
- **Primary**: `#1a2332` — Deep navy charcoal (headings, dark backgrounds)
- **Secondary**: `#c9a45c` — Warm gold/brass accent (highlights, CTAs, borders)
- **Background Light**: `#f5f5f0` — Warm off-white (main page background)
- **Background Dark**: `#1a2332` — Deep navy (footer, dark sections)
- **Text Primary**: `#1a2332` — Dark on light backgrounds
- **Text Secondary**: `#6b7280` — Muted gray for body text
- **Text Light**: `#f5f5f0` — Light text on dark backgrounds
- **Surface**: `#ffffff` — Pure white for cards
- **Border**: `#e5e5e0` — Subtle warm gray borders

## Typography
- **Headings**: Inter, weight 700-800, tight letter-spacing (-0.02em)
- **Body**: Inter, weight 400-500, line-height 1.6
- **Accent text**: Inter, weight 600, uppercase, letter-spacing 0.1em
- **Hero title**: 48px mobile, 72px desktop
- **H2**: 32px mobile, 48px desktop
- **Body**: 16px-18px

## Spacing
- Section padding: 80px mobile, 120px desktop
- Container max-width: 1280px
- Card gap: 24px-32px
- Component padding: 24px-32px

## Elevation & Borders
- Card shadow: `0 4px 20px rgba(26, 35, 50, 0.08)`
- Card border-radius: 8px
- Button border-radius: 4px
- Subtle border on cards: 1px solid `#e5e5e0`

## Components
- **Buttons**: 
  - Primary: bg `#c9a45c`, text `#1a2332`, hover bg `#b8934d`
  - Secondary: border 2px `#c9a45c`, text `#1a2332`, hover bg `#c9a45c` /10
  - Dark: bg `#1a2332`, text `#f5f5f0`, hover bg `#2a3342`
- **Cards**: bg white, rounded-lg, subtle shadow, hover lift with shadow increase
- **Navigation**: Fixed top, bg white/95 backdrop-blur, bottom border subtle

## Do's
- Use generous whitespace
- Use gold accent sparingly for impact
- Keep layouts clean and grid-aligned
- Use high-contrast text for readability
- Use imagery of industrial machinery with warm tones

## Don'ts
- Don't use bright or neon colors
- Don't crowd elements together
- Don't use more than 2 font sizes within a section
- Don't use pure black — use `#1a2332` instead
