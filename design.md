# MicroCosmos Design System

## Brand Identity
MicroCosmos is a visually rich, science-meets-art website exploring the microscopic world. The design evokes wonder, depth, and scientific beauty — like peering through a microscope into an alien universe.

## Color Palette

### Primary Colors
- Deep Black: `#050810` — page background, hero backdrop
- Dark Navy: `#0a0f1e` — section backgrounds
- Midnight Blue: `#0d1530` — card backgrounds, panels
- Dark Teal: `#0e2a2a` — alternate section backgrounds

### Accent Colors
- Cyan Glow: `#00d4ff` — primary accent, headings, highlights (`text-cyan-400`)
- Teal: `#14b8a6` — secondary accent, tags, badges (`text-teal-400`)
- Emerald: `#10b981` — success states, category labels (`text-emerald-400`)
- Purple: `#a855f7` — special highlights (`text-purple-400`)

### Text Colors
- Primary Text: `#f0f9ff` — headings on dark backgrounds (`text-sky-50`)
- Secondary Text: `#94a3b8` — body copy, descriptions (`text-slate-400`)
- Muted Text: `#475569` — captions, metadata (`text-slate-600`)

### Borders & Dividers
- Subtle Border: `border-slate-800` — card borders
- Accent Border: `border-cyan-900` — highlighted cards
- Glow Border: `border-cyan-500/30` — featured elements

## Typography

### Font Family
- Primary: `Inter` (Google Fonts) — clean, scientific, readable
- Monospace: `font-mono` — labels, data, scientific names

### Scale
- Hero Title: `text-6xl md:text-8xl font-black tracking-tight`
- Section Title: `text-4xl md:text-5xl font-bold`
- Card Title: `text-xl font-semibold`
- Body: `text-base font-normal leading-relaxed`
- Caption/Label: `text-sm font-medium tracking-widest uppercase`
- Scientific Name: `text-sm font-mono italic text-cyan-400`

## Spacing & Layout
- Max content width: `max-w-7xl mx-auto`
- Section padding: `py-20 px-4 md:px-8`
- Card padding: `p-6`
- Grid gaps: `gap-6` for cards, `gap-4` for tight grids

## Component Styles

### Cards
- Background: `bg-slate-900/80`
- Border: `border border-slate-800`
- Hover: `hover:border-cyan-500/50 hover:shadow-lg hover:shadow-cyan-500/10`
- Border radius: `rounded-2xl`
- Transition: `transition-all duration-300`

### Buttons
- Primary: `bg-cyan-500 hover:bg-cyan-400 text-black font-semibold px-6 py-3 rounded-full`
- Secondary: `border border-cyan-500/50 text-cyan-400 hover:bg-cyan-500/10 px-6 py-3 rounded-full`

### Badges / Tags
- `bg-cyan-500/10 text-cyan-400 border border-cyan-500/20 text-xs font-mono px-3 py-1 rounded-full`

### Section Dividers
- Use subtle gradient dividers or thin `border-t border-slate-800`

## Visual Effects
- Glow effects: `shadow-lg shadow-cyan-500/20` on featured images
- Gradient overlays on hero: `bg-gradient-to-b from-transparent to-black/80`
- Subtle grid pattern on hero background
- Image hover: `scale-105 transition-transform duration-500`

## Image Guidelines
- Hero: 16x9, full-width, dramatic microscopic subject
- Gallery cards: 4x3 or 1x1 ratio
- Feature spotlight: 3x2 or 16x9
- Category icons: 1x1 square

## Do's
- Use dark backgrounds throughout for a dramatic, immersive feel
- Layer cyan/teal glows on key elements
- Use monospace font for scientific names and data
- Include generous whitespace between sections
- Use image-heavy layouts — this site is visual-first

## Don'ts
- No light/white backgrounds (breaks the immersive dark theme)
- No bright primary colors (red, orange) — keep to cool tones
- No cluttered layouts — let images breathe
- No small, hard-to-read text on dark backgrounds
