# ARTITECT MACHINERY Visual Design Guide

## Brand feel
Elegant, precise, and trustworthy. The website should feel like a premium industrial machinery manufacturer: engineered, strong, polished, and easy to navigate.

## Color system
- Background: deep slate and soft warm white surfaces using Tailwind classes like `bg-slate-950`, `bg-slate-900`, `bg-stone-50`, and `bg-white`.
- Primary text: high-contrast slate/white text using `text-slate-950`, `text-slate-900`, `text-white`, and `text-slate-100`.
- Secondary text: legible muted tones using `text-slate-600`, `text-slate-300`, and `text-slate-400`.
- Accent: refined amber/gold for calls to action and highlights using `bg-amber-400`, `text-amber-300`, `border-amber-300`, and `from-amber-300`.

## Typography
- Use Inter from Google Fonts for clean, modern readability.
- Headlines should be bold and spacious: `text-4xl`, `md:text-6xl`, `font-semibold`, `tracking-tight`.
- Body copy should be calm and readable: `text-base`, `text-lg`, `leading-7`.
- Use uppercase tracking for small industrial labels: `uppercase tracking-[0.24em] text-xs`.

## Layout and spacing
- Use generous whitespace with `px-6`, `py-20`, `lg:py-28`.
- Keep content centered with `max-w-7xl mx-auto`.
- Desktop layouts should use multi-column grids; mobile layouts should stack cleanly.
- Cards should have comfortable padding: `p-6`, `p-8`.

## Components
- Buttons: rounded-full, clear contrast, amber primary action and transparent secondary action.
- Cards: subtle borders, soft shadows, readable text, and strong foreground colors.
- Navigation: simple, sticky-feeling visual style, compact and user-friendly.
- Product sections: use precise specs, benefit-led copy, and clean machinery-oriented imagery.

## Do's
- Keep all text readable with explicit foreground colors.
- Use refined industrial visuals: metal, precision machinery, fabrication, folding lines.
- Use consistent rounded corners and border treatments.
- Prioritize clear product categories and easy contact actions.

## Don'ts
- Do not use low-contrast gray text on dark backgrounds.
- Do not make the desktop layout look like a mobile single column.
- Do not use loud colors or playful styling that conflicts with industrial elegance.
