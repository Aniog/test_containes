# Design System

## Color Palette
- Background: `bg-slate-900` (dark base)
- Surface / Card: `bg-slate-800`
- Surface elevated: `bg-slate-700`
- Border: `border-slate-600`
- Primary accent: `bg-violet-600`, hover `bg-violet-500`
- Danger: `bg-red-500`, hover `bg-red-400`
- Success / completed: `text-slate-400 line-through`
- Text primary: `text-slate-100`
- Text secondary: `text-slate-400`
- Text muted: `text-slate-500`

## Typography
- Font family: Inter (Google Fonts)
- Page title: `text-3xl font-bold text-slate-100`
- Section heading: `text-lg font-semibold text-slate-200`
- Body: `text-sm text-slate-300`
- Muted / helper: `text-xs text-slate-500`

## Spacing & Layout
- Page padding: `p-6` on mobile, `p-10` on md+
- Card padding: `p-6`
- Gap between items: `gap-3`
- Max content width: `max-w-xl mx-auto`

## Borders & Radius
- Cards: `rounded-2xl`
- Buttons: `rounded-lg`
- Inputs: `rounded-lg`
- Todo items: `rounded-xl`

## Shadows
- Cards: `shadow-xl`
- Buttons: no shadow, rely on color contrast

## Buttons
- Primary: `bg-violet-600 hover:bg-violet-500 text-white font-medium rounded-lg px-4 py-2 transition-colors`
- Danger: `bg-red-500 hover:bg-red-400 text-white rounded-lg px-3 py-1.5 text-sm transition-colors`
- Ghost: `text-slate-400 hover:text-slate-200 transition-colors`

## Inputs
- `bg-slate-700 border border-slate-600 text-slate-100 placeholder-slate-500 rounded-lg px-4 py-2 focus:outline-none focus:ring-2 focus:ring-violet-500`

## Do's
- Always pair dark backgrounds with light text
- Use violet as the single accent color
- Keep spacing generous and consistent
- Use transitions on interactive elements

## Don'ts
- Don't use white backgrounds
- Don't use more than 2 accent colors
- Don't use very small text (below text-xs)
