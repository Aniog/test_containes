# SSourcing China - Design System

## Visual Style
- **Theme**: Professional B2B, clean, trustworthy, international
- **Primary Color**: #1E3A5F (deep navy blue) - trust, professionalism
- **Secondary Color**: #F97316 (warm orange) - CTAs, energy, action
- **Accent Color**: #10B981 (green) - success, reliability, growth
- **Background Light**: #F8FAFC (off-white)
- **Background Dark**: #0F172A (dark slate)
- **Text Primary**: #1E293B (slate-800)
- **Text Secondary**: #64748B (slate-500)
- **Text Light**: #F1F5F9 (slate-100)

## Typography
- **Font Family**: Inter (Google Fonts)
- **Headings**: 700-800 weight, tight letter-spacing
- **Body**: 400-500 weight, comfortable line-height (1.6-1.8)
- **Scale**: h1 (3.5rem), h2 (2.5rem), h3 (1.75rem), h4 (1.25rem)

## Spacing System
- **Base unit**: 4px (Tailwind default)
- **Section padding**: py-20 (80px) for desktop, py-12 (48px) for mobile
- **Container max-width**: max-w-7xl (1280px)
- **Grid gap**: gap-8 for major elements, gap-6 for cards

## Component Style
- **Border radius**: rounded-lg (8px) for cards, rounded-xl (12px) for larger elements
- **Shadows**: shadow-md for cards, shadow-lg for elevated elements
- **Borders**: border-1 slate-200 for subtle dividers

## Image Strategy
- Use stock images for factory, QC, shipping contexts
- Hero sections with overlay gradients
- Card images with consistent 16:9 or 4:3 aspect ratios
- Use Unsplash for realistic industrial/manufacturing imagery

## Animations
- Smooth transitions (300-500ms)
- Subtle hover effects on cards and buttons
- No excessive or distracting animations

## Do's
- Clean white space between sections
- Clear visual hierarchy
- Professional photography
- Consistent button styling
- Trust indicators (stats, badges, testimonials)

## Don'ts
- Cluttered layouts
- Multiple competing CTAs
- Excessive color variation
- Low contrast text
- Amateur stock photos
