# Design Guidelines - SSourcing China

## Visual Style
Clean, trustworthy, international B2B appearance. Professional and clear.

## Typography
- Main Font: Inter (Google Fonts)
- Headings: Sans-serif, bold, professional (Inter)
- Body: Sans-serif, clear legibility (Inter)

## Colors
- Primary: Navy Blue (#1a365d) - Represents trust and professionalism.
- Secondary: Sky Blue (#3182ce) - Reliable and modern.
- Accent: Deep Slate (#2d3748) - For high-contrast text and elements.
- Background: Light Gray/White (#f7fafc, #ffffff) - Clean and spacious.
- Success/CTA: Industrial Orange or Deep Green (e.g., #dd6b20 or #38a169) for "Get a Quote".

## Spacing & Layout
- Use standard Tailwind spacing (p-4, m-8, etc.)
- Responsive grid layouts (cols-1 for mobile, cols-2 or cols-3 for desktop)
- Consistent padding within cards and sections (p-6 to p-10)

## UI Elements
- Cards: Subtle shadows, rounded corners (rounded-lg)
- Buttons: Rounded (rounded-md), hover effects (hover:bg-opacity-90)
- Forms: Clear labels, spacious inputs, focus states

## Tailwind Class Examples
- `text-slate-900`, `text-slate-600`
- `bg-slate-50`, `bg-white`
- `border-slate-200`
- `primary-btn`: `bg-blue-800 text-white px-6 py-3 rounded-lg font-semibold hover:bg-blue-700 transition-colors`
- `section-title`: `text-3xl font-bold text-slate-900 mb-6 text-center`
