# ARTITECT MACHINERY Design System

## Brand Identity
Industrial elegance meets user-friendly clarity. The design conveys precision engineering, reliability, and sophistication — reflecting the quality of sheet metal folding machinery.

## Typography
- **Headings**: Inter (weights 700, 800) — bold, confident, architectural
- **Body**: Inter (weights 300, 400, 500) — clean, readable
- **Accent text**: Inter (weight 600) — for labels, CTAs, highlights

### Example Tailwind Classes
- Hero title: `text-5xl md:text-7xl font-extrabold tracking-tight`
- Section heading: `text-3xl md:text-4xl font-bold tracking-tight`
- Card title: `text-xl font-semibold`
- Body: `text-base font-normal leading-relaxed`
- Small label: `text-sm font-medium uppercase tracking-wider`

## Colors
- **Primary**: `#1e293b` (slate-800) — deep, authoritative navy for headers and primary surfaces
- **Primary Light**: `#334155` (slate-700) — hover states, secondary surfaces
- **Accent**: `#b45309` (amber-700) — warm metallic accent for CTAs, highlights
- **Accent Light**: `#f59e0b` (amber-500) — hover accent, subtle glows
- **Background**: `#fafaf9` (stone-50) — warm off-white page background
- **Card Background**: `#ffffff` — clean white cards
- **Text Primary**: `#1a1a1a` — near-black for body text
- **Text Secondary**: `#64748b` (slate-500) — muted text
- **Text Light**: `#94a3b8` (slate-400) — placeholder, subtle
- **Border**: `#e2e8f0` (slate-200) — subtle borders
- **Success**: `#16a34a` (green-600)

## Spacing
- Section padding: `py-20 md:py-28`
- Container max-width: `max-w-7xl mx-auto px-6`
- Card padding: `p-8`
- Grid gap: `gap-8`
- Element gap: `gap-4` or `gap-6`

## Borders & Shadows
- Cards: `rounded-xl shadow-sm border border-slate-200`
- Buttons: `rounded-lg`
- Inputs: `rounded-lg border border-slate-300`
- Hero: no border, full-bleed
- Subtle shadows: `shadow-sm` (never heavy shadows)

## Do's
- Use generous whitespace for breathing room
- Use the accent color sparingly — only for CTAs and key highlights
- Keep text left-aligned with clear hierarchy
- Use iconography to reinforce industrial/engineering theme
- Keep navigation minimal and clean

## Don'ts
- Don't use heavy box shadows or gradients
- Don't overcrowd — let each section breathe
- Don't use the accent color on large surfaces
- Don't use more than 2 font weights per text block
- Don't use animated transitions that are too flashy
