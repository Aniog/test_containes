# ARTITECT MACHINERY - Design System

## Visual Style
- **Overall Aesthetic**: Elegant industrial sophistication with user-friendly accessibility
- **Tone**: Professional, trustworthy, premium quality

## Color Palette
- Primary: Deep charcoal (#1a1f2e) - sophisticated and industrial
- Secondary: Warm bronze (#b8860b) - accent for CTAs and highlights
- Neutral: Soft cream (#faf8f5) - clean backgrounds
- Text: Dark slate (#2d3748) - readable body text
- Light gray (#e8e4de) - subtle dividers and cards

## Typography
- Headings: Inter, bold weights (600-700), clean and modern
- Body: Inter, regular (400-500), highly legible
- Emphasis: Medium weights for key information

## Layout Principles
- Generous whitespace for breathing room
- Clear visual hierarchy
- Consistent 16px base spacing unit
- Maximum content width: 1280px
- Responsive: Mobile-first approach

## Component Guidelines
- Buttons: Rounded corners (8px), clear hover states, sufficient padding
- Cards: Subtle shadows, clean borders, generous internal spacing
- Navigation: Sticky header, clear active states
- Forms: Clean inputs with focus states

## Do's
- Use bronze accents sparingly for emphasis
- Maintain consistent spacing throughout
- Ensure all text has excellent contrast
- Use high-quality product imagery

## Don'ts
- Avoid cluttered layouts
- Don't use overly bright or neon colors
- Avoid small touch targets on mobile
- Don't use decorative elements that distract from content
