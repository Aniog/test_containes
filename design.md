# Design System

## Visual Style
A modern, dark-themed design inspired by technology and AI. Clean, minimal, and professional.

## Colors
- Background (dark): `#0a0f1e` — deep navy/space black
- Surface: `#111827` (gray-900), `#1f2937` (gray-800)
- Primary accent: `#6366f1` (indigo-500) — electric indigo
- Secondary accent: `#8b5cf6` (violet-500)
- Highlight: `#06b6d4` (cyan-500)
- Text primary: `#f9fafb` (gray-50)
- Text secondary: `#9ca3af` (gray-400)
- Text muted: `#6b7280` (gray-500)
- Border: `#1f2937` (gray-800), `#374151` (gray-700)

## Typography
- Font: Inter (Google Fonts)
- Hero heading: `text-5xl md:text-7xl font-extrabold`
- Section heading: `text-3xl md:text-4xl font-bold`
- Subheading: `text-xl font-semibold`
- Body: `text-base font-normal`
- Caption/label: `text-sm font-medium`

## Spacing
- Section padding: `py-20 px-6`
- Card padding: `p-6` or `p-8`
- Gap between cards: `gap-6` or `gap-8`

## Borders & Shadows
- Card border: `border border-gray-800`
- Card radius: `rounded-2xl`
- Glow effect: `shadow-lg shadow-indigo-500/10`
- Hover glow: `hover:shadow-indigo-500/20`

## Gradients
- Hero gradient text: `bg-gradient-to-r from-indigo-400 via-violet-400 to-cyan-400 bg-clip-text text-transparent`
- Section gradient bg: `bg-gradient-to-br from-gray-900 to-gray-800`
- Accent gradient: `bg-gradient-to-r from-indigo-500 to-violet-500`

## Do's
- Use dark backgrounds with light text for readability
- Use gradient accents sparingly for emphasis
- Use rounded corners (`rounded-2xl`) on cards
- Use subtle borders (`border-gray-800`) to define surfaces
- Use icon + text combos in feature cards

## Don'ts
- Don't use light backgrounds in the main content
- Don't use low-contrast text (e.g., gray-600 on gray-900)
- Don't use too many colors — stick to indigo/violet/cyan palette
- Don't use heavy drop shadows that look dated
