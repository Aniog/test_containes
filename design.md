# ARTITECT MACHINERY - Design Guidelines

## Brand Identity
- **Brand Name**: ARTITECT MACHINERY
- **Industry**: Industrial sheet metal folding machines
- **Tone**: Elegant, professional, industrial, trustworthy

## Color Palette
- **Primary**: `#1a1a2e` (Deep navy/charcoal - sophistication)
- **Secondary**: `#16213e` (Dark blue - industrial feel)
- **Accent**: `#e94560` (Coral red - call-to-action, energy)
- **Accent Secondary**: `#0f3460` (Medium blue - depth)
- **Light**: `#f8f9fa` (Off-white - clean backgrounds)
- **Text Primary**: `#1a1a2e` (Dark for readability)
- **Text Secondary**: `#6c757d` (Muted gray)
- **Text Light**: `#ffffff` (White for dark backgrounds)
- **Border**: `#e9ecef` (Light gray)

## Typography
- **Headings**: "Playfair Display", serif - elegant, sophisticated
- **Body**: "DM Sans", sans-serif - clean, readable, modern
- **Font Sizes**:
  - Hero Title: 4rem (64px)
  - Section Title: 2.5rem (40px)
  - Subtitle: 1.25rem (20px)
  - Body: 1rem (16px)
  - Small: 0.875rem (14px)

## Layout & Spacing
- **Max Content Width**: 1200px
- **Section Padding**: 80px vertical, 24px horizontal
- **Card Padding**: 24px
- **Grid Gap**: 32px
- **Border Radius**: 8px (cards), 4px (buttons)

## Visual Effects
- **Shadows**: Subtle, elegant shadows
  - Card: `0 4px 20px rgba(0, 0, 0, 0.08)`
  - Hover: `0 8px 30px rgba(0, 0, 0, 0.12)`
- **Transitions**: 0.3s ease for all interactive elements
- **Backgrounds**: Subtle gradients, geometric patterns

## Components

### Header/Navigation
- Fixed position, white background with subtle shadow
- Logo on left, navigation links center, CTA button right
- Smooth scroll to sections

### Hero Section
- Full viewport height
- Dark gradient background with subtle industrial pattern
- Large headline with brand name emphasis
- Subheadline describing the products
- Two CTA buttons: primary (accent color) and secondary (outline)

### Products Section
- Section title with accent underline
- Product cards in 3-column grid (responsive to 2 and 1)
- Each card: image placeholder, title, description, "Learn More" link
- Hover effect: lift and shadow increase

### Features Section
- Icon-based feature cards
- 4-column grid
- Icons with accent color
- Title and description per feature

### About Section
- Two-column layout: image + text
- Company story and expertise
- Statistics/achievements

### Contact Section
- Clean form layout
- Fields: Name, Email, Phone, Message
- Submit button with accent color

### Footer
- Dark background
- 4-column layout: About, Products, Contact, Social
- Copyright at bottom

## Responsive Breakpoints
- **Desktop**: 1024px+
- **Tablet**: 768px - 1023px
- **Mobile**: < 768px

## Do's and Don'ts
- DO: Use clean whitespace, consistent spacing
- DO: Use accent color sparingly for CTAs
- DO: Maintain professional, industrial feel
- DON'T: Use too many colors (keep it minimal)
- DON'T: Overcrowd content
- DON'T: Use flashy animations (keep it subtle)
