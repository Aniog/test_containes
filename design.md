# ARTITECT MACHINERY - Design Guidelines

## Brand Identity
- **Brand Name**: ARTITECT MACHINERY
- **Industry**: Industrial sheet metal folding machines
- **Tone**: Elegant, professional, trustworthy, user-friendly

## Color Palette
- **Primary**: `#1a1a2e` (Deep navy/charcoal - sophistication)
- **Secondary**: `#16213e` (Dark blue - industrial reliability)
- **Accent**: `#e94560` (Coral red - action/CTA buttons)
- **Accent Secondary**: `#0f3460` (Medium blue - depth)
- **Light**: `#f8f9fa` (Off-white - backgrounds)
- **Text Primary**: `#1a1a2e` (Dark for readability)
- **Text Secondary**: `#6c757d` (Muted gray)
- **Text Light**: `#ffffff` (White for dark backgrounds)
- **Border**: `#dee2e6` (Light gray)

## Typography
- **Headings**: "Playfair Display", serif - elegant, premium feel
- **Body**: "Inter", sans-serif - clean, readable, modern
- **Font Sizes**:
  - Hero: 56px (desktop), 36px (mobile)
  - H1: 42px
  - H2: 32px
  - H3: 24px
  - Body: 16px
  - Small: 14px

## Layout & Spacing
- **Max Width**: 1200px container
- **Section Padding**: 80px vertical (desktop), 48px (mobile)
- **Grid Gap**: 32px
- **Card Padding**: 24px
- **Border Radius**: 8px (cards), 4px (buttons)

## Visual Effects
- **Shadows**: Subtle box-shadow for cards: `0 4px 20px rgba(0,0,0,0.08)`
- **Hover Effects**: Smooth transitions (0.3s ease)
- **Backgrounds**: Subtle gradients on hero sections

## Components

### Header/Navigation
- Fixed top navigation
- Logo on left
- Navigation links center
- CTA button on right
- White background with subtle shadow on scroll

### Hero Section
- Full-width with dark gradient overlay
- Large headline with brand name emphasis
- Subheadline describing products
- Two CTA buttons: primary (accent color) and secondary (outline)
- Background: industrial/machinery image or gradient

### Product Categories Section
- Section title with underline accent
- Grid of product category cards (3 columns desktop, 1 mobile)
- Each card: icon, title, brief description
- Hover: subtle lift effect

### Products Showcase
- Featured products grid
- Product cards with image placeholder, name, brief specs
- "View Details" button

### Features/Benefits Section
- Alternating layout (image + text)
- Key selling points with icons

### Contact/CTA Section
- Dark background
- Compelling headline
- Contact form or contact information

### Footer
- Dark background
- Logo, navigation links, contact info
- Copyright

## Product Keywords (for SEO/content)
- Double folding machine
- Double folder
- Sheet metal folder
- Sheet metal folding machine
- Metal folder
- Metal folder machine
- Metal folding machine

## Do's
- Use clean, generous whitespace
- Maintain consistent spacing
- Use high-quality imagery
- Keep text concise and scannable
- Use clear CTAs
- Ensure mobile responsiveness

## Don'ts
- Don't use too many colors (keep it cohesive)
- Don't clutter the design
- Don't use generic stock photos if possible
- Don't make it feel cold or impersonal - add warmth through design
