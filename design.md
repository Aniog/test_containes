# ARTITECT MACHINERY - Design System

## Overview
An elegant, industrial-themed website for premium sheet metal machinery. Dark, sophisticated palette with warm gold accents. Conveys precision, reliability, and engineering excellence.

## Color Palette
| Token | Hex | Usage |
|-------|-----|-------|
| Primary Dark | `#0f172a` | Page backgrounds, nav |
| Secondary Dark | `#1e293b` | Cards, sections |
| Accent | `#d97706` | CTAs, highlights, borders |
| Accent Light | `#fbbf24` | Hover states |
| Text Primary | `#f1f5f9` | Headings, body text |
| Text Muted | `#94a3b8` | Descriptions, labels |
| Border | `#334155` | Card borders, dividers |
| Surface | `#111827` | Elevated surfaces |

## Typography
- **Display font**: Playfair Display (serif) - for all headings
- **Body font**: Inter (sans-serif) - for body text, UI elements
- H1: 48px / 56px, weight 700
- H2: 36px / 44px, weight 600
- H3: 24px / 32px, weight 600
- Body: 16px / 24px, weight 400
- Small: 14px / 20px, weight 400

## Spacing
- Section padding: 80px vertical (py-20)
- Container max-width: 1280px (max-w-7xl)
- Horizontal padding: 16px mobile, 24px tablet, 48px desktop
- Card gap: 24px (gap-6)
- Component gap: 32px (gap-8)

## Components
- Buttons: Rounded-lg, px-6 py-3, bg-amber-600 hover:bg-amber-500
- Cards: Rounded-xl, bg-slate-800, border border-slate-700, hover:border-amber-600/50
- Inputs: Rounded-lg, bg-slate-800, border border-slate-700, focus:border-amber-600
- Section titles: Centered, uppercase tracking-widest small label above, large serif heading below

## Effects
- Subtle gradient overlays on hero images
- Card hover: translateY(-4px), border color change, shadow increase
- Smooth transitions: 300ms ease-out
- Subtle glow on accent elements

## Do's
- Use generous whitespace
- Keep text readable with strong contrast
- Use gold/amber sparingly for maximum impact
- Include stock imagery of machinery

## Don'ts
- Don't use multiple accent colors
- Don't crowd elements on mobile
- Don't use pure white (#fff) for large text areas
- Don't use light backgrounds for sections
