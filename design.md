# Velmora Fine Jewelry — Design System

## Visual Identity
- Mood: Quiet luxury, warm, editorial. Premium demi-fine jewelry.
- NOT loud, NOT discount-looking, NOT generic e-commerce.

## Color Palette
- **Background**: `#FAF7F2` (warm cream) — primary page background
- **Surface**: `#F5EDE3` (warm beige) — cards, secondary surfaces
- **Border/Divider**: `#E8DDD0` (sand) — hairline dividers, borders
- **Accent**: `#C9A96E` (warm gold) — CTAs, highlights, interactive elements
- **Accent Hover**: `#A8864E` (dark gold) — hover states
- **Text Primary**: `#2C2C2C` (charcoal) — headings, body text
- **Text Muted**: `#8A7E72` (warm gray) — secondary text, captions
- **Text Light Muted**: `#B5A99A` — placeholders, disabled states
- **Dark Surface**: `#1A1612` (espresso) — hero overlays, dark sections

## Typography
- **Headings**: Cormorant Garamond (serif), weights 400-600
- **Body/UI**: Inter (sans-serif), weights 300-500
- **Product names**: UPPERCASE, letter-spacing 0.2em-0.25em, serif
- **Section headings**: Serif, large, generous whitespace above/below
- **Body text**: 16px base, line-height 1.6-1.8

## Spacing & Layout
- Generous whitespace throughout
- Max content width: 1280px
- Section padding: 80px-120px vertical (desktop), 48px-64px (mobile)
- Hairline dividers (1px, sand color)

## Buttons
- Primary: Solid gold (#C9A96E) background, cream text, rounded-none or rounded-sm
- Secondary: Outlined, gold border, gold text, transparent background
- Hover: Darken gold, subtle transition (200ms)
- Font: Sans-serif, uppercase, wide letter-spacing (0.15em), medium weight

## Images
- Large editorial imagery
- Warm gold jewelry on dark/neutral backgrounds
- Hover: subtle scale or second-image reveal
- Aspect ratios: 3x4 for products, 16x9 for hero, 9x16 for UGC/reels

## Animations
- Subtle, smooth transitions (200-400ms)
- Fade-in on scroll
- Hover: gentle scale (1.02-1.05), opacity changes
- No bouncy or playful animations

## Do's
- Use warm cream backgrounds
- Use serif for all headings and product names
- Keep generous whitespace
- Use thin hairline dividers
- Use gold as the primary accent sparingly

## Don'ts
- Don't use bright or saturated colors
- Don't use playful/bouncy animations
- Don't crowd elements — always leave breathing room
- Don't use sans-serif for product names or section headings
- Don't use heavy shadows — keep them soft and subtle
