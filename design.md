# ARTITECT MACHINERY Design System

## Visual Style
- **Overall:** Elegant yet user-friendly, combining industrial strength with refined aesthetics
- **Color Palette:**
  - Primary: Deep metallic blue (#1a365d) - trustworthy, technical
  - Secondary: Brushed steel gray (#64748b) - industrial elegance
  - Accent: Warm gold (#b45309) - premium quality
  - Background: Light gray (#f8fafc) with subtle steel textures
  - Text: Dark charcoal (#1e293b) for readability
- **Typography:**
  - Headings: Inter (700-800 weight) for modern, clean look
  - Body: Inter (400-500 weight) for readability
  - Line heights: 1.5-1.6 for comfortable reading
- **Layout:**
  - Generous white space
  - Grid-based layouts with clear hierarchy
  - Balanced asymmetry for visual interest
- **Components:**
  - Cards with subtle shadows and rounded corners
  - Buttons with clear hover states
  - Clean icons from Lucide
- **Imagery:**
  - High-quality industrial machine photography
  - Professional, clean backgrounds
  - Consistent lighting and perspective

## Design Principles
1. **Clarity First:** Easy to understand product information
2. **Professional Trust:** Build credibility through design quality
3. **Visual Hierarchy:** Guide eyes to important information
4. **Responsive:** Works on all devices
5. **Fast Loading:** Optimize images and code splitting

## Specific Elements
- Hero section with compelling headline and machine imagery
- Product cards with clear specs and call-to-actions
- Feature highlights with icons and brief descriptions
- Customer testimonials for social proof
- Contact form for inquiries
- Footer with company information

## Color Usage
- Use primary blue for headers, buttons, and key elements
- Use steel gray for secondary text and borders
- Use gold accent for highlights, CTAs, and premium elements
- Light backgrounds to keep focus on content
- Dark text for readability on light backgrounds

## Responsive Design
- Mobile-first approach
- Collapsible navigation
- Stacked layouts on mobile
- Appropriate font size adjustments
- Touch-friendly buttons and links
