# SSourcing China - Design Guide

## Brand Colors
- Primary: Blue (#1e40af / blue-800) - trust, professionalism
- Primary Light: (#3b82f6 / blue-500) - CTAs, links
- Primary Dark: (#1e3a8a / blue-900) - headings, dark backgrounds
- Accent: (#ea580c / orange-600) - highlights, badges, emphasis
- Background: White (#ffffff) and gray-50 (#f9fafb) for sections
- Text: gray-900 (#111827) for headings, gray-600 (#4b5563) for body
- Borders: gray-200 (#e5e7eb)
- Success: green-600 (#16a34a)

## Typography
- Font: Inter (Google Fonts)
- Headings: font-bold, tracking-tight
- Body: font-normal, leading-relaxed
- Sizes: h1 = text-4xl/5xl, h2 = text-3xl, h3 = text-xl/2xl

## Spacing
- Section padding: py-16 md:py-24
- Container: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8
- Card padding: p-6 md:p-8
- Gap between sections: next section starts with py-16

## Components
- Cards: white bg, rounded-xl, shadow-sm, border border-gray-100, hover:shadow-md transition
- Buttons (primary): bg-blue-600 hover:bg-blue-700 text-white px-6 py-3 rounded-lg font-medium
- Buttons (secondary): bg-white border border-gray-300 hover:border-blue-500 text-gray-700 px-6 py-3 rounded-lg font-medium
- Links: text-blue-600 hover:text-blue-700 font-medium
- Section titles: text-center mb-12 md:mb-16

## Visual Style
- Clean, white backgrounds with alternating gray sections
- Subtle shadows and rounded corners
- Professional, trustworthy feel
- No exaggerated claims, practical B2B tone
- Icons use Lucide React
- Responsive: mobile-first with md: and lg: breakpoints

## Don't
- Don't use dark mode
- Don't use gradients or decorative flourishes
- Don't use emojis
- Don't make exaggerated claims
- Don't use stock photo placeholders that look fake
