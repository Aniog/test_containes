# Design System — APPLE mini PC Landing Page

## Visual Style
Clean, minimal, premium product aesthetic inspired by modern tech brands.
Light backgrounds with dark text for readability. Bold typography for headlines.

## Colors
- Background primary: `bg-white` / `#ffffff`
- Background secondary: `bg-gray-50` / `#f9fafb`
- Background dark section: `bg-gray-950` / `#030712`
- Text primary (dark bg): `text-white`
- Text primary (light bg): `text-gray-900`
- Text secondary: `text-gray-500`
- Accent / CTA: `bg-blue-600` hover `bg-blue-700` — `#2563eb`
- Accent text: `text-blue-600`
- Border: `border-gray-200`

## Typography
- Font: Inter (Google Fonts, loaded in index.html)
- Hero headline: `text-5xl md:text-7xl font-bold tracking-tight text-gray-900`
- Section headline: `text-3xl md:text-4xl font-bold text-gray-900`
- Subheadline: `text-lg md:text-xl text-gray-500 font-normal`
- Body: `text-base text-gray-600`
- Label/caption: `text-sm text-gray-400 uppercase tracking-widest`

## Spacing
- Section padding: `py-20 md:py-28 px-6`
- Container max width: `max-w-6xl mx-auto`
- Card padding: `p-8`
- Gap between grid items: `gap-8`

## Borders & Shadows
- Card border: `border border-gray-100 rounded-2xl`
- Card shadow: `shadow-sm hover:shadow-md transition-shadow`
- Pill badge: `rounded-full px-4 py-1 text-sm font-medium`

## Buttons
- Primary CTA: `bg-blue-600 hover:bg-blue-700 text-white font-semibold px-8 py-3 rounded-full transition-colors`
- Secondary: `border border-gray-300 hover:border-gray-400 text-gray-700 font-semibold px-8 py-3 rounded-full transition-colors`

## Do's
- Use generous whitespace between sections
- Use large, bold numbers for specs (e.g. "16GB", "M4 chip")
- Keep nav minimal and sticky
- Use subtle gradient overlays on hero backgrounds

## Don'ts
- Don't use bright neon colors
- Don't crowd elements — keep breathing room
- Don't use more than 2 font weights per section
- Don't use low-contrast text on colored backgrounds
