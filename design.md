# SSourcing China - Design System

## Visual Style

Professional, clean, trustworthy B2B website for a China-based sourcing agent.

### Colors
- Primary: #0F172A (slate-900) - Deep navy for headers and CTAs
- Secondary: #1E40AF (blue-800) - Trustworthy blue for accents
- Accent: #0EA5E9 (sky-500) - Light blue for highlights
- Background: #FFFFFF (white) - Clean white backgrounds
- Surface: #F8FAFC (slate-50) - Light gray for cards/sections
- Text Primary: #0F172A (slate-900) - Dark text for readability
- Text Secondary: #475569 (slate-600) - Muted text for descriptions
- Border: #E2E8F0 (slate-200) - Subtle borders
- Success: #059669 (emerald-600) - For trust indicators

### Typography
- Font: Inter (system-ui fallback)
- Headings: 600-700 weight, tight tracking
- Body: 400-500 weight, 1.6 line-height
- Small text: 14px for captions and meta

### Spacing
- Section padding: 80px top/bottom (desktop), 48px (mobile)
- Card padding: 32px
- Element gaps: 16px, 24px, 32px
- Container max-width: 1280px

### Components
- Buttons: Rounded corners (6px), 48px height for primary
- Cards: White background, subtle shadow, 6px radius
- Forms: Clean inputs with 8px radius, clear focus states
- Navigation: Fixed header, clean links, mobile hamburger

### Images
- Use realistic factory, QC inspection, and shipping visuals
- Professional photography style
- Consistent aspect ratios

### Do's
- Use ample white space
- Maintain consistent alignment
- Use icons from lucide-react for visual clarity
- Ensure all text has proper contrast
- Keep CTAs prominent but not overwhelming

### Don'ts
- Avoid bright or saturated colors
- No exaggerated claims or hype language
- Don't overcrowd sections
- Avoid decorative elements that don't serve purpose
- No low-contrast text
