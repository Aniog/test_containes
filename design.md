# ARTITECT MACHINERY Design System

## Brand Personality
Elegant industrial precision. Clean, confident, and approachable. The design should feel premium but not cold — like a well-engineered machine that's also a pleasure to use.

## Color Palette
- Primary: `#0f172a` (slate-900) — deep charcoal for headers and strong accents
- Secondary: `#334155` (slate-700) — softer dark for body text and secondary elements
- Accent: `#2563eb` (blue-600) — confident blue for CTAs and highlights
- Accent hover: `#1d4ed8` (blue-700)
- Background: `#f8fafc` (slate-50) — warm light gray for page background
- Surface: `#ffffff` (white) — cards and elevated surfaces
- Border: `#e2e8f0` (slate-200) — subtle borders
- Muted text: `#64748b` (slate-500) — helper text and captions

## Typography
- Font family: Inter (Google Fonts) — clean, highly legible, modern
- Headings: Bold weights (700-800), tight line height
- Body: Regular weight (400-500), comfortable line height (1.6-1.7)
- Scale: Use Tailwind default scale (text-sm, text-base, text-lg, text-xl, text-2xl, text-3xl, text-4xl, text-5xl)

## Spacing & Layout
- Generous whitespace — sections should breathe
- Max content width: `max-w-7xl` (1280px) centered
- Section padding: `py-20` to `py-28` on desktop
- Card padding: `p-6` to `p-8`
- Grid gaps: `gap-6` to `gap-8`

## Components
- Buttons: Rounded corners (`rounded-lg`), clear padding, subtle shadow on hover
- Cards: White background, subtle border (`border border-slate-200`), soft shadow (`shadow-sm` to `shadow-md`), rounded corners (`rounded-xl`)
- Navigation: Clean, minimal, with subtle backdrop blur when scrolled
- Forms: Clean inputs with slate borders, focus ring in blue

## Visual Style
- Use subtle gradients sparingly — prefer solid colors with depth through shadows
- Icons: Lucide React, consistent stroke width
- Images: Use stock image system for machinery/industrial photography feel
- Rounded corners: `rounded-lg` for buttons, `rounded-xl` for cards, `rounded-2xl` for hero elements
- Shadows: `shadow-sm` default, `shadow-md` on hover, `shadow-lg` for hero

## Do's
- Use plenty of whitespace
- Keep text readable with strong contrast
- Use the accent blue sparingly for CTAs and key highlights
- Maintain consistent spacing rhythm
- Use subtle animations (fade-in, slide-up) for polish

## Don'ts
- Don't use pure black (`#000`) — use slate-900 instead
- Don't overcrowd sections
- Don't use more than 2-3 colors per section
- Don't use harsh borders or heavy outlines
- Don't use decorative elements that don't serve the content
