# Velmora Fine Jewelry - Design System

## Visual Identity

### Mood
Quiet luxury, warm, editorial. Premium demi-fine jewelry aesthetic — refined, intimate, not loud or discount-looking.

### Color Palette
- **Background**: `#0D0D0D` (deep charcoal black)
- **Surface**: `#1A1A1A` (elevated surfaces)
- **Surface Hover**: `#242424` (interactive surfaces)
- **Primary Text**: `#F5F0EB` (warm cream white)
- **Secondary Text**: `#8A8680` (muted warm gray)
- **Accent Gold**: `#C9A962` (warm champagne gold - primary accent)
- **Accent Gold Light**: `#E5D4A1` (highlight gold)
- **Border**: `#2A2A2A` (subtle dividers)
- **Error**: `#D64545`
- **Success**: `#4A9D6E`

### Typography
- **Headings**: Cormorant Garamond (Google Fonts) - elegant serif with beautiful italics
  - Display: 600 weight, tracking: 0.02em
  - Product names: UPPERCASE, weight 500, tracking: 0.15em
  
- **Body/UI**: Inter (Google Fonts) - clean, modern sans-serif
  - Body: 400 weight, line-height 1.6
  - UI Elements: 500 weight
  - Small/Labels: 400-500 weight, tracking: 0.05em

### Spatial System
- Base unit: 4px
- Section padding: 80px-120px vertical (desktop), 48px-64px (mobile)
- Component gaps: 16px-32px
- Generous whitespace throughout
- Max content width: 1280px

### Visual Elements
- **Borders**: 1px hairline borders, color `#2A2A2A`
- **Shadows**: Soft, subtle shadows for elevation
  - Card: `0 4px 20px rgba(0,0,0,0.3)`
  - Modal: `0 8px 40px rgba(0,0,0,0.5)`
- **Buttons**: 
  - Primary: Solid gold background, dark text, subtle hover lift
  - Secondary: Outlined gold border, transparent bg, gold text on hover
- **Images**: Dark/neutral backgrounds for jewelry photography
- **Transitions**: 300ms ease-out for all interactions

### Accessibility
- WCAG AA contrast ratios maintained
- Focus states visible on all interactive elements
- Semantic HTML structure
