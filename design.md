# Design System — Pizzeria Site

## Brand Identity
Warm, rustic Italian pizzeria. Feels authentic, inviting, and appetizing.

## Colors
- Background (cream): `bg-[#FFF8F0]`
- Primary red (tomato): `bg-[#C0392B]` / `text-[#C0392B]`
- Dark charcoal (text): `text-[#1A1A1A]`
- Warm gold (accent): `text-[#E67E22]` / `bg-[#E67E22]`
- Muted text: `text-[#6B5B4E]`
- Card background: `bg-white`
- Section alt background: `bg-[#1A1A1A]`

## Typography
- Font: Inter (loaded via Google Fonts in index.html)
- Hero headline: `text-5xl md:text-7xl font-extrabold`
- Section headings: `text-3xl md:text-4xl font-bold`
- Body: `text-base md:text-lg`
- Muted/caption: `text-sm text-[#6B5B4E]`

## Spacing
- Section padding: `py-20 px-6`
- Card padding: `p-6`
- Gap between cards: `gap-8`

## Borders & Shadows
- Cards: `rounded-2xl shadow-md`
- Buttons: `rounded-full`
- Dividers: `border-[#E67E22]`

## Buttons
- Primary CTA: `bg-[#C0392B] text-white rounded-full px-8 py-3 font-semibold hover:bg-[#A93226] transition`
- Secondary: `border-2 border-white text-white rounded-full px-8 py-3 font-semibold hover:bg-white hover:text-[#1A1A1A] transition`

## Do's
- Use warm cream `#FFF8F0` as the main page background
- Use deep red `#C0392B` for CTAs and highlights
- Use dark charcoal `#1A1A1A` for primary text on light backgrounds
- Use white text on dark/red backgrounds
- Keep sections alternating between cream and dark for visual rhythm

## Don'ts
- Don't use pure white `#FFFFFF` as the page background
- Don't use low-contrast text (e.g. gray on cream)
- Don't use more than 3 font weights per section
