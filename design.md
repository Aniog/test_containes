# Design System — Pink Website

## Color Palette
- Primary: `#ec4899` (pink-500) — main brand color
- Primary Light: `#fce7f3` (pink-100) — backgrounds, tints
- Primary Dark: `#be185d` (pink-700) — hover states, accents
- Secondary: `#f9a8d4` (pink-300) — soft accents
- Background: `#fff0f6` — page background (very light pink)
- Surface: `#ffffff` — cards, modals
- Text Primary: `#1f2937` (gray-800) — headings
- Text Secondary: `#6b7280` (gray-500) — body text
- Text On Primary: `#ffffff` — text on pink backgrounds

## Typography
- Font Family: Inter (Google Fonts)
- Heading 1: `text-5xl font-bold text-gray-800`
- Heading 2: `text-3xl font-bold text-gray-800`
- Heading 3: `text-xl font-semibold text-gray-800`
- Body: `text-base text-gray-600`
- Small: `text-sm text-gray-500`

## Spacing
- Section padding: `py-20 px-6`
- Card padding: `p-8`
- Gap between items: `gap-8`

## Borders & Radius
- Cards: `rounded-2xl`
- Buttons: `rounded-full`
- Inputs: `rounded-xl`
- Subtle border: `border border-pink-100`

## Shadows
- Card shadow: `shadow-md hover:shadow-xl`
- Button shadow: `shadow-pink-200 shadow-lg`

## Buttons
- Primary: `bg-pink-500 hover:bg-pink-600 text-white rounded-full px-8 py-3 font-semibold shadow-lg`
- Outline: `border-2 border-pink-500 text-pink-500 hover:bg-pink-50 rounded-full px-8 py-3 font-semibold`

## Do's
- Use pink gradients for hero sections: `from-pink-50 to-rose-50`
- Use pink-500 for primary CTAs and highlights
- Keep backgrounds light and airy
- Use rounded corners generously

## Don'ts
- Don't use dark backgrounds (this is a light, feminine pink theme)
- Don't use harsh contrasts — keep it soft and elegant
- Don't mix too many accent colors
