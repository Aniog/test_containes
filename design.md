# Design System

## Color Palette
- Background: `bg-slate-900` (dark base)
- Card surface: `bg-slate-800`
- Input surface: `bg-slate-700`
- Primary accent: `bg-violet-600`, hover `bg-violet-500`
- Danger: `bg-red-500`, hover `bg-red-400`
- Muted text: `text-slate-400`
- Body text: `text-slate-100`
- Completed text: `text-slate-500 line-through`
- Border: `border-slate-600`

## Typography
- Font: Inter (Google Fonts)
- Heading: `text-3xl font-bold text-slate-100`
- Subheading / label: `text-sm font-medium text-slate-400`
- Body: `text-base text-slate-200`

## Spacing & Layout
- Page max-width: `max-w-lg mx-auto`
- Card padding: `p-6`
- Section gap: `gap-4`
- Item padding: `px-4 py-3`

## Borders & Radius
- Card: `rounded-2xl`
- Input / button: `rounded-lg`
- Todo item: `rounded-xl`

## Shadows
- Card: `shadow-xl`

## Buttons
- Primary: `bg-violet-600 hover:bg-violet-500 text-white rounded-lg px-4 py-2 font-medium transition-colors`
- Ghost / icon: `text-slate-400 hover:text-red-400 transition-colors`
- Filter active: `bg-violet-600 text-white`
- Filter inactive: `bg-slate-700 text-slate-300 hover:bg-slate-600`

## Do's
- Always pair a dark background with light text
- Use violet as the single accent color
- Keep interactive elements clearly distinguishable with hover states

## Don'ts
- Don't use white backgrounds
- Don't use low-contrast text on dark surfaces
- Don't mix multiple accent colors
