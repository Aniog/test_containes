# SSourcing China — Design System

## Brand Identity
Professional B2B sourcing agent website. Clean, trustworthy, international, modern industrial aesthetic.

## Colors
- **Primary Navy**: `#0f1d36` — headers, footer backgrounds, deep accents
- **Primary Blue**: `#1a3a5c` — section backgrounds, cards
- **Accent Red**: `#c41e3a` — CTAs, highlights, links
- **Gold**: `#d4a853` — subtle highlights, badges, icons
- **White**: `#ffffff` — page backgrounds, cards
- **Gray-50**: `#f8f9fa` — light section backgrounds
- **Gray-100**: `#f0f2f5` — subtle card borders
- **Gray-300**: `#cbd5e1` — borders
- **Gray-500**: `#64748b` — body text
- **Gray-700**: `#334155` — headings
- **Gray-900**: `#0f172a` — dark text

## Typography
- **Font Family**: Inter (Google Fonts, already loaded in index.html)
- **Headings**: Bold, tracking-tight, Gray-900
  - h1: text-5xl/none md:text-6xl font-bold
  - h2: text-3xl md:text-4xl font-bold
  - h3: text-xl md:text-2xl font-semibold
- **Body**: text-base md:text-lg text-gray-500 leading-relaxed
- **Small/Caption**: text-sm text-gray-500

## Spacing
- Sections: py-16 md:py-24
- Container: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8

## Buttons
- **Primary CTA**: bg-red-600 hover:bg-red-700 text-white font-semibold rounded-lg px-8 py-3.5 shadow-lg
- **Secondary**: border-2 border-navy text-navy hover:bg-navy hover:text-white rounded-lg px-8 py-3.5
- **Ghost/Text**: text-red-600 hover:text-red-700 font-medium

## Cards
- bg-white rounded-xl shadow-sm border border-gray-100 p-6 md:p-8
- Hover: shadow-md transition-shadow

## Visual Style
- Clean, modern, ample whitespace
- Subtle shadows, no harsh borders
- Professional icon usage (Lucide icons)
- Data/statistics displays with large numbers
- Professional factory/QC imagery via stock images
- Do: use trust signals, certifications, numbers, clean layouts
- Don't: use exaggerated claims, flashy animations, unprofessional colors
