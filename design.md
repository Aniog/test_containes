# MicroCosmos Design System

## Typography
- **Heading**: 'Inter', sans-serif; font-weight: 700-800.
- **Body**: 'Inter', sans-serif; font-weight: 400.
- **Accents**: Monospace for data-like feels.

## Colors
- **Primary (Deep Space)**: #0a0a0a (background)
- **Secondary (Neon Teal)**: #2dd4bf (accents)
- **Neutral (Gray)**: #a1a1aa (supporting text)
- **White**: #ffffff (main headings)

## Visual Style
- **Dark Mode by default**: Dark background to make micro-visuals pop.
- **Glassmorphism**: Use semi-transparent backgrounds for overlays.
- **Micro-interactions**: Subtle hover scaling on images.
- **Immersive Imagery**: Full-width hero and high-density galleries.

## Tailwind Classes
- `bg-stone-950` for main background.
- `text-teal-400` for highlights.
- `backdrop-blur-md bg-black/40` for card overlays.
- `hover:scale-105 transition-transform duration-300` for images.
