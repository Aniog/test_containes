# Velmora Fine Jewelry — Design System

## Visual Identity
- **Mood**: Quiet luxury, warm, editorial. Premium demi-fine jewelry.
- **Not**: Loud, discount-looking, generic e-commerce.

## Color Palette
| Token | Hex | Usage |
|-------|-----|-------|
| velmora-cream | #FAF8F5 | Page background |
| velmora-sand | #F5F0EB | Card backgrounds, subtle surfaces |
| velmora-gold | #C8A96E | Primary accent, buttons, links, highlights |
| velmora-gold-light | #D4BC8B | Hover states, subtle gold accents |
| velmora-gold-dark | #A6844A | Active states, deeper gold |
| velmora-charcoal | #1E1E1E | Primary text, dark backgrounds |
| velmora-smoke | #3D3D3D | Secondary text |
| velmora-stone | #8C8C8C | Tertiary text, placeholders |
| velmora-pearl | #E8E3DC | Dividers, subtle borders |
| velmora-rose | #B8A0A0 | Accent blush tone |

## Typography
- **Headings / Product names**: Cormorant Garamond (serif) — uppercase with wide letter-spacing
- **Body / UI**: Inter (sans-serif) — clean, readable
- Product names: UPPERCASE, tracking-widest (0.25em)
- Section titles: serif, 3xl–5xl, leading-tight

## Spacing & Layout
- Generous whitespace: sections get py-20 to py-32
- Section padding: px-6 md:px-12 lg:px-20
- Hairline dividers: border-t border-velmora-pearl/50

## Components
- **Buttons**: .btn-primary (gold bg, white text), .btn-outline (gold border), .btn-ghost (text-only)
- **Cards**: Subtle hover shadow, smooth transitions
- Product cards: hover reveals second image, quick add-to-cart
- **Accordions**: Clean, thin borders, serif titles

## Imagery
- Warm-lit gold jewelry on dark/neutral backgrounds
- Editorial, model-worn shots for hero and UGC
- Clean product-only shots for cards
