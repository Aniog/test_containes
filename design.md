# Velmora Fine Jewelry Design System

## Visual Style
- **Mood:** Quiet luxury, warm, editorial, premium.
- **Palette:** 
  - Background (Beige/Off-white): `bg-[#FDFCFB]` / `hsl(20, 20%, 98%)`
  - Text (Charcoal): `text-[#1A1A1A]` / `hsl(20, 10%, 10%)`
  - Primary (Deep Bronze Gold): `bg-[#6B532F]` / `hsl(38, 40%, 30%)`
  - Accent (Warm Gold): `bg-[#C2A370]` / `hsl(38, 30%, 60%)`
  - Border (Light Taupe): `border-[#EBE2D5]` / `hsl(38, 10%, 90%)`
- **Typography:**
  - Headings/Product Names: `font-serif` (Cormorant Garamond)
  - Body/UI: `font-sans` (Inter)
  - Product Names styling: `.serif-uppercase` (Uppercase, wide letter spacing `tracking-widest`)
- **Elements:**
  - Hairline Dividers: `.hairline-divider` (Thin, light borders)
  - Buttons: `.btn-premium` (Solid dark bronze) or `.btn-outline` (Outlined)
  - Shadow: Soft, subtle elevations.
- **Imagery:** Warm-lit, editorial, high quality, gold focus on neutral/dark backgrounds.

## Do's
- Use generous whitespace.
- Keep UI elements minimal and clean.
- Use wide letter-spacing for uppercase serif text.
- Maintain strong contrast for accessibility.

## Don'ts
- Use loud, saturated colors.
- Use rounded corners (keep it sharp/square for premium feel).
- Use generic e-commerce badges or "SALE" labels that look cheap.
- Overcrowd the layout.
