# Velmora Fine Jewelry — Design System

## Visual Identity
- **Mood**: Quiet luxury, warm, editorial. Premium demi-fine jewelry — NOT loud, NOT discount-looking.
- **Target**: Women 25–45, gifting and self-purchase, $30–$120 price point.

## Color Palette
- **Background Primary**: `#FAF7F2` (brand-cream) — warm off-white, main page background
- **Background Secondary**: `#F5F0E8` (brand-ivory) — slightly deeper warm neutral for cards/sections
- **Accent Sand**: `#E8DFD0` (brand-sand) — subtle dividers, hover states
- **Gold Accent**: `#C9A96E` (brand-gold) — primary accent, CTAs, highlights
- **Gold Light**: `#D4BA8A` (brand-gold-light) — hover state for gold buttons
- **Gold Dark**: `#A8863F` (brand-gold-dark) — active/pressed state for gold
- **Text Primary**: `#1A1714` (brand-warm-black) — headings, body text
- **Text Secondary**: `#6B6560` (brand-warm-gray) — descriptions, captions
- **Text Muted**: `#9B9590` (brand-warm-gray-light) — placeholders, subtle text
- **Border**: `#E5DFD8` (brand-warm-border) — hairline dividers, card borders
- **Dark Sections**: `#2C2C2C` (brand-charcoal) — footer, dark feature sections

## Typography
- **Headings**: Cormorant Garamond (serif), weights 400–600
- **Body/UI**: Inter (sans-serif), weights 300–500
- **Product Names**: UPPERCASE, letter-spacing 0.2em–0.25em, serif
- **Section Headings**: Serif, large (3xl–5xl), tracking wide
- **Body Text**: Sans-serif, text-sm–text-base, tracking normal

## Spacing & Layout
- Generous whitespace — sections have py-20 to py-28
- Max content width: max-w-7xl (1280px)
- Hairline dividers: h-px bg-brand-warm-border
- Card padding: p-6 to p-8
- Grid gaps: gap-6 to gap-8

## Buttons
- **Primary (Gold)**: bg-brand-gold text-white, hover:bg-brand-gold-light, rounded-none, tracking-ultra-wide, uppercase, font-sans, text-xs, px-8 py-3
- **Secondary (Outline)**: border-brand-gold text-brand-gold, hover:bg-brand-gold hover:text-white, same sizing
- **Subtle**: text-brand-warm-gray, hover:text-brand-warm-black, underline on hover

## Cards
- Product cards: bg-white, subtle shadow on hover, image aspect-ratio consistent
- Hover: second image reveal, slight scale, shadow increase
- Quick-add overlay on hover

## Animations
- Subtle, refined transitions (200–300ms ease)
- No bouncy or playful animations
- Fade-in on scroll, gentle hover lifts
- Cart drawer slides from right

## Do's
- Use serif for all headings and product names
- Use generous whitespace between sections
- Keep gold as the single accent color
- Use hairline dividers between sections
- Ensure high contrast for readability

## Don'ts
- Don't use bright or saturated colors
- Don't use playful/bouncy animations
- Don't mix too many font weights
- Don't use rounded buttons (use sharp/slightly rounded)
- Don't use generic e-commerce patterns (badges, countdown timers, etc.)
