# Velmora Fine Jewelry — Design System

## Brand Mood
Quiet luxury, warm, editorial. Premium demi-fine jewelry — NOT loud, NOT discount-looking, NOT generic e-commerce.

## Color Palette
- **Ivory** `#FFFEF9` — Primary background, cards, clean surfaces
- **Cream** `#FAF7F2` — Secondary background, subtle section alternation
- **Warm** `#F5EDE3` — Accent background, hover states, UGC overlay
- **Rose** `#E8D5C4` — Soft accent, testimonial backgrounds
- **Gold** `#C8A96E` — Primary accent, CTAs, highlights, borders
- **Gold Light** `#D4BA8A` — Gold hover state
- **Gold Dark** `#A68B5B` — Gold active/pressed state
- **Charcoal** `#1A1A1A` — Primary text, nav, headings
- **Charcoal Light** `#2D2D2D` — Secondary text
- **Taupe** `#8B7E74` — Muted text, captions, subtle labels
- **Taupe Light** `#B5A99A` — Borders, dividers, disabled states

## Typography
### Headings & Product Names
- Font: Cormorant Garamond (serif)
- Weights: 300 (light), 400 (regular), 500 (medium), 600 (semibold)
- Product names: UPPERCASE, letter-spacing 0.12em
- Section titles: Title case, letter-spacing -0.02em

### Body & UI
- Font: Inter (sans-serif)
- Weights: 400 (regular), 500 (medium), 600 (semibold)
- Button text: UPPERCASE, letter-spacing 0.15em, font-size 0.75rem–0.875rem

## Spacing & Layout
- Max content width: 1440px
- Section padding: 2rem–6rem vertical, responsive horizontal
- Card gap: 1.5rem–2rem
- Generous whitespace between sections (6rem–10rem)

## Components

### Buttons
- `.btn-primary`: Dark charcoal bg, ivory text, hover turns gold
- `.btn-outline`: Charcoal border/text, hover fills charcoal
- `.btn-gold`: Gold bg, charcoal text, hover darkens gold
- All buttons: uppercase, tracked, no border-radius (sharp edges)

### Cards
- No border-radius (editorial feel)
- Subtle shadow on hover
- Image hover: slight scale or crossfade to second image

### Dividers
- Thin hairline (1px) with taupe/20 opacity
- Used between sections and product details

### Navigation
- Transparent over hero, solid on scroll
- Serif logo, sans-serif links
- Sticky with backdrop blur on scroll

## Do's
- Use generous whitespace
- Keep imagery warm-toned and editorial
- Use gold accents sparingly — for CTAs and highlights
- Maintain high contrast for text readability
- Use subtle transitions (300ms ease)

## Don'ts
- Don't use border-radius on buttons or cards (editorial feel)
- Don't use bright or neon colors
- Don't use discount/sale badges prominently
- Don't use generic stock photo aesthetics
- Don't mix too many accent colors
