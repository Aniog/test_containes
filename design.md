# ARTITECT MACHINERY - Design Guidelines

## Brand Identity
- **Brand Name**: ARTITECT MACHINERY
- **Industry**: Industrial sheet metal folding machines
- **Tone**: Elegant, professional, trustworthy, user-friendly

## Color Palette
- **Primary**: `#1a1a2e` (Deep navy/charcoal - sophistication)
- **Secondary**: `#16213e` (Dark blue - industrial trust)
- **Accent**: `#e94560` (Coral red - action/CTA buttons)
- **Accent Secondary**: `#0f3460` (Medium blue - depth)
- **Light**: `#f8f9fa` (Off-white - backgrounds)
- **Text Primary**: `#1a1a2e` (Dark for readability)
- **Text Secondary**: `#6c757d` (Muted gray)
- **Text Light**: `#ffffff` (White for dark backgrounds)
- **Border**: `#e9ecef` (Light gray)

## Typography
- **Headings**: "Playfair Display" (elegant serif) - for brand name and main headings
- **Body**: "DM Sans" (clean sans-serif) - for readability and user-friendliness
- **Font Sizes**:
  - Hero title: 56px (desktop), 36px (mobile)
  - Section titles: 42px (desktop), 28px (mobile)
  - Subheadings: 24px
  - Body: 16px
  - Small: 14px

## Layout & Spacing
- **Max content width**: 1200px
- **Section padding**: 80px vertical (desktop), 48px (mobile)
- **Component spacing**: 24px between elements
- **Card padding**: 32px
- **Border radius**: 8px (cards), 4px (buttons)

## Visual Style
- Clean, minimalist design with generous whitespace
- Subtle shadows for depth: `0 4px 20px rgba(0,0,0,0.08)`
- Smooth transitions: 0.3s ease
- Professional imagery with industrial aesthetic
- Grid-based layouts for products

## Components

### Header
- Fixed position, white background with subtle shadow on scroll
- Logo on left (brand name in Playfair Display)
- Navigation links centered
- CTA button on right ("Get Quote")

### Hero Section
- Full-width with dark gradient overlay
- Background: industrial machinery image
- Headline + subheadline + CTA buttons
- Two buttons: Primary (Get Quote) and Secondary (View Products)

### Products Section
- Section title with decorative underline
- 3-column grid (desktop), 1-column (mobile)
- Product cards with:
  - Icon/illustration
  - Product name
  - Brief description
  - "Learn More" link

### About Section
- Two-column layout (image + text)
- Company description
- Key benefits/features list with checkmarks

### Contact Section
- Dark background for contrast
- Contact form with fields: Name, Email, Phone, Message
- Submit button with loading state

### Footer
- Dark background matching brand
- Logo, navigation links, contact info
- Copyright notice

## Responsive Breakpoints
- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px

## Animations
- Fade-in on scroll for sections
- Hover effects on cards (slight lift + shadow increase)
- Button hover: background color shift
- Smooth scroll between sections
