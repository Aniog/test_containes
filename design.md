# Velmora Fine Jewelry — Design System

## Visual Identity
- Mood: Quiet luxury, warm, editorial. Premium demi-fine jewelry.
- NOT loud, NOT discount-looking, NOT generic e-commerce.

## Color Palette
- **Background**: `#FAF7F2` (warm cream) — primary page background
- **Surface**: `#FFFFFF` (white) — cards, modals, overlays
- **Ivory**: `#F5F0E8` — alternate section backgrounds
- **Sand**: `#E8DFD0` — borders, dividers, subtle backgrounds
- **Gold Accent**: `#C9A96E` — primary accent, buttons, highlights
- **Gold Light**: `#D4BA8A` — hover states, secondary gold
- **Gold Dark**: `#A88B52` — active/pressed gold states
- **Charcoal**: `#2C2C2C` — primary text
- **Warm Gray**: `#6B6560` — secondary text, descriptions
- **Cool Gray**: `#9B9590` — placeholder, disabled text
- **Soft Gray**: `#D4CEC7` — borders, dividers
- **Black**: `#1A1A1A` — headings, strong emphasis

## Typography
- **Headings**: Cormorant Garamond (serif), weights 400-600
- **Body/UI**: Inter (sans-serif), weights 300-500
- **Product names**: UPPERCASE, letter-spacing 0.25em (ultra-wide), serif
- **Section headings**: serif, regular case, generous spacing
- **Nav links**: sans-serif, uppercase, letter-spacing 0.15em (wide), small size

## Spacing
- Section padding: `py-20 md:py-28`
- Container max-width: `max-w-7xl mx-auto px-4 md:px-6 lg:px-8`
- Card gaps: `gap-6 md:gap-8`
- Generous whitespace throughout — never crowded

## Components
- **Buttons**: Solid gold (`bg-brand-gold text-white`) or outlined (`border-brand-gold text-brand-gold`). Rounded-none or rounded-sm. Font-sans, uppercase, wide letter-spacing, medium weight. Hover: darken gold.
- **Dividers**: Thin hairline (`border-t border-brand-soft-gray` or `border-brand-sand`)
- **Cards**: White bg, subtle shadow on hover, no border by default. Image-first layout.
- **Inputs**: Clean, minimal. Border-bottom or thin border. Serif or sans depending on context.
- **Accordions**: Clean, minimal. Plus/minus icon. Serif headings.

## Images
- Large, editorial, warm-lit
- Gold jewelry on dark/neutral backgrounds
- Generous image sizes — hero full-bleed, product cards 4:3 or 3:4
- Hover: subtle scale or second-image reveal

## Animations
- Subtle, smooth transitions (200-400ms ease)
- Hover: opacity, scale(1.02), translate
- Scroll: fade-in, slide-up
- No bouncy or flashy animations

## Do's
- Use generous whitespace
- Use serif for emotional/editorial content
- Use warm cream backgrounds
- Keep product names uppercase with wide spacing
- Use thin hairline dividers

## Don'ts
- Don't use bright/saturated colors
- Don't use rounded/bubbly UI elements
- Don't crowd content
- Don't use sans-serif for headings
- Don't use heavy shadows or gradients
- Don't use generic e-commerce patterns (badges, countdown timers, etc.)
