# Design System — Animal Kingdom

## Visual Style
Nature-inspired, warm and earthy tones with lush greens and deep browns. Clean, modern layout with generous whitespace. Photography-forward design that lets animal images shine.

## Colors
- Primary green: `#2d6a4f` (deep forest green) — `text-[#2d6a4f]`, `bg-[#2d6a4f]`
- Light green: `#52b788` — `text-[#52b788]`
- Accent amber: `#e9c46a` — `text-[#e9c46a]`, `bg-[#e9c46a]`
- Warm brown: `#6b4226` — `text-[#6b4226]`
- Background light: `#f8f5f0` — `bg-[#f8f5f0]`
- Background dark: `#1a2e1a` — `bg-[#1a2e1a]`
- Text primary: `#1c1c1c` — `text-[#1c1c1c]`
- Text muted: `#6b7280` — `text-gray-500`
- White: `#ffffff` — `text-white`, `bg-white`

## Typography
- Font: Inter (loaded via Google Fonts in index.html)
- Hero heading: `text-5xl md:text-7xl font-extrabold`
- Section heading: `text-3xl md:text-4xl font-bold`
- Card title: `text-xl font-semibold`
- Body: `text-base font-normal`
- Caption/label: `text-sm font-medium`

## Spacing
- Section padding: `py-16 md:py-24 px-4 md:px-8`
- Card padding: `p-5 md:p-6`
- Gap between grid items: `gap-6 md:gap-8`

## Borders & Shadows
- Card border radius: `rounded-2xl`
- Card shadow: `shadow-md hover:shadow-xl`
- Image border radius: `rounded-xl`
- Button border radius: `rounded-full`

## Components
- Cards: white background, rounded-2xl, shadow-md, overflow-hidden
- Buttons: rounded-full, px-6 py-3, font-semibold
- Primary button: `bg-[#2d6a4f] text-white hover:bg-[#1b4332]`
- Accent button: `bg-[#e9c46a] text-[#1c1c1c] hover:bg-[#f4a261]`
- Section backgrounds alternate: white and `bg-[#f8f5f0]`
- Hero: full-screen with dark overlay, white text

## Do's
- Use large, high-quality animal images
- Alternate section backgrounds for visual rhythm
- Use green tones for nature/animal theme
- Keep text clearly readable against all backgrounds

## Don'ts
- Don't use dark text on dark backgrounds
- Don't use light text on light backgrounds
- Don't use more than 3 font weights per section
- Don't crowd cards — maintain generous padding
