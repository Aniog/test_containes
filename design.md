# ARTITECT MACHINERY - Design Guidelines

## Brand Identity
- **Brand Name**: ARTITECT MACHINERY
- **Industry**: Industrial machinery - metal folding machines
- **Style**: Elegant but user-friendly — premium industrial feel with approachable design

## Typography
- **Primary Font**: Inter (Google Fonts) — clean, modern, professional
- **Headings**: Inter 700-800 weight, tracking tight
- **Body**: Inter 400 weight, line-height 1.6
- **Accent/Label**: Inter 500-600 weight, uppercase tracking-wide for labels and tags

## Color Palette
- **Primary Dark (Navy)**: #0F172A — headers, hero backgrounds, footer
- **Primary Accent (Steel Blue)**: #2563EB — CTAs, links, highlights
- **Secondary Accent (Gold)**: #D4A853 — premium accents, borders, decorative elements
- **Neutral Light**: #F8FAFC — page backgrounds, card backgrounds
- **Neutral Mid**: #E2E8F0 — borders, dividers
- **Text Primary**: #0F172A — main body text on light backgrounds
- **Text Secondary**: #64748B — secondary/muted text
- **Text on Dark**: #F8FAFC — text on dark backgrounds
- **Success**: #16A34A
- **White**: #FFFFFF

## Spacing
- Section padding: py-20 md:py-28
- Container max-width: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8
- Card padding: p-6 md:p-8
- Element gaps: gap-6 md:gap-8

## Borders & Shadows
- Card border: border border-slate-200
- Card shadow: shadow-sm hover:shadow-md transition
- Gold accent border: border-l-4 border-[#D4A853]
- Rounded corners: rounded-lg for cards, rounded-full for buttons/pills

## Visual Style
- Clean, spacious layouts with generous whitespace
- Subtle gold accents to convey premium quality
- Dark navy sections for contrast and authority
- Professional product imagery with clean backgrounds
- Icons from Lucide React for feature highlights
- Smooth hover transitions on interactive elements

## Do's
- Use navy + gold color combination for premium feel
- Keep layouts clean and uncluttered
- Use clear, readable text with good contrast
- Make CTAs prominent with steel blue
- Use subtle shadows and borders for depth
- Ensure mobile-responsive layouts

## Don'ts
- Don't use overly bright or neon colors
- Don't clutter sections with too much information
- Don't use light text on light backgrounds
- Don't use more than 2 accent colors at once
- Don't use decorative fonts for body text
