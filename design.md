# SSourcing China - Design Guidelines

## Brand Identity
- **Company**: SSourcing China
- **Industry**: B2B Sourcing / Supply Chain
- **Tone**: Professional, trustworthy, clear, practical — no exaggerated claims

## Color Palette

### Primary Colors
- **Navy Blue** (primary): `#1B3A5C` — Trust, authority, professionalism. Used for headings, nav, primary buttons.
- **Ocean Blue** (primary-light): `#2563EB` — Action, links, hover states. Used for CTAs, links, accents.
- **Sky Blue** (primary-50): `#EFF6FF` — Light backgrounds for sections.

### Neutral Colors
- **Slate 900**: `#0F172A` — Body text, dark backgrounds.
- **Slate 700**: `#334155` — Secondary text.
- **Slate 500**: `#64748B` — Muted text, placeholders.
- **Slate 300**: `#CBD5E1` — Borders, dividers.
- **Slate 100**: `#F1F5F9` — Light section backgrounds.
- **White**: `#FFFFFF` — Card backgrounds, main background.

### Accent Colors
- **Amber** (accent): `#F59E0B` — Highlights, badges, trust indicators.
- **Green** (success): `#10B981` — Success states, checkmarks, positive indicators.
- **Red** (alert): `#EF4444` — Warning states, error messages.

## Typography
- **Font Family**: Inter (Google Fonts)
- **Headings**: font-weight 700 (bold), tracking tight
  - H1: 48px / 3rem (desktop), 32px / 2rem (mobile)
  - H2: 36px / 2.25rem (desktop), 28px / 1.75rem (mobile)
  - H3: 24px / 1.5rem (desktop), 20px / 1.25rem (mobile)
- **Body**: font-weight 400, line-height 1.75
  - Base: 16px / 1rem
  - Large: 18px / 1.125rem
- **Small**: 14px / 0.875rem

## Spacing
- Section padding: `py-20` (80px vertical) on desktop, `py-12` on mobile
- Container max-width: `max-w-7xl` (1280px) with `mx-auto px-4 sm:px-6 lg:px-8`
- Card padding: `p-6` to `p-8`
- Element gaps: `gap-6` to `gap-8`

## Component Styles

### Buttons
- **Primary CTA**: `bg-ocean-blue text-white px-8 py-3 rounded-lg font-semibold hover:bg-blue-700 transition-colors`
- **Secondary CTA**: `border-2 border-navy text-navy px-8 py-3 rounded-lg font-semibold hover:bg-navy hover:text-white transition-colors`
- **Ghost**: `text-slate-600 hover:text-navy transition-colors`

### Cards
- White background, subtle shadow: `bg-white rounded-xl shadow-sm border border-slate-100 p-6`
- Hover: `hover:shadow-md transition-shadow`

### Section Headers
- Centered with subtitle: heading in navy, subtitle in slate-500
- Left-aligned for content sections

### Navigation
- Sticky top, white background with subtle shadow
- Logo left, nav links center/right, CTA button right
- Mobile: hamburger menu with slide-out drawer

## Visual Style Rules
- Clean, spacious layouts with generous whitespace
- Professional stock imagery of factories, quality control, shipping, containers
- Icons from Lucide React for service features
- Subtle gradients only for hero sections
- No decorative patterns or busy backgrounds
- Cards and sections with clear visual hierarchy
- Data and statistics prominently displayed

## Do's
- Use navy blue for trust and authority
- Keep layouts clean and scannable
- Use real industry terminology
- Show process steps clearly
- Include trust indicators (certifications, experience, client logos)
- Make CTAs prominent and clear

## Don'ts
- Don't use bright or neon colors
- Don't use cartoon illustrations
- Don't make exaggerated claims ("best in the world", "guaranteed")
- Don't clutter layouts with too many elements
- Don't use dark backgrounds for content sections (only hero/footer)
- Don't use more than 2 font families
