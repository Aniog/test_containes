# ARTITECT MACHINERY Design Guide

## Brand Identity
- Brand: ARTITECT MACHINERY
- Industry: Industrial sheet metal folding machinery
- Style: Elegant, refined, user-friendly, professional
- Mood: Sophisticated industrial, precision engineering, approachable expertise

## Color Palette
- Primary: slate-900 (#0f172a) - Deep navy/charcoal for headers and primary elements
- Primary Light: slate-800 (#1e293b) - For secondary backgrounds
- Accent: amber-600 (#d97706) - Warm gold/copper metallic accent for CTAs and highlights
- Accent Light: amber-50 (#fffbeb) - Light warm tint for feature cards
- Background: white (#ffffff) - Clean white backgrounds
- Surface: gray-50 (#f8fafc) - Light gray for alternate sections
- Text Primary: slate-900 (#0f172a) - Main body text
- Text Secondary: slate-500 (#64748b) - Supporting text
- Border: gray-200 (#e5e7eb) - Subtle borders

## Typography
- Font Family: 'Inter', system-ui, sans-serif (all text)
- Headings: font-semibold or font-bold, tracking-tight
- Body: font-normal, leading-relaxed
- Sizes follow Tailwind scale

## Spacing
- Section padding: py-16 md:py-24
- Container max-width: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8
- Card padding: p-6 md:p-8
- Between sections: space-y-16 md:space-y-24

## Visual Styles
- Cards: bg-white rounded-xl shadow-sm border border-gray-100
- Buttons (Primary): bg-amber-600 hover:bg-amber-700 text-white rounded-lg px-6 py-3 font-medium transition-all
- Buttons (Outline): border-2 border-amber-600 text-amber-600 hover:bg-amber-50 rounded-lg px-6 py-3 font-medium transition-all
- Links: text-amber-600 hover:text-amber-700 underline-offset-2
- Dividers: thin border-t border-gray-100

## Example Tailwind Classes
- Section heading: text-3xl md:text-4xl font-bold text-slate-900 tracking-tight
- Section subtitle: text-lg text-slate-500 max-w-2xl mx-auto
- Card heading: text-xl font-semibold text-slate-900
- Card text: text-slate-500 leading-relaxed

## Do's
- Use ample whitespace
- Use warm gold/amber sparingly as accent
- Keep layouts clean and airy
- Use subtle shadows for depth
- Use professional, precise language
- Ensure high contrast for readability

## Don'ts
- Don't use overly bright or neon colors
- Don't overcrowd sections
- Don't use heavy industrial grunge aesthetics
- Don't use Comic Sans or playful fonts
- Don't use low-contrast text
