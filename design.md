# ARTITECT MACHINERY Design System

## Typography
- Font family: Inter (Google Fonts)
- Headings: font-bold, tracking-tight
- Body: font-normal, leading-relaxed
- Hero title: text-5xl md:text-7xl font-extrabold
- Section title: text-3xl md:text-4xl font-bold
- Subtitle: text-lg md:text-xl text-muted-foreground

## Colors
- Primary: slate-900 (dark backgrounds, headings)
- Primary accent: blue-600 (CTAs, links, highlights)
- Secondary: slate-100 (light backgrounds)
- Background: white
- Muted foreground: slate-500
- Card background: slate-50
- Border: slate-200

## Spacing
- Section padding: py-20 md:py-32
- Container max-width: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8
- Card gap: gap-6 md:gap-8
- Element spacing: space-y-4, space-y-6

## Borders & Shadows
- Cards: rounded-2xl shadow-lg border border-slate-200
- Buttons: rounded-lg
- Images: rounded-xl

## Visual Style
- Clean, industrial elegance
- Generous whitespace
- Subtle gradients for hero sections
- Professional product photography feel
- Smooth hover transitions on interactive elements

## Do's
- Use consistent spacing scale
- Keep text readable with proper contrast
- Use subtle animations on hover
- Maintain professional industrial aesthetic

## Don'ts
- Don't use bright/neon colors
- Don't overcrowd sections
- Don't use decorative fonts
- Don't use low-contrast text
