# Design System

## Visual Style
Inspired by ExampleFile.com — bold geometric shapes, vibrant colors, and a modern brand-focused layout.

## Colors
- Background (blue-indigo): `#5B5EA6` / Tailwind: `bg-[#5B5EA6]`
- Purple accent: `#7B2FBE` / Tailwind: `bg-[#7B2FBE]`
- Cyan accent: `#00D4E8` / Tailwind: `bg-[#00D4E8]`
- Orange accent: `#F05A1A` / Tailwind: `bg-[#F05A1A]`
- White/frosted glass panel: `bg-white/20` with `backdrop-blur`
- Dark text on light surfaces: `text-gray-900`
- Light text on dark surfaces: `text-white`

## Typography
- Font: Inter (Google Fonts)
- Brand name: `text-5xl font-black tracking-tight text-gray-900`
- Tagline: `text-lg font-medium text-gray-700`
- Body: `text-base font-normal`

## Layout
- Full-viewport hero section
- Geometric SVG shapes as decorative background elements
- Centered brand panel with frosted glass effect
- Responsive: stacks gracefully on mobile

## Borders & Shadows
- Glass panel: `rounded-2xl shadow-2xl border border-white/30`
- No hard borders on geometric shapes

## Spacing
- Section padding: `p-8 md:p-16`
- Component gap: `gap-4 md:gap-8`

## Do's
- Use bold, saturated colors for geometric shapes
- Use frosted glass (backdrop-blur + bg-white/20) for overlay panels
- Keep the layout clean and centered
- Use SVG for geometric decorative elements

## Don'ts
- Don't use muted or pastel colors for main shapes
- Don't add too many UI elements — keep it minimal and bold
- Don't use drop shadows on geometric shapes
