# SSourcing China — Design System

## Brand Identity
- **Company:** SSourcing China
- **Tagline:** China Sourcing Agent for Global Buyers
- **Tone:** Professional, clear, practical, trustworthy, international B2B

## Color Palette

### Primary — Deep Navy Blue (trust, professionalism)
- `navy-900`: #0D1B2A — darkest navy, hero backgrounds
- `navy-800`: #1B2E45 — dark sections
- `navy-700`: #1E3A5F — primary brand color
- `navy-600`: #2A4F7C — hover states
- `navy-500`: #3B6FA0 — accents

### Accent — China Red (energy, action)
- `red-600`: #C0392B — CTA buttons, highlights
- `red-500`: #E74C3C — hover CTA

### Neutral
- `slate-50`: #F8FAFC — page background
- `slate-100`: #F1F5F9 — section alternates
- `slate-200`: #E2E8F0 — borders
- `slate-500`: #64748B — secondary text
- `slate-700`: #334155 — body text
- `slate-900`: #0F172A — headings

### Success / Trust
- `green-600`: #16A34A — verified badges, checkmarks

## Typography
- **Font:** Inter (Google Fonts)
- **Headings:** font-bold, tracking-tight, text-slate-900
- **Body:** font-normal, text-slate-700, leading-relaxed
- **Small/Caption:** text-slate-500, text-sm

### Scale
- H1: `text-4xl md:text-5xl lg:text-6xl font-bold`
- H2: `text-3xl md:text-4xl font-bold`
- H3: `text-xl md:text-2xl font-semibold`
- Body: `text-base md:text-lg`
- Small: `text-sm`

## Spacing
- Section padding: `py-16 md:py-24`
- Container: `max-w-7xl mx-auto px-4 sm:px-6 lg:px-8`
- Card padding: `p-6 md:p-8`
- Gap between cards: `gap-6 md:gap-8`

## Components

### Buttons
- Primary CTA: `bg-red-600 hover:bg-red-500 text-white font-semibold px-6 py-3 rounded-lg transition-colors`
- Secondary: `bg-navy-700 hover:bg-navy-600 text-white font-semibold px-6 py-3 rounded-lg transition-colors`
- Outline: `border-2 border-navy-700 text-navy-700 hover:bg-navy-700 hover:text-white font-semibold px-6 py-3 rounded-lg transition-colors`

### Cards
- `bg-white rounded-xl shadow-sm border border-slate-200 p-6 hover:shadow-md transition-shadow`

### Section Headers
- Centered label: `text-red-600 text-sm font-semibold uppercase tracking-widest mb-3`
- Main heading: `text-3xl md:text-4xl font-bold text-slate-900 mb-4`
- Subtext: `text-lg text-slate-600 max-w-2xl mx-auto`

### Navbar
- Background: `bg-white border-b border-slate-200 shadow-sm`
- Links: `text-slate-700 hover:text-navy-700 font-medium`
- Active: `text-navy-700 font-semibold`

### Hero
- Background: dark navy gradient with subtle pattern overlay
- Text: white on dark
- CTA button: red-600

## Do's
- Use navy-700 as the primary brand color for headings and nav
- Use red-600 exclusively for primary CTAs and important highlights
- Keep sections alternating between white and slate-50/slate-100
- Use generous whitespace — B2B buyers need clarity
- Use checkmark icons (green-600) for trust/feature lists
- Keep images realistic: factories, QC inspectors, shipping containers

## Don'ts
- Don't use bright/neon colors
- Don't use dark text on dark backgrounds
- Don't use more than 2 accent colors per section
- Don't use decorative fonts — Inter only
- Don't use exaggerated marketing language
