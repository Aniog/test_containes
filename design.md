# ARTITECT MACHINERY - Design Guidelines

## Brand Identity
- **Brand Name**: ARTITECT MACHINERY
- **Industry**: Industrial Machinery / Sheet Metal Folding Equipment
- **Style**: Elegant but user-friendly — premium industrial feel with clean, approachable design

## Color Palette

### Primary Colors
- **Deep Navy**: `#0F1B2D` — Main backgrounds, hero sections, headers
- **Steel Blue**: `#2E5090` — Primary accent, buttons, links, highlights
- **Silver**: `#C0C8D4` — Secondary text, borders, subtle accents

### Neutral Colors
- **White**: `#FFFFFF` — Card backgrounds, text on dark
- **Light Gray**: `#F4F6F9` — Page backgrounds, section alternation
- **Medium Gray**: `#6B7A8D` — Body text, secondary content
- **Dark Charcoal**: `#1A2332` — Footer, deep backgrounds

### Accent Colors
- **Gold Accent**: `#C8973E` — Premium highlights, CTAs, badges
- **Success Green**: `#2D8A56` — Status indicators

## Typography
- **Headings**: Inter, font-weight 700-800, tracking tight
- **Body**: Inter, font-weight 400-500, tracking normal
- **Accent/Labels**: Inter, font-weight 600, uppercase tracking-wide for small labels

### Scale
- Hero Title: `text-5xl md:text-6xl lg:text-7xl` font-extrabold
- Section Title: `text-3xl md:text-4xl lg:text-5xl` font-bold
- Subsection Title: `text-xl md:text-2xl` font-semibold
- Body: `text-base md:text-lg` font-normal
- Small/Label: `text-sm` font-medium uppercase tracking-wider

## Spacing
- Section padding: `py-20 md:py-28`
- Container max-width: `max-w-7xl mx-auto px-4 sm:px-6 lg:px-8`
- Card padding: `p-6 md:p-8`
- Element gaps: `gap-6 md:gap-8`

## Borders & Shadows
- Card border: `border border-gray-200` or subtle `border border-white/10` on dark
- Card shadow: `shadow-lg` or `shadow-xl`
- Button shadow: `shadow-md hover:shadow-lg`
- Border radius: `rounded-lg` for cards, `rounded-full` for buttons/badges

## Visual Style Rules
- **Do**: Use generous whitespace for an elegant feel
- **Do**: Use dark navy backgrounds for hero and key sections to convey premium quality
- **Do**: Use gold accents sparingly for premium highlights
- **Do**: Use clean, geometric layouts with clear visual hierarchy
- **Do**: Use subtle hover transitions (0.3s ease)
- **Don't**: Use overly bright or saturated colors
- **Don't**: Crowd sections with too much content
- **Don't**: Use decorative fonts or excessive ornamentation
- **Don't**: Use low-contrast text combinations

## Component Patterns
- Navigation: Fixed top, transparent on hero, solid on scroll
- Hero: Full-width with background image, centered text, CTA buttons
- Product Cards: Clean white cards with image, title, description, and link
- Feature Grid: Icon + title + description in 3-column layout
- Contact: Clean form with dark background
- Footer: Dark charcoal with organized link columns
