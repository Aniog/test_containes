# Velmora Fine Jewelry — Design System

## Visual Identity
- **Mood**: Quiet luxury, warm, editorial. Premium demi-fine jewelry — NOT loud, NOT discount-looking.
- **Target**: Women 25–45, gifting and self-purchase, $30–$120 price point.

## Color Palette
- **Base (deep warm charcoal)**: `#1C1917` — primary background for hero/dark sections
- **Cream (warm off-white)**: `#FAF7F2` — page background, light sections
- **Gold Accent**: `#C9A96E` — CTAs, accents, hover states, decorative elements
- **Gold Light**: `#D4BA8A` — secondary gold for subtle accents
- **Text Primary**: `#1C1917` — on light backgrounds
- **Text Secondary**: `#78716C` — muted text, descriptions
- **Text On Dark**: `#FAF7F2` — on dark backgrounds
- **Border**: `#E7E0D6` — hairline dividers, card borders
- **Card BG**: `#FFFFFF` — product cards on cream

## Typography
- **Headings**: Cormorant Garamond (serif), weights 400/500/600
- **Body/UI**: Inter (sans-serif), weights 400/500/600
- **Product names**: UPPERCASE, letter-spacing 0.15em, serif
- **Section headings**: UPPERCASE, letter-spacing 0.2em, smaller size
- **Body text**: 16px base, line-height 1.6

## Spacing & Layout
- Generous whitespace: section padding 80–120px desktop, 48–64px mobile
- Max content width: 1280px
- Product grid gap: 24px
- Hairline dividers: 1px solid border color

## Buttons
- **Primary (accent)**: bg gold, text dark, hover slightly darker gold, tracking-wide uppercase
- **Secondary (outlined)**: border gold, text gold, hover fill gold
- **Pill buttons**: rounded-full for variant selectors
- Transitions: all 300ms ease

## Cards
- Minimal: image + text below, no visible border on cream bg
- Hover: subtle shadow lift, second image reveal
- Product name: serif uppercase tracking-wide

## Shadows
- Subtle only: `0 2px 8px rgba(0,0,0,0.06)` for cards
- `0 4px 16px rgba(0,0,0,0.08)` for elevated elements

## Animations
- Subtle hover transitions (300ms ease)
- Smooth scroll
- Cart drawer slide-in from right
- No bouncy or playful animations

## Do's
- Use generous whitespace
- Use serif for all headings and product names
- Use warm, muted tones
- Keep layouts clean and editorial
- Ensure strong text contrast

## Don'ts
- Don't use bright/saturated colors
- Don't use playful/bouncy animations
- Don't crowd sections with too many elements
- Don't use generic e-commerce patterns (badges, countdown timers)
- Don't use dark text on gold backgrounds (use dark text on light, light text on dark)
