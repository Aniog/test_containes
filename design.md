# SSourcing China - Visual Style Guide

## Brand Identity
- Professional B2B sourcing agency for global buyers
- Trustworthy, international, clean, practical
- No exaggerated claims - factual and straightforward

## Colors
- Primary: Blue (#1e40af) - trust, professionalism
- Primary Light: (#3b82f6) - interactive elements
- Secondary: Slate (#475569) - supporting text
- Accent: Amber (#d97706) - CTAs, highlights
- Background: White (#ffffff) - main backgrounds
- Surface: Gray (#f8fafc) - alternate sections
- Border: Light gray (#e2e8f0)
- Text Primary: Dark slate (#0f172a)
- Text Muted: Slate (#64748b)

## Typography
- Font: Inter (Google Fonts)
- Headings: font-bold, tracking-tight
- Body: font-normal, leading-relaxed
- Size hierarchy: text-4xl → text-3xl → text-2xl → text-xl → text-lg → text-base

## Spacing
- Section padding: py-16 md:py-24
- Container max-width: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8
- Component gap: gap-6 md:gap-8

## Borders
- Cards: rounded-xl border border-gray-100 shadow-sm
- Images: rounded-lg
- Buttons: rounded-lg

## Buttons
- Primary CTA: bg-amber-600 hover:bg-amber-700 text-white font-semibold rounded-lg px-6 py-3
- Secondary: bg-primary hover:bg-primary-dark text-white font-semibold rounded-lg px-6 py-3
- Outline: border-2 border-primary text-primary hover:bg-primary hover:text-white

## Navigation
- Sticky header with white background
- Text links with hover underline effect
- Mobile: hamburger menu

## Visual Style
- Clean, lots of whitespace
- Icons: Lucide React icons
- Professional imagery: factory, QC inspection, shipping, products
- Card-based layouts for services and case studies
- Grid layouts for product categories
