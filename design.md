# Velmora Fine Jewelry — Design System

## Mood
Quiet luxury, warm, editorial. Premium demi-fine jewelry. Not loud, not discount-looking, not generic e-commerce.

## Color Palette
- Background: #FDFBF7 (brand-warm) and #F5F0E8 (brand-cream)
- Text: #1A1A1A (brand-charcoal)
- Accent gold: #C9A96E (brand-gold), #DFC28C (gold-light), #A88B54 (gold-dark)
- Secondary: #B88A7A (brand-rose), #8C7A6B (brand-taupe)
- Light neutrals: #E8E0D0 (brand-cream-dark), #2A2A2A (brand-charcoal-light)

## Typography
- Headings: Cormorant Garamond (serif), generous tracking (0.15em+)
- Product names: Cormorant Garamond, uppercase, 0.25em+ letter-spacing
- Body/UI: Inter (sans-serif)
- Weight hierarchy: Serif 300/400/500/600 for headings, Sans 400/500 for UI

## Layout
- Generous whitespace throughout
- Large editorial imagery
- Thin hairline dividers (1px border-t brand-cream-dark)
- Max-width containers (1280px) with padding
- Mobile-first responsive

## Components
- Buttons: full-width on mobile, pill-like, uppercase tracking-wider
  - btn-primary: Dark bg, white text
  - btn-outline: Border only, dark text
  - btn-accent: Gold bg, white text
- Cards: clean, no borders, image top, thin divider, product name uppercase
- Navigation: transparent over hero, solid on scroll, hairline bottom border
- Cart drawer: slides in from right, overlay

## Interactions
- Subtle opacity/transform transitions (duration-300)
- Image hover zoom (scale-105)
- Smooth scroll behavior
- Cart slide-in animation
- Product card hover with secondary image reveal
