# Velmora Fine Jewelry — Design System

## Visual Direction
Quiet luxury, warm editorial. Premium demi-fine jewelry brand. Not loud, not discount-looking.

## Color Palette
- **Base:** Cream (#FDFBF7) — warm off-white page background
- **Deep:** Espresso (#1C1816) — primary text, deep backgrounds
- **Accent:** Gold (#B8944E) — CTAs, links, accents, subtle highlights
- **Muted:** Warm (#8B8580) — secondary text, dividers, subtle UI
- **Soft Background:** Sand (#F5F1EB) — card backgrounds, section alternates

## Typography
- **Headings/Product Names:** Cormorant Garamond (serif) — uppercase with wide letter-spacing (0.25em)
- **Body/UI:** Inter (sans-serif) — clean, 300–500 weights

## Spacing
- Generous whitespace: sections use py-20 to py-28
- Cards use thin hairline borders (border-cream-200)
- Section dividers: thin 1px lines, warm/cream tones

## Buttons
- Primary: bg-gold text-cream, hover bg-gold-500, subtle transition
- Outline: border-gold text-gold, hover bg-gold text-cream
- All buttons: px-8 py-3, rounded-none (sharp/architectural), tracking-wide uppercase sans-serif text-sm

## Cards
- bg-sand-50 or bg-cream, border border-cream-200
- Subtle shadow on hover (shadow-sm → shadow-md)
- Transition duration-300
