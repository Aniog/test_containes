# ARTITECT MACHINERY - Design Guidelines

## Brand Overview
- **Brand Name**: ARTITECT MACHINERY
- **Industry**: Industrial sheet metal folding machines
- **Products**: Double folding machine, double folder, sheet metal folder, sheet metal folding machine, metal folder, metal folder machine, metal folding machine
- **Style**: Elegant but user-friendly

## Color Palette
- **Primary**: `#1a1a2e` (Deep navy/charcoal - professional, industrial)
- **Secondary**: `#16213e` (Dark blue-gray)
- **Accent**: `#e94560` (Vibrant red - machinery/industrial feel)
- **Accent Secondary**: `#0f3460` (Medium blue)
- **Light**: `#f8f9fa` (Off-white for backgrounds)
- **Text Primary**: `#1a1a2e` (Dark for readability)
- **Text Secondary**: `#6c757d` (Muted gray)
- **Text Light**: `#ffffff` (White for dark backgrounds)
- **Border**: `#dee2e6` (Light gray)

## Typography
- **Primary Font**: "Outfit" (Google Fonts) - Modern, clean, professional
- **Headings**: Outfit, weights 600-700
- **Body**: Outfit, weight 400
- **Logo/Brand**: Outfit, weight 700, uppercase, letter-spacing 0.15em

## Layout & Spacing
- **Max Content Width**: 1280px
- **Section Padding**: 80px vertical (desktop), 48px (mobile)
- **Component Spacing**: 24px between elements
- **Card Padding**: 32px
- **Border Radius**: 12px (cards), 8px (buttons), 4px (small elements)

## Visual Effects
- **Shadows**: Subtle box-shadow for cards: `0 4px 20px rgba(0,0,0,0.08)`
- **Hover Effects**: Smooth transitions (0.3s ease), slight lift on cards
- **Backgrounds**: Clean white/off-white with subtle geometric patterns
- **Hero**: Dark gradient overlay on industrial imagery

## Component Guidelines

### Navigation
- Fixed top navigation
- Logo on left, nav links on right
- Transparent initially, solid on scroll
- Mobile: hamburger menu

### Hero Section
- Full-width, 80vh height
- Dark background with subtle pattern
- Large headline with brand name
- Subheadline with value proposition
- CTA button with accent color

### Product Cards
- White background with subtle shadow
- Product image (placeholder with gradient)
- Product name
- Brief description
- "Learn More" link
- Hover: slight lift and shadow increase

### Features Section
- Icon + title + description format
- 3-4 columns on desktop
- Clean grid layout

### Contact/CTA Section
- Dark background
- Clear call-to-action
- Contact information

### Footer
- Dark background
- Logo, navigation links, contact info
- Copyright

## Do's and Don'ts
- DO: Use clean whitespace, consistent spacing
- DO: Use high-quality imagery (industrial/machinery)
- DO: Keep text concise and scannable
- DON'T: Use too many colors (stick to palette)
- DON'T: Overcrowd sections
- DON'T: Use decorative fonts (keep it professional)
