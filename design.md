# ARTITECT MACHINERY Design System

## Brand Identity
- Name: ARTITECT MACHINERY
- Industry: Industrial sheet-metal folding machinery
- Tone: Precise, trustworthy, premium engineering

## Color Palette
- Background: #0B0F17 (deep navy/black)
- Primary accent: #D4AF37 (brushed gold)
- Secondary accent: #5A7D9A (steel blue)
- Card surface: #131A24
- Muted text: #94A3B8
- Body text: #F1F5F9
- Border: #1E293B
- Success/CTA hover: #C49B2C

## Typography
- Headings: Inter, 600-800 weight, tight tracking
- Body: Inter, 400-500 weight, line-height 1.6
- Hero title: clamp(2.5rem, 5vw, 4.5rem)
- Section title: clamp(1.75rem, 3vw, 2.5rem)

## Spacing
- Section vertical padding: py-20 md:py-28
- Container max-width: max-w-7xl
- Card padding: p-6 md:p-8
- Grid gaps: gap-6 md:gap-8

## Components
- Buttons: rounded-lg, px-6 py-3, bg-[#D4AF37], text-[#0B0F17], hover:bg-[#C49B2C]
- Cards: bg-[#131A24], border border-[#1E293B], rounded-2xl, hover:border-[#D4AF37]/40
- Links: text-[#F1F5F9], hover:text-[#D4AF37]
- Inputs: bg-[#131A24], border border-[#1E293B], rounded-lg, focus:border-[#D4AF37]

## Elevation
- Subtle shadow: shadow-lg shadow-black/20
- Card hover: -translate-y-1 transition-transform

## Do's and Don'ts
- Do use generous whitespace to convey premium quality.
- Do use gold only for accents, CTAs, and highlights.
- Don't use bright or playful colors.
- Don't overcrowd sections; keep layouts clean and grid-based.
