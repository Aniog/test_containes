# Velmora Fine Jewelry - Design System

## Visual Identity

### Brand Philosophy
- **Mood**: Quiet luxury, warm, editorial
- **Positioning**: Premium demi-fine jewelry — NOT loud, NOT discount-looking, NOT generic e-commerce
- **Target**: Women 25-45, gifting and self-purchase, $30–$120 price point

### Color Palette

**Primary Colors**
- `cream`: #FAF7F2 (warm off-white background)
- `ivory`: #F5F0E8 (secondary background)
- `charcoal`: #1A1A1A (primary text, dark)
- `warm-black`: #2C2825 (soft black for headings)

**Accent Colors**
- `champagne`: #D4AF37 (gold accent - primary CTA)
- `antique-gold`: #B8964C (hover states)
- `rose-gold`: #B76E79 (subtle accent for variety)

**Neutral Colors**
- `stone`: #8B8580 (secondary text)
- `pebble`: #E8E4DE (borders, dividers)
- `mist`: #F0EDE8 (card backgrounds)

### Typography

**Headings & Product Names**
- Primary: `Cormorant Garamond` (serif)
- Weights: 400 (regular), 500 (medium), 600 (semibold)
- Letter-spacing: 0.08em for uppercase product names
- Line-height: 1.2

**Body & UI**
- Primary: `Manrope` (sans-serif)
- Weights: 400 (regular), 500 (medium), 600 (semibold)
- Letter-spacing: 0.01em
- Line-height: 1.6

### Spacing System
- Base unit: 4px
- Spacing scale: 4, 8, 12, 16, 24, 32, 48, 64, 96, 128px
- Section padding: 96px (desktop), 64px (tablet), 48px (mobile)
- Container max-width: 1400px

### Visual Effects

**Shadows**
- `shadow-soft`: 0 4px 20px rgba(44, 40, 37, 0.06)
- `shadow-hover`: 0 8px 32px rgba(44, 40, 37, 0.1)
- `shadow-card`: 0 2px 12px rgba(44, 40, 37, 0.04)

**Borders**
- Hairline dividers: 1px solid #E8E4DE
- Border radius: 0px (sharp, editorial) or 2px (subtle)

**Animations**
- Default transition: 0.3s ease
- Hover transitions: 0.25s ease-out
- Page transitions: fade 0.4s ease

### Component Guidelines

**Buttons**
- Primary: Solid champagne (#D4AF37), white text, uppercase, letter-spacing 0.1em
- Secondary: Outlined with champagne border, transparent background
- Height: 48px (default), 56px (large)
- Hover: Darken to antique-gold (#B8964C)

**Cards**
- Background: #F0EDE8 (mist)
- Padding: 24px
- Hover: Lift with shadow-hover, reveal second image

**Form Inputs**
- Border: 1px solid #E8E4DE
- Focus: 2px solid #D4AF37
- Height: 48px
- Background: white

### Responsive Breakpoints
- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px

### Do's and Don'ts

**DO:**
- Use generous whitespace
- Keep imagery large and editorial
- Use thin hairline dividers
- Apply subtle hover transitions
- Keep accent color restrained
- Use uppercase with wide letter-spacing for product names

**DON'T:**
- Use loud, bright colors
- Create discount-looking layouts
- Use generic e-commerce patterns
- Overcrowd content
- Use heavy shadows or borders
- Mix too many font weights
