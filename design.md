# ARTITECT MACHINERY - Design System

## Brand Identity
- **Brand**: ARTITECT MACHINERY
- **Industry**: Industrial machinery / Sheet metal fabrication
- **Personality**: Professional, authoritative, precision-focused, modern elegance

## Color Palette
- **Primary (Deep Charcoal)**: `#1a1a2e` - Main brand color, used for headers, hero backgrounds
- **Secondary (Steel Blue)**: `#16213e` - Accent backgrounds, secondary elements
- **Accent (Copper/Gold)**: `#c9a959` - CTA buttons, highlights, accent lines, premium feel
- **Surface Light**: `#f8f7f4` - Page background, light sections
- **Surface Warm**: `#eef0f2` - Alternate section backgrounds
- **Text Primary**: `#1a1a2e` - Main body text on light backgrounds
- **Text Secondary**: `#5a6275` - Subtitles, descriptions, captions
- **Text Light**: `#f8f7f4` - Text on dark backgrounds
- **Border**: `#d4d8e0` - Subtle dividers and borders
- **Success**: `#2d7a4f` - Success states
- **Error**: `#c53030` - Error states

## Typography
- **Headings**: Space Grotesk (400-700 weight) - Industrial, geometric feel
- **Body**: Inter (300-700 weight) - Clean, readable
- **Display**: Space Grotesk 700-900 weight, uppercase with letter-spacing

## Spacing Scale
- Section padding: py-20 lg:py-28
- Container max-width: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8
- Card padding: p-6 lg:p-8
- Component gap: gap-6 lg:gap-8

## Component Styles
- **Cards**: White bg, subtle shadow (shadow-sm), rounded-lg, hover shadow transition
- **Buttons Primary**: bg-accent text-dark, rounded-md, font-semibold, hover brighten
- **Buttons Secondary**: border-2 border-accent text-accent, bg-transparent
- **Section Headers**: Space Grotesk, uppercase tracking-wide, accent underline
- **Hero**: Full-width, dark bg, large typography, subtle gradient overlay

## Visual Style
- Clean, modern industrial aesthetic
- Generous white space
- Sharp lines with subtle rounded corners (rounded-lg)
- Professional photography of machinery
- Subtle animations on scroll (fade-in)
- Copper/gold accent creates premium industrial feel

## Do's
- Use generous whitespace
- Maintain consistent dark/light section alternation
- Use accent color sparingly for maximum impact
- Keep typography hierarchy clear and bold
- Use shadcn UI components

## Don'ts
- Don't use overly bright or saturated colors
- Don't use more than 3 font sizes per section
- Don't crowd elements together
- Don't use decorative borders or excessive ornamentation
