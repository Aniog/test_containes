# Yoga Shop Membership Website Design System

## Visual Style
A calming, zen-inspired design that reflects the peaceful nature of yoga practice.

## Colors
- **Primary**: Sage Green (`#87A96B`) - calming, natural
- **Secondary**: Warm Beige (`#F5F1E8`) - soft, welcoming
- **Accent**: Deep Teal (`#2D5A5A`) - grounding, stability
- **Text Primary**: Charcoal (`#2C3E50`) - readable, professional
- **Text Secondary**: Medium Gray (`#6B7280`) - subtle information
- **Background**: Off White (`#FEFEFE`) - clean, spacious
- **Success**: Soft Green (`#10B981`) - positive actions
- **Warning**: Warm Orange (`#F59E0B`) - attention
- **Error**: Coral Red (`#EF4444`) - alerts

## Typography
- **Headings**: Inter font family, weights 600-800
- **Body**: Inter font family, weights 400-500
- **Accent**: Inter font family, weight 300 for quotes/testimonials

## Spacing
- **Base unit**: 4px (0.25rem)
- **Small**: 8px (0.5rem)
- **Medium**: 16px (1rem)
- **Large**: 24px (1.5rem)
- **XL**: 32px (2rem)
- **XXL**: 48px (3rem)

## Borders & Shadows
- **Border radius**: 8px for cards, 4px for buttons
- **Border color**: Light Gray (`#E5E7EB`)
- **Shadow**: Soft, subtle shadows using `shadow-sm` and `shadow-md`

## Components Style
- **Cards**: White background, subtle shadow, rounded corners
- **Buttons**: Rounded, medium padding, hover states
- **Forms**: Clean inputs with focus states, proper spacing
- **Navigation**: Minimal, clear hierarchy

## Layout
- **Container**: Max width 1200px, centered
- **Grid**: 12-column responsive grid
- **Breakpoints**: Mobile-first approach using Tailwind breakpoints

## Do's
- Use plenty of whitespace for breathing room
- Maintain consistent spacing using the base unit system
- Use the sage green primary color for CTAs and important elements
- Keep text readable with proper contrast ratios
- Use soft, rounded corners throughout

## Don'ts
- Don't use harsh, bright colors that clash with the zen aesthetic
- Don't overcrowd elements - maintain spacious layouts
- Don't use more than 3 colors in a single component
- Don't use heavy, bold typography for body text
- Don't create layouts that feel cramped or busy

## Example Tailwind Classes
- Primary button: `bg-sage-600 hover:bg-sage-700 text-white`
- Card: `bg-white rounded-lg shadow-md border border-gray-200`
- Heading: `text-charcoal-800 font-semibold`
- Body text: `text-gray-600 font-normal`
