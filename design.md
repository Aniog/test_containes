# SSourcing China - Visual Design Guide

## Brand Colors
- Primary: `#1a365d` (dark navy blue) - trust, professionalism
- Primary Light: `#2b6cb0` (medium blue) - CTAs, links
- Accent: `#c53030` (deep red) - highlights, urgency
- Background: `#ffffff` (white) - clean, open
- Section BG: `#f7fafc` (very light gray) - alternating sections
- Text: `#1a202c` (near black) - body text
- Text Muted: `#718096` (gray) - secondary text
- Border: `#e2e8f0` (light gray) - cards, dividers
- Success: `#276749` (green) - trust badges, verification

## Typography
- Font: Inter (sans-serif)
- H1: 3rem/3.5rem, font-bold, tracking-tight (#1a202c)
- H2: 2rem/2.5rem, font-bold (#1a202c)
- H3: 1.25rem/1.75rem, font-semibold (#1a202c)
- Body: 1rem/1.625rem, font-normal (#4a5568)
- Small: 0.875rem/1.25rem (#718096)

## Spacing
- Section padding: py-16 md:py-24
- Container max-width: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8
- Card padding: p-6 md:p-8
- Gap between cards: gap-6 md:gap-8
- Element spacing: space-y-4 within sections, space-y-16 between sections

## Visual Style
- Cards: white bg, rounded-lg border, subtle shadow-sm
- Buttons:
  - Primary CTA: bg-primary-light text-white, px-8 py-3 rounded-lg, hover:bg-primary
  - Secondary: bg-white text-primary-light border, hover:bg-gray-50
- Section backgrounds: alternate between white and gray-50
- Icons: Lucide React, 5-6rem size for feature icons, 1.25rem for inline
- Images: use data-strk-img attributes with realistic factory/QC/shipping visuals
- Dividers: thin border-b border-gray-100 between sections

## Layout
- Navigation: fixed top, white bg, border-bottom, logo left, links right, CTA button
- Footer: dark navy bg, white text, 4-column grid, social links, copyright
- Grid: 1-col mobile, 2-col tablet, 3-col desktop for feature grids
- Process steps: horizontal on desktop, vertical on mobile, numbered steps

## SEO
- H1 on each page, unique meta descriptions
- Semantic HTML structure
- Alt text for all images
- Clean URLs matching page names

## Do's and Don'ts
- DO use clean, professional imagery of factories, QC inspectors, shipping
- DO use trust signals: badges, certifications, number counts
- DO keep copy concise and benefit-focused
- DON'T use exaggerated claims or "guaranteed" language
- DON'T use overly bright colors or playful design
- DON'T use generic stock photos of people in suits
- DON'T use cluttered layouts or excessive animations
