# Velmora Fine Jewelry — Design System

## Visual Identity
- Mood: quiet luxury, warm, editorial. Premium demi-fine jewelry — NOT loud, NOT discount-looking.
- Target: women 25–45, gifting and self-purchase, $30–$120 price point.

## Color Palette
- **Background**: `#FAF7F2` (brand-cream) — warm off-white, primary page background
- **Warm accent bg**: `#F5EDE3` (brand-warm) — subtle section backgrounds
- **Sand**: `#E8DDD0` (brand-sand) — borders, dividers, hover states
- **Gold accent**: `#C9A96E` (brand-gold) — primary accent, buttons, links, highlights
- **Gold light**: `#D4BA8A` (brand-gold-light) — hover states on gold elements
- **Gold dark**: `#A88B52` (brand-gold-dark) — active/pressed gold states
- **Charcoal**: `#2C2C2C` (brand-charcoal) — primary text color
- **Charcoal light**: `#3D3D3D` (brand-charcoal-light) — secondary text
- **Espresso**: `#1A1A1A` (brand-espresso) — darkest, hero overlays, footer
- **Ivory**: `#FFFEFA` (brand-ivory) — text on dark backgrounds
- **Muted**: `#8A8278` (brand-muted) — tertiary text, captions
- **Muted light**: `#B5AFA7` (brand-muted-light) — placeholders, disabled states

## Typography
- **Headings**: Cormorant Garamond (serif), weights 400–600
- **Body/UI**: Inter (sans-serif), weights 300–500
- **Product names**: UPPERCASE, letter-spacing 0.2em–0.25em, serif font
- **Section headings**: serif, larger sizes, normal case
- **Body text**: 14px–16px, line-height 1.6–1.8

## Spacing & Layout
- Generous whitespace throughout
- Max content width: 1280px
- Section padding: 80px–120px vertical (desktop), 48px–64px (mobile)
- Grid gaps: 24px–32px
- Thin hairline dividers: 1px, brand-sand color

## Buttons
- **Primary (accent)**: bg-brand-gold, text-brand-ivory, hover:bg-brand-gold-light, rounded-none or subtle rounded, uppercase, wide letter-spacing, serif or sans depending on context
- **Secondary (outlined)**: border-brand-gold, text-brand-gold, hover:bg-brand-gold hover:text-brand-ivory
- **Ghost**: text-brand-charcoal, hover:text-brand-gold, underline on hover

## Cards
- Minimal: no visible border or shadow by default
- Hover: subtle shadow or slight scale, second image reveal
- Product cards: image top, name + price below, clean spacing

## Images
- Large, editorial, warm-lit
- Jewelry on model or styled flat-lays
- Warm gold tones on dark/neutral backgrounds
- Aspect ratios: 3x4 for product, 16x9 for hero, 9x16 for UGC/reels

## Animations
- Subtle, smooth transitions (200–400ms ease)
- Fade-in on scroll
- Hover: gentle scale (1.02), shadow lift
- No bouncy or playful animations

## Do's
- Use serif for all headings and product names
- Keep generous whitespace
- Use gold accent sparingly and intentionally
- Ensure strong contrast (charcoal on cream, ivory on espresso)
- Use thin hairline dividers between sections

## Don'ts
- Don't use bright/saturated colors
- Don't use rounded/bubbly UI elements
- Don't use playful or bouncy animations
- Don't crowd elements — let them breathe
- Don't use generic e-commerce patterns (badges, countdown timers, etc.)
