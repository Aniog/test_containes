# Velmora Fine Jewelry — Design System

## Mood
Quiet luxury, warm, editorial. Premium demi-fine jewelry. NOT loud, NOT discount-looking.

## Color Palette
- **Background (cream):** `#FAF7F2` — warm off-white page base
- **Foreground (charcoal):** `#1A1A1A` — primary text
- **Accent (gold):** `#B8860B` — warm dark gold for CTAs, highlights
- **Accent hover:** `#9A7209` — deeper gold on hover
- **Muted:** `#F5F0E8` — subtle warm backgrounds, cards
- **Muted foreground:** `#6B5E4F` — secondary text, captions
- **Border:** `#E8E0D4` — hairline dividers, card borders
- **Dark surface:** `#1A1A1A` — footer, dark sections
- **Dark surface text:** `#FAF7F2` — text on dark surfaces

## Typography
- **Headings / Product names:** `Cormorant Garamond`, serif — elegant, editorial
- **Body / UI:** `Inter`, sans-serif — clean, modern
- **Product names:** UPPERCASE, letter-spacing: 0.12em
- **Heading sizes:** Hero: text-5xl md:text-7xl; Section: text-3xl md:text-4xl; Card: text-lg
- **Body:** text-sm md:text-base, leading-relaxed

## Spacing & Layout
- Generous whitespace: sections py-16 md:py-24
- Container max-w-7xl mx-auto px-4 md:px-8
- Card gaps: gap-4 md:gap-6
- Thin hairline dividers: border-t border-border (1px)

## Components
- **Buttons (primary):** bg-accent text-white px-8 py-3 text-sm uppercase tracking-widest hover:bg-accent-hover transition-colors
- **Buttons (outlined):** border border-accent text-accent px-8 py-3 text-sm uppercase tracking-widest hover:bg-accent hover:text-white transition-colors
- **Cards:** bg-white rounded-none border border-border overflow-hidden
- **Hover transitions:** transition-all duration-300 ease-in-out
- **Shadows:** shadow-none or shadow-sm only

## Do's
- Use generous whitespace
- Keep imagery large and editorial
- Use thin hairline dividers
- Subtle hover transitions (opacity, translate-y)
- Restrained accent color usage

## Don'ts
- No rounded corners on cards (sharp/editorial)
- No heavy shadows or gradients
- No bright/loud colors
- No cluttered layouts
- No generic e-commerce feel
