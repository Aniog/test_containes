# Velmora Fine Jewelry - Design System

## Brand Identity
- **Brand Name**: Velmora Fine Jewelry
- **Positioning**: Premium demi-fine jewelry, quiet luxury, editorial aesthetic
- **Target Audience**: Women 25-45, gifting and self-purchase
- **Price Range**: $30–$120 (premium-but-accessible)

## Visual Identity

### Color Palette
- **Primary Background**: `#FAF9F7` (warm off-white/cream)
- **Secondary Background**: `#1A1814` (deep warm charcoal/brown-black)
- **Text Primary**: `#1A1814` (deep warm charcoal)
- **Text Secondary**: `#6B635A` (warm taupe/bronze)
- **Text on Dark**: `#FAF9F7` (cream)
- **Accent Gold**: `#C9A962` (warm muted gold)
- **Accent Gold Hover**: `#B8954F` (deeper gold)
- **Border/Divider**: `#E8E4DF` (warm light gray)
- **Card Background**: `#FFFFFF` (pure white)

### Typography
- **Headings**: Cormorant Garamond (serif)
  - H1: 56px / 64px line-height / font-weight 500
  - H2: 40px / 48px line-height / font-weight 500
  - H3: 28px / 36px line-height / font-weight 500
  - Product Names: 18px / uppercase / 0.15em letter-spacing / font-weight 500
- **Body/UI**: Manrope (sans-serif)
  - Body: 16px / 26px line-height / font-weight 400
  - Small: 14px / 22px line-height
  - Button: 14px / uppercase / 0.1em letter-spacing / font-weight 600

### Spacing System
- **Section Padding**: 80px vertical (desktop), 48px (mobile)
- **Container Max Width**: 1280px
- **Grid Gap**: 24px (desktop), 16px (mobile)
- **Component Padding**: 24px

### Visual Effects
- **Shadows**: 
  - Card hover: `0 20px 40px rgba(26, 24, 20, 0.08)`
  - Button: `0 4px 12px rgba(201, 169, 98, 0.25)`
- **Border Radius**: 0px (sharp, editorial) for buttons; 4px for cards
- **Dividers**: 1px solid #E8E4DF (hairline)
- **Transitions**: 0.3s ease for all interactive elements

## Component Guidelines

### Buttons
- **Primary**: 
  - Background: #C9A962 (gold)
  - Text: #1A1814 (dark)
  - Padding: 16px 32px
  - Uppercase, letter-spacing 0.1em
  - Hover: darken background, subtle shadow
- **Secondary/Outline**:
  - Border: 1px solid #1A1814
  - Text: #1A1814
  - Background: transparent
  - Hover: background #1A1814, text #FAF9F7

### Product Cards
- White background
- Sharp corners (0px radius)
- Image with aspect ratio 4:5
- Product name: uppercase, wide spacing
- Price in body font
- Hover: reveal second image, show "Add to Cart" overlay

### Navigation
- Transparent over hero, solid #FAF9F7 on scroll
- Logo: serif, uppercase, letter-spacing 0.2em
- Links: sans-serif, 14px, uppercase, 0.08em spacing

### Forms
- Input: 1px border #E8E4DF, 16px padding
- Focus: border-color #C9A962
- Placeholder: #6B635A

## Responsive Breakpoints
- **Mobile**: < 640px
- **Tablet**: 640px - 1024px
- **Desktop**: > 1024px

## Do's and Don'ts
- DO: Use generous whitespace, elegant typography, subtle animations
- DO: Keep imagery warm and editorial
- DON'T: Use loud discount styling or generic e-commerce look
- DON'T: Use bright/neon accent colors
- DON'T: Use rounded corners on buttons (keep sharp/editorial)
