# Design Guidelines

## Typography
- Font: Inter (Google Fonts), loaded via index.html
- Headings: font-bold, tracking-tight
- Body: font-normal, text-base leading-relaxed

## Colors
- Primary: indigo-600 (#4F46E5) for buttons, links, accents
- Primary hover: indigo-700
- Background: white (#FFFFFF) for cards, gray-50 (#F9FAFB) for page
- Text: gray-900 for headings, gray-600 for body
- Muted: gray-400 for secondary text
- Success: emerald-600 for success states
- Error: red-500 for error states

## Spacing
- Section padding: py-16 md:py-24
- Container max-width: max-w-4xl, centered with mx-auto
- Card padding: p-6 md:p-8
- Form input spacing: gap-5

## Borders & Shadows
- Cards: rounded-xl shadow-sm border border-gray-100
- Inputs: rounded-lg border border-gray-200 focus:ring-2 focus:ring-indigo-500 focus:border-indigo-500
- Buttons: rounded-lg

## Do's
- Use white cards on light gray backgrounds
- Use clear, readable text contrast
- Keep forms simple and focused
- Use consistent spacing throughout

## Don'ts
- Don't use dark backgrounds for text-heavy areas
- Don't use low-contrast text
- Don't overcrowd the layout
