# Velmora Fine Jewelry — Design System

## Visual Identity
- **Mood**: Quiet luxury, warm, editorial. Premium demi-fine jewelry — NOT loud, NOT discount-looking.
- **Target**: Women 25–45, gifting and self-purchase, $30–$120 price point.

## Color Palette
- **Background**: `#FAF7F2` (warm cream) — primary page background
- **Warm accent**: `#F5EDE3` (warm beige) — section backgrounds, cards
- **Sand**: `#E8DFD1` — borders, dividers, subtle backgrounds
- **Gold**: `#C9A96E` — primary accent, buttons, highlights
- **Gold Light**: `#D4BA8A` — hover states
- **Gold Dark**: `#A8863F` — active/pressed states
- **Charcoal**: `#2C2C2C` — primary text
- **Charcoal Light**: `#3D3D3D` — secondary text
- **Espresso**: `#1A1612` — hero overlays, deep backgrounds
- **Warm Gray**: `#8A8279` — muted text, placeholders
- **Muted**: `#6B6560` — captions, helper text

## Typography
- **Headings**: Cormorant Garamond (serif), uppercase, letter-spacing 0.25em
- **Body**: Inter (sans-serif), regular weight, normal tracking
- **Product names**: Cormorant Garamond, uppercase, letter-spacing 0.2em
- **Buttons/Labels**: Inter, uppercase, letter-spacing 0.2em, text-xs

## Spacing
- Section padding: `py-20 md:py-28`
- Container max-width: `max-w-7xl mx-auto px-5 md:px-8`
- Card gaps: `gap-6 md:gap-8`
- Generous whitespace throughout

## Components
- **Buttons**: Solid gold bg or gold outline, uppercase, wide tracking, rounded-none, hover transitions
- **Dividers**: 1px border in sand color (`border-brand-sand`)
- **Cards**: White or cream bg, subtle shadow on hover, no border radius (sharp corners for editorial feel)
- **Images**: Full-bleed hero, editorial aspect ratios, hover reveals second image

## Do's
- Use generous whitespace
- Use serif for all headings and product names
- Use uppercase with wide letter-spacing for product names
- Use warm, gold-toned accents sparingly
- Ensure strong text contrast on all backgrounds
- Use thin hairline dividers between sections

## Don'ts
- Don't use bright/saturated colors
- Don't use rounded corners on cards or buttons (editorial, not playful)
- Don't use heavy shadows
- Don't mix too many font styles
- Don't use generic e-commerce patterns (no starburst badges, no aggressive CTAs)
