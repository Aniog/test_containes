# ARTITECT MACHINERY Design System

## Visual direction
ARTITECT MACHINERY should feel elegant, precise, industrial, and user friendly. The design should balance premium engineering confidence with clear, approachable content.

## Typography
- Use Inter as the primary font for all interface and marketing text.
- Headlines should be bold, compact, and confident with tracking tightened slightly where appropriate.
- Body copy should stay readable with generous line height.
- Example Tailwind classes: `font-sans`, `text-4xl`, `md:text-6xl`, `font-semibold`, `leading-tight`, `tracking-tight`.

## Colors
- Primary surface: deep slate and zinc tones for a premium industrial look.
- Accent: amber for precision highlights, calls to action, and key metrics.
- Supporting surfaces: white, slate-50, slate-100, and slate-900.
- Text must always be high contrast. Use `text-white` on dark surfaces and `text-slate-950` or `text-slate-700` on light surfaces.
- Example Tailwind classes: `bg-slate-950`, `bg-zinc-900`, `bg-slate-50`, `text-white`, `text-slate-950`, `text-slate-700`, `text-amber-400`, `bg-amber-400`.

## Layout and spacing
- Use spacious sections with clear hierarchy and strong alignment.
- Desktop layouts should use multi-column grids where useful; mobile layouts should stack cleanly.
- Example Tailwind classes: `max-w-7xl`, `mx-auto`, `px-6`, `lg:px-8`, `py-20`, `grid`, `lg:grid-cols-2`, `gap-10`.

## Components
- Cards should use subtle borders, rounded corners, and soft shadows.
- Navigation should be simple, sticky-feeling visually, and easy to scan.
- Buttons should be large, rounded, clear, and high contrast.
- Example Tailwind classes: `rounded-3xl`, `border`, `border-slate-200`, `shadow-xl`, `hover:-translate-y-1`, `transition`.

## Imagery
- Use refined industrial imagery for folding machines, sheet metal fabrication, workshop precision, and engineered metal components.
- Image areas should have rounded corners and subtle overlays so text remains readable.

## Do's
- Keep copy concise and confidence-building.
- Show product categories clearly: double folding machine, double folder, sheet metal folder, sheet metal folding machine, metal folder, metal folder machine, and metal folding machine.
- Maintain strong contrast and accessible text sizes.

## Don'ts
- Do not use low-contrast grey text on dark backgrounds.
- Do not crowd mobile screens with too many columns.
- Do not use flashy effects that distract from product credibility.
