# ARTITECT MACHINERY Visual Style

## Brand feeling
Elegant, precise, industrial, and user friendly. The site should feel like a premium manufacturing partner: engineered, trustworthy, and easy to navigate.

## Colors
- Primary dark: deep slate/charcoal surfaces for premium industrial contrast. Example Tailwind: `bg-slate-950`, `text-slate-950`.
- Secondary surface: clean white and warm light gray for readable cards. Example Tailwind: `bg-white`, `bg-slate-50`.
- Accent metal: amber/gold highlights for refined machinery details. Example Tailwind: `text-amber-500`, `bg-amber-500`, `border-amber-400`.
- Support tones: slate grays for labels and descriptive text. Example Tailwind: `text-slate-600`, `text-slate-400`.

## Typography
- Use Inter for all text, with strong headings and clear body copy.
- Headings should be bold, compact, and confident. Example Tailwind: `font-bold tracking-tight`.
- Labels and badges should use uppercase tracking for a technical feel. Example Tailwind: `uppercase tracking-[0.18em]`.

## Layout
- Desktop layouts should use generous two-column sections and large hero spacing.
- Mobile layouts should stack clearly with comfortable padding.
- Use rounded cards, subtle borders, and restrained shadows.

## Components
- Buttons: high contrast, rounded-full, clear hover states.
- Cards: white or translucent dark cards with explicit readable text colors.
- Product cards: show names, use cases, and feature bullets in a scannable layout.
- CTAs: direct, clear, and friendly.

## Do
- Keep every visible text element readable with explicit foreground colors.
- Use professional industrial imagery through the Strikingly image tagging system where appropriate.
- Maintain consistent spacing and alignment.

## Don’t
- Do not use low-contrast text on dark or image backgrounds.
- Do not use overly playful colors or decorative fonts.
- Do not overcrowd the homepage with dense technical copy.
