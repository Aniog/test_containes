# SSourcing China - Design System

## Brand Identity
- **Company**: SSourcing China
- **Industry**: B2B Sourcing / Supply Chain
- **Tone**: Professional, trustworthy, practical, international

## Color Palette
- **Primary**: `#1E40AF` (Deep Blue) - Trust, professionalism, corporate
- **Primary Dark**: `#1E3A8A` - Hover states
- **Primary Light**: `#3B82F6` - Accents, links
- **Secondary**: `#F59E0B` (Amber) - CTAs, highlights, urgency
- **Secondary Dark**: `#D97706` - Hover state for CTAs
- **Neutral 900**: `#0F172A` - Headings, primary text
- **Neutral 700**: `#334155` - Body text
- **Neutral 500**: `#64748B` - Secondary text, captions
- **Neutral 200**: `#E2E8F0` - Borders, dividers
- **Neutral 100**: `#F1F5F9` - Section backgrounds (alternating)
- **Neutral 50**: `#F8FAFC` - Light backgrounds
- **White**: `#FFFFFF` - Card backgrounds, main bg
- **Success**: `#059669` - Success states
- **Error**: `#DC2626` - Error states

## Typography
- **Font Family**: Inter (Google Fonts)
- **H1**: `text-4xl md:text-5xl lg:text-6xl font-extrabold`
- **H2**: `text-3xl md:text-4xl font-bold`
- **H3**: `text-xl md:text-2xl font-semibold`
- **Body**: `text-base md:text-lg leading-relaxed`
- **Small**: `text-sm`
- **Lead**: `text-lg md:text-xl text-neutral-500`

## Spacing
- **Section padding**: `py-16 md:py-24`
- **Container**: `max-w-7xl mx-auto px-4 sm:px-6 lg:px-8`
- **Card padding**: `p-6 md:p-8`
- **Grid gap**: `gap-6 md:gap-8`

## Components
- **Primary Button**: `bg-primary hover:bg-primary-dark text-white font-semibold px-6 py-3 rounded-lg transition-all shadow-md hover:shadow-lg`
- **Secondary Button**: `bg-secondary hover:bg-secondary-dark text-white font-semibold px-6 py-3 rounded-lg`
- **Outline Button**: `border-2 border-primary text-primary hover:bg-primary hover:text-white rounded-lg px-6 py-3`
- **Card**: `bg-white rounded-xl shadow-sm hover:shadow-md transition-shadow p-6 md:p-8 border border-neutral-200`
- **Badge**: `bg-primary/10 text-primary text-sm font-medium px-3 py-1 rounded-full`

## Do's
- Use whitespace generously for a clean, international feel
- Use real stock images for factory, QC, and shipping visuals
- Keep text concise and practical
- Use consistent iconography from Lucide React
- Maintain strong visual hierarchy

## Don'ts
- Don't use exaggerated claims or hype language
- Don't use overly decorative or playful design elements
- Don't mix more than 2 accent colors
- Don't use low-contrast text combinations
- Don't crowd sections - B2B audiences prefer clarity
