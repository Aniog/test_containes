# ARTITECT MACHINERY - Design System

## Brand Overview
Industrial machinery brand specializing in metal folding equipment. Design should feel premium, trustworthy, and sophisticated while remaining approachable and easy to navigate.

## Colors
- **Primary (navy)**: `#1B2A4A` - main brand color, headers, primary buttons
- **Primary light**: `#2C3E6B` - hover states, secondary elements
- **Accent (gold/brass)**: `#C9A84C` - call-to-action buttons, highlights, accents
- **Accent light**: `#DFC66A` - hover states for accent
- **Background**: `#F8F9FC` - page background
- **Surface**: `#FFFFFF` - cards, sections, containers
- **Text primary**: `#1A1A2E` - body text
- **Text secondary**: `#5A6A7E` - muted text, captions
- **Border**: `#E2E8F0` - dividers, card borders
- **Success**: `#10B981`
- **Error**: `#EF4444`

## Typography
- **Font**: Inter (Google Font) - weights 300, 400, 500, 600, 700
- **Headings**: font-semibold or font-bold, tracking-tight
- **Body**: font-normal leading-relaxed
- **Captions**: text-sm text-secondary

## Spacing
- Section padding: py-20 md:py-28
- Container max width: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8
- Component gap: gap-8 md:gap-12
- Card padding: p-6 md:p-8

## Component Styles
- **Cards**: bg-white rounded-xl shadow-sm border border-border
- **Buttons - Primary**: bg-[#1B2A4A] text-white hover:bg-[#2C3E6B] rounded-lg px-6 py-3 font-medium transition
- **Buttons - Accent**: bg-[#C9A84C] text-white hover:bg-[#DFC66A] rounded-lg px-6 py-3 font-medium transition
- **Inputs**: bg-white border border-border rounded-lg px-4 py-3 focus:ring-2 focus:ring-[#C9A84C] focus:border-transparent
- **Section titles**: text-3xl md:text-4xl font-bold text-[#1B2A4A] mb-4
- **Section subtitles**: text-lg text-[#5A6A7E] max-w-2xl

## Effects
- Subtle shadow on hover: hover:shadow-md transition-shadow
- Smooth transitions: transition-all duration-300
- Border radius: rounded-lg (8px), rounded-xl (12px)
- Gradient hero: bg-gradient-to-br from-[#1B2A4A] to-[#0F1A32]

## Do's
- Use generous whitespace for readability
- Keep CTAs prominent with gold accent color
- Use high-quality imagery for machinery
- Maintain consistent spacing and alignment
- Use clean, minimal layouts

## Don'ts
- Don't use overly bright or saturated colors
- Don't overcrowd sections with too much text
- Don't use small fonts below 14px for body text
- Don't mix multiple accent colors
