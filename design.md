# ARTITECT MACHINERY - Design Guide

## Brand Identity
- Brand: ARTITECT MACHINERY
- Industry: Industrial sheet metal and folding machinery
- Style: Elegant, professional, user-friendly, modern industrial

## Color Palette
- Primary: Navy Blue (#1e293b / slate-800) - Used for headers, navbars, and primary backgrounds
- Primary Light: (#334155 / slate-700) - Hover states, secondary backgrounds
- Accent: Amber/Copper (#d97706 / amber-600) - CTA buttons, highlights, accents
- Accent Light: (#f59e0b / amber-500) - Hover states for accent elements
- Background: Off-white (#f8fafc / slate-50) - Page background
- Card Background: White (#ffffff) - Cards, sections
- Text Primary: (#0f172a / slate-900) - Main body text
- Text Secondary: (#475569 / slate-600) - Secondary text, descriptions
- Text Muted: (#94a3b8 / slate-400) - Muted hints
- Border: (#e2e8f0 / slate-200) - Light borders

## Typography
- Headings: Inter, bold weight (700-800), dark slate color
- Body: Inter, regular weight (400-500), secondary slate color
- Hero title: Large (4-5xl), bold, white text
- Section titles: 3xl, bold, centered
- Product names: xl, semibold

## Spacing
- Section padding: py-16 md:py-24 (generous whitespace)
- Container max-width: max-w-7xl mx-auto
- Card padding: p-6 md:p-8
- Gap between elements: gap-6 md:gap-8

## Component Styles
### Navigation
- Fixed/sticky top, transparent on hero, solid on scroll
- Dark background (slate-800/900)
- White text, hover: amber accent
- Logo: bold, uppercase, tracking-wider

### Cards (Products)
- White background, rounded-xl, shadow-md
- Hover: shadow-lg, subtle lift transform
- Image on top, title and description below
- Accent border-top or icon color

### Buttons
- Primary: Solid amber-600 bg, white text, rounded-lg, px-6 py-3
- Secondary: Outlined white, transparent bg, white border
- Hover effects: subtle scale or brightness

### Forms
- Clean inputs with border, rounded-lg
- Focus: ring-2 ring-amber-500
- Labels above inputs

## Layout Rules
- Single column on mobile, multi-column on md+ screens
- Centered content with max-width constraint
- Generous whitespace between sections
- Smooth scrolling between anchor links
