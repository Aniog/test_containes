# Velmora Fine Jewelry - Design System

## Visual Identity

### Brand Essence
- **Mood**: Quiet luxury, warm, editorial
- **Positioning**: Premium demi-fine jewelry — NOT loud, NOT discount-looking, NOT generic e-commerce
- **Target**: Women 25-45, gifting and self-purchase, $30-$120 price point

### Color Palette
A refined, warm palette that flatters gold jewelry:

| Role | Color | Hex |
|------|-------|-----|
| Primary Background | Warm Ivory | `#FDFBF7` |
| Secondary Background | Soft Cream | `#F7F4EF` |
| Deep Background | Charcoal | `#1C1C1C` |
| Primary Text | Rich Black | `#1A1A1A` |
| Secondary Text | Warm Gray | `#6B6560` |
| Muted Text | Soft Taupe | `#9B9590` |
| Accent Gold | Champagne Gold | `#C9A962` |
| Accent Gold Dark | Deep Gold | `#A88B45` |
| Border | Hairline Tan | `#E8E4DC` |
| Success | Sage | `#7D9B76` |
| Error | Muted Rose | `#C27D7D` |

### Typography
- **Headings**: Cormorant Garamond (serif) — elegant, editorial, luxury feel
- **Body/UI**: Inter (sans-serif) — clean, modern, highly readable
- **Product Names**: Uppercase with letter-spacing: 0.15em

### Font Scale
| Element | Font | Size | Weight | Letter-spacing |
|---------|------|------|--------|----------------|
| Hero H1 | Cormorant | 4rem/64px | 400 | 0.02em |
| Section H2 | Cormorant | 2.5rem/40px | 400 | 0.02em |
| Product H3 | Cormorant | 1.5rem/24px | 500 | 0.15em uppercase |
| Body | Inter | 1rem/16px | 400 | normal |
| Small | Inter | 0.875rem/14px | 400 | normal |
| Caption | Inter | 0.75rem/12px | 500 | 0.05em |

### Spacing System
- Base unit: 4px
- Section padding: 80px (desktop), 48px (mobile)
- Component gaps: 24px
- Card padding: 24px
- Button padding: 16px 32px

### Motion Philosophy
- **Timing**: 300ms ease for hover transitions
- **Hover effects**: Subtle scale (1.02), soft shadow lift
- **Page transitions**: Fade-in with slight upward motion
- **Scroll animations**: Gentle fade-in on intersection

## Components

### Buttons
- **Primary**: Solid gold background, white text, rounded-sm
- **Secondary**: Outlined with gold border, gold text
- **Ghost**: Text only with underline on hover
- **States**: Hover darkens, disabled reduces opacity

### Cards
- Background: White or warm ivory
- Border: 1px hairline tan
- Shadow: None or very subtle on hover
- Radius: 0px (sharp corners for editorial feel)

### Form Inputs
- Border: 1px hairline tan
- Focus: Gold accent border
- Background: White
- Radius: 0px

### Dividers
- Hairline style: 1px, hairline tan color
- Optional decorative: centered diamond or flourish

## Layout

### Container
- Max width: 1400px
- Padding: 24px (mobile), 48px (tablet), 64px (desktop)

### Grid
- Product grid: 4 columns (desktop), 2 columns (tablet), 1-2 columns (mobile)
- Content sections: Asymmetric layouts for visual interest

### Breakpoints
- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px

## Imagery
- **Style**: Warm-lit, editorial photography
- **Background**: Dark/neutral surfaces
- **Aspect ratios**: 3:4 for product cards, 4:5 for hero, 9:16 for UGC
- **Hover**: Second image reveal for products

## Do's and Don'ts

### Do
- Use generous whitespace
- Maintain editorial, magazine-like layouts
- Use thin hairline dividers
- Keep accent colors restrained
- Use subtle, premium hover transitions
- Ensure strong contrast for accessibility

### Don't
- Use loud promotional language
- Show discount badges or sale stickers
- Use generic stock photography
- Cram too many elements together
- Use rounded corners everywhere (keep editorial feel)
- Overuse the accent gold color
