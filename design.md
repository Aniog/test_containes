# SSourcing China - Design Guidelines

## Project Overview
- **Project Name**: SSourcing China Website
- **Type**: B2B Corporate Website
- **Goal**: Generate qualified sourcing inquiries from overseas buyers
- **Target Audience**: International B2B buyers seeking China sourcing agents

## Visual Style

### Color Palette
- **Primary**: #1E3A5F (Deep Navy Blue) - Trust, professionalism
- **Secondary**: #2D5A87 (Steel Blue) - Supporting blue
- **Accent**: #E67E22 (Industrial Orange) - CTAs, highlights
- **Success**: #27AE60 (Green) - Trust indicators
- **Background Light**: #F8FAFC (Off-white)
- **Background Dark**: #0F172A (Dark navy for footer)
- **Text Primary**: #1E293B (Dark slate)
- **Text Secondary**: #64748B (Muted slate)
- **Border**: #E2E8F0 (Light gray)

### Typography
- **Headings**: "DM Sans", sans-serif (Bold, professional)
- **Body**: "DM Sans", sans-serif (Regular, readable)
- **Font Sizes**:
  - H1: 48px/56px (hero)
  - H2: 36px/44px (section titles)
  - H3: 24px/32px (card titles)
  - Body: 16px/24px
  - Small: 14px/20px

### Layout
- **Max Width**: 1280px
- **Section Padding**: 80px vertical, 24px horizontal
- **Card Padding**: 24px
- **Grid**: 12-column system
- **Responsive Breakpoints**:
  - Mobile: < 640px
  - Tablet: 640px - 1024px
  - Desktop: > 1024px

### Visual Effects
- **Shadows**: Subtle, professional (0 4px 6px -1px rgba(0,0,0,0.1))
- **Border Radius**: 8px (cards), 4px (buttons), 50% (avatars)
- **Transitions**: 200ms ease-in-out
- **Hover States**: Slight lift, color shift

## Component Guidelines

### Header/Navigation
- Fixed position, white background
- Logo on left, nav links center, CTA button right
- Mobile: hamburger menu

### Hero Section
- Full-width, dark navy background with subtle pattern
- Headline + subheadline + primary CTA
- Trust badges below CTA

### Service Cards
- White background, subtle shadow
- Icon (Lucide), title, description
- Hover: slight lift effect

### Process Steps
- Numbered circles with connecting line
- Icon + title + description per step

### Case Study Cards
- Image, category tag, title, brief description
- "Read more" link

### Form Elements
- Clean input fields with labels
- Focus states with accent color
- Clear validation messages

### Buttons
- Primary: Accent orange background, white text
- Secondary: Navy outline, navy text
- Hover: Darken by 10%

## Page Structure

### Home Page
1. Hero Section
2. Services Overview (4 cards)
3. Sourcing Process (5 steps)
4. Products We Source (categories)
5. Problems We Solve
6. Trust Points
7. Case Studies Preview
8. FAQ Section
9. CTA Section with Inquiry Form

### Services Page
- Detailed service descriptions
- Pricing/Process info where applicable

### How It Works Page
- Step-by-step guide
- Timeline visualization

### Products We Source Page
- Category grid with descriptions

### Case Studies Page
- Success story cards

### Blog Page
- Article previews

### Contact Page
- Full inquiry form
- Contact information

## Image Guidelines
- Use professional B2B imagery
- Factory, QC inspection, shipping container visuals
- Avoid overly "stock" looking images
- Use data-strk-img for dynamic stock images where appropriate
