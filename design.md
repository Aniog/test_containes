# Serengeti Pulse — Design Guidelines

## Visual Identity: "Warm Earth"

### Color Palette
- **Burnt Orange**: earth-500 (#d4842f), earth-600 (#c46b24), earth-700 (#a35320)
- **Deep Ochre**: ochre-400 (#f6bf1c), ochre-600 (#c77f0a), ochre-700 (#9f5b0d)
- **Acacia Green**: acacia-400 (#7da376), acacia-600 (#4a6e45), acacia-800 (#31472e)
- **Neutrals**: savanna-cream (#fdf8f0), savanna-sand (#e8d5b7), savanna-dust (#c4a882), savanna-earth (#5c3317), savanna-shadow (#2d1a0c)

### Typography
- **Primary font**: Playfair Display — a tall, elegant serif
- Headings: font-display / font-serif, weight 600-800
- Body: font-serif, weight 400-500
- Use generous letter spacing on uppercase labels (tracking-widest)
- Use italic variants for pull quotes and accent phrases

### Spacing
- Sections: generous vertical padding (py-24 to py-32)
- Cards: p-6 to p-8
- Use large negative space to evoke the vastness of the savanna

### Visual Effects
- Parallax: slow-motion CSS transform on hero images (20s animation)
- Dissolve transitions: blur + brightness filter for heat-haze effect
- Hover states: subtle scale + background shift
- Gradient overlays: dark-to-transparent for text legibility over images

### Do's
- Use warm, earthy tones throughout
- Keep text sparse and impactful over hero images
- Use full-bleed images wherever possible
- Maintain generous whitespace

### Don'ts
- No cool blues or purples
- No sans-serif for body text
- No busy patterns or clutter
- No jarring, fast transitions
