# Velmora Fine Jewelry - Design System

## Visual Identity

### Mood
- Quiet luxury, warm, editorial
- Premium demi-fine jewelry
- NOT loud, NOT discount-looking, NOT generic e-commerce

### Color Palette
- **Primary Background**: #0D0D0D (rich black)
- **Secondary Background**: #1A1A1A (soft black)
- **Tertiary Background**: #262626 (card surfaces)
- **Primary Text**: #F5F5F0 (warm white)
- **Secondary Text**: #A3A3A3 (muted gray)
- **Accent**: #C9A962 (warm gold)
- **Accent Hover**: #D4B978 (lighter gold)
- **Border**: #333333 (subtle dividers)
- **Success**: #4A7C59 (muted green)
- **Error**: #9B4A4A (muted red)

### Typography
- **Headings**: Cormorant Garamond (serif) - elegant, editorial
  - H1: 56px / 64px line-height / font-weight 500
  - H2: 40px / 48px line-height / font-weight 500
  - H3: 28px / 36px line-height / font-weight 500
  - H4: 20px / 28px line-height / font-weight 500
  
- **Body/UI**: Manrope (sans-serif) - clean, modern
  - Body: 16px / 24px line-height / font-weight 400
  - Small: 14px / 20px line-height / font-weight 400
  - Caption: 12px / 16px line-height / font-weight 500

- **Product Names**: UPPERCASE with letter-spacing: 0.15em

### Spacing System
- Base unit: 4px
- xs: 4px, sm: 8px, md: 16px, lg: 24px, xl: 32px, 2xl: 48px, 3xl: 64px, 4xl: 96px

### Visual Effects
- **Shadows**: 
  - Card: 0 4px 20px rgba(0,0,0,0.3)
  - Elevated: 0 8px 40px rgba(0,0,0,0.4)
- **Border Radius**: 0px (sharp edges for luxury feel) or 2px for buttons
- **Dividers**: 1px solid #333333 (hairline)
- **Transitions**: 300ms ease-out for all interactions

### Components

#### Buttons
- **Primary**: 
  - Background: #C9A962 (gold)
  - Text: #0D0D0D (black)
  - Padding: 16px 32px
  - Hover: #D4B978
  - Uppercase, letter-spacing: 0.1em
  
- **Secondary/Outline**:
  - Border: 1px solid #C9A962
  - Text: #C9A962
  - Background: transparent
  - Hover: rgba(201, 169, 98, 0.1)

#### Cards
- Background: #1A1A1A
- Border: 1px solid #333333
- Padding: 24px
- Hover: subtle lift + border color change to #C9A962

#### Form Inputs
- Background: #1A1A1A
- Border: 1px solid #333333
- Focus border: #C9A962
- Text: #F5F5F0
- Placeholder: #666666

### Responsive Breakpoints
- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px
- Max content width: 1400px

### Do's and Don'ts
- DO: Use generous whitespace
- DO: Use large editorial imagery
- DO: Use thin hairline dividers
- DO: Use subtle hover transitions
- DO: Keep accent color restrained
- DON'T: Use bright/chunky colors
- DON'T: Use rounded corners (except buttons)
- DON'T: Use generic stock photos
- DON'T: Create cluttered layouts
