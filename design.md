# Landing Page Design

A clean, friendly landing page for collecting contacts and viewing them later.

## Visual Style

- **Palette**
  - Background: `bg-slate-50`
  - Surface: `bg-white`
  - Primary brand: `indigo-600` / `#4f46e5`
  - Primary hover: `indigo-700`
  - Text primary: `slate-900`
  - Text secondary: `slate-600`
  - Muted/borders: `slate-200`
  - Danger: `rose-600`
  - Success: `emerald-600`
- **Typography**
  - Sans-serif stack: `Inter, system-ui, sans-serif`
  - Headings: `font-semibold` to `font-bold`
  - Body: `text-base` (`1rem`)
  - Small/meta: `text-sm`
- **Spacing**
  - Page max width: `max-w-5xl`
  - Section vertical padding: `py-16 md:py-24`
  - Card padding: `p-6 md:p-8`
  - Form field gap: `gap-4`
- **Shape**
  - Border radius: `rounded-xl` for cards, `rounded-lg` for inputs/buttons
  - Subtle shadow: `shadow-sm`
  - Card shadow: `shadow-md`
- **Form Inputs**
  - Border: `border-slate-300`
  - Focus ring: `ring-2 ring-indigo-500 ring-offset-1`
  - Placeholder: `placeholder:text-slate-400`

## Do's

- Keep the layout light and breathable.
- Use the primary indigo for actions and links.
- Provide clear feedback after a form submission.
- Ensure all text has strong contrast on its background.
- Use `min-h-screen` for the page shell.

## Don'ts

- Don't use dark text on dark backgrounds.
- Don't rely on inherited colors for form fields or cards.
- Don't add unnecessary visual clutter.
