# SSourcing China - Design Guidelines

## Project Overview
- **Project Name**: SSourcing China Website
- **Type**: B2B Corporate Website
- **Core Functionality**: China sourcing agent website to generate qualified sourcing inquiries
- **Target Users**: Overseas buyers seeking reliable Chinese suppliers

## Visual Style

### Color Palette
- **Primary**: `#1E3A5F` (Deep Navy Blue - trust, professionalism)
- **Secondary**: `#2D5A87` (Medium Blue - headers, accents)
- **Accent**: `#E67E22` (Warm Orange - CTAs, highlights)
- **Success**: `#27AE60` (Green - trust indicators)
- **Background Light**: `#F8FAFC` (Off-white)
- **Background Dark**: `#0F172A` (Dark navy for footer)
- **Text Primary**: `#1E293B` (Dark slate)
- **Text Secondary**: `#64748B` (Muted gray)
- **Border**: `#E2E8F0` (Light gray)

### Typography
- **Headings**: "DM Sans", sans-serif (700 weight)
- **Body**: "DM Sans", sans-serif (400, 500 weight)
- **Font Sizes**:
  - H1: 56px (hero), 48px (pages)
  - H2: 40px
  - H3: 28px
  - H4: 22px
  - Body: 17px
  - Small: 15px

### Layout
- **Max Width**: 1280px
- **Spacing Scale**: 4px base (4, 8, 12, 16, 24, 32, 48, 64, 96)
- **Border Radius**: 8px (cards), 4px (buttons), 12px (large containers)

### Visual Effects
- **Shadows**: 
  - Card: `0 4px 20px rgba(30, 58, 95, 0.08)`
  - Hover: `0 8px 30px rgba(30, 58, 95, 0.12)`
- **Transitions**: 300ms ease for all interactive elements

## Components

### Header
- Fixed position, white background with subtle shadow on scroll
- Logo (left), Navigation (center), CTA button (right)
- Mobile: hamburger menu

### Hero Section (Home)
- Full-width with subtle factory/warehouse background
- Headline + subheadline + primary CTA
- Trust badges row below

### Service Cards
- Icon + title + description
- Hover lift effect
- White background with subtle border

### Process Steps

- Icon + title + description
- Alternating layout on desktop

### Product Categories
- Grid of category cards
- Image + title + item count
- Hover zoom effect

### Case Study Cards
- Image + client + results metrics
- White card with shadow

### FAQ Accordion
- Clean expand/collapse
- Plus/minus icon indicator

### Contact Form
- Clean input fields with labels

- Submit button with loading state

### Footer
- Dark navy background
- 4-column layout: About, Services, Resources, Contact

## Responsive Breakpoints
- **Mobile**: < 640px (single column)
- **Tablet**: 640px - 1024px (2 columns)
- **Desktop**: > 1024px (full layout)

## Do's and Don'ts
- DO: Use professional, clean layouts
- DO: Include realistic factory/QC/shipping imagery
- DO: Keep text concise and scannable
- DO: Use clear CTAs with contrasting colors
- DON'T: Use overly flashy animations
- DON'T: Make exaggerated claims
- DON'T: Clutter pages with too much content
- DON'T: Use low-quality or generic stock photos
