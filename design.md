# SSourcing China - B2B Website Design Guidelines

## Project Overview
- **Project Name**: SSourcing China Website
- **Type**: B2B Corporate Website
- **Purpose**: Generate qualified sourcing inquiries for China-based sourcing agent services
- **Target Audience**: Overseas buyers seeking reliable Chinese suppliers

## Visual Style
- **Tone**: Professional, trustworthy, practical, international B2B
- **Aesthetic**: Clean, modern, corporate, no exaggerated claims

## Color Palette
- **Primary**: `#1E3A5F` (Deep Navy Blue - trust, professionalism)
- **Secondary**: `#2D5A87` (Medium Blue - corporate)
- **Accent**: `#E67E22` (Warm Orange - CTA buttons, energy)
- **Accent Hover**: `#D35400` (Darker Orange)
- **Background Light**: `#F8FAFC` (Off-white)
- **Background White**: `#FFFFFF`
- **Text Primary**: `#1A1A2E` (Near Black)
- **Text Secondary**: `#4A5568` (Dark Gray)
- **Text Muted**: `#718096` (Medium Gray)
- **Border**: `#E2E8F0` (Light Gray)
- **Success**: `#10B981` (Green)
- **Warning**: `#F59E0B` (Amber)

## Typography
- **Font Family**: "Inter", system-ui, sans-serif
- **Headings**: 
  - H1: 48px/56px, font-weight 700
  - H2: 36px/44px, font-weight 600
  - H3: 24px/32px, font-weight 600
  - H4: 20px/28px, font-weight 600
- **Body**: 16px/24px, font-weight 400
- **Small**: 14px/20px, font-weight 400

## Spacing System
- **Section Padding**: 80px vertical (desktop), 48px (mobile)
- **Container Max Width**: 1200px
- **Grid Gap**: 32px (desktop), 24px (mobile)
- **Component Spacing**: 16px, 24px, 32px, 48px

## Layout Principles
- Clean, generous whitespace
- Card-based content sections
- Clear visual hierarchy
- Responsive grid layouts (1 col mobile, 2 col tablet, 3-4 col desktop)
- Sticky header navigation

## Component Guidelines

### Buttons
- **Primary CTA**: Orange background (#E67E22), white text, rounded-md
- **Secondary**: Navy outline, navy text
- **Hover**: Darker shade, subtle shadow
- **Padding**: 12px 24px

### Cards
- White background
- Subtle shadow (shadow-sm)
- Rounded corners (8px)
- Hover: shadow-md transition

### Icons
- Lucide React icons
- Consistent sizing (24px standard, 20px small, 32px large)
- Navy or gray colors

### Images
- Realistic factory/QC/shipping visuals
- Use placeholder images with appropriate aspect ratios
- 16:9 for hero sections
- 4:3 for cards
- 1:1 for team/avatar images

## Page Structure

### Header
- Logo (left)
- Navigation links (center)
- CTA button "Get a Free Quote" (right)
- Sticky on scroll

### Footer
- Company info
- Quick links
- Contact details
- Social links
- Copyright

## Responsive Breakpoints
- **Mobile**: < 640px
- **Tablet**: 640px - 1024px
- **Desktop**: > 1024px

## Do's and Don'ts
- DO: Use professional, measured language
- DO: Include trust signals (certifications, stats, testimonials)
- DO: Make CTA prominent and consistent
- DON'T: Use exaggerated claims or "guaranteed" language
- DON'T: Clutter pages with too much information
- DON'T: Use casual or overly friendly tone
- DON'T: Use bright, playful colors (keep it corporate)
