# Velmora Fine Jewelry - Design System

## Visual Identity
- **Mood**: Quiet luxury, warm, editorial, premium demi-fine jewelry.
- **Color Palette**:
  - Primary: `#1A1A1A` (Deep Charcoal - for deep refined base)
  - Accent: `#D4AF37` (Gold Metallic - for warm metallic accents)
  - Neutral: `#F9F7F2` (Soft Bone/Parchment - for background and editorial feel)
  - Muted: `#717171` (Muted Grey - for secondary text)
- **Typography**:
  - Headings: `Cormorant Garamond`, serif (Elegant, premium feel)
  - Body/UI: `Inter` or `Manrope`, sans-serif (Clean, readable)
  - Product Names: UPPERCASE with wide letter-spacing (`tracking-widest`)

## UI Components
- **Buttons**:
  - Primary: Solid Charcoal background with Gold or White text.
  - Secondary: Outlined Charcoal or Gold.
  - Hover: Subtle transitions (`transition-all duration-300`).
- **Dividers**: Thin hairline dividers (`border-t border-black/10`).
- **Cards**: Large editorial imagery, generous whitespace.
- **Transitions**: Smooth, subtle animations using Framer Motion.

## Do's and Don'ts
### Do:
- Use generous whitespace to create a luxury feel.
- Use large, high-quality editorial images.
- Keep the color palette restrained and consistent.
- Ensure high contrast for readability.

### Don't:
- Use loud or discount-looking design elements (no big red "SALE" badges).
- Overcrowd the layout.
- Use generic e-commerce patterns that feel "cheap".
