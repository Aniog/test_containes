# MicroCosmos Design Guidelines

## Theme
- Dark mode by default
- Scientific, exploratory, and premium feel
- Focus on large, high-quality images

## Colors
- Background: `bg-slate-950`
- Foreground text: `text-slate-50`
- Accent / Primary: `text-cyan-400`
- Secondary text: `text-slate-400`
- Borders: `border-slate-800`

## Typography
- Font Family: Inter (sans-serif)
- Headings: Bold, clean, slightly tracked out
- Body: Legible, medium weight

## Layout & Components
- Full-width hero sections with background images
- Masonry or Grid layouts for image galleries
- Generous padding (`py-20`, `px-6`)
- Cards with subtle gradients and dark backgrounds
- Minimalist navigation

## Do's
- Do use dark, slate-based backgrounds to make vibrant microscopic images pop.
- Do use glowing accent colors (cyan, teal, magenta) sparingly for highlights.
- Do ensure text on background images has sufficient contrast (e.g. using dark overlays).
- Do use responsive grid classes (`grid-cols-1 md:grid-cols-2 lg:grid-cols-3`).

## Don'ts
- Don't use bright, light backgrounds which might wash out the detailed microscopic pictures.
- Don't clutter the UI with too many borders and boxes; let the images define the layout.
