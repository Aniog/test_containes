# ARTITECT MACHINERY - Design Guidelines

## Brand Identity
- **Brand Name**: ARTITECT MACHINERY
- **Industry**: Sheet metal folding machines and equipment
- **Tone**: Elegant, professional, industrial, trustworthy

## Color Palette
- **Primary**: `#1a1a2e` (Deep navy/charcoal - sophistication)
- **Secondary**: `#16213e` (Dark blue - trust and reliability)
- **Accent**: `#e94560` (Coral red - energy, action)
- **Light**: `#f8f9fa` (Off-white - clean, modern)
- **Text Primary**: `#1a1a2e` (Dark for readability)
- **Text Secondary**: `#6c757d` (Muted gray)
- **Border/Divider**: `#dee2e6` (Light gray)

## Typography
- **Headings**: "Playfair Display" - elegant serif for sophistication
- **Body**: "Inter" - clean, readable sans-serif
- **Font Sizes**:
  - Hero title: 4rem (64px)
  - Section titles: 2.5rem (40px)
  - Subheadings: 1.5rem (24px)
  - Body: 1rem (16px)
  - Small: 0.875rem (14px)

## Layout Principles
- **Max content width**: 1200px
- **Section padding**: 80px vertical (desktop), 48px (mobile)
- **Grid gap**: 32px
- **Card padding**: 24px
- **Border radius**: 8px (subtle, professional)

## Visual Style
- Clean, minimalist design with generous whitespace
- Subtle shadows for depth: `0 4px 20px rgba(0,0,0,0.08)`
- Smooth transitions: 0.3s ease
- Professional imagery with industrial aesthetic
- Clear visual hierarchy

## Components

### Header
- Fixed/sticky navigation
- Logo on left, nav links on right
- Transparent on hero, solid on scroll
- Mobile: hamburger menu

### Hero Section
- Full-width background with industrial image
- Overlay gradient for text readability
- Large headline + subheadline + CTA button
- Height: 90vh

### Product Cards
- White background with subtle shadow
- Product image (aspect ratio 4:3)
- Product name, brief description
- "Learn More" button
- Hover: lift effect with shadow increase

### Features Section
- Icon + title + description layout
- 3-4 columns on desktop
- Subtle background color

### Contact Section
- Clean form layout
- Name, email, message fields
- Submit button with accent color

### Footer
- Dark background
- Logo, quick links, contact info
- Copyright notice

## Responsive Breakpoints
- **Mobile**: < 640px (single column)
- **Tablet**: 640px - 1024px (2 columns)
- **Desktop**: > 1024px (full layout)

## Do's and Don'ts
- DO: Use clean whitespace, professional imagery
- DO: Keep text concise and scannable
- DO: Use accent color sparingly for CTAs
- DON'T: Use too many colors (stick to palette)
- DON'T: Overcrowd content
- DON'T: Use flashy animations (keep professional)
