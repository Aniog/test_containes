# SSourcing China - Visual Design System

## Overview
Professional B2B website for a China-based sourcing agent. Clean, trustworthy, international aesthetic focused on generating qualified sourcing inquiries.

## Color Palette

### Primary
- **Navy Blue**: #0A2540 - Primary brand color, headers, navigation
- **Deep Blue**: #1E3A5F - Secondary dark, cards, accents

### Accent
- **Trust Blue**: #2563EB - CTAs, links, highlights
- **Success Green**: #059669 - Trust signals, success states

### Neutral
- **Off White**: #F8FAFC - Backgrounds, cards
- **Light Gray**: #E2E8F0 - Borders, dividers
- **Medium Gray**: #64748B - Secondary text, captions
- **Dark Gray**: #1E293B - Body text, headings

### Text
- **Primary Text**: #0F172A - Main content text
- **Secondary Text**: #475569 - Supporting text
- **Muted Text**: #94A3B8 - Placeholders, footnotes

## Visual Style Guidelines

### Do's
- Use generous whitespace
- Maintain consistent icon sizing (24px for inline, 48px for feature)
- Use subtle shadows for depth
- Keep text contrast high (4.5:1 minimum)
- Use professional, realistic imagery
- Align elements to grid

### Don'ts
- No bright or saturated colors
- No decorative flourishes or gradients
- No exaggerated claims or hype language
- No overlapping elements
- No low-contrast text
- No cluttered layouts

## Imagery Guidelines
- Factory interiors, production lines, quality control
- Shipping containers, logistics
- Professional team meetings
- Product samples and inspection
- Clean, well-lit, documentary style
- No stock photo clichés

## Section Patterns
- Hero: Full-width, centered content, strong headline
- Feature Grid: 3-4 columns, icon + title + description
- Process Steps: Numbered or icon-based horizontal flow
- Testimonials: Quote + attribution, subtle card
- FAQ: Accordion style, clean borders
- CTA Banner: Full-width, centered, strong action

## Responsive Behavior
- Desktop: Full multi-column layouts
- Tablet: Reduced columns, maintained spacing
- Mobile: Single column, stacked elements, touch-friendly targets (min 44px)
