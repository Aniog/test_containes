# Velmora Fine Jewelry — Design System

## Mood
Quiet luxury, warm, editorial. Premium demi-fine jewelry. NOT loud, NOT discount-looking.

## Color Palette
- **Background (cream):** `cream` → `#FAF7F2` — warm off-white page background
- **Surface (ivory):** `ivory` → `#F5F0E8` — card/section backgrounds
- **Foreground (charcoal):** `charcoal` → `#1A1A1A` — primary text
- **Muted text:** `stone` → `#6B5E54` — secondary/body text
- **Accent (gold):** `gold` → `#B8860B` — CTAs, highlights, hover states
- **Accent hover:** `gold-dark` → `#8B6508` — darker gold for hover
- **Accent light:** `gold-light` → `#D4A843` — lighter gold for subtle accents
- **Border:** `border` → `#E8E0D4` — hairline dividers, card borders

## Typography
- **Headings / Product names:** `Cormorant Garamond`, serif — elegant, editorial
- **Body / UI:** `Inter`, sans-serif — clean, modern, readable
- **Product names:** UPPERCASE, letter-spacing: 0.15em
- **Heading sizes:** Hero: 4xl–6xl, Section: 3xl–4xl, Card: lg–xl

## Spacing & Layout
- Generous whitespace: sections py-16 to py-24
- Max content width: max-w-7xl (1280px)
- Grid gaps: gap-6 to gap-8
- Card padding: p-6 to p-8

## Components
- **Buttons:** Solid gold bg with cream text (primary), outlined gold border (secondary). Rounded-none (sharp edges). px-8 py-3. Uppercase tracking-wider text-sm.
- **Cards:** bg-ivory, no border or very subtle border-border. Soft shadow on hover (shadow-sm). Transition-all duration-300.
- **Dividers:** 1px border-border, full width or partial.
- **Hover effects:** opacity transitions, subtle scale (scale-[1.02]), underline animations.

## Do's
- Use generous whitespace
- Keep imagery large and editorial
- Use gold accent sparingly for CTAs and highlights
- Maintain serif/sans-serif hierarchy
- Use thin hairline dividers between sections

## Don'ts
- Don't use bright/neon colors
- Don't use rounded buttons (use sharp/square edges)
- Don't crowd elements — let them breathe
- Don't use heavy shadows or borders
- Don't use generic e-commerce patterns (sale badges, countdown timers)
