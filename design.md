# ARTITECT MACHINERY Design System

## Typography
- Font family: Inter (Google Fonts)
- Headings: font-bold, tracking-tight
- Body: font-normal, leading-relaxed
- Hero title: text-5xl md:text-6xl lg:text-7xl
- Section title: text-3xl md:text-4xl
- Subtitle: text-lg md:text-xl text-muted-foreground

## Colors
- Primary: slate-900 (dark backgrounds, headings)
- Primary accent: amber-600 (CTAs, highlights)
- Secondary: slate-600 (subtle text)
- Background: white, slate-50 (alternating sections)
- Card background: white with subtle shadow
- Border: slate-200
- Text on dark: white, slate-300
- Text on light: slate-900, slate-600

## Layout
- Max container width: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8
- Section padding: py-16 md:py-24
- Grid gaps: gap-6 md:gap-8
- Card padding: p-6

## Components
- Buttons: rounded-lg, px-6 py-3, font-medium
- Primary button: bg-amber-600 hover:bg-amber-700 text-white
- Secondary button: border border-slate-300 hover:bg-slate-50
- Cards: bg-white rounded-xl shadow-sm border border-slate-200
- Navigation: sticky top-0, bg-white/95 backdrop-blur, border-b

## Spacing
- Section spacing: py-16 md:py-24
- Element spacing: space-y-4, space-y-6
- Grid: grid-cols-1 md:grid-cols-2 lg:grid-cols-3

## Visual Style
- Clean, industrial elegance
- Subtle shadows and borders
- Generous whitespace
- Professional photography placeholders
- Smooth transitions on hover
- Mobile-first responsive design
