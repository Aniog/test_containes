# Design System

## Color Palette
- Primary: Indigo (#6366f1) — `bg-indigo-500`, `text-indigo-600`
- Primary Dark: `bg-indigo-600`, hover: `hover:bg-indigo-700`
- Background: Light gray `bg-gray-50` for page, `bg-white` for cards
- Surface: White `bg-white` with subtle shadow `shadow-md`
- Border: `border-gray-200`
- Text Primary: `text-gray-900`
- Text Secondary: `text-gray-500`
- Text Muted: `text-gray-400`
- Danger: `text-red-500`, `hover:text-red-600`
- Success/Complete: `text-green-500`, `line-through text-gray-400`

## Typography
- Font: Inter (Google Fonts)
- Heading: `text-3xl font-bold text-gray-900`
- Subheading: `text-lg font-semibold text-gray-700`
- Body: `text-sm text-gray-700`
- Muted: `text-sm text-gray-400`

## Spacing & Layout
- Page padding: `px-4 py-8`
- Card padding: `p-6`
- Section gap: `gap-4`
- Max width: `max-w-2xl mx-auto`

## Borders & Radius
- Card: `rounded-2xl`
- Input: `rounded-lg`
- Button: `rounded-lg`
- Checkbox: `rounded`

## Shadows
- Card: `shadow-md`
- Input focus: `focus:ring-2 focus:ring-indigo-400`

## Do's
- Use indigo as the primary action color
- Use white cards on gray-50 background
- Keep text clearly readable — dark text on light backgrounds
- Use subtle hover states for interactive elements

## Don'ts
- Don't use dark backgrounds for the main page
- Don't use low-contrast text (e.g. gray-300 on white)
- Don't use multiple accent colors
