# Visual Style Guide - Velmora Fine Jewelry

## Typography
- **Headings & Product Names**: `Cormorant Garamond`, serif. Uppercase for product names with wide letter-spacing (`tracking-[0.2em]`).
- **Body & UI**: `Inter`, sans-serif. Clean and readable.

## Palette
- **Background (Base)**: `#FDFCFB` (Warm Off-white / Cream)
- **Foreground (Text)**: `#1C1917` (Deep Charcoal / Stone-900)
- **Accent (Gold)**: `#C5A059` (Soft Metallic Gold)
- **Muted**: `#78716C` (Stone-500)
- **Border**: `#E7E5E4` (Stone-200 / Thin Hairlines)

## Components
- **Buttons**:
  - Primary: Solid Charcoal background with Gold text, or vice versa if emphasized.
  - Secondary: Outlined/Transparent with thin borders.
- **Dividers**: Thin hairlines (`h-[1px] bg-stone-200`).
- **Cards**: Minimalist, no borders or light shadows, focus on imagery.

## Spacing
- Generous whitespace between sections.
- Large padding on mobile edges (at least `px-6`).

## Animations
- Subtle fades and scale-ups using `framer-motion`.
- Smooth transitions for hover effects.
