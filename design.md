# Velmora Fine Jewelry - Design System

## Visual Identity

### Mood
- Quiet luxury, warm, editorial
- Premium demi-fine jewelry
- NOT loud, NOT discount-looking, NOT generic e-commerce

### Color Palette
| Name | Hex | Usage |
|------|-----|-------|
| velmora-cream | #F9F7F4 | Primary background |
| velmora-charcoal | #1A1A1A | Primary text, dark elements |
| velmora-gold | #C9A962 | Accent color, CTAs, highlights |
| velmora-gold-light | #E5D4A1 | Gold hover states, subtle accents |
| velmora-gold-dark | #A68B4B | Gold pressed states |
| velmora-warm | #F5EDE4 | Secondary background, cards |
| velmora-sage | #9CAF88 | Secondary accent (subtle) |
| velmora-rose | #D4A5A5 | Tertiary accent (subtle) |

### Typography
- **Headings & Product Names**: Cormorant Garamond (serif)
  - Weights: 300, 400, 500, 600, 700
  - Product names: UPPERCASE with wide letter-spacing (0.15em)
- **Body & UI**: Inter (sans-serif)
  - Weights: 300, 400, 500, 600

### Spacing System
- Base unit: 4px
- Generous whitespace throughout
- Section padding: 80px+ on desktop, 48px on mobile

### Visual Elements
- Thin hairline dividers (1px, light gray)
- Subtle hover transitions (200-300ms ease)
- Soft shadows: `0 4px 20px rgba(0, 0, 0, 0.06)`
- Buttons: solid or outlined accent, premium feel

## Component Guidelines

### Buttons
- Primary: Gold background (#C9A962), white text, subtle shadow
- Secondary: Outlined gold, transparent background
- Hover: Slight lift with enhanced shadow
- Border-radius: 0 (sharp) or 2px (minimal)

### Cards
- Cream/warm background
- Subtle shadow on hover
- Image with second image reveal on hover
- Quick "Add to Cart" on hover

### Dividers
- Hairline style: 1px solid rgba(0,0,0,0.08)
- Used between sections and within content

## Responsive Breakpoints
- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px
- Mobile-first approach (most traffic is mobile)

## Animations
- Subtle fade-in on scroll
- Smooth hover transitions (200-300ms)
- Image zoom on hover (subtle, 5-10%)
- Cart drawer slide-in from right
