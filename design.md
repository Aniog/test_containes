# Velmora Fine Jewelry — Design System

## Visual Identity
- **Mood**: Quiet luxury, warm, editorial. Premium demi-fine jewelry.
- **Feel**: Refined, not loud. Warm, not cold. Accessible luxury, not discount.

## Color Palette
- **Background**: `#FAF7F2` (warm cream) — primary page background
- **Surface**: `#F5EDE3` (warm beige) — cards, sections
- **Accent Surface**: `#E8DDD0` (sand) — hover states, subtle backgrounds
- **Gold Accent**: `#C9A96E` — primary accent, CTAs, highlights
- **Gold Light**: `#D4BA8A` — hover states for gold elements
- **Gold Dark**: `#A8864E` — pressed/active gold states
- **Text Primary**: `#2C2C2C` (charcoal) — headings, body text
- **Text Muted**: `#8A8279` — secondary text, descriptions
- **Espresso**: `#1A1612` — dark sections, hero overlays, footer

## Typography
- **Headings**: Cormorant Garamond (serif), weights 400-600
- **Body/UI**: Inter (sans-serif), weights 300-500
- **Product names**: Uppercase, letter-spacing 0.2em, serif
- **Section headings**: Serif, large, generous spacing

## Spacing & Layout
- Generous whitespace throughout
- Max content width: 1280px
- Section padding: 80-120px vertical (desktop), 48-64px (mobile)
- Hairline dividers: 1px, color #E8DDD0

## Buttons
- Primary: solid gold (#C9A96E) background, cream text, hover gold-dark
- Secondary: outlined gold border, gold text, hover fills gold
- Pill buttons for variant selectors
- All buttons: subtle transition (0.3s ease)

## Images
- Large, editorial-style imagery
- Warm lighting, gold jewelry on neutral/dark backgrounds
- Hover reveals second image on product cards
- Aspect ratios: 3x4 for products, 16x9 for hero, 9x16 for UGC

## Animations
- Subtle fade-in on scroll
- Smooth hover transitions (0.3s ease)
- Cart drawer slides in from right
- No jarring or bouncy animations

## Do's
- Use serif for all headings and product names
- Keep generous whitespace
- Use gold accent sparingly and intentionally
- Ensure strong text contrast on all backgrounds
- Use thin hairline dividers between sections

## Don'ts
- Don't use bright/saturated colors
- Don't crowd elements — let them breathe
- Don't use sans-serif for headings
- Don't use heavy shadows — keep them soft and subtle
- Don't use discount/sale visual language
