# Design Guide — SSourcing China

## Brand Colors
- Primary (CTA/links): #2563eb (blue-600)
- Primary hover: #1d4ed8 (blue-700)
- Dark text: #1e293b (slate-800)
- Muted text: #64748b (slate-500)
- Light bg: #f8fafc (slate-50)
- Section alt bg: #f1f5f9 (slate-100)
- Border: #e2e8f0 (slate-200)
- Success/trust: #059669 (emerald-600)
- White: #ffffff

## Typography
- Headings: Inter, font-bold, tracking-tight
- Body: Inter, font-normal, leading-relaxed
- Use Tailwind font-weight classes: font-bold, font-semibold, font-medium, font-normal

## Spacing
- Section padding: py-16 md:py-24
- Container max width: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8
- Card padding: p-6 md:p-8
- Gap between elements: gap-6 md:gap-8

## Visual Style
- Cards: bg-white rounded-xl shadow-sm border border-slate-200
- Buttons: 
  - Primary: bg-blue-600 hover:bg-blue-700 text-white font-semibold rounded-lg px-6 py-3
  - Outline: border-2 border-blue-600 text-blue-600 hover:bg-blue-50 font-semibold rounded-lg px-6 py-3
- Section titles: text-3xl md:text-4xl font-bold text-center mb-4
- Section subtitles: text-lg text-slate-500 text-center max-w-2xl mx-auto mb-12
- Dividers: border-t border-slate-200

## Do's
- Use clean, professional layout with plenty of white space
- Keep consistent spacing throughout
- Use icons (Lucide React) for service cards and feature lists
- Use the stock image system for realistic factory/QC/shipping visuals
- Keep CTA buttons prominent with blue color
- Use breadcrumb-style navigation on inner pages

## Don'ts
- Don't use dark mode colors
- Don't use exaggerated claims or superlatives
- Don't clutter pages with too much text
- Don't use decorative fonts
- Don't use heavy shadows or gradients
