# SSourcing China - B2B Website Design System

## Visual Style

**Overall Aesthetic**: Clean, trustworthy, international B2B. Professional and practical without exaggeration. Uses a restrained color palette with strong typography hierarchy.

**Color Palette**:
- Primary: #0F172A (Slate 900) - Deep navy for headers and CTAs
- Secondary: #1E40AF (Blue 800) - Trustworthy blue for accents and links
- Accent: #0EA5E9 (Sky 500) - Light blue for highlights
- Neutral: #F8FAFC (Slate 50) - Light background
- Text: #334155 (Slate 700) - Body text
- Muted: #64748B (Slate 500) - Secondary text
- Success: #059669 (Emerald 600) - Trust indicators
- Border: #E2E8F0 (Slate 200) - Subtle dividers

**Typography**:
- Font: Inter (system-ui fallback)
- Headings: 600-700 weight, tight tracking
- Body: 400-500 weight, 1.6 line-height
- Scale: H1 48px, H2 36px, H3 24px, Body 16px

**Spacing**:
- Section padding: 80px vertical (desktop), 48px (mobile)
- Container max-width: 1280px
- Consistent 8px grid system

**Components**:
- Buttons: Rounded corners (6px), clear hover states, primary/secondary variants
- Cards: Subtle shadow, clean borders, generous padding
- Forms: Clean inputs with focus states, clear labels
- Navigation: Fixed header, clear active states

**Imagery**:
- Realistic factory, QC inspection, and shipping visuals
- Professional photography style
- Consistent aspect ratios (16:9 for hero, 4:3 for cards)

**Do's**:
- Use ample white space
- Maintain consistent alignment
- Use semantic HTML
- Ensure 4.5:1 minimum contrast
- Keep CTAs prominent but not aggressive

**Don'ts**:
- Avoid overly bright colors
- No exaggerated claims or hype language
- No decorative flourishes or gradients
- No low-contrast text
- Avoid cluttered layouts
