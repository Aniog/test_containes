# SSourcing China - Design System

## Brand Colors
- Primary (Brand Blue): #1e40af (brand-500) - Used for buttons, links, key accents
- Dark Blue: #0f172a (brand-900) - Headings, footer backgrounds
- Light Blue: #eff6ff (brand-50) - Section backgrounds, cards
- Accent Red: #dc2626 (accent-500) - Highlights, CTAs, emphasis
- Surface neutrals: slate gray scale (#f8fafc through #0f172a)

## Typography
- Font: Inter (Google Fonts)
- Headings: weight 700, color #0f172a
- Body: weight 400, color #1e293b, line-height 1.6
- Small/caption: text-sm text-surface-500

## Spacing
- Section padding: py-16 md:py-24
- Container max-width: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8
- Card padding: p-6 md:p-8
- Gap between cards: gap-6 md:gap-8

## Buttons
- Primary CTA: bg-brand-500 text-white px-6 py-3 rounded-lg font-semibold hover:bg-brand-600 transition
- Secondary: bg-white text-brand-500 border border-brand-500 px-6 py-3 rounded-lg font-semibold hover:bg-brand-50 transition
- Accent CTA: bg-accent-500 text-white px-8 py-4 rounded-lg font-bold text-lg hover:bg-accent-600 transition

## Cards
- bg-white rounded-xl shadow-sm border border-surface-100 p-6
- Hover: shadow-md transition

## Section Backgrounds
- Light sections: bg-surface-50
- White sections: bg-white
- Dark sections: bg-brand-900 text-white

## Image Style
- Use data-strk-img for realistic factory/QC/shipping visuals
- Aspect ratios: 16x9 for hero, 4x3 for cards, 3x2 for thumbnails

## Layout
- Fixed header with white background, shadow-sm
- Footer: bg-brand-900 text-white
- Max content width: max-w-7xl
- Responsive: mobile-first, md: breakpoint for desktop layouts

## Do's
- Use brand-500 for primary actions and links
- Use surface-50 for alternating section backgrounds
- Keep layouts clean with plenty of white space
- Use consistent card styles
- Use the accent red sparingly for key CTAs only

## Don'ts
- Don't use bright/playful colors
- Don't use overly casual language
- Don't use low-contrast text
- Don't use decorative flourishes or illustrations
- Don't use hero sliders or auto-rotating carousels
