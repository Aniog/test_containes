# SSourcing China - Design Guidelines

## Brand Identity
- Professional B2B sourcing agency
- Trustworthy, clean, international feel
- Navy blue primary with orange accent

## Colors
- Primary: #1e3a5f (Navy) - headers, nav, trust elements
- Primary Light: #2a5298 (Blue) - links, hover states
- Accent: #e8763a (Orange) - CTAs, highlights, important actions
- Accent Light: #f09050 - hover on CTAs
- Text: #1a202c (Near Black) - body text
- Steel: #4a5568 - secondary text
- Slate: #64748b - muted text
- Surface: #f8f9fb - section backgrounds
- White: #ffffff - cards, main background

## Typography
- Font: Inter (Google Fonts)
- Headings: font-bold, tracking-tight
- H1: text-4xl md:text-5xl lg:text-6xl
- H2: text-3xl md:text-4xl
- H3: text-xl md:text-2xl
- Body: text-base (16px), leading-relaxed

## Spacing
- Section padding: py-16 md:py-24
- Container: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8
- Card padding: p-6 md:p-8
- Gap between elements: gap-6 md:gap-8

## Components
- Buttons: rounded-lg, font-semibold, transition-all
- Primary CTA: bg-accent text-white hover:bg-accent-light
- Secondary: border-2 border-primary text-primary hover:bg-primary hover:text-white
- Cards: bg-white rounded-xl shadow-sm border border-gray-100
- Cards hover: shadow-md transition-shadow

## Visual Style
- Clean, minimal design with generous whitespace
- Subtle shadows and borders
- Professional imagery (factory, QC, shipping)
- Icons from Lucide React
- Consistent rounded corners (rounded-xl for cards, rounded-lg for buttons)

## Do's
- Use navy for trust and authority
- Use orange sparingly for CTAs and highlights
- Keep layouts clean with clear hierarchy
- Use responsive design (mobile-first)
- Ensure high contrast for readability

## Don'ts
- Don't use bright or flashy colors
- Don't overcrowd sections
- Don't use informal language or emojis
- Don't use low-contrast text
- Don't use more than 2 primary colors
