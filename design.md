# ARTITECT MACHINERY - Design System

## Brand Identity
Industrial machinery brand specializing in metal folding machines. Elegant, premium, user-friendly.

## Colors
- Primary: `#1a2332` (Deep Navy/Slate) - For headers, footers, primary buttons
- Accent: `#c4963c` (Warm Gold) - For highlights, CTAs, links, decorative elements
- Secondary Accent: `#2d4a6e` (Steel Blue) - For secondary elements, cards
- Background: `#f7f8fa` (Light Gray-White) - Main page background
- Surface: `#ffffff` (White) - Cards, content areas
- Text Primary: `#1a2332` (Navy)
- Text Secondary: `#5a6470` (Muted Slate)
- Text on Dark: `#ffffff` (White)
- Border: `#e2e6eb` (Light Gray)

## Typography
- Font Family: Inter (from Google Fonts, already in index.html)
- Headings: Bold, tracking-tight
- Body: Regular, leading-relaxed
- Sizes: Use Tailwind defaults (text-sm through text-5xl)

## Spacing
- Section padding: py-16 md:py-24
- Container max-width: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8
- Card padding: p-6 md:p-8

## Visual Elements
- Subtle shadows for depth: shadow-sm, shadow-md
- Rounded corners: rounded-lg, rounded-xl
- Gold accent lines/borders for elegance
- Clean, minimal icons from Lucide React
- Generous whitespace for readability

## Do's
- Use gold accent sparingly for emphasis
- Maintain high contrast for readability
- Use professional, clean layouts
- Keep navigation simple and intuitive

## Don'ts
- No overly bright or neon colors
- No cluttered layouts
- No low-contrast text
- No excessive animations
