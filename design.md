# SSourcing China - Design Guidelines

## Brand
- Company: SSourcing China - China Sourcing Agent
- Tone: Professional, clear, practical, trustworthy, B2B
- No exaggerated claims. Focus on proven expertise, transparency, and reliability.

## Colors
- Primary: Blue (#1e40af / blue-900) - trust, professionalism, corporate
- Primary Light: (#3b82f6 / blue-500) - CTAs, interactive elements
- Secondary: (#ea580c / orange-600) - accent, highlights, badges
- Background: (#f8fafc / slate-50) - page backgrounds
- Card Background: (#ffffff / white)
- Text Primary: (#0f172a / slate-900) - headings
- Text Secondary: (#475569 / slate-600) - body text
- Muted: (#94a3b8 / slate-400) - secondary info
- Border: (#e2e8f0 / slate-200) - dividers, cards
- Success: (#16a34a / green-600) - verification, trust badges

## Typography
- Headings: Inter, bold/semibold, clean
- Body: Inter, regular weight
- Sizes: h1: text-4xl/tight md:text-5xl, h2: text-3xl md:text-4xl, h3: text-xl md:text-2xl, body: text-base/text-lg

## Spacing
- Section padding: py-16 md:py-20
- Container max-width: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8
- Between sections: space-y-0 (each section is its own block)
- Card padding: p-6 md:p-8

## Borders & Shadows
- Cards: rounded-xl border border-slate-200 shadow-sm
- Cards hover: shadow-md transition-shadow
- Buttons: rounded-lg font-medium

## Visual Style
- Clean, minimal, lots of whitespace
- Consistent icon usage from lucide-react
- Use data-strk-img for factory/QC/shipping visuals with realistic B2B context
- Grid layouts for service cards and product categories
- Centered section headings with brief subtitle
- Trust indicators: logos, certificates, numbers/stats

## Layout
- Fixed top nav with transparent background, white on scroll
- Footer: dark (slate-900) with logo, quick links, contact info
- Mobile: hamburger menu, stacked layouts
