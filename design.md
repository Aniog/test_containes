# Visual Style Guide - ARTITECT MACHINERY

## Typography
- **Primary Font**: Inter, sans-serif
- **Headings**: Semi-bold to Bold (font-semibold, font-bold)
- **Body**: Regular (font-normal)
- **Accents**: Muted-foreground for secondary text (text-muted-foreground)

## Colors
- **Primary**: Slate 900 (`bg-slate-900`, `text-slate-900`) - Represents industrial strength.
- **Secondary**: Slate 600 (`bg-slate-600`, `text-slate-600`) - For subtle accents.
- **Background**: White/Slate 50 (`bg-white`, `bg-slate-50`) - Clean, elegant look.
- **Accent**: Cool Gray (`text-gray-500`) - Professional machinery feel.

## Spacing & Layout
- **Container**: Max width 1200px (container mx-auto)
- **Padding**: Generous section padding (py-12 to py-24)
- **Grid**: Responsive grids (grid-cols-1 md:grid-cols-2 lg:grid-cols-3)

## UI Elements
- **Buttons**: Rounded-md, elegant transitions (transition-all duration-300)
- **Cards**: Subtle borders and light shadows (border border-slate-200 shadow-sm)
- **Images**: High-quality machinery shots with rounded corners (rounded-lg)

## Do's
- Use high-contrast headings for readability.
- Maintain a clean, minimalist industrial aesthetic.
- Ensure all machinery labels are clear.

## Don'ts
- Avoid overly bright or neon colors.
- Don't crowd sections; allow for white space.
- Avoid low-contrast text on dark backgrounds.
