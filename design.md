# Velmora Fine Jewelry — Design System

## Visual Identity
- **Mood**: Quiet luxury, warm, editorial. Premium demi-fine jewelry — NOT loud, NOT discount-looking, NOT generic e-commerce.
- **Direction**: Deep refined base + warm metallic accents. Soft neutral/editorial scheme.

## Color Palette

### Primary Colors
- **Deep Charcoal (base)**: `#1C1917` — backgrounds, nav, hero overlays
- **Warm Cream (light base)**: `#FAF7F2` — page backgrounds, cards
- **Rich Gold (accent)**: `#C9A96E` — CTAs, highlights, decorative elements
- **Soft Ivory**: `#F5F0E8` — secondary backgrounds, newsletter section

### Text Colors
- **Primary Text**: `#1C1917` — headings, body on light
- **Secondary Text**: `#78716C` — muted text, descriptions
- **Light Text**: `#FAF7F2` — text on dark backgrounds
- **Gold Text**: `#C9A96E` — accent text, links

### Border & Divider
- **Hairline**: `#E7E0D5` — thin dividers, card borders
- **Medium**: `#D6CFC4` — stronger borders

## Typography
- **Headings**: Cormorant Garamond (serif), elegant, refined
  - H1: 3.5rem–4rem, tracking-wide, uppercase for product names
  - H2: 2rem–2.5rem, tracking-wide
  - H3: 1.25rem–1.5rem
- **Body**: Inter (sans-serif), clean, modern
  - Body: 0.875rem–1rem, leading-relaxed
  - Small: 0.75rem–0.8125rem
  - Buttons: 0.75rem, uppercase, tracking-widest
- **Product Names**: UPPERCASE, letter-spacing: 0.15em–0.2em, serif font

## Spacing
- Section padding: py-20 to py-28 (desktop), py-12 to py-16 (mobile)
- Container max-width: max-w-7xl (1280px)
- Card gaps: gap-6 to gap-8
- Generous whitespace throughout

## Borders & Dividers
- Hairline dividers: border-t with color #E7E0D5
- Card borders: subtle, 1px, #E7E0D5
- No heavy shadows — soft, barely-there shadows only

## Buttons
- **Primary (accent)**: bg-[#C9A96E] text-white, uppercase, tracking-widest, px-8 py-3, hover:bg-[#B8964F], transition
- **Secondary (outlined)**: border-[#C9A96E] text-[#C9A96E], uppercase, tracking-widest, px-8 py-3, hover:bg-[#C9A96E] hover:text-white
- **Dark**: bg-[#1C1917] text-[#FAF7F2], uppercase, tracking-widest
- Rounded: rounded-none or rounded-sm (minimal rounding for premium feel)

## Shadows
- Cards: shadow-sm or shadow-none with border
- Hover: subtle shadow-md
- No heavy drop shadows

## Transitions
- Default: transition-all duration-300
- Hover effects: subtle scale (scale-[1.02]), opacity changes
- Image reveals: opacity 0→1 on hover

## Do's
- Use generous whitespace
- Use serif for headings and product names
- Use uppercase with wide letter-spacing for product names
- Use thin hairline dividers between sections
- Keep color palette restrained and warm
- Ensure strong contrast for accessibility

## Don'ts
- Don't use bright/saturated colors
- Don't use heavy shadows or thick borders
- Don't use rounded-full buttons (not premium feel)
- Don't use generic e-commerce patterns (badges, loud CTAs)
- Don't use more than 2 fonts
- Don't use low-contrast text
