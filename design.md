# Velmora Fine Jewelry — Design System

## Color Palette
- **Background**: `#FAF8F5` (warm ivory) — main page background
- **Surface**: `#FFFFFF` (pure white) — cards, modals, inputs
- **Text Primary**: `#1A1A1A` (near black) — headings, body text
- **Text Secondary**: `#6B6560` (warm taupe) — captions, secondary text
- **Text Muted**: `#9B9590` (soft warm gray) — placeholders, subtle labels
- **Accent / Gold**: `#B8952E` (warm metallic gold) — CTAs, links, highlights
- **Accent Hover**: `#9A7B24` (deeper gold) — hover states
- **Border**: `#E8E2DA` (warm light border) — dividers, card borders
- **Border Light**: `#F0EBE4` (very subtle border) — section dividers
- **Star Rating**: `#C9A84C` (soft gold) — review stars
- **Success**: `#4A7C59` (muted sage green) — success states
- **Error**: `#B5443E` (muted terracotta) — error states

## Typography
- **Headings / Brand**: Cormorant Garamond (serif) — elegant, editorial
  - Logo: 28px, weight 400, letter-spacing 0.15em
  - H1: 48-64px, weight 400, line-height 1.1
  - H2: 32-40px, weight 400, line-height 1.2
  - H3: 24-28px, weight 400, line-height 1.3
  - Product names: UPPERCASE, letter-spacing 0.12em
- **Body / UI**: Inter (sans-serif) — clean, modern
  - Body: 15-16px, weight 400, line-height 1.6
  - Small: 13-14px, weight 400
  - Buttons: 13px, weight 500, letter-spacing 0.08em, UPPERCASE
  - Labels: 12px, weight 500, letter-spacing 0.06em, UPPERCASE

## Spacing
- Section padding: 80-120px vertical on desktop, 48-64px on mobile
- Card gap: 24px desktop, 16px mobile
- Component padding: 16-24px
- Button padding: 14px 32px

## Borders & Dividers
- Hairline dividers: 1px solid `#E8E2DA`
- Card borders: 1px solid `#F0EBE4`
- Border radius: 2px (subtle, premium feel — NOT rounded)

## Shadows
- Card hover: `0 4px 20px rgba(0,0,0,0.06)`
- Dropdown / drawer: `0 8px 40px rgba(0,0,0,0.12)`
- Button hover: `0 2px 12px rgba(184,149,46,0.2)`

## Transitions
- Hover transitions: 300ms ease
- Page transitions: 200ms ease
- Cart drawer slide: 400ms cubic-bezier(0.16, 1, 0.3, 1)

## Buttons
- **Primary (Accent)**: Solid gold background `#B8952E`, white text, uppercase, letter-spacing 0.08em
- **Secondary (Outline)**: Transparent background, gold border `#B8952E`, gold text
- **Tertiary (Text)**: No background/border, gold text with underline on hover

## Do's
- Use generous whitespace — let products breathe
- Keep text minimal and editorial
- Use uppercase for product names and navigation
- Maintain warm, muted tones throughout
- Use thin hairline dividers between sections
- Subtle hover animations only

## Don'ts
- No bright/neon colors
- No heavy shadows or gradients
- No large border radius (keep it sharp/premium)
- No discount badges or loud sale indicators
- No cluttered layouts
- No generic e-commerce patterns (countdown timers, popups)
