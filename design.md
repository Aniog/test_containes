# Velmora Fine Jewelry - Design System

## Brand Identity
- **Brand Name**: Velmora Fine Jewelry
- **Tagline**: Crafted to be Treasured
- **Mood**: Quiet luxury, warm, editorial. Premium demi-fine jewelry — NOT loud, NOT discount-looking, NOT generic e-commerce.

## Target Audience
- Women 25–45
- Gifting and self-purchase
- Premium-but-accessible price point ($30–$120)

## Color Palette

### Primary Colors
- **Rich Charcoal** (Background): `#1A1816` - Deep, warm dark base
- **Warm Ivory** (Text/Light): `#FAF8F5` - Soft cream white
- **Antique Gold** (Accent): `#C9A962` - Warm metallic gold accent
- **Muted Bronze** (Secondary Accent): `#A68B5B` - Deeper gold tone

### Neutral Colors
- **Soft Taupe**: `#E8E4DF` - Light backgrounds
- **Warm Gray**: `#6B6560` - Secondary text
- **Stone**: `#3D3835` - Borders, dividers

### Functional Colors
- **Success**: `#7D9B76` - Muted sage green
- **Error**: `#B85C5C` - Muted coral red

## Typography

### Headings & Product Names
- **Primary Font**: Cormorant Garamond (Google Fonts)
- **Weights**: 400 (regular), 500 (medium), 600 (semibold)
- **Product Names**: UPPERCASE with letter-spacing: `0.15em`

### Body & UI
- **Primary Font**: Manrope (Google Fonts)
- **Weights**: 300 (light), 400 (regular), 500 (medium), 600 (semibold)

### Font Sizes
- Hero Headline: 56px / 3.5rem (mobile: 36px)
- Section Headings: 40px / 2.5rem (mobile: 28px)
- Product Names: 14px / 0.875rem (uppercase)
- Body: 16px / 1rem
- Small/Caption: 14px / 0.875rem
- Tiny: 12px / 0.75rem

## Spacing System
- Base unit: 4px
- Spacing scale: 4, 8, 12, 16, 24, 32, 48, 64, 96, 128px
- Section padding: 96px vertical (mobile: 64px)
- Container max-width: 1400px
- Content max-width: 1200px

## Visual Effects

### Shadows
- **Soft**: `0 4px 20px rgba(0, 0, 0, 0.08)`
- **Medium**: `0 8px 30px rgba(0, 0, 0, 0.12)`
- **Elevated**: `0 16px 48px rgba(0, 0, 0, 0.16)`

### Borders & Dividers
- Hairline dividers: `1px solid rgba(255, 255, 255, 0.1)`
- Subtle borders: `1px solid #3D3835`
- Border radius: 0px (sharp, editorial) or 4px (buttons)

### Animations
- Default transition: `all 0.3s ease`
- Hover transitions: `all 0.4s cubic-bezier(0.4, 0, 0.2, 1)`
- Fade in: `opacity 0.5s ease`
- Slide up: `transform 0.5s ease`

## Component Styles

### Buttons
- **Primary**: Solid Antique Gold background, Rich Charcoal text
  - Hover: Slightly lighter gold `#D4B872`
  - Padding: 16px 32px
  - Border-radius: 0px (sharp) or 4px
  - Font: Manrope, 500 weight, uppercase, letter-spacing: 0.1em

- **Secondary**: Transparent with Antique Gold border
  - Hover: Antique Gold background with dark text
  - Padding: 16px 32px

### Cards
- Background: Transparent or slightly elevated
- Hover: Subtle shadow, slight scale (1.02)
- Product cards: Image with hover reveal second image + quick add

### Form Inputs
- Border: 1px solid Stone
- Focus: Antique Gold border
- Background: Transparent or Soft Taupe
- Padding: 12px 16px

### Accordions
- Thin hairline dividers
- Smooth expand/collapse animation
- Plus/minus icon indicator

## Layout Principles
- Generous whitespace
- Large editorial imagery
- Thin hairline dividers
- Restrained accent color
- Mobile-first responsive design

## Responsive Breakpoints
- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px
- Large Desktop: > 1400px
