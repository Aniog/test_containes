# SSourcing China - Design System

## Overview
Professional B2B website for a China-based sourcing agent targeting overseas buyers.

## Color Palette

### Primary Colors
- **Primary Orange**: `#c2410c` (burnt orange) - Main CTA, highlights, accents
- **Primary Orange Dark**: `#9a3412` - Hover states
- **Primary Orange Light**: `#fef3c7` - Backgrounds, tags

### Neutral Colors
- **Dark Navy**: `#1a1a2e` - Headings, footer background
- **Dark Navy Light**: `#2d2d44` - Secondary dark backgrounds
- **Gray 900**: `#111827` - Primary text
- **Gray 700**: `#374151` - Secondary text
- **Gray 600**: `#4b5563` - Body text
- **Gray 500**: `#6b7280` - Muted text
- **Gray 400**: `#9ca3af` - Placeholder text
- **Gray 200**: `#e5e7eb` - Borders
- **Gray 100**: `#f3f4f6` - Light backgrounds
- **Gray 50**: `#f8fafc` - Page background

### Accent Colors
- **Success Green**: `#16a34a` - Checkmarks, success states
- **Error Red**: `#dc2626` - Error states
- **Info Blue**: `#4f46e5` - Category badges

## Typography

### Font Families
- **Headings**: 'Playfair Display', serif - For logo and hero highlights
- **Body**: 'Inter', system-ui, sans-serif - For all body text

### Font Sizes
- Hero Title: 3.5rem (56px)
- Section Title: 2.5rem (40px)
- Card Title: 1.25-1.5rem (20-24px)
- Body: 1rem (16px)
- Small: 0.85-0.9rem (14px)
- Caption: 0.75rem (12px)

### Font Weights
- Regular: 400
- Medium: 500
- Semibold: 600
- Bold: 700
- Extra Bold: 800

## Spacing System
- Base unit: 0.25rem (4px)
- Section padding: 5rem (80px) vertical
- Container max-width: 1280px
- Card padding: 2rem (32px)
- Grid gap: 2rem (32px)

## Layout Patterns

### Section Structure
- Max width: 1280px centered
- Padding: 5rem vertical, 2rem horizontal
- Header: Centered with tag, title, subtitle

### Grid Systems
- 4-column: For services, trust points
- 3-column: For case studies, process steps
- 2-column: For problems, form rows
- 1-column: Mobile, FAQs

### Responsive Breakpoints
- Desktop: 1024px+
- Tablet: 768px - 1023px
- Mobile: < 768px

## Components

### Buttons
- Primary: Orange background, white text, rounded 8px
- Secondary: White background, dark border, dark text
- Hover: Darker shade, slight lift with shadow

### Cards
- Background: White or light gray
- Border: 1px solid gray-200
- Border-radius: 12-16px
- Hover: Lift effect with shadow

### Tags/Badges
- Background: Light orange (primary light)
- Text: Primary orange
- Border-radius: 20px (pill shape)

### Form Inputs
- Border: 1px solid gray-300
- Border-radius: 8px
- Focus: Orange border with subtle shadow
- Padding: 0.875rem 1rem

## Visual Effects
- Card hover: translateY(-5px) with shadow
- Button hover: translateY(-2px) with shadow
- Transitions: 0.2s ease for all interactive elements
- Background patterns: Subtle geometric patterns for hero sections

### Inner Pages

- Content sections

- CTA section

## Do's and Don'ts

### Do
- Use orange (#c2410c) for primary actions and highlights
- Keep text readable with proper contrast (gray-600 on white)
- Use consistent spacing (2rem gaps, 5rem section padding)
- Include trust signals (stats, testimonials, certifications)
- Make CTAs prominent and consistent

### Don't
- Use too many colors - stick to the palette
- Use black (#000) for text - use gray-900 instead
- Overcrowd layouts - maintain breathing room
- Use generic placeholder images
- Mix font families unnecessarily
