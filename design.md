# Velmora Fine Jewelry — Design System

## Visual Identity
- **Mood**: Quiet luxury, warm, editorial. Premium demi-fine jewelry — NOT loud, NOT discount-looking.
- **Target**: Women 25–45, gifting and self-purchase, premium-but-accessible ($30–$120).

## Color Palette
- **Base (deep warm charcoal)**: `#1C1917` (stone-900) — primary background for hero/dark sections
- **Cream (warm off-white)**: `#FAF7F2` — primary light background
- **Gold Accent**: `#C9A96E` — primary accent, buttons, highlights
- **Gold Light**: `#D4B87A` — hover states
- **Text Dark**: `#1C1917` — on light backgrounds
- **Text Light**: `#FAF7F2` — on dark backgrounds
- **Muted**: `#78716C` (stone-500) — secondary text, dividers
- **Border**: `#E7E5E4` (stone-200) — hairline dividers

## Typography
- **Headings**: Cormorant Garamond (serif), elegant, uppercase with wide letter-spacing for product names
- **Body/UI**: Inter (sans-serif), clean and readable
- **Product names**: UPPERCASE, tracking-widest (0.2em+), font-semibold
- **Section headings**: Uppercase, tracking-wider, Cormorant Garamond

## Spacing & Layout
- Generous whitespace throughout
- Max content width: 1280px
- Section padding: 80px–120px vertical (desktop), 48px–64px (mobile)
- Hairline dividers (1px, stone-200)
- Large editorial imagery

## Buttons
- Primary: solid gold (#C9A96E) background, cream text, rounded-none, uppercase, tracking-wider, font-semibold
- Secondary: outlined (1px gold border), gold text, transparent bg, same typography
- Hover: slight darkening of gold, smooth transition (200ms)
- Padding: py-3 px-8

## Cards
- Minimal, no visible border by default
- Subtle shadow on hover (shadow-md)
- Image-first, generous image area
- Product name in uppercase serif with wide tracking

## Animations
- Subtle hover transitions (200–300ms ease)
- Soft shadows on card hover
- Smooth scroll behavior
- No jarring animations

## Do's
- Use warm, muted tones that flatter gold jewelry
- Keep whitespace generous
- Use serif for emotional/editorial moments
- Ensure strong contrast for accessibility
- Use thin hairline dividers

## Don'ts
- Don't use bright/saturated colors
- Don't crowd layouts
- Don't use generic e-commerce patterns (badges, loud CTAs)
- Don't use rounded buttons (keep them sharp/rectangular for editorial feel)
- Don't use low-contrast text
