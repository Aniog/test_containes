# SSourcing China - Design Guidelines

## Project Overview
- **Project Name**: SSourcing China Website
- **Type**: B2B Corporate Website
- **Purpose**: Generate qualified sourcing inquiries for China sourcing agent services
- **Target Audience**: Overseas buyers, procurement managers, business owners looking to source from China

## Visual Style

### Typography
- **Primary Font**: Inter (Google Fonts) - clean, professional, highly readable
- **Headings**: 
  - H1: 48px/56px, font-weight 700
  - H2: 36px/44px, font-weight 600
  - H3: 24px/32px, font-weight 600
  - H4: 20px/28px, font-weight 500
- **Body**: 16px/24px, font-weight 400
- **Small**: 14px/20px, font-weight 400

### Color Palette
- **Primary**: #1E3A5F (Deep Navy Blue) - trust, professionalism
- **Primary Light**: #2D5A8A (Medium Blue)
- **Secondary**: #F7931E (Orange Accent) - CTAs, energy
- **Secondary Hover**: #E0850D (Darker Orange)
- **Background Primary**: #FFFFFF (White)
- **Background Secondary**: #F8FAFC (Light Gray-Blue)
- **Background Tertiary**: #F1F5F9 (Slate 100)
- **Text Primary**: #1E293B (Slate 800)
- **Text Secondary**: #64748B (Slate 500)
- **Text Muted**: #94A3B8 (Slate 400)
- **Border**: #E2E8F0 (Slate 200)
- **Success**: #10B981 (Emerald 500)
- **Error**: #EF4444 (Red 500)

### Spacing System
- **Base unit**: 4px
- **Section padding**: 80px vertical (desktop), 48px (mobile)
- **Container max-width**: 1200px
- **Grid gap**: 24px (desktop), 16px (mobile)
- **Card padding**: 24px
- **Button padding**: 12px 24px

### Visual Effects
- **Border radius**: 8px (cards), 6px (buttons), 4px (inputs)
- **Shadows**: 
  - Card: 0 4px 6px -1px rgba(0,0,0,0.1)
  - Elevated: 0 10px 15px -3px rgba(0,0,0,0.1)
  - Button hover: translateY(-2px) with shadow increase
- **Transitions**: 200ms ease-in-out

## Layout Structure

### Header
- Fixed position, white background with subtle shadow on scroll
- Logo (left): "SSourcing China" text logo
- Navigation (center): Home, Services, How It Works, Products, Case Studies, Blog, Contact
- CTA Button (right): "Get a Free Quote" - orange accent color

### Footer
- Dark navy background (#1E3A5F)
- 4-column layout: Company info, Services, Resources, Contact
- Social links and copyright
- Newsletter signup

### Responsive Breakpoints
- **Mobile**: < 640px (single column, hamburger menu)
- **Tablet**: 640px - 1024px (2 columns)
- **Desktop**: > 1024px (full layout)

## Page Sections

### Homepage
1. **Hero Section**
   - Full-width with subtle factory/warehouse background image
   - Headline: "China Sourcing Agent for Global Buyers"
   - Subheadline: Supporting overseas businesses with supplier verification, quality control, and logistics
   - Primary CTA: "Get a Free Sourcing Quote" (orange)
   - Secondary CTA: "Learn How It Works"
   - Trust badges: Years in business, Clients served, Suppliers verified

2. **Services Section**
   - 6 service cards in 3x2 grid
   - Services: Supplier Verification, Quality Inspection, Production Follow-up, Shipping & Logistics, Product Sourcing, Custom Solutions
   - Each card: Icon, title, brief description, "Learn more" link

3. **Problems We Solve Section**
   - 2-column layout with image on left
   - Common pain points: Language barriers, Quality issues, Scam prevention, Logistics complexity
   - How we solve each problem

4. **Sourcing Process Section**
   - 5-step horizontal timeline/process
   - Steps: Submit Request → Supplier Matching → Factory Verification → Production & QC → Shipping & Delivery

5. **Products We Source Section**
   - Category grid (6 categories)
   - Categories: Electronics, Machinery, Textiles, Furniture, Packaging, Consumer Goods

6. **Trust Points Section**
   - Statistics: 500+ Suppliers Verified, 1000+ Inspections, 50+ Countries, 98% Client Satisfaction
   - Certifications, partnerships

7. **Case Studies Section**
   - 3 featured case study cards
   - Each: Industry, challenge, solution, result

8. **FAQ Section**
   - Accordion with 6-8 common questions

9. **CTA/Contact Section**
   - Inquiry form: Name, Email, Company, Product Details, Message
   - Contact information sidebar

10. **Blog Preview Section**
    - 3 latest blog posts preview

## Components

### Buttons
- **Primary**: Orange background (#F7931E), white text, rounded
- **Secondary**: Navy outline, navy text
- **Ghost**: No background, text link style

### Cards
- White background, subtle shadow
- Hover: slight lift effect
- Icon + title + description layout

### Form Inputs
- Light gray background (#F8FAFC)
- Border on focus (primary blue)
- Clear labels and placeholders

### Icons
- Lucide React icons
- Consistent sizing: 24px (default), 32px (featured)

## Image Guidelines
- Use professional B2B imagery
- Factory/warehouse photos for backgrounds
- QC inspection photos
- Shipping/logistics imagery
- Professional team photos
- Use stock images for placeholders

## Tone & Voice
- Professional and confident
- Clear and practical
- No exaggerated claims
- Focus on results and reliability
- Use "we" for company, "you" for prospective clients
