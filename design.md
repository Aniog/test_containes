# Design Guidelines

## Visual Style
A modern, clean, and professional look for a creative agency.

- **Typography**: 
  - Sans-serif font (Inter) for a modern feel.
  - Large, bold headings for impact.
- **Colors**: 
  - Primary: Slate-900 (Backgrounds, Headings)
  - Secondary: Slate-600 (Subtext)
  - Accent: Blue-600 (Links, Buttons, Highlights)
  - Background: White / Slate-50
- **Borders & Spacing**:
  - Generous padding and margins for a "breathable" layout.
  - Subtle rounded corners (rounded-lg) for cards and buttons.
- **Visual Styles**:
  - Subtle shadows (shadow-sm) for depth.
  - Responsive grids for content.

## Do's
- Use Tailwind spacing utilities consistently.
- Ensure high contrast for readability.
- Use Lucide icons for visual interest.

## Don'ts
- Don't use magic values/hex codes outside of Tailwind configuration.
- Don't overload the page with too much text.
- Don't neglect mobile responsiveness.
