# ARTITECT MACHINERY — Design Guidelines

## Brand Identity
ARTITECT MACHINERY is a premium industrial machinery brand. It combines the precision of architecture ("Architect") with the technical expertise of machinery. The visual identity must feel:
- **Elegant** — refined, premium, sophisticated
- **User-friendly** — approachable, clear, inviting
- **Professional** — trustworthy, precise, capable

## Color Palette
- **Primary**: Deep navy blue (#1a2744) — professionalism, trust, industrial strength
- **Primary Light**: (#2a3f64) — hover states, secondary elements
- **Accent**: Warm copper/amber (#c8553d or #d4875e) — elegance, warmth, CTAs
- **Accent Light**: (#e8a87c) — hover states for accent elements
- **Background**: Warm off-white (#f7f5f0) — softer than pure white
- **Surface**: White (#ffffff) — cards, sections
- **Text Primary**: Dark charcoal (#1f2937) — body text
- **Text Muted**: (#6b7280) — secondary text, captions
- **Border**: Light warm gray (#e5e0d8) — dividers, borders

## Typography
- **Headings**: Inter, 700 weight, tracking-tight
- **Body**: Inter, 400 weight, leading-relaxed
- **Font sizes**: 
  - Hero heading: text-5xl md:text-7xl
  - Section titles: text-3xl md:text-4xl
  - Card titles: text-xl
  - Body: text-base
  - Small/captions: text-sm

## Spacing
- Section padding: py-20 md:py-28
- Container max-width: max-w-7xl mx-auto
- Gap between cards: gap-8
- Section title bottom margin: mb-12 md:mb-16

## Components Style

### Navigation
- Fixed top, transparent initially, slight blur background on scroll
- Logo on left, nav links on right
- White/light text on dark sections, dark text on light sections

### Hero Section
- Full-viewport height with dark overlay
- Large heading, descriptive subtitle
- Two CTAs: primary (accent) and secondary (outline)
- Background image showcasing machinery

### Cards (Products/Features)
- Rounded corners (rounded-xl)
- Subtle shadow (shadow-sm hover:shadow-md)
- Clean borders (border border-border)
- Smooth transitions on hover
- Icon + title + short description

### Section Layout
- Alternating light and dark backgrounds
- Centered titles with subtle decoration
- Generous whitespace

### Buttons
- Primary: filled accent color, rounded-lg, px-8 py-3
- Secondary: outline with border, same radius
- Hover: slight lift/scale effect

## Do's
- Use warm off-white backgrounds for readability
- Keep generous whitespace
- Use subtle shadows for depth
- Use Inter font consistently
- Maintain consistent section spacing

## Don'ts
- Don't use pure black text (#000)
- Don't overcrowd sections
- Don't use overly bright or saturated colors
- Don't skip responsive design
- Don't use multiple accent colors
