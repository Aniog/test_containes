# ARTITECT MACHINERY - Design Specification

## Brand Identity
- **Brand Name**: ARTITECT MACHINERY
- **Industry**: Industrial machinery manufacturing (sheet metal folding machines)
- **Products**: Double folding machine, double folder, sheet metal folder, sheet metal folding machine, metal folder, metal folder machine, metal folding machine
- **Style**: Elegant but user-friendly

## Visual Style

### Color Palette
- **Primary**: Deep Navy Blue `#1a365d` - conveys professionalism and industrial strength
- **Secondary**: Steel Gray `#64748b` - represents metal/machinery
- **Accent**: Golden Amber `#d97706` - adds warmth and highlights CTAs
- **Background Light**: `#f8fafc` - clean, professional white-gray
- **Background Dark**: `#0f172a` - for contrast sections
- **Text Primary**: `#1e293b` - dark slate for readability
- **Text Light**: `#f1f5f9` - for dark backgrounds

### Typography
- **Headings**: Inter (Google Font) - clean, modern, professional
- **Body**: Inter - consistent, highly readable
- **Font Weights**: 300 (light), 400 (regular), 500 (medium), 600 (semibold), 700 (bold), 800 (extra bold)

### Design Principles
1. **Elegant**: Clean lines, generous whitespace, sophisticated typography
2. **User-Friendly**: Clear navigation, intuitive layout, accessible CTAs
3. **Industrial Touch**: Subtle geometric patterns, metallic accents, precision imagery
4. **Professional**: Trust-building layout, clear value proposition

## Layout Structure

### Header/Navigation
- Logo on left
- Navigation links: Home, Products, About, Contact
- CTA button: "Get Quote"
- Sticky on scroll with subtle shadow

### Hero Section
- Full-width background with gradient overlay
- Bold headline with brand name
- Subheadline describing products
- Primary CTA button
- Secondary info link

### Products Section
- Section title with subtle underline
- Grid layout (3 columns desktop, 1 mobile)
- Product cards with:
  - Icon or image placeholder
  - Product name
  - Brief description
  - "Learn More" link

### Features/Benefits Section
- Icon-based feature list
- 3-4 key benefits displayed in grid
- Professional imagery suggestion

### About Section
- Company introduction
- Key statistics or trust indicators
- Brief company story

### Contact Section
- Contact form (Name, Email, Phone, Message)
- Company contact information
- Location/address if applicable

### Footer
- Brand name and tagline
- Quick links
- Contact information
- Copyright

## Component Guidelines

### Buttons
- Primary: Golden amber background, dark text, rounded corners
- Secondary: Outlined with navy border, navy text
- Hover: Slight scale and shadow increase

### Cards
- White background with subtle shadow
- Rounded corners (8px)
- Hover lift effect
- Clear hierarchy: title, description, action

### Form Inputs
- Clean bordered inputs
- Focus state with accent color
- Clear labels
- Proper spacing

### Icons
- Use Lucide React icons
- Consistent stroke width
- Match color scheme

## Responsive Breakpoints
- Mobile: < 640px (single column)
- Tablet: 640px - 1024px (2 columns)
- Desktop: > 1024px (3+ columns)

## Do's and Don'ts
- DO: Use clean whitespace generously
- DO: Keep CTAs prominent and accessible
- DO: Use professional, high-quality imagery placeholders
- DO: Maintain consistent spacing
- DON'T: Use too many colors (stick to palette)
- DON'T: Clutter the design with too many elements
- DON'T: Use low contrast text
- DON'T: Make navigation confusing
