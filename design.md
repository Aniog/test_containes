# ARTITECT MACHINERY — Design System

An elegant, engineering-forward website for an industrial sheet-metal folding machine manufacturer. The visual language is refined, confident, and user-friendly.

## Brand Voice
- Precise, confident, trustworthy, and modern.
- Industrial elegance: steel, graphite, and clean lines.
- Friendly clarity for non-technical buyers.

## Color Palette
- Background: `#0b0f19` (near-black, industrial)
- Surface: `#111827` (dark card / section)
- Surface highlight: `#1f2937`
- Primary accent: `#38bdf8` (cyan blue — precision, technology)
- Secondary accent: `#f59e0b` (amber — warning, highlights)
- Text primary: `#f8fafc` (off-white)
- Text secondary: `#94a3b8` (cool gray)
- Border: `#334155`
- Muted: `#475569`

## Typography
- Headings: Inter, 700–800, tight line-height.
- Body: Inter, 400–500, line-height 1.6.
- Hero headline: 3.5rem / 56px on desktop, 2.25rem / 36px on mobile.
- Section headline: 2rem / 32px.
- Body: 1rem / 16px.

## Spacing
- Page horizontal padding: `px-4 sm:px-6 lg:px-12 xl:px-20`.
- Section vertical padding: `py-16 md:py-24`.
- Grid gaps: `gap-6 md:gap-8`.
- Card padding: `p-6 md:p-8`.

## Components
- Buttons: rounded-md, uppercase tracking-wide text-xs font-semibold.
  - Primary: `bg-cyan-500 text-slate-950 hover:bg-cyan-400`.
  - Secondary / outline: `border border-slate-600 text-slate-100 hover:border-cyan-500 hover:text-cyan-400`.
- Cards: `bg-slate-900 border border-slate-800 rounded-xl`.
- Inputs: `bg-slate-900 border border-slate-700 rounded-md focus:border-cyan-500 focus:ring-1 focus:ring-cyan-500`.

## Do's
- Use high-contrast text on dark surfaces.
- Use generous whitespace and clear visual hierarchy.
- Include real product names and benefits prominently.
- Use stock-image tags for product / factory visuals.

## Don'ts
- No neon-on-dark low-contrast body text.
- No cluttered single-column desktop layouts.
- No purely decorative imagery without alt text or tags.
