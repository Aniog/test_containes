# Visual Style Guide: SSourcing China

## Brand Identity
- **Name:** SSourcing China
- **Goal:** Generate qualified sourcing inquiries from overseas buyers.
- **Tone:** Professional, clear, practical, no exaggerated claims, concise, trustworthy.
- **Style:** Clean, international B2B, robust, data-focused, realistic factory/QC/shipping visuals.

## Color Palette
- **Primary:** Deep Blue `bg-blue-800` (represents trust, professionalism, reliability)
- **Secondary:** Slate `bg-slate-700` (for secondary elements, borders, subtle text)
- **Accent:** Orange `bg-orange-500` (for calls to action, alerts, highlights)
- **Background (Main):** White `bg-white`
- **Background (Alternate):** Off-White/Light Gray `bg-slate-50`
- **Text (Main):** Slate 900 `text-slate-900`
- **Text (Muted):** Slate 600 `text-slate-600`

## Typography
- **Font Family:** Inter (or similar clean sans-serif like Roboto, standard system fonts)
- **Headings:** Bold `font-bold`, tight tracking `tracking-tight`
- **Body:** Regular `font-normal`, leading-relaxed `leading-relaxed`

## Component Styles
- **Buttons (Primary):** `bg-orange-500 hover:bg-orange-600 text-white font-medium py-3 px-6 rounded-md transition-colors shadow-sm`
- **Buttons (Secondary):** `bg-blue-800 hover:bg-blue-900 text-white font-medium py-3 px-6 rounded-md transition-colors shadow-sm`
- **Buttons (Outline):** `border-2 border-blue-800 text-blue-800 hover:bg-blue-50 font-medium py-2.5 px-6 rounded-md transition-colors`
- **Cards:** `bg-white rounded-lg border border-slate-200 shadow-sm p-6 hover:shadow-md transition-shadow`
- **Inputs:** `w-full border border-slate-300 rounded-md px-4 py-2 focus:ring-2 focus:ring-blue-500 focus:border-blue-500 outline-none text-slate-900 bg-white`
- **Borders:** `border-slate-200`
- **Shadows:** Subtle shadows to create depth without being overwhelming (`shadow-sm` for normal state, `shadow-md` on hover).

## Layout & Spacing
- Use generous padding for sections (`py-16` to `py-24`).
- Content width usually bound by `max-w-7xl mx-auto`.
- Grid layouts for services, products, and process steps.
- Clear separation between distinct sections using alternate background colors.

## Images
- Use `data-strk-img` and `data-strk-bg` tags for dynamic stock image loading.
- Focus on practical B2B imagery: factory inspections, shipping containers, quality control processes, manufacturing lines, meetings.
- Image styles: high quality, professional, realistic (no overly abstract or cartoony SVGs).

## Do's and Don'ts
- **Do:** Use clear, straightforward language. Add plenty of whitespace. Ensure high contrast for text.
- **Don't:** Don't use overly vibrant, playful tech-startup colors (like neon pink or lime green). Don't crowd too much information into small spaces without structure. Don't center-align long blocks of text.
