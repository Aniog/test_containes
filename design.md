# Velmora Fine Jewelry — Design System

## Color Palette
- **Background**: `#FAF8F5` (warm ivory) — primary bg
- **Background Alt**: `#F3EFEA` (slightly deeper warm neutral)
- **Foreground**: `#1A1714` (warm near-black) — primary text
- **Muted**: `#8B8580` — secondary text, labels
- **Accent**: `#B8860B` (dark goldenrod / `gold-dark`) — CTAs, highlights
- **Accent Light**: `#D4A843` (warm gold) — hover states, icons
- **Accent Soft**: `#F5ECD7` — light gold tint for backgrounds
- **Border**: `#E8E2DA` — hairline dividers
- **Card**: `#FFFFFF` — card backgrounds
- **Dark**: `#1A1714` — navbar solid, footer

## Typography
- **Headings**: Cormorant Garamond (serif) — elegant, editorial
- **Body/UI**: Inter (sans-serif) — clean, readable
- **Product Names**: Uppercase, `tracking-[0.2em]`, serif font
- **Logo**: Cormorant Garamond, `text-2xl`, `tracking-[0.3em]`, uppercase

## Spacing & Layout
- Generous whitespace: `py-16` to `py-24` for sections
- Max content width: `max-w-7xl mx-auto`
- Grid gaps: `gap-6` to `gap-8`
- Hairline dividers: `border border-border`

## Buttons
- **Primary**: bg-accent, text-white, uppercase, tracking-wider, rounded-none
- **Outlined**: border-accent, text-accent, bg-transparent
- Hover: darken slightly, subtle shadow

## Cards
- White background, subtle shadow on hover
- Image aspect ratio: 4/5 or 3/4
- Hover: slight scale, shadow increase

## Transitions
- `transition-all duration-300 ease-in-out`
- Hover scale: `hover:scale-[1.02]`
- Nav scroll: smooth backdrop transition
