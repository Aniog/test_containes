# SSourcing China — Design System

## Brand Identity
- **Company**: SSourcing China
- **Tagline**: China Sourcing Agent for Global Buyers
- **Tone**: Professional, trustworthy, practical, international B2B

## Color Palette
- **Primary Blue**: `#1A3C6E` — deep navy, trust and authority (`bg-[#1A3C6E]`, `text-[#1A3C6E]`)
- **Accent Red**: `#C0392B` — China-inspired accent, CTAs (`bg-[#C0392B]`, `text-[#C0392B]`)
- **Light Blue**: `#EBF2FA` — section backgrounds (`bg-[#EBF2FA]`)
- **Neutral Gray**: `#F7F8FA` — alternate section backgrounds
- **Text Dark**: `#1A1A2E` — primary body text
- **Text Muted**: `#6B7280` — secondary text, captions
- **White**: `#FFFFFF` — cards, nav, footer text on dark

## Typography
- **Font**: Inter (Google Fonts)
- **Headings**: `font-bold`, sizes: `text-4xl` (h1), `text-3xl` (h2), `text-2xl` (h3), `text-xl` (h4)
- **Body**: `text-base` or `text-lg`, `text-[#1A1A2E]`
- **Muted**: `text-[#6B7280]`
- **CTA text**: `text-white font-semibold`

## Spacing
- Section padding: `py-16 md:py-24`
- Container: `max-w-7xl mx-auto px-4 sm:px-6 lg:px-8`
- Card padding: `p-6` or `p-8`
- Gap between cards: `gap-6` or `gap-8`

## Components

### Buttons
- **Primary CTA**: `bg-[#C0392B] hover:bg-[#a93226] text-white font-semibold px-6 py-3 rounded-lg transition-colors`
- **Secondary**: `bg-[#1A3C6E] hover:bg-[#15305a] text-white font-semibold px-6 py-3 rounded-lg transition-colors`
- **Outline**: `border-2 border-[#1A3C6E] text-[#1A3C6E] hover:bg-[#1A3C6E] hover:text-white px-6 py-3 rounded-lg transition-colors`

### Cards
- `bg-white rounded-xl shadow-sm border border-gray-100 p-6 hover:shadow-md transition-shadow`

### Section Headers
- Centered label: `text-sm font-semibold text-[#C0392B] uppercase tracking-wider mb-2`
- Main heading: `text-3xl md:text-4xl font-bold text-[#1A3C6E] mb-4`
- Subtext: `text-lg text-[#6B7280] max-w-2xl mx-auto`

### Navbar
- Background: `bg-white` with `shadow-sm`
- Logo text: `text-[#1A3C6E] font-bold text-xl`
- Nav links: `text-[#1A1A2E] hover:text-[#1A3C6E] font-medium`
- CTA button in nav: primary red

### Footer
- Background: `bg-[#1A3C6E]`
- Text: `text-white` and `text-blue-200`

## Do's
- Use navy blue for headings and structural elements
- Use red only for CTAs and accent highlights
- Keep layouts clean with generous whitespace
- Use card grids for services and features
- Use numbered steps for process sections
- Always show trust signals (years, clients, countries)

## Don'ts
- No dark text on dark backgrounds
- No light gray text on white (use `text-[#6B7280]` minimum)
- No exaggerated claims or superlatives
- No cluttered layouts
- No more than 3 columns on mobile
