# Velmora Fine Jewelry — Design System

## Overview
Quiet luxury, editorial, warm. Premium demi-fine jewelry. Target audience: women 25–45.

## Color Palette
- **Background:** `#0f0f0f` (deep charcoal — near-black, elegant base)
- **Surface / Cards:** `#1a1a1a` (dark charcoal)
- **Surface Elevated:** `#222222` (cards, product tiles)
- **Text Primary:** `#f5f0eb` (warm off-white)
- **Text Secondary:** `#a8a29e` (warm gray)
- **Text Muted:** `#6b6560` (for meta)
- **Accent / Gold:** `#c9a96e` (warm gold metallic)
- **Accent Hover:** `#d4b98a` (lighter gold)
- **Border / Divider:** `#2a2a2a` (hairline)
- **Border Light:** `#333333`
- **Rose Gold accent:** `#c97e6e` (for variant)
- **Success:** `#4ade80`
- **Error:** `#f87171`

## Typography
- **Headings:** 'Cormorant Garamond', serif — weights 400, 500, 600, 700
- **Body / UI:** 'Inter', sans-serif — weights 300, 400, 500
- **Product names:** UPPERCASE, letter-spacing: 0.15em, Inter 500
- **Price:** Inter 400 or Cormorant Garamond 500
- **Logo:** Cormorant Garamond 700, uppercase, letter-spacing: 0.3em

## Spacing
- Page max-width: 1440px, centered
- Section padding: py-24 (96px) desktop, py-16 (64px) mobile
- Grid gap: gap-6 (24px) / gap-8 (32px)
- Card padding: p-6 (24px)
- Content max-width: 1200px

## Borders & Dividers
- Hairline dividers: border-t border-[#2a2a2a]
- Thin borders on cards: border border-[#2a2a2a]
- Focus rings: ring-1 ring-[#c9a96e]

## Shadows
- Card hover: shadow-lg shadow-black/30
- Subtle glow on gold elements: shadow-[0_0_15px_rgba(201,169,110,0.15)]

## Buttons
- **Primary (accent):** bg-[#c9a96e] text-[#0f0f0f] font-medium hover:bg-[#d4b98a] transition-all
- **Outline:** border border-[#c9a96e] text-[#c9a96e] hover:bg-[#c9a96e] hover:text-[#0f0f0f]
- **Ghost:** text-[#a8a29e] hover:text-[#f5f0eb]

## Animations
- Hover transitions: 300ms ease-out
- Fade-in on scroll: opacity-0 translate-y-4 → opacity-100 translate-y-0
- Image hover: scale-105 with 400ms ease-out

## Do's
- Use generous whitespace
- Use large editorial-style images
- Keep UI minimal and uncluttered
- Use thin lines and subtle separators
- Gold accents should be used sparingly for emphasis

## Don'ts
- No bright colors or loud patterns
- No heavy borders or thick lines
- No discount/voucher aesthetic
- No generic e-commerce patterns
- No cluttered layouts
