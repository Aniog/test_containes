# Velmora Fine Jewelry — Design System

## Visual Identity
- **Mood**: Quiet luxury, warm, editorial. Premium demi-fine jewelry — NOT loud, NOT discount-looking.
- **Direction**: Warm neutral base with gold metallic accents. Deep refined base + warm metallic.

## Color Palette
- **Background**: `#FAF7F2` (brand-cream) — warm off-white, primary background
- **Surface**: `#F5F0E8` (brand-ivory) — cards, elevated surfaces
- **Border/Divider**: `#E8DFD0` (brand-sand) — hairline dividers, subtle borders
- **Accent**: `#C9A96E` (brand-gold) — CTAs, links, highlights, hover states
- **Accent Hover**: `#A8863F` (brand-gold-dark) — button hover
- **Text Primary**: `#2C2C2C` (brand-charcoal) — headings, body text
- **Text Secondary**: `#6B5E50` (brand-warm-gray) — descriptions, captions
- **Text Muted**: `#9B8E7E` (brand-muted) — placeholders, disabled states
- **Dark Surface**: `#1A1A1A` (brand-onyx) — hero overlays, dark sections

## Typography
- **Headings**: Cormorant Garamond (serif), weights 400-600
- **Body/UI**: Inter (sans-serif), weights 300-500
- **Product Names**: UPPERCASE, letter-spacing 0.2em, serif font
- **Section Headings**: UPPERCASE, letter-spacing 0.15em, serif font
- **Nav Links**: UPPERCASE, letter-spacing 0.15em, sans-serif, weight 500

## Spacing & Layout
- Generous whitespace throughout
- Max content width: 1280px
- Section padding: 80px vertical (desktop), 48px (mobile)
- Card gaps: 24px
- Hairline dividers: 1px solid brand-sand

## Buttons
- **Primary (Accent)**: bg-brand-gold, text-white, hover:bg-brand-gold-dark, rounded-none, uppercase, tracking-super-wide, font-sans, weight 500, padding 14px 32px
- **Secondary (Outlined)**: border-brand-gold, text-brand-gold, hover:bg-brand-gold hover:text-white, rounded-none, uppercase, tracking-super-wide
- **Ghost**: text-brand-charcoal, hover:text-brand-gold, no border

## Cards
- Product cards: no border, subtle shadow on hover, image aspect 3x4
- Hover: second image fade-in, "Add to Cart" overlay slide-up

## Images
- Editorial, warm-lit, gold jewelry on dark/neutral backgrounds
- Hero: full-bleed, 16x9 aspect
- Product: 3x4 portrait
- UGC: 9x16 vertical
- Category tiles: 4x3

## Animations
- Subtle, smooth transitions (200-300ms ease)
- Hover: opacity transitions, gentle scale
- Page transitions: fade-in
- Cart drawer: slide-in from right

## Do's
- Use serif for all headings and product names
- Use generous whitespace
- Use hairline dividers
- Use warm, muted tones
- Keep layouts clean and editorial

## Don'ts
- Don't use bright/saturated colors
- Don't use rounded buttons (use sharp/slightly rounded)
- Don't crowd content
- Don't use generic e-commerce patterns (badges, aggressive sales copy)
- Don't use dark text on dark backgrounds without contrast check
