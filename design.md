# Velmora Fine Jewelry - Design System

## Visual Identity
Quiet luxury, warm, editorial. Premium demi-fine jewelry. Generous whitespace, thin hairline dividers.

## Color Palette
- **Deep Base**: `#121212` (Tailwind: `bg-stone-950`) - For dark sections and text.
- **Parchment**: `#F9F7F2` (Tailwind: `bg-[#F9F7F2]`) - Primary light background.
- **Worn Gold**: `#C5A059` (Tailwind: `text-[#C5A059]`, `bg-[#C5A059]`) - Primary brand color for buttons and accents.
- **Muted Bronze**: `#8B735B` (Tailwind: `text-[#8B735B]`) - Secondary accents.
- **Border**: `#E5E7EB` (Tailwind: `border-stone-200`) - Hairline dividers.

## Typography
- **Headings/Serif**: `Cormorant Garamond`, serif. Used for headlines, product names, and editorial text.
- **Body/Sans**: `Inter`, sans-serif. Used for UI elements, descriptions, and functional text.

**Styles:**
- Product Names: `font-serif uppercase tracking-[0.2em] font-medium`
- Section Titles: `font-serif text-3xl md:text-5xl italic`
- Body Text: `font-sans leading-relaxed text-stone-600`
- CTA/Buttons: `font-sans uppercase tracking-widest text-xs font-semibold`

## UI Elements
- **Buttons**:
  - Solid: `bg-[#C5A059] text-white px-8 py-4 uppercase tracking-widest text-xs hover:bg-[#B48F4A] transition-colors`
  - Outline: `border border-[#C5A059] text-[#C5A059] px-8 py-4 uppercase tracking-widest text-xs hover:bg-[#C5A059] hover:text-white transition-all`
- **Cards**: Minimalist, no borders, soft shadows on hover.
- **Dividers**: `h-[1px] bg-stone-200 w-full`
- **Transitions**: `duration-500 ease-in-out`
