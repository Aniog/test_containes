# Velmora Fine Jewelry — Design System

## Visual Identity
- **Mood**: Quiet luxury, warm, editorial. Premium demi-fine jewelry — NOT loud, NOT discount-looking.
- **Target**: Women 25–45, gifting and self-purchase, $30–$120 price point.

## Color Palette
- **Background**: `#FAF7F2` (warm cream) — primary page background
- **Surface**: `#F5F0E8` (ivory) — cards, elevated surfaces
- **Sand**: `#E8DFD0` — subtle backgrounds, hover states
- **Gold Accent**: `#C9A96E` — primary accent, CTAs, links, highlights
- **Gold Light**: `#D4BA8A` — hover state for gold
- **Gold Dark**: `#A8864A` — active/pressed state for gold
- **Text Primary**: `#2C2C2C` (charcoal) — headings, body text
- **Text Secondary**: `#6B6560` (warm gray) — descriptions, captions
- **Text Muted**: `#9B9490` — placeholders, disabled states
- **Border**: `#D9D0C4` — hairline dividers, card borders
- **Dark Surface**: `#1A1714` (warm black) — hero overlays, footer

## Typography
- **Headings**: Cormorant Garamond (serif), weights 400–600
- **Body/UI**: Inter (sans-serif), weights 300–500
- **Product Names**: UPPERCASE, letter-spacing 0.25em, serif
- **Section Headings**: UPPERCASE, letter-spacing 0.15em, serif
- **Body Text**: 14–16px, line-height 1.6–1.8
- **Small/Caption**: 12px, letter-spacing 0.05em

## Spacing & Layout
- Generous whitespace throughout
- Max content width: 1280px
- Section padding: 80–120px vertical (desktop), 48–64px (mobile)
- Card padding: 24–32px
- Hairline dividers: 1px, color brand-warm-border

## Buttons
- **Primary (Accent)**: bg-brand-gold, text-white, rounded-none, uppercase, letter-spacing 0.15em, font-sans, text-sm, px-8 py-3
- **Primary Hover**: bg-brand-gold-dark
- **Secondary (Outlined)**: border-brand-gold, text-brand-gold, bg-transparent, same sizing
- **Secondary Hover**: bg-brand-gold, text-white

## Cards
- Minimal: no border or 1px border-brand-warm-border
- Soft shadow on hover: shadow-md
- Image-first layout, generous padding below image

## Images
- Editorial, warm-lit, close-up jewelry photography
- Dark/neutral backgrounds for product shots
- 3x2 or 4x3 ratio for product cards
- 16x9 for hero/banner
- 9x16 for UGC/reel cards

## Animations
- Subtle, smooth transitions (200–400ms ease)
- Hover: slight scale (1.02), shadow increase
- Page transitions: fade-in, slide-up
- No bouncy or playful animations

## Do's
- Use serif for all headings and product names
- Use generous whitespace between sections
- Use hairline dividers between sections
- Keep accent color (gold) restrained — for CTAs and highlights only
- Ensure strong text contrast on all backgrounds

## Don'ts
- Don't use bright/saturated colors
- Don't use rounded buttons (use sharp/slightly rounded)
- Don't use playful or bouncy animations
- Don't mix too many font weights
- Don't use generic e-commerce patterns (badges, countdown timers, etc.)
