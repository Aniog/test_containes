# Velmora Fine Jewelry — Design System

## Brand Identity
Quiet luxury. Warm. Editorial. Premium demi-fine jewelry for women 25–45.

## Color Palette
| Token | Hex | Usage |
|---|---|---|
| base | #1A1613 | Dark chocolate — primary backgrounds, nav |
| base-light | #2A2420 | Slightly lighter dark for cards/sections |
| cream | #FAF6F1 | Light backgrounds, page base |
| cream-dark | #F0EBE3 | Alternate section backgrounds |
| gold | #C9A96E | Primary accent — buttons, highlights |
| gold-light | #D4B882 | Hover state for gold |
| gold-dark | #B89555 | Pressed state for gold |
| text-primary | #1A1613 | Main body text on light |
| text-secondary | #6B5E50 | Muted text |
| text-light | #FAF6F1 | Text on dark backgrounds |
| border | #E5DDD3 | Hairline dividers |
| star | #C9A96E | Rating stars |

## Typography
- **Headings / Product names:** Cormorant Garamond, serif, UPPERCASE, tracking-[0.2em]
- **Body / UI:** Inter, sans-serif, regular weight
- **Buttons:** Inter, medium weight, tracking-wide

## Spacing & Layout
- Generous whitespace (sections: py-20 md:py-28)
- Max content width: max-w-7xl mx-auto
- Card border-radius: 8px
- Thin hairline dividers: 1px solid #E5DDD3

## Components
- Buttons: solid gold bg with dark text, or outlined gold with gold text
- Hover: subtle scale(1.02), soft shadow, color transitions (200ms)
- Cards: minimal, large image, cream bg, subtle shadow on hover
- Nav: transparent over hero → solid dark on scroll
