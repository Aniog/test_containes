# ARTITECT MACHINERY - Design System

## Brand Identity
- Industrial precision meets refined elegance
- Professional, trustworthy, high-end
- Clean, modern, and user-friendly

## Colors
- Primary: Deep navy blue (#1B2A4A) - trust, professionalism, industrial strength
- Primary light: (#2C3E6B)
- Accent: Warm brass/gold (#C9A84C) - elegance, quality, premium feel
- Accent light: (#E8D5A3)
- Background: Off-white (#F8F6F3) - warm, approachable
- Surface: White (#FFFFFF)
- Text primary: (#1A1A2E) - deep charcoal
- Text secondary: (#5A6478) - muted navy-gray
- Text light: (#8E95A8) - subtle
- Border: (#E2E0DC) - soft warm gray

## Typography
- Headings: 'Playfair Display', serif - elegant, refined
- Body: 'Inter', sans-serif - clean, readable
- Heading sizes: text-4xl (hero), text-3xl (section), text-2xl (sub), text-xl (cards)
- Body: text-base (16px), text-sm (14px)

## Spacing
- Section padding: py-20 md:py-28
- Container max width: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8
- Card padding: p-6 md:p-8
- Grid gaps: gap-6 md:gap-8 lg:gap-10

## Borders & Shadows
- Card border: border border-[#E2E0DC]
- Card shadow: shadow-sm hover:shadow-md
- Transition: transition-all duration-300

## Components Style
- Buttons: Rounded-none (sharp/industrial), uppercase tracking-wider, font-medium
- Primary button: bg-[#1B2A4A] text-white hover:bg-[#2C3E6B]
- Accent button: bg-[#C9A84C] text-white hover:bg-[#B89530]
- Cards: clean borders, subtle shadows, hover elevation
- Navigation: transparent to solid on scroll, elegant logo

## Icons
- Use Lucide React icons consistently
- Accent color for decorative icons

## Imagery
- Industrial/product photography
- Clean backgrounds, professional lighting aesthetic
