# SSourcing China - Design Guidelines

## Brand Identity
- Professional B2B sourcing agent website
- Trustworthy, clean, international feel
- Navy blue + gold accent color scheme

## Colors
- **Primary (Navy):** `#1e3a5f` - Headers, navigation, primary buttons, trust elements
- **Primary Light:** `#2a5298` - Hover states, links, secondary accents
- **Primary Dark:** `#0f1f33` - Footer background, deep emphasis
- **Accent (Gold):** `#d4a843` - CTA buttons, highlights, important callouts
- **Accent Light:** `#e8c76a` - Hover states for gold elements
- **Background:** `#ffffff` - Main background
- **Surface Light:** `#f8fafc` - Alternate section backgrounds
- **Surface Medium:** `#f1f5f9` - Card backgrounds, subtle surfaces
- **Text Primary:** `#1e293b` - Body text, headings
- **Text Secondary:** `#475569` - Descriptions, secondary text
- **Text Muted:** `#64748b` - Captions, labels
- **Border:** `#e2e8f0` - Dividers, card borders

## Typography
- **Font Family:** Inter (Google Fonts)
- **H1:** 3rem (48px), font-weight 800, tracking tight
- **H2:** 2.25rem (36px), font-weight 700, tracking tight
- **H3:** 1.5rem (24px), font-weight 600
- **H4:** 1.25rem (20px), font-weight 600
- **Body:** 1rem (16px), font-weight 400, line-height 1.6
- **Small:** 0.875rem (14px)
- **Button:** 1rem, font-weight 600

## Spacing
- Section padding: `py-20 md:py-28`
- Container max-width: `max-w-7xl mx-auto px-4 sm:px-6 lg:px-8`
- Card padding: `p-6 md:p-8`
- Grid gaps: `gap-6 md:gap-8`

## Components
- **Buttons:** Rounded-lg, px-6 py-3, font-semibold
  - Primary CTA: bg-gold text-white hover:bg-gold-light
  - Secondary: bg-navy text-white hover:bg-navy-light
  - Outline: border-2 border-navy text-navy hover:bg-navy hover:text-white
- **Cards:** bg-white rounded-xl shadow-sm border border-slate-200, hover:shadow-md transition
- **Section headers:** Centered, with gold underline accent

## Visual Style
- Clean, spacious layouts with generous whitespace
- Professional photography of factories, shipping, QC processes
- Icons from Lucide React for service features
- Subtle shadows and borders, no heavy gradients
- Gold accent used sparingly for emphasis and CTAs
- Navy for authority and trust

## Do's
- Use navy + gold color scheme consistently
- Keep layouts clean with ample whitespace
- Use professional, realistic imagery
- Ensure high contrast for readability
- Use responsive design for all screen sizes

## Don'ts
- Don't use bright or neon colors
- Don't use cartoonish icons or illustrations
- Don't overcrowd sections with too much text
- Don't use light text on light backgrounds
- Don't use exaggerated marketing language
