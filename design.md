# MicroCosmos Design System

## Brand Identity
MicroCosmos is a science-meets-wonder website exploring the invisible world. The visual style is dark, immersive, and scientific — like peering through a microscope into another universe.

## Color Palette
- Background: `bg-gray-950` (#030712) — near-black deep space
- Surface: `bg-gray-900` (#111827) — card/panel backgrounds
- Surface elevated: `bg-gray-800` (#1f2937)
- Border: `border-gray-700` (#374151)
- Primary accent: `text-teal-400` / `bg-teal-500` (#14b8a6 / #06b6d4) — bioluminescent teal
- Secondary accent: `text-violet-400` / `bg-violet-500` (#a78bfa) — microscope purple
- Highlight: `text-emerald-400` (#34d399) — living green
- Text primary: `text-white` / `text-gray-100`
- Text secondary: `text-gray-400`
- Text muted: `text-gray-500`

## Typography
- Font: Inter (Google Fonts)
- Page titles: `text-5xl md:text-7xl font-black tracking-tight`
- Section headings: `text-3xl md:text-4xl font-bold`
- Card titles: `text-xl font-semibold`
- Body: `text-base text-gray-300 leading-relaxed`
- Labels/captions: `text-sm text-gray-400`

## Spacing & Layout
- Page max width: `max-w-7xl mx-auto px-4 md:px-8`
- Section padding: `py-16 md:py-24`
- Card gap: `gap-6 md:gap-8`
- Card padding: `p-6`
- Border radius: `rounded-2xl` for cards, `rounded-full` for badges/pills

## Components
- Cards: `bg-gray-900 border border-gray-800 rounded-2xl overflow-hidden hover:border-teal-500/50 transition-all`
- Buttons primary: `bg-teal-500 hover:bg-teal-400 text-gray-950 font-semibold px-6 py-3 rounded-full`
- Buttons outline: `border border-teal-500 text-teal-400 hover:bg-teal-500/10 px-6 py-3 rounded-full`
- Badges: `bg-teal-500/10 text-teal-400 text-xs font-medium px-3 py-1 rounded-full border border-teal-500/20`
- Nav: `bg-gray-950/90 backdrop-blur border-b border-gray-800`

## Image Style
- Use full-bleed images with `object-cover`
- Apply subtle overlays: `bg-gradient-to-t from-gray-950 via-transparent to-transparent`
- Aspect ratios: hero `16x9`, cards `4x3` or `3x2`, portraits `1x1`

## Do's
- Use dark backgrounds everywhere
- Use teal/violet/emerald accents for interactive elements
- Use generous whitespace
- Use image-heavy layouts — images are the star
- Use subtle gradients and glows for depth

## Don'ts
- No white or light backgrounds
- No low-contrast text on dark surfaces
- No cluttered layouts
- No bright/saturated non-accent colors
