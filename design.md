# Velmora Fine Jewelry — Design System

## Visual Identity
- **Mood**: Quiet luxury, warm, editorial. Premium demi-fine jewelry — NOT loud, NOT discount-looking.
- **Target**: Women 25–45, gifting and self-purchase, $30–$120 price point.

## Color Palette
| Token | Hex | Usage |
|---|---|---|
| cream | #FAF7F2 | Page background |
| ivory | #F5F0E8 | Card backgrounds, alternate sections |
| sand | #E8DFD0 | Borders, dividers, subtle backgrounds |
| gold | #C9A96E | Primary accent, CTAs, links, highlights |
| gold-light | #D4BA8A | Hover states, secondary gold |
| gold-dark | #A88B52 | Active/pressed gold |
| charcoal | #2C2C2C | Primary text, nav background |
| charcoal-light | #3D3D3D | Secondary text |
| warm | #8B7355 | Supporting text, captions |
| muted | #9B8E7E | Muted text, placeholders |
| warm-gray | #6B6156 | Tertiary text |

## Typography
- **Headings**: Cormorant Garamond (serif), weights 400–600
- **Body / UI**: Inter (sans-serif), weights 300–500
- **Product names**: UPPERCASE, letter-spacing 0.25em (ultra-wide), serif
- **Section headings**: UPPERCASE, letter-spacing 0.15em (wide), serif
- **Body text**: 14–16px, line-height 1.6
- **Small text / captions**: 12–13px, letter-spacing 0.05em

## Spacing & Layout
- Generous whitespace: sections 80–120px vertical padding (desktop), 48–64px (mobile)
- Max content width: 1280px
- Grid gaps: 24–32px
- Hairline dividers: 1px solid sand (#E8DFD0)

## Buttons
- **Primary (accent)**: bg-gold text-charcoal, hover:bg-gold-light, uppercase, letter-spacing wide, serif font, px-8 py-3
- **Secondary (outlined)**: border-gold text-gold, hover:bg-gold hover:text-charcoal, same typography
- **Subtle**: text-gold underline, hover:text-gold-dark

## Cards
- White or ivory background
- Subtle shadow on hover: shadow-md
- Rounded corners: rounded-sm (2px) for editorial feel
- Image aspect ratios: 4x5 for products, 3x4 for editorial

## Animations
- Subtle, smooth transitions: 300ms ease
- Hover: slight scale (1.02) or shadow increase
- Page transitions: fade-in, slide-up
- No bouncy or playful animations

## Do's
- Use warm, neutral backgrounds that flatter gold jewelry
- Keep typography elegant and restrained
- Use generous whitespace
- Ensure strong contrast (charcoal on cream, gold on charcoal)
- Use serif for emotional/editorial moments, sans for functional UI

## Don'ts
- Don't use bright or saturated colors
- Don't use rounded/bubbly UI elements
- Don't crowd content — whitespace is luxury
- Don't use playful or bouncy animations
- Don't use generic e-commerce patterns (badges, countdowns, aggressive CTAs)
