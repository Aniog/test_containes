# Velmora Fine Jewelry — Design System

## Mood
Quiet luxury, warm, editorial. Premium demi-fine jewelry. NOT loud, NOT discount-looking.

## Color Palette
- **Background (cream):** `cream` → `#FAF7F2` — warm off-white page base
- **Foreground (charcoal):** `charcoal` → `#1A1A1A` — primary text
- **Accent (gold):** `gold` → `#B8860B` — warm dark gold for CTAs, highlights
- **Accent light:** `gold-light` → `#D4A853` — hover states, secondary gold
- **Muted:** `taupe` → `#8C7B6B` — secondary text, captions
- **Surface:** `ivory` → `#F5F0E8` — card backgrounds, sections
- **Border:** `sand` → `#E8E0D4` — hairline dividers, borders
- **Dark surface:** `espresso` → `#2C2420` — dark sections, footer

## Typography
- **Headings / Product names:** `Cormorant Garamond`, serif — elegant, editorial
- **Body / UI:** `Inter`, sans-serif — clean, modern
- **Product names:** UPPERCASE, letter-spacing: 0.15em
- **Heading sizes:** Hero: text-5xl md:text-7xl; Section: text-3xl md:text-4xl; Card: text-lg

## Spacing & Layout
- Generous whitespace: sections py-16 md:py-24
- Container max-w-7xl mx-auto px-4 md:px-8
- Grid gaps: gap-4 md:gap-6
- Card padding: p-4 md:p-6

## Components
- **Buttons:** Solid gold bg with cream text (primary), outlined with gold border (secondary). Rounded-none (square edges). Uppercase tracking-wider text-sm font-medium. Hover: slight opacity or darker shade.
- **Cards:** No border-radius, subtle shadow-sm on hover, transition-all duration-300
- **Dividers:** Thin 1px border-sand, sometimes with opacity
- **Hover effects:** scale-[1.02] on cards, opacity transitions on images

## Do's
- Use generous whitespace
- Keep imagery large and editorial
- Use serif for emotional/brand text
- Use thin hairline dividers
- Subtle transitions (duration-300)

## Don'ts
- No rounded corners on cards/buttons (use rounded-none or very subtle rounded-sm max)
- No bright/neon colors
- No heavy shadows
- No cluttered layouts
- No generic e-commerce feel
