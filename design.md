# SSourcing China - Design System

## Brand Identity
- Professional, trustworthy, international B2B website
- Clean, modern design that conveys reliability and expertise
- Color palette inspired by trust (blues) and action (orange accents)

## Typography
- Primary font: Inter (Google Fonts) - clean, professional, highly readable
- Headings: font-bold to font-extrabold, tight line-heights
- Body: font-normal, relaxed line-height (1.6-1.75)
- Use tracking-tight for headings, tracking-normal for body

## Color Palette
- Primary: #1e3a5f (deep navy blue) - trust, professionalism
- Primary Light: #2d5a8e - hover states, accents
- Accent: #e8753a (warm orange) - CTAs, highlights, action items
- Accent Hover: #d0622b - CTA hover state
- Background: #ffffff (white) - main content areas
- Background Alt: #f8fafc (slate-50) - alternating sections
- Background Dark: #0f172a (slate-900) - footer, dark sections
- Text Primary: #1e293b (slate-800) - main body text
- Text Secondary: #475569 (slate-600) - descriptions, secondary text
- Text Light: #94a3b8 (slate-400) - muted text on dark backgrounds
- Border: #e2e8f0 (slate-200) - borders, dividers
- Success: #059669 - trust indicators, positive states
- White: #ffffff

## Spacing
- Section padding: py-16 md:py-24 lg:py-28
- Container: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8
- Card padding: p-6 md:p-8
- Gap between cards: gap-6 md:gap-8

## Shadows & Borders
- Cards: shadow-md hover:shadow-lg transition-shadow
- Buttons: no shadow, clean borders
- Subtle rounded corners: rounded-lg (8px), rounded-xl (12px)

## Components
- Primary CTA: bg-[#e8753a] text-white hover:bg-[#d0622b] px-6 py-3 rounded-lg font-semibold
- Secondary CTA: border-2 border-[#1e3a5f] text-[#1e3a5f] hover:bg-[#1e3a5f] hover:text-white px-6 py-3 rounded-lg font-semibold
- Cards: bg-white rounded-xl shadow-md hover:shadow-lg transition-all duration-300
- Section headings: text-3xl md:text-4xl font-bold text-[#1e293b]
- Section subheadings: text-lg text-[#475569] max-w-3xl

## Responsive Design
- Mobile-first approach
- Breakpoints: sm(640), md(768), lg(1024), xl(1280)
- Navigation: hamburger on mobile, full on desktop
- Grid layouts: 1 col mobile -> 2-3 cols tablet -> 3-4 cols desktop

## Image Guidelines
- Use stock images for: factory tours, quality inspection, shipping/logistics, warehouse scenes
- Professional, clean photography style
- Avoid overly staged or stock-looking images

## Don'ts
- Do not use emoji in professional UI text
- Do not use bright neon colors
- Do not use decorative fonts for body text
- Do not use very thin fonts at small sizes (poor readability)
- Do not mix more than 2 accent colors
