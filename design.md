# SSourcing China Visual Design System

## Brand feel
Clean, trustworthy, international B2B, practical, and operational. The site should feel like a reliable sourcing partner with real factory, quality control, production, and shipping context.

## Colors
- Primary navy: use `slate-950` / `blue-950` for headers, navigation, and strong text.
- Deep blue: use `blue-700` for primary CTAs and active states.
- Teal accent: use `teal-700` for process highlights and trust indicators.
- Warm amber: use `amber-600` for small emphasis badges only.
- Soft background: use `slate-50` for page sections.
- Card background: `white` with `slate-200` borders and readable `slate-900` text.

## Typography
- Use Inter from Google Fonts.
- Headings: strong, compact, `font-bold` or `font-semibold`, navy/slate text.
- Body: clear, practical, `text-slate-600` or `text-slate-700`, never light gray on white.

## Layout and spacing
- Use max-width containers (`max-w-7xl mx-auto`) with responsive padding (`px-4 sm:px-6 lg:px-8`).
- Desktop should use multi-column layouts where useful; mobile should stack cleanly.
- Cards use rounded corners (`rounded-2xl`), subtle borders, and soft shadows.

## Components
- Primary button: navy/blue background with white text, rounded full, confident spacing.
- Secondary button: white or transparent with navy text and visible border.
- Forms: white card, explicit slate text, visible labels, clear input borders, readable placeholder text.
- Badges: use pale tinted backgrounds with high-contrast text.

## Imagery
- Use Strikingly stock image tags for factory floors, QC inspection, packaging, logistics, and shipping visuals.
- Images should be realistic and professional; avoid abstract or exaggerated visuals.

## Do's
- Keep copy direct, practical, and buyer-focused.
- Use trust signals such as verification, inspection, documentation, reporting, and communication.
- Keep all text clearly readable against backgrounds.

## Don'ts
- Do not use exaggerated promises, hype, or unrealistic claims.
- Do not use low-contrast text or invisible placeholder text.
- Do not hardcode image URLs.
