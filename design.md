# ARTITECT MACHINERY - Design Guidelines

## Brand Identity
- **Brand Name**: ARTITECT MACHINERY
- **Industry**: Industrial sheet metal folding machines
- **Tone**: Elegant, professional, industrial, trustworthy

## Color Palette
- **Primary**: `#1a1a2e` (Deep navy/charcoal - sophistication)
- **Secondary**: `#16213e` (Dark blue - industrial trust)
- **Accent**: `#e94560` (Coral red - energy/action)
- **Light**: `#f8f9fa` (Off-white - clean, modern)
- **Text Primary**: `#1a1a2e`
- **Text Secondary**: `#6c757d`
- **Border**: `#dee2e6`

## Typography
- **Headings**: "Playfair Display" (elegant serif) - for brand name and hero
- **Body**: "Inter" (clean sans-serif) - for readability
- **Sizes**:
  - Hero title: 56px (desktop), 36px (mobile)
  - Section titles: 36px (desktop), 28px (mobile)
  - Body: 16px
  - Small: 14px

## Layout
- Max content width: 1200px
- Section padding: 80px vertical (desktop), 48px (mobile)
- Grid: 12-column system
- Responsive breakpoints: 768px (tablet), 1024px (desktop)

## Visual Style
- Clean, minimalist design with generous whitespace
- Subtle shadows for depth: `0 4px 20px rgba(0,0,0,0.08)`
- Rounded corners: 8px for cards, 4px for buttons
- Smooth transitions: 0.3s ease

## Components

### Header
- Fixed navigation bar
- Logo on left (brand name in Playfair Display)
- Navigation links: Home, Products, About, Contact
- CTA button: "Get Quote"

### Hero Section
- Full-width background with subtle gradient overlay
- Large headline with brand name
- Subheadline describing products
- Two CTA buttons: "View Products" (primary), "Contact Us" (outline)
- Industrial machinery imagery feel

### Products Section
- Section title: "Our Products"
- Grid of product cards (3 columns desktop, 1 mobile)
- Each card: image placeholder, product name, brief description, "Learn More" link
- Products to feature:
  1. Double Folding Machine
  2. Double Folder
  3. Sheet Metal Folder
  4. Sheet Metal Folding Machine
  5. Metal Folder
  6. Metal Folder Machine

### Features Section
- Why choose us
- 4 key features in a row:
  1. Precision Engineering
  2. Durable Construction
  3. Easy Operation
  4. Global Support

### About Section
- Brief company introduction
- Mission statement
- Years of experience badge

### Contact Section
- Contact form with fields: Name, Email, Phone, Message
- Company contact information
- Map placeholder

### Footer
- Company info
- Quick links
- Contact details
- Copyright

## Do's and Don'ts
- DO: Use clean whitespace, consistent spacing
- DO: Use high contrast for readability
- DO: Keep navigation simple and intuitive
- DON'T: Use too many colors (stick to palette)
- DON'T: Clutter the design with too many elements
- DON'T: Use generic-looking fonts (stick to specified typography)
