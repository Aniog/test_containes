# MicroCosmos Design System

## Visual Theme
A dark, immersive design evoking the feeling of looking through a microscope into an alien world.
Deep space-like backgrounds with luminous, glowing accents in teal/cyan and violet.

## Colors
- **Background**: Deep navy black `#0a0e1a`
- **Surface/Cards**: Dark indigo `#131832`
- **Primary Accent**: Teal/Cyan `#00d4aa` (bioluminescent glow)
- **Secondary Accent**: Violet/Purple `#8b5cf6` (microscopic structures)
- **Highlight**: Electric blue `#38bdf8`
- **Text Primary**: Off-white `#e8edf5`
- **Text Secondary**: Muted lavender `#94a3b8`
- **Border/Glow**: `#1e293b`

## Typography
- Headings: Inter 700/800, with slight letter-spacing
- Body: Inter 400/500
- Accent numbers/stats: Inter 300 (light weight, large size)

## Visual Effects
- Soft glowing borders on cards using box-shadow with primary accent color
- Gradient overlays on images (dark to transparent)
- Smooth hover transitions (300ms ease)
- Subtle parallax-like depth with layered sections

## Layout
- Full-width hero with background image
- Grid-based gallery sections (responsive: 1 col mobile, 2-3 cols tablet, 3-4 cols desktop)
- Max content width: 1280px centered
- Generous vertical spacing between sections (py-20 to py-32)

## Card Style
- Rounded corners (16px)
- Dark surface background with subtle border
- Image with overlay gradient
- Glowing border on hover

## Do's
- Use stock images generously via data-strk-img system
- Use dark backgrounds throughout
- Keep text readable (high contrast on dark)
- Use subtle animations and glow effects

## Don'ts
- Don't use bright/light backgrounds
- Don't use harsh, saturated colors
- Don't overcrowd sections
- Don't use low-contrast text
