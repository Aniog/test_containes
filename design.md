# SSourcing China - B2B Website Design Specification

## Project Overview
- **Project Name**: SSourcing China Website
- **Type**: B2B Corporate Website
- **Core Functionality**: China sourcing agent website to generate qualified sourcing inquiries
- **Target Users**: Overseas buyers seeking reliable Chinese suppliers

## Visual Style Guidelines

### Color Palette
- **Primary**: `#1a365d` (Deep Navy Blue - trust, professionalism)
- **Primary Light**: `#2c5282` (Medium Navy)
- **Secondary**: `#c9a227` (Gold/Amber - premium, quality)
- **Accent**: `#38a169` (Green - success, reliability)
- **Background**: `#f8fafc` (Light gray-white)
- **Surface**: `#ffffff` (White)
- **Text Primary**: `#1a202c` (Near black)
- **Text Secondary**: `#4a5568` (Gray)
- **Text Muted**: `#718096` (Light gray)
- **Border**: `#e2e8f0` (Light border)
- **Error**: `#c53030` (Red)

### Typography
- **Font Family**: "Inter", system-ui, sans-serif
- **Headings**: 
  - H1: 48px/56px, font-weight 700
  - H2: 36px/44px, font-weight 700
  - H3: 24px/32px, font-weight 600
  - H4: 20px/28px, font-weight 600
- **Body**: 16px/24px, font-weight 400
- **Small**: 14px/20px, font-weight 400

### Spacing System
- Base unit: 4px
- Section padding: 80px vertical (desktop), 48px (mobile)
- Container max-width: 1200px
- Grid gap: 32px

### Visual Effects
- Card shadows: `0 4px 6px -1px rgba(0, 0, 0, 0.1), 0 2px 4px -1px rgba(0, 0, 0, 0.06)`
- Hover transitions: 200ms ease
- Border radius: 8px (cards), 4px (buttons)

## Layout Structure

### Header
- Fixed navigation bar
- Logo (left): "SSourcing China" text logo
- Navigation links (center): Home, Services, How It Works, Products, Case Studies, Blog, Contact
- CTA button (right): "Get a Free Quote"
- Mobile: Hamburger menu

### Footer
- 4-column layout
- Company info, Quick Links, Services, Contact
- Copyright and social links

## Page Specifications

### 1. Homepage
**Sections:**
1. **Hero Section**
   - Full-width background with subtle factory/warehouse image
   - Headline: "China Sourcing Agent for Global Buyers"
   - Subheadline: "Find verified suppliers, ensure quality, and streamline shipping from China"
   - Primary CTA: "Get a Free Sourcing Quote"
   - Secondary CTA: "Learn How It Works"

2. **Services Section**
   - 6 service cards in 3x2 grid
   - Services: Supplier Verification, Factory Audit, Quality Control, Production Follow-up, Shipping & Logistics, Sourcing Consultation
   - Each card: Icon, title, brief description

3. **Sourcing Process Section**
   - 5-step horizontal timeline
   - Steps: Submit Request → Supplier Matching → Sample Evaluation → Order & Production → Shipping & Delivery

4. **Products We Source Section**
   - Category grid (4x3)
   - Categories: Electronics, Furniture, Textiles, Machinery, Packaging, Toys, Automotive Parts, Home Goods, Sports Equipment, Medical Supplies, Beauty & Personal Care, Industrial Components

5. **Problems We Solve Section**
   - 4 problem/solution cards
   - Common pain points: Language barriers, Quality issues, Supplier scams, Logistics complexity

6. **Trust Points Section**
   - Statistics: Years in business, Suppliers verified, Orders completed, Client satisfaction
   - Trust badges/certifications

7. **Case Studies Section**
   - 3 featured case study cards
   - Each: Industry, challenge, solution, result

8. **FAQ Section**
   - Accordion with 6-8 common questions

9. **Inquiry Form Section**
   - Contact form with fields: Name, Company, Email, Phone, Product Details, Quantity, Message
   - Submit button: "Submit Inquiry"

### 2. Services Page

## Component Guidelines

### Buttons
- **Primary**: Navy background (#1a365d), white text, gold hover
- **Secondary**: White background, navy border, navy text
- **CTA**: Gold background (#c9a227), dark text

### Cards
- White background
- Subtle shadow
- 8px border radius
- 24px padding

### Form Inputs
- Full width
- 12px padding
- Border: #e2e8f0
- Focus: Navy border
- 4px border radius

### Icons
- Lucide React icons
- Consistent sizing (24px default)
- Navy or gold color

## Responsive Breakpoints
- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px

## Do's and Don'ts
- **Do**: Use professional imagery, clear typography, ample white space
- **Don't**: Use overly flashy animations, exaggerated claims, cluttered layouts
- **Do**: Maintain consistent spacing and alignment
- **Don't**: Use too many different colors or fonts
