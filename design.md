# SSourcing China — Visual Style Guide

## Brand Identity
- **Name**: SSourcing China
- **Tagline**: China Sourcing Agent for Global Buyers
- **Tone**: Professional, clear, practical, trustworthy. No exaggerated claims.

## Color Palette

### Primary Colors
- **Navy Blue** (primary): `#1B3A5C` — trust, professionalism, B2B authority
- **Sky Blue** (primary-light): `#2E6DA4` — accents, links, hover states
- **Teal** (accent): `#0D9488` — CTAs, highlights, success indicators

### Neutral Colors
- **White**: `#FFFFFF` — backgrounds, cards
- **Light Gray**: `#F8FAFC` — section backgrounds, alternating rows
- **Mid Gray**: `#E2E8F0` — borders, dividers
- **Dark Gray**: `#475569` — body text
- **Charcoal**: `#1E293B` — headings, nav text

### Semantic Colors
- **Success**: `#16A34A` — positive indicators
- **Warning**: `#D97706` — caution
- **Error**: `#DC2626` — errors, required fields

## Typography
- **Font Family**: Inter (Google Fonts), system-ui fallback
- **Headings**: font-weight 700 (bold), color charcoal `#1E293B`
  - H1: 3rem (48px) — hero headlines
  - H2: 2.25rem (36px) — section titles
  - H3: 1.5rem (24px) — card titles, subsections
  - H4: 1.25rem (20px) — small headings
- **Body**: font-weight 400, color dark gray `#475569`
  - Base: 1rem (16px), line-height 1.75
  - Large: 1.125rem (18px) — hero subtitles, lead paragraphs
  - Small: 0.875rem (14px) — captions, meta text
- **CTA Buttons**: font-weight 600, uppercase optional, letter-spacing 0.025em

## Spacing
- Section padding: `py-20` (80px vertical)
- Card padding: `p-6` to `p-8`
- Grid gaps: `gap-6` to `gap-8`
- Container max-width: `max-w-7xl` (1280px), centered

## Layout
- Clean, spacious layout with generous whitespace
- Max content width: 1280px
- Cards with subtle shadows: `shadow-md` or `shadow-lg` on hover
- Rounded corners: `rounded-lg` (8px) for cards, `rounded-md` (6px) for buttons
- Alternating white/light-gray section backgrounds

## Buttons
- **Primary CTA**: bg teal `#0D9488`, text white, `px-8 py-3 rounded-md font-semibold`
  - Hover: darker teal `#0F766E`
- **Secondary**: border navy, text navy, `px-6 py-3 rounded-md font-semibold`
  - Hover: bg navy, text white
- **Ghost**: text navy, no border, underline on hover

## Cards
- White background, `shadow-md`, `rounded-lg`, `p-6`
- Hover: `shadow-lg`, slight translateY(-2px) transition
- Icon + title + description pattern

## Images
- Use `data-strk-img` for stock images (factory, QC, shipping, logistics)
- Ratios: `16x9` for heroes, `4x3` for cards, `3x2` for features
- Professional, realistic visuals — factories, warehouses, shipping containers, inspection scenes

## Navigation
- Sticky top nav, white background, subtle bottom border
- Logo left, nav links center/right, CTA button right
- Mobile: hamburger menu with slide-in drawer
- Active link: teal underline or bold

## Footer
- Dark navy background `#1B3A5C`, white/light text
- 4-column layout: About, Services, Resources, Contact
- Copyright bar at bottom

## Do's
- Use generous whitespace between sections
- Keep text concise and scannable
- Use icons alongside feature descriptions
- Use real-world numbers and specifics where possible
- Ensure high contrast for readability

## Don'ts
- Don't use bright/saturated colors as backgrounds
- Don't use cartoon or overly stylized illustrations
- Don't use exaggerated marketing language
- Don't crowd sections with too much text
- Don't use light text on light backgrounds
