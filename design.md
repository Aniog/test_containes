# Strikingly Generators Hub Design System

This project follows Strikingly's marketing style for the AI Generators hub page.

## Typography
- Heading font: `Josefin Sans`, `Poppins`, sans-serif as the fallback stack for Brandon Grotesque
- Body font: `Open Sans`, Arial, sans-serif
- Headings use uppercase, bold weight, and compact line height
- Body text uses a 14px base with relaxed readability

## Color tokens
- Brand purple: `#8159BB`
- AI gradient start: `#7671FF`
- AI gradient end: `#CB0C9F`
- Body text: `#636972`
- Section heading text: `#4B5056`
- Hero heading text: `#2E2E2F`
- Card border: `#C6C9CD`
- Divider: `#E2E4E7`
- Light background: `#F4F6F8`
- Surface: `#FFFFFF`

## Surfaces and borders
- Default page background stays white
- Cards use white surfaces, 1px neutral borders, and 3px radius
- Hover states use a subtle shadow and brand-purple border
- Only the hero may use a faint purple/pink wash

## Spacing
- Use a 10px spacing rhythm
- 20px between nearby blocks
- 40px between sections
- 60px to 80px vertical padding for the hero
- Content width maxes out at 1200px and stays centered

## Buttons and badges
- Filled CTAs use the AI gradient with white text
- Ghost buttons use transparent backgrounds with purple borders and purple text
- Buttons use uppercase heading typography
- Category badges use a compact uppercase ghost style

## Do
- Keep layouts clean, airy, and easy to scan
- Use semantic sections and visible focus states
- Make card text and supporting copy clearly readable on white backgrounds
- Keep SVG illustrations simple, line-based, and light

## Don't
- Do not use dark black text or heavy shadows
- Do not use tinted backgrounds across every section
- Do not add fake testimonials, ratings, or inflated metrics
- Do not rely on JavaScript to render the directory content
