# ARTITECT MACHINERY Design System

## Brand direction
ARTITECT MACHINERY should feel precise, elegant, industrial, and trustworthy. The site must present heavy-duty sheet metal folding equipment in a polished, easy-to-understand way for buyers and factory decision makers.

## Typography
- Primary font: Inter
- Headings: bold, clean, high-contrast, slightly tight tracking
- Body text: calm and readable with medium line height
- Example Tailwind classes:
  - Hero heading: `text-4xl md:text-6xl font-semibold tracking-tight`
  - Section heading: `text-3xl md:text-4xl font-semibold tracking-tight`
  - Body copy: `text-base md:text-lg leading-7`
  - Eyebrow labels: `text-xs md:text-sm uppercase tracking-[0.24em]`

## Color system
Use a refined industrial palette with warm metallic accents.
- Background: `bg-slate-950`
- Surface: `bg-slate-900`, `bg-slate-800/70`
- Light section background: `bg-stone-100`
- Text on dark: `text-white`, `text-slate-300`
- Text on light: `text-slate-900`, `text-slate-600`
- Accent: `bg-amber-400`, `text-amber-300`, `border-amber-400/40`
- Dividers: `border-white/10`, `border-slate-300`

## Layout and spacing
- Use wide desktop containers with comfortable breathing room: `max-w-7xl mx-auto px-6 md:px-10`
- Section spacing: `py-16 md:py-24`
- Card spacing: `p-6 md:p-8`
- Keep desktop layouts multi-column when content allows. Stack only on smaller screens.

## Components
- Use rounded corners sparingly: `rounded-2xl` for feature cards and visual panels
- Use subtle borders and layered backgrounds instead of strong shadows
- Buttons should be high-contrast and easy to read
- Data and labels must always have explicit readable text colors

## Imagery
- Use industrial machinery, sheet metal fabrication, precision engineering, and factory production imagery
- Prefer wide landscape visuals in hero and feature sections
- Pair images with nearby text references using Strikingly image attributes

## Do
- Keep the experience premium but approachable
- Highlight product categories and benefits clearly
- Use strong contrast on every section
- Make calls to action obvious and simple

## Don't
- Do not use playful colors
- Do not rely on low-contrast muted text for important details
- Do not overcrowd desktop sections into a single narrow column
- Do not use arbitrary hex values directly in class names
