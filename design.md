# ARTITECT MACHINERY - Website Design Guidelines

## Brand Overview
- **Brand Name**: ARTITECT MACHINERY
- **Industry**: Industrial machinery manufacturing (sheet metal folding machines)
- **Products**: Double folding machine, double folder, sheet metal folder, sheet metal folding machine, metal folder, metal folder machine, metal folding machine

## Visual Style
- **Overall Aesthetic**: Elegant but user-friendly, industrial professionalism
- **Design Philosophy**: Clean, sophisticated, trustworthy

## Color Palette
- **Primary**: `#1a1a2e` (Deep navy/charcoal - sophistication)
- **Secondary**: `#16213e` (Dark blue - industrial trust)
- **Accent**: `#e94560` (Coral red - call-to-action, energy)
- **Light**: `#f8f9fa` (Off-white - clean backgrounds)
- **Text Primary**: `#1a1a2e` (Dark for readability)
- **Text Secondary**: `#6c757d` (Muted gray)
- **Border/Divider**: `#dee2e6` (Light gray)

## Typography
- **Headings**: "Playfair Display" - elegant serif for brand prestige
- **Body**: "Inter" - clean, readable sans-serif
- **Font Sizes**:
  - Hero: 56px (desktop), 36px (mobile)
  - H1: 42px
  - H2: 32px
  - H3: 24px
  - Body: 16px
  - Small: 14px

## Layout & Spacing
- **Max Content Width**: 1200px
- **Section Padding**: 80px vertical (desktop), 48px (mobile)
- **Component Spacing**: 24px between elements
- **Border Radius**: 8px (cards), 4px (buttons)

## UI Components

### Navigation
- Fixed top navigation with logo on left
- Navigation links centered
- CTA button on right ("Get Quote")
- Background: white with subtle shadow on scroll

### Hero Section
- Full-width with dark gradient overlay
- Large headline with brand name emphasis
- Subheadline describing products
- Two CTA buttons: primary (Get Quote) and secondary (View Products)
- Background: industrial/machinery image with overlay

### Features Section
- 3-column grid (stacked on mobile)
- Icon + title + description per feature
- Subtle hover animations

### Products Section
- Product cards with image, title, description
- "Learn More" link per card
- Grid layout: 3 columns desktop, 1 column mobile

### About Section
- Split layout: image left, text right
- Company story and expertise
- Key statistics (years, clients, etc.)

### Contact/CTA Section
- Dark background
- Compelling headline
- Contact form or CTA button

### Footer
- 4-column layout: About, Products, Contact, Social
- Dark background
- Copyright notice

## Animations & Interactions
- Smooth scroll behavior
- Fade-in animations on scroll
- Button hover: slight lift + color shift
- Card hover: subtle shadow increase
- Navigation: smooth background transition on scroll

## Responsive Breakpoints
- **Mobile**: < 640px
- **Tablet**: 640px - 1024px
- **Desktop**: > 1024px

## Do's
- Use high-quality industrial imagery
- Maintain consistent spacing
- Use accent color sparingly for CTAs
- Keep text concise and scannable
- Show product images prominently

## Don'ts
- Don't use too many colors (keep it minimal)
- Don't clutter the design
- Don't use decorative fonts for body text
- Don't make it look like a generic template
