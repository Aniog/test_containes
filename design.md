# SSourcing China — Visual Design System

## Brand Identity
Professional, trustworthy, international B2B sourcing agency.

## Color Palette
- **Primary Blue**: `#1e3a5f` — Deep navy (trust, professionalism)
- **Primary Light**: `#2c5282` — Medium blue
- **Accent Orange**: `#e8740c` — CTA buttons, highlights (action-oriented)
- **Accent Orange Hover**: `#cf660b`
- **Background White**: `#ffffff`
- **Background Light**: `#f7f8fa` — Section alternating bg
- **Background Muted**: `#edf2f7` — Cards, subtle sections
- **Text Primary**: `#1a202c` — Headings, important text
- **Text Secondary**: `#4a5568` — Body text, descriptions
- **Text Muted**: `#718096` — Labels, secondary info
- **Border Light**: `#e2e8f0`
- **Success Green**: `#22543d` — Trust indicators

## Typography
- **Font**: Inter (Google Fonts)
- **Heading 1**: text-4xl md:text-5xl lg:text-6xl font-bold leading-tight
- **Heading 2**: text-3xl md:text-4xl font-bold leading-tight
- **Heading 3**: text-xl md:text-2xl font-semibold
- **Body**: text-base md:text-lg leading-relaxed
- **Small**: text-sm leading-normal
- **Font weights**: 300 (light), 400 (regular), 500 (medium), 600 (semibold), 700 (bold), 800 (extrabold)

## Spacing
- Section padding: py-16 md:py-24
- Container max-width: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8
- Card padding: p-6 md:p-8
- Grid gaps: gap-6 md:gap-8

## Border & Shadows
- Cards: rounded-xl shadow-sm border border-[#e2e8f0]
- Buttons: rounded-lg
- Images: rounded-lg

## Visual Style
- Clean white backgrounds with light gray alternating sections
- No heavy gradients — subtle, professional feel
- Trust badges and statistics prominently displayed
- Real factory/QC/shipping imagery (via stock image system)
- Icons from Lucide React for consistency

## Do's
- Use consistent color tokens from this palette
- Maintain generous whitespace
- Use semantic HTML for accessibility
- Keep CTAs prominent with accent orange
- Use data-strk-img for realistic B2B visuals

## Don'ts
- Don't use gradients on backgrounds
- Don't use decorative animations
- Don't use bright or neon colors
- Don't overcrowd sections
