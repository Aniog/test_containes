# Design Guidelines

## Visual Style
- **Typography**: Inter (sans-serif)
- **Colors**:
  - Primary: `bg-slate-900`, `text-white`
  - Secondary: `bg-slate-100`, `text-slate-900`
  - Accent: `bg-blue-600`, `text-white`
- **Borders**: `border-slate-200`, `rounded-lg`
- **Spacing**: `p-4`, `m-4`, `gap-4`

## Components
- Cards should have a subtle shadow: `shadow-sm`
- Buttons should have a hover effect: `hover:bg-opacity-90`
- Use Tailwind's container for page layouts: `container mx-auto px-4`

## Responsive Design
- Mobile first approach
- Use `md:` and `lg:` for desktop adjustments
