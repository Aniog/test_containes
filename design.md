# ARTITECT MACHINERY Website Design Guidelines

## Brand Identity
- **Brand Name:** ARTITECT MACHINERY
- **Industry:** Industrial machinery manufacturing, specifically metal sheet folding machines
- **Tone:** Professional, elegant, trustworthy, user-friendly

## Visual Style

### Color Palette
- **Primary:** Deep Navy Blue `#0A1628` - conveys trust, professionalism, industrial strength
- **Secondary:** Steel Gray `#64748B` - represents metal, machinery, precision
- **Accent:** Copper Orange `#D97706` - adds warmth, highlights CTAs, suggests precision engineering
- **Light Background:** `#F8FAFC` - clean, professional backdrop
- **Card Background:** `#FFFFFF` - crisp white for content cards
- **Text Primary:** `#1E293B` - dark readable text
- **Text Secondary:** `#64748B` - muted descriptions

### Typography
- **Headings:** Inter (700, 600) - modern, professional, highly readable
- **Body:** Inter (400, 500) - clean, easy to read
- **Fallback:** system-ui, -apple-system, sans-serif
- **Scale:** h1 (3.5rem), h2 (2.5rem), h3 (1.5rem), body (1rem)

### Layout & Spacing
- **Container:** max-width 1280px, centered with px-6 padding
- **Section spacing:** py-20 (80px vertical)
- **Component spacing:** gap-6 to gap-8
- **Border radius:** rounded-lg (8px) for cards, rounded-xl (12px) for larger elements

### Borders & Shadows
- **Cards:** subtle shadow-lg with light borders
- **Hover states:** elevated shadow-xl with slight scale transform
- **Transitions:** 300ms ease for all interactive elements

### Do's
- Use generous whitespace for elegance
- Include clear visual hierarchy with consistent spacing
- Add subtle animations on hover for interactivity
- Use high-contrast text for readability
- Include professional imagery placeholders

### Don'ts
- Avoid cluttered layouts
- Don't use too many colors
- Don't use low-contrast text
- Avoid harsh edges - keep design smooth and refined

## Section Structure
1. **Navigation** - Fixed top, logo + nav links + CTA button
2. **Hero** - Full height, brand tagline, primary CTA, visual element
3. **Products Showcase** - Grid of product cards with descriptions
4. **Features/Benefits** - Icon cards highlighting key advantages
5. **Why Choose Us** - Trust indicators and quality guarantees
6. **Contact CTA** - Contact form or call-to-action section
7. **Footer** - Company info, quick links, contact details

## Animations
- Fade-in on scroll for sections
- Smooth hover transitions on cards (scale 1.02, shadow increase)
- Button hover effects (background shift, subtle scale)
- Navigation link underlines on hover
