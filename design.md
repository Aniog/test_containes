# ARTITECT MACHINERY - Design System

## Brand
ARTITECT MACHINERY produces industrial sheet metal folding machines. The design should convey precision engineering, elegance, and trustworthiness while remaining user-friendly and approachable.

## Typography
- Font: Inter (Google Fonts), weights 300, 400, 500, 600, 700, 800
- Headings: font-bold tracking-tight
- Body: font-normal leading-relaxed

## Color Palette
- Primary: slate-900 (#0f172a) - Deep charcoal for headers, footers
- Primary Light: slate-800 (#1e293b) - Section backgrounds
- Accent: amber-500 (#f59e0b) - CTAs, highlights, hover states
- Accent Dark: amber-600 (#d97706) - Button hover
- Surface: white / slate-50 - Card backgrounds
- Text Primary: slate-900
- Text Secondary: slate-600
- Text Muted: slate-400
- Border: slate-200

## Do's
- Use clean, spacious layouts with generous padding
- Use amber/gold as the primary accent for CTAs and key elements
- Use white/slate-50 card surfaces with subtle shadows
- Keep navigation simple and uncluttered
- Use rounded-lg for cards and buttons
- Maintain high contrast for readability

## Don'ts
- Avoid overly bright or neon colors
- Avoid cluttered layouts with too many elements
- Avoid harsh box shadows - use subtle ones
- Avoid multiple competing accent colors
