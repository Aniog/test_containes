# ARTITECT MACHINERY - Design Guidelines

## Visual Style Overview
A sophisticated, industrial-elegant design that balances professionalism with approachability. The aesthetic evokes precision engineering while remaining warm and inviting to potential customers.

## Typography
- **Headings**: Playfair Display (serif) - elegant, premium feel
- **Body/UI**: Montserrat (sans-serif) - clean, modern, highly readable
- **Font weights**: 
  - Headlines: 600-700
  - Subheadings: 500-600
  - Body: 400-500
  - Small text: 300-400

## Color Palette
### Primary Colors
- **Deep Navy**: `#1a2744` - Primary brand color, used for headers, CTAs
- **Steel Gray**: `#4a5568` - Secondary text, supporting elements
- **Charcoal**: `#2d3748` - Body text, dark backgrounds

### Accent Colors
- **Gold/Amber**: `#d4a574` - Premium accent, highlights, special elements
- **Warm Bronze**: `#b8956f` - Secondary accent
- **Light Gold**: `#e8d4b8` - Subtle accents, borders

### Neutral Palette
- **Pure White**: `#ffffff` - Primary backgrounds, text on dark
- **Off-White**: `#f8f6f3` - Section backgrounds, cards
- **Light Gray**: `#e8e6e3` - Borders, dividers
- **Medium Gray**: `#9ca3af` - Muted text, placeholders

### Functional Colors
- **Success Green**: `#059669` - Success states
- **Error Red**: `#dc2626` - Error states

## Spacing System
- Base unit: 4px
- Section padding: 80px-120px vertical
- Container max-width: 1280px
- Grid gap: 24px-32px
- Component spacing: 16px-24px

## Component Styling
### Cards
- Background: White or off-white
- Border radius: 8px-12px
- Shadow: Subtle, layered shadows for depth
- Border: Light gray or gold accent line

### Buttons
- Primary: Deep navy background, white text
- Secondary: Gold/bronze background, dark text
- Outline: Transparent with navy border
- Hover: Slight scale (1.02-1.05) + shadow enhancement
- Border radius: 6px-8px
- Padding: 14px-24px horizontal

### Navigation
- Sticky header with blur backdrop
- Logo prominent on left
- Clean navigation links
- CTA button on right
- Mobile: Hamburger menu with slide-out drawer

### Section Layouts
- Hero: Full-width, dramatic imagery/gradient
- Products: Grid layout (3 columns desktop, 1 mobile)
- Features: Icon-led cards or alternating layout
- About: Split layout with image and text
- Contact: Form + contact info side by side

## Visual Elements
- Subtle geometric shapes as decorative elements
- Thin gold accent lines/dividers
- Clean, minimal iconography (Lucide icons)
- High-quality industrial imagery
- Gradient overlays for depth

## Responsive Breakpoints
- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px
- Large Desktop: > 1280px

## Do's
- Use high contrast for readability
- Maintain generous white space
- Keep animations subtle and purposeful
- Use gold accents sparingly for premium feel
- Ensure clear visual hierarchy
- Make CTAs prominent and accessible

## Don'ts
- Don't use too many colors
- Don't clutter sections with too many elements
- Don't use small font sizes for important content
- Don't create low-contrast text
- Don't use flashy or distracting animations
