# Design System

## Visual Style
Clean, modern, professional. Light background with subtle gradients. Card-based layout with soft shadows.

## Colors
- Primary: Indigo `#6366f1` (indigo-500) — buttons, active states, accents
- Primary Dark: `#4f46e5` (indigo-600) — hover states
- Background: `#f8fafc` (slate-50) — page background
- Surface: `#ffffff` — cards, modals
- Border: `#e2e8f0` (slate-200) — dividers, input borders
- Text Primary: `#0f172a` (slate-900) — headings
- Text Secondary: `#475569` (slate-600) — body text, labels
- Text Muted: `#94a3b8` (slate-400) — placeholders, hints
- Success: `#10b981` (emerald-500)
- Error: `#ef4444` (red-500)
- Warning: `#f59e0b` (amber-500)
- Status Active: `#10b981` (emerald-500)
- Status Inactive: `#94a3b8` (slate-400)
- Status Pending: `#f59e0b` (amber-500)

## Typography
- Font: Inter (Google Fonts)
- Heading XL: `text-3xl font-bold text-slate-900`
- Heading L: `text-2xl font-semibold text-slate-900`
- Heading M: `text-xl font-semibold text-slate-800`
- Heading S: `text-base font-semibold text-slate-800`
- Body: `text-sm text-slate-600`
- Caption: `text-xs text-slate-400`

## Spacing
- Page padding: `px-4 py-8 md:px-8`
- Card padding: `p-6`
- Section gap: `gap-6`
- Form field gap: `gap-4`

## Borders & Shadows
- Card: `rounded-2xl shadow-sm border border-slate-200`
- Input: `rounded-lg border border-slate-200 focus:ring-2 focus:ring-indigo-500`
- Button primary: `rounded-lg bg-indigo-500 hover:bg-indigo-600 text-white`
- Button secondary: `rounded-lg border border-slate-200 bg-white hover:bg-slate-50 text-slate-700`

## Do's
- Use indigo as the primary accent color
- Use card-based layout with rounded-2xl and soft shadows
- Keep form labels above inputs, text-sm font-medium text-slate-700
- Use status badges with colored dots
- Responsive: single column on mobile, multi-column on desktop

## Don'ts
- Don't use dark backgrounds for main content areas
- Don't use low-contrast text (e.g. gray on white without sufficient contrast)
- Don't use harsh box shadows
- Don't stack everything in a single column on desktop
