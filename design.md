# Velmora Fine Jewelry - Design System

## Visual Identity
- **Mood**: Quiet luxury, warm, editorial, premium.
- **Vibe**: High-end DTC, not discount-focused.

## Typography
- **Headings/Product Names**: Cormorant Garamond (Serif).
  - Headings: `font-serif`
  - Product Names: `font-serif uppercase tracking-[0.2em] font-medium`
- **Body/UI**: Inter (Sans).
  - Body: `font-sans text-stone-600 leading-relaxed`
  - UI Labels: `font-sans uppercase text-xs tracking-widest font-semibold`

## Palette
- **Primary (Background)**: `#FFFFFF` (White) / `#FAF9F6` (Off-white)
- **Secondary (Dark/Text)**: `#121212` (Ebonized Wood)
- **Accent (Gold)**: `#C5A059` (Soft Gold)
- **Muted**: `#E5E5E5` (Hairline Dividers)
- **Background Accent**: `#F9F5F1` (Warm Neutral)

## UI Elements
- **Buttons**:
  - Primary: Solid `#121212` with White text, square or slightly rounded (2px).
  - Secondary: Outlined `#121212`, clear background.
- **Dividers**: 1px Hairline stones in `#E5E5E5`.
- **Transitions**: 400ms ease-in-out for hovers.
- **Images**: Large, editorial focus, warm lighting.

## Layout
- Generous whitespace.
- Proportional spacing (Tailwind spacing scale).
- Mobile-first responsive design.
