# Design System: Velmora Fine Jewelry

## Visual Mood
Quiet luxury, warm, editorial, premium demi-fine jewelry.

## Palette
- Base: `bg-stone-50` / `bg-[#fcfaf7]` (Warm Pearl)
- Text Primary: `text-stone-900` / `text-[#1a1a1a]` (Deep Onyx)
- Accent/Metallic: `text-amber-800` / `text-[#9a7b4f]` (Muted Gold)
- Border: `border-stone-200`
- Surface: `bg-white`

## Typography
- Headings: `font-serif` (Cormorant Garamond)
- UI/Body: `font-sans` (Inter)
- Product Names: `font-serif uppercase tracking-[0.2em]`

## UI Components
- Buttons: Solid primary or outlined stone. Rounded none/sm.
- Dividers: `h-[1px] bg-stone-200` (Hairline)
- Transitions: Subtle fade-in, scale hover.

## Icons
- Lucide React (thin stroke width)
