# Velmora Fine Jewelry - Design System

## Brand Identity
- **Brand Name**: Velmora Fine Jewelry
- **Type**: Direct-to-Consumer (B2C) Demi-Fine Jewelry Store
- **Target Audience**: Women 25-45, gifting and self-purchase
- **Price Range**: $30–$120 (premium-but-accessible)

## Visual Identity

### Mood
- Quiet luxury, warm, editorial
- Premium demi-fine jewelry
- NOT loud, NOT discount-looking, NOT generic e-commerce

### Color Palette
**Primary Colors:**
- `cream`: #FAF7F2 (warm off-white background)
- `charcoal`: #1A1A1A (primary text, dark)
- `warm-gray`: #6B6560 (secondary text)
- `gold`: #C9A962 (accent color - warm gold)
- `gold-light`: #E8DCC4 (subtle gold accents)
- `ivory`: #FFFEF9 (card backgrounds)

**Semantic Colors:**
- `success`: #4A7C59 (muted green)
- `error`: #9B4A4A (muted red)

### Typography
**Headings & Product Names:**
- Font: Cormorant Garamond (serif)
- Weight: 500-600
- Product names: UPPERCASE with letter-spacing: 0.15em

**Body & UI:**
- Font: Manrope (sans-serif)
- Weight: 400-500
- Clean, readable, modern

### Spacing System
- Base unit: 4px
- Section padding: 80px (desktop), 48px (mobile)
- Card padding: 24px
- Generous whitespace throughout

### Visual Effects
- Hairline dividers: 1px solid #E8E4DE
- Soft shadows: 0 4px 20px rgba(0,0,0,0.06)
- Hover transitions: 300ms ease
- Subtle scale on hover: scale(1.02)

### Buttons
**Primary (Accent):**
- Background: #1A1A1A (charcoal)
- Text: #FAF7F2 (cream)
- Padding: 16px 32px
- Border-radius: 0 (sharp edges for premium feel)
- Hover: background lightens slightly

**Secondary (Outlined):**
- Border: 1px solid #1A1A1A
- Background: transparent
- Text: #1A1A1A

## Component Guidelines

### Navigation
- Transparent over hero, solid on scroll
- Logo: serif "VELMORA" left-aligned
- Center links: Shop, Collections, About, Journal
- Right: search + cart icons

### Product Cards
- Image with hover reveal second image
- Quick "Add to Cart" on hover
- Product name: uppercase serif
- Price below name

### Cards & Containers
- Background: #FFFEF9 (ivory)
- Border: 1px solid #E8E4DE
- Border-radius: 0 (premium feel)
- Soft shadow on hover

### Forms
- Input border: 1px solid #E8E4DE
- Focus: border-color #C9A962 (gold)
- Placeholder: #6B6560 (warm-gray)

## Responsive Breakpoints
- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px

## Animations
- Page transitions: fade in 300ms
- Hover effects: 300ms ease
- Scroll animations: subtle fade-up
- Cart drawer: slide from right

## Do's and Don'ts
- DO: Use generous whitespace
- DO: Use elegant serif fonts for headings
- DO: Keep gold accents subtle and refined
- DO: Use hairline dividers
- DON'T: Use loud discount-style colors
- DON'T: Use generic e-commerce blue
- DON'T: Overcrowd elements
- DON'T: Use rounded corners on buttons (keep sharp for premium feel)
