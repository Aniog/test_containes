# ARTITECT MACHINERY - Design System

## Brand Identity
ARTITECT MACHINERY manufactures premium sheet metal folding machines. The brand conveys precision, strength, and industrial elegance.

## Color Palette
- **Primary (Steel Blue)**: `#1e293b` (slate-800) — Used for headers, navigation, primary buttons
- **Primary Dark**: `#0f172a` (slate-900) — Used for footer, deep backgrounds
- **Accent (Industrial Orange)**: `#f59e0b` (amber-500) — Used for CTAs, highlights, key actions
- **Accent Hover**: `#d97706` (amber-600) — Button hover states
- **Surface**: `#ffffff` — Cards, content areas
- **Background**: `#f8fafc` (slate-50) — Page background
- **Muted Text**: `#64748b` (slate-500) — Secondary text, descriptions
- **Border**: `#e2e8f0` (slate-200) — Card borders, dividers

## Typography
- **Font Family**: Inter (Google Fonts) — Clean, modern, highly readable
- **Headings**: Bold, tracking-tight. Scale: text-4xl (hero), text-3xl (section), text-2xl (card), text-xl (sub)
- **Body**: text-base leading-relaxed text-slate-600
- **Small/Caption**: text-sm text-slate-500

## Spacing
- **Section padding**: py-20 md:py-28
- **Container**: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8
- **Card padding**: p-6 md:p-8
- **Gap between cards**: gap-6 md:gap-8

## Visual Style
- Clean white cards with subtle borders and shadow-sm
- Rounded corners: rounded-lg for cards, rounded-md for buttons
- Subtle hover effects: shadow-md transition on cards
- Industrial blue headers against light backgrounds
- Orange accent sparingly for CTAs and highlights
- Elegant whitespace, not cluttered

## Do's
- Use slate-800 for main headings
- Use slate-600 for body text
- Use white cards on slate-50 background
- Keep navigation clean and simple
- Use amber-500 only for primary CTAs and accents

## Don'ts
- Don't use too many colors
- Don't overcrowd layouts
- Don't use dark background for main content
- Don't use harsh shadows (stick to shadow-sm/shadow-md)
- Don't use gradients
