# SSourcing China - Design System

## Visual Style

**Overall Aesthetic:** Clean, trustworthy, international B2B. Professional and practical without exaggeration.

**Color Palette:**
- Primary: #0A2540 (Deep navy - trust, professionalism)
- Secondary: #1E3A5F (Medium navy)
- Accent: #2563EB (Professional blue for CTAs)
- Accent Light: #3B82F6
- Text Primary: #0F172A (Near black for readability)
- Text Secondary: #475569 (Slate gray for supporting text)
- Text Muted: #64748B
- Background: #FFFFFF (White)
- Background Alt: #F8FAFC (Light gray for sections)
- Border: #E2E8F0
- Success: #059669 (For trust indicators)

**Typography:**
- Font Family: Inter, system-ui, -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif
- Headlines: 600-700 weight, tight tracking
- Body: 400-500 weight, 1.6 line-height
- Small text: 14px, 500 weight for labels

**Spacing Scale:**
- xs: 4px
- sm: 8px
- md: 16px
- lg: 24px
- xl: 32px
- 2xl: 48px
- 3xl: 64px
- 4xl: 80px

**Border Radius:**
- sm: 4px
- md: 6px
- lg: 8px
- xl: 12px
- full: 9999px

**Shadows:**
- sm: 0 1px 2px 0 rgb(0 0 0 / 0.05)
- md: 0 4px 6px -1px rgb(0 0 0 / 0.1), 0 2px 4px -2px rgb(0 0 0 / 0.1)
- lg: 0 10px 15px -3px rgb(0 0 0 / 0.1), 0 4px 6px -4px rgb(0 0 0 / 0.1)

## Component Guidelines

**Buttons:**
- Primary: Navy background, white text, 600 weight
- Secondary: White background, navy border, navy text
- Large CTA: 48px height, 18px font, rounded-lg
- Hover: Slight darken or blue accent

**Cards:**
- White background
- Subtle border (#E2E8F0)
- 8px border radius
- 24px internal padding
- Hover: subtle shadow lift

**Forms:**
- Clean inputs with 1px border
- Focus: 2px blue ring
- Labels: 14px, 600 weight, slate
- Error: Red text below field

**Navigation:**
- Fixed header with white background
- Subtle bottom border
- Logo left, nav center, CTA right
- Mobile: hamburger menu

## Image Usage

Use data-strk-img system for all visuals:
- Factory interiors and production lines
- Quality control inspection
- Shipping and logistics
- Professional team meetings
- Product samples

All images should feel authentic and documentary-style, not stock-photo polished.

## Tone & Language

- Professional, clear, practical
- No exaggerated claims ("best", "guaranteed", "100%")
- Focus on process, verification, and risk reduction
- Use specific numbers where realistic (e.g., "200+ factories visited", "15 years experience")
- Lead with value, not hype

## Page Structure Principles

- Generous whitespace
- Clear visual hierarchy
- Section dividers with subtle backgrounds
- Consistent CTA placement
- Mobile-first responsive
- Fast, scannable content blocks
