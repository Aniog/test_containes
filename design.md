# MicroCosmos Design System

## Theme
Strict black and white monochrome UI. All text, backgrounds, borders, and UI elements use only black, white, and shades of gray. Color appears ONLY in photographs/images.

## Colors
- Background primary: `bg-black` (#000000)
- Background secondary: `bg-neutral-950` (#0a0a0a)
- Background card: `bg-neutral-900` (#171717)
- Background subtle: `bg-neutral-800` (#262626)
- Border: `border-neutral-700` (#404040) or `border-white/10`
- Text primary: `text-white`
- Text secondary: `text-neutral-300`
- Text muted: `text-neutral-500`
- Accent (white): `text-white`, `bg-white`, `text-black`

## Typography
- Font: Inter (Google Fonts)
- Display headings: `font-black tracking-tighter` (very bold, tight)
- Section headings: `font-bold tracking-tight`
- Body: `font-normal leading-relaxed`
- Labels/captions: `text-xs font-medium tracking-widest uppercase`

## Spacing
- Section padding: `py-24 md:py-32`
- Container: `max-w-7xl mx-auto px-6 md:px-12`
- Card gap: `gap-6 md:gap-8`

## Borders & Dividers
- Thin white lines: `border border-white/10`
- Dividers: `border-t border-white/10`

## Images
- All images are in full color (no filters applied)
- Images are displayed in rounded containers: `rounded-2xl overflow-hidden`
- Images use `object-cover w-full h-full`
- Hero images: large, full-bleed or near full-bleed
- Gallery images: varied aspect ratios for visual interest

## Buttons
- Primary: `bg-white text-black font-semibold px-8 py-3 rounded-full hover:bg-neutral-200`
- Secondary/outline: `border border-white text-white px-8 py-3 rounded-full hover:bg-white hover:text-black`

## Cards
- Background: `bg-neutral-900 rounded-2xl overflow-hidden`
- Hover: subtle scale or border highlight

## Do's
- Use stark contrast between black backgrounds and white text
- Let colorful images "pop" against the monochrome UI
- Use generous whitespace
- Use uppercase tracking-widest for section labels

## Don'ts
- No colored text (no blue links, no colored badges)
- No colored backgrounds on UI elements
- No gradients (except subtle black-to-transparent overlays on images)
