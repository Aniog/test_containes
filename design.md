# 秋易广告公司 Design System

## Brand Identity
Professional advertising agency with a premium, modern aesthetic. Conveys creativity, trust, and expertise.

## Color Palette

### Primary Colors
- Deep Navy (background): `#0A0F1E` — `bg-[#0A0F1E]`
- Dark Charcoal (sections): `#111827` — `bg-gray-900`
- Card Background: `#1A2035` — `bg-[#1A2035]`

### Accent Colors
- Gold (primary accent): `#D4A853` — `text-[#D4A853]` / `bg-[#D4A853]`
- Warm Orange (secondary accent): `#E8834A` — `text-[#E8834A]`
- Gradient: from `#D4A853` to `#E8834A`

### Text Colors
- Primary Text: `#F9FAFB` — `text-gray-50`
- Secondary Text: `#9CA3AF` — `text-gray-400`
- Muted Text: `#6B7280` — `text-gray-500`

### Border Colors
- Subtle Border: `rgba(212, 168, 83, 0.2)` — `border-[#D4A853]/20`
- Card Border: `rgba(255,255,255,0.08)` — `border-white/10`

## Typography

### Font Family
- Primary: Inter (Google Fonts)
- Chinese: System default CJK fonts

### Scale
- Hero Title: `text-5xl md:text-7xl font-bold`
- Section Title: `text-3xl md:text-4xl font-bold`
- Card Title: `text-xl font-semibold`
- Body: `text-base text-gray-300`
- Caption: `text-sm text-gray-400`

### Do's
- Use bold weights (700) for headings
- Use gold accent for key words in headings
- Keep body text at gray-300 for readability on dark backgrounds

### Don'ts
- Never use white text on gold/amber backgrounds
- Never use gray-400 or lighter on white backgrounds
- Avoid mixing too many font weights

## Spacing
- Section padding: `py-20 md:py-28`
- Container: `max-w-7xl mx-auto px-6 md:px-12`
- Card padding: `p-6 md:p-8`
- Gap between cards: `gap-6 md:gap-8`

## Components

### Buttons
- Primary: `bg-gradient-to-r from-[#D4A853] to-[#E8834A] text-white font-semibold px-8 py-3 rounded-full hover:opacity-90 transition`
- Outline: `border border-[#D4A853] text-[#D4A853] px-8 py-3 rounded-full hover:bg-[#D4A853]/10 transition`

### Cards
- Background: `bg-[#1A2035]`
- Border: `border border-white/10`
- Radius: `rounded-2xl`
- Hover: `hover:border-[#D4A853]/40 transition`

### Section Dividers
- Use subtle gold gradient lines: `h-px bg-gradient-to-r from-transparent via-[#D4A853]/40 to-transparent`

## Visual Style
- Dark, premium aesthetic with gold accents
- Subtle gradients and glows for depth
- Clean grid layouts
- Professional photography for all images
- Smooth hover transitions (200-300ms)
