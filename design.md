# Visual Identity: Velmora Fine Jewelry

## Mood
Quiet luxury, warm, editorial, premium demi-fine jewelry.

## Palette
- **Cream (Base)**: `#f9f7f2` (Tailwind: `bg-velmora-cream`)
- **Stone (Secondary)**: `#f1ede4` (Tailwind: `bg-velmora-stone`)
- **Gold (Accent)**: `#c5a059` (Tailwind: `text-velmora-gold`)
- **Black (Contrast)**: `#1a1a1a` (Tailwind: `text-velmora-black`)
- **White**: `#ffffff`

## Typography
- **Headings / Product Names**: Serif - Cormorant Garamond. High-end, elegant.
  - Example: `font-serif tracking-[0.2em] uppercase`
- **Body / UI**: Sans-serif - Manrope. Clean, modern, accessible.
  - Example: `font-sans tracking-wide`

## Design Elements
- **Borders**: Thin hairline dividers (`border-velmora-black/5`)
- **Spacing**: Generous whitespace for an editorial feel.
- **Imagery**: Large full-bleed hero, high-quality product grids, hover image swaps.
- **Transitions**: Subtle hover effects, smooth transitions (`duration-500` or `duration-1000`).
- **Buttons**: Structured, premium, uppercase labels with wide letter-spacing.

## Do's
- Use uppercase for labels and navigation.
- Maintain wide tracking for uppercase text.
- Use `velmora-cream` for page backgrounds.
- Keep shadows soft and subtle.

## Don'ts
- Use neon colors or heavy gradients.
- Overcrowd the layout.
- Use rounded corners for primary call-to-action buttons (keep them sharp or slightly rounded).
- Use generic sans-serif for primary branding.
