# Velmora Fine Jewelry Design System

## Brand Identity
- **Mood**: Quiet luxury, warm, editorial, premium demi-fine jewelry.
- **Tone**: Sophisticated, timeless, accessible.

## Typography
- **Headings**: Cormorant Garamond (Serif)
  - `font-serif`
  - Uppercase for product names with wide letter-spacing (`tracking-widest`).
- **Body/UI**: Inter / Manrope (Sans-serif)
  - `font-sans`

## Color Palette
- **Background**: Soft Cream / Off-white (`#FCF9F5`)
- **Primary Accent**: Deep Charcoal / Bronze (`#1C1C1C`)
- **Secondary Accent**: Warm Gold (`#D4AF37`)
- **Text**: Deep Espresso (`#2C2C2C`)
- **Muted**: Soft Gray (`#F5F5F5`)

## UI Elements
- **Buttons**:
  - Solid: Deep charcoal background, light cream text.
  - Outlined: Thin hairline divider, minimal padding.
- **Dividers**: Hairline (`0.5px`), subtle.
- **Shadows**: Very soft, minimal (`shadow-sm`).
- **Transitions**: Smooth, subtle hover effects.

## Space & Layout
- Generous whitespace.
- Large editorial imagery.
- Balanced proportions.
