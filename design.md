# SSourcing China - Design Guidelines

## Project Overview
- **Project Name**: SSourcing China Website
- **Type**: B2B Corporate Website
- **Purpose**: Generate qualified sourcing inquiries for China sourcing agent services
- **Target Audience**: Overseas buyers, procurement managers, business owners looking to source products from China

## Visual Style

### Color Palette
- **Primary**: `#1a365d` (Deep Navy Blue - trust, professionalism)
- **Primary Light**: `#2c5282` (Medium Blue)
- **Secondary**: `#c53030` (Professional Red - action, China connection)
- **Accent**: `#d69e2e` (Gold - premium, quality)
- **Background**: `#f7fafc` (Light Gray)
- **Surface**: `#ffffff` (White)
- **Text Primary**: `#1a202c` (Near Black)
- **Text Secondary**: `#4a5568` (Gray)
- **Text Muted**: `#718096` (Light Gray)
- **Border**: `#e2e8f0` (Light Border)
- **Success**: `#38a169` (Green)
- **Warning**: `#dd6b20` (Orange)

### Typography
- **Font Family**: "Inter", system-ui, sans-serif
- **Headings**: 
  - H1: 48px/56px, font-weight 700
  - H2: 36px/44px, font-weight 700
  - H3: 24px/32px, font-weight 600
  - H4: 20px/28px, font-weight 600
- **Body**: 16px/24px, font-weight 400
- **Small**: 14px/20px, font-weight 400

### Spacing System
- Base unit: 4px
- Section padding: 80px vertical (desktop), 48px (mobile)
- Container max-width: 1200px
- Grid gap: 32px

### Visual Effects
- Card shadows: `0 4px 6px -1px rgba(0, 0, 0, 0.1), 0 2px 4px -1px rgba(0, 0, 0, 0.06)`
- Hover transitions: 200ms ease-in-out
- Border radius: 8px (cards), 4px (buttons), 50% (avatars)

## Layout Structure

### Header
- Fixed position, white background with subtle shadow on scroll
- Logo (left): "SSourcing China" text logo
- Navigation (center): Home, Services, How It Works, Products, Case Studies, Blog
- CTA Button (right): "Get a Free Quote" - primary color

### Footer
- Dark background (#1a365d)
- 4 columns: Company, Services, Resources, Contact
- Social links, copyright

### Responsive Breakpoints
- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px

## Components

### Buttons
- **Primary**: Navy background, white text, red hover
- **Secondary**: White background, navy border, navy text
- **Ghost**: Transparent, navy text

### Cards
- White background
- Subtle shadow
- 8px border radius
- Hover: slight lift effect

### Icons
- Lucide React icons
- Size: 24px default, 20px for inline

### Form Inputs
- Border: 1px solid #e2e8f0
- Focus: 2px navy outline
- Padding: 12px 16px
- Border radius: 4px

## Image Guidelines
- Use professional B2B imagery
- Factory/warehouse photos
- Quality inspection visuals
- Shipping/logistics images
- Professional business settings
- Avoid: overly stock-looking photos, generic business people

## Do's and Don'ts
- DO: Clean, professional, trustworthy appearance
- DO: Clear value propositions
- DO: Easy navigation
- DO: Prominent CTAs
- DON'T: Exaggerated claims
- DON'T: Cluttered layouts
- DON'T: Too many colors
- DON'T: Generic stock photos
