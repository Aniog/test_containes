# Velmora Fine Jewelry - Design System

## Visual Identity

### Mood
- Quiet luxury, warm, editorial
- Premium demi-fine jewelry
- NOT loud, NOT discount-looking, NOT generic e-commerce

### Color Palette
- **Primary Background**: `#FAF9F7` (warm off-white/cream)
- **Secondary Background**: `#F5F3EF` (warm light gray)
- **Dark Background**: `#1A1815` (warm charcoal)
- **Primary Text**: `#1A1815` (warm charcoal)
- **Secondary Text**: `#6B6560` (warm gray)
- **Accent Gold**: `#C9A962` (warm gold)
- **Accent Gold Hover**: `#B8954F` (darker gold)
- **Border/Divider**: `#E5E0D8` (warm light border)
- **White**: `#FFFFFF`

### Typography
- **Headings**: Cormorant Garamond (serif) - elegant, editorial
  - H1: 56px / 64px line-height / font-weight 500
  - H2: 40px / 48px line-height / font-weight 500
  - H3: 28px / 36px line-height / font-weight 500
  - Product names: 16px / uppercase / 0.15em letter-spacing / font-weight 500
  
- **Body/UI**: Manrope (sans-serif) - clean, modern
  - Body: 16px / 24px line-height / font-weight 400
  - Small: 14px / 20px line-height
  - Button: 14px / uppercase / 0.1em letter-spacing / font-weight 500

### Spacing System
- Base unit: 4px
- Section padding: 80px vertical (desktop), 48px (mobile)
- Container max-width: 1280px
- Grid gap: 24px (desktop), 16px (mobile)

### Visual Effects
- **Shadows**: 
  - Soft: `0 4px 20px rgba(26, 24, 21, 0.08)`
  - Card hover: `0 8px 30px rgba(26, 24, 21, 0.12)`
- **Border radius**: 0px (sharp, editorial) or 4px (subtle)
- **Dividers**: 1px solid #E5E0D8 (hairline)
- **Transitions**: 300ms ease-out (subtle, premium feel)

### Components

#### Buttons
- **Primary**: 
  - Background: #C9A962 (gold)
  - Text: #1A1815 (dark)
  - Padding: 16px 32px
  - Hover: #B8954F (darker gold)
- **Secondary/Outline**:
  - Border: 1px solid #1A1815
  - Text: #1A1815
  - Background: transparent
  - Hover: #1A1815 background with #FAF9F7 text

#### Product Cards
- Sharp corners (0px radius)
- Image aspect ratio: 4:5 (portrait)
- Hover: reveal second image + quick add to cart
- Product name: uppercase, wide letter-spacing

#### Form Inputs
- Border: 1px solid #E5E0D8
- Focus: border-color #C9A962
- Padding: 12px 16px

### Responsive Breakpoints
- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px

### Do's and Don'ts
- DO: Use generous whitespace, large imagery, thin dividers
- DO: Keep accent color restrained (gold only for CTAs and highlights)
- DON'T: Use bright/chunky colors, generic gradients, loud discount badges
- DON'T: Use rounded corners > 4px (keep it editorial)
- DON'T: Use more than 2 font weights per font family
