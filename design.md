# ARTITECT MACHINERY - Design System

## Brand Identity
Industrial elegance meets modern web design. The brand conveys precision engineering,
reliability, and sophistication. Avoid overly playful or casual styling.

## Color Palette
- **Primary (Deep Navy):** #0C1B2A — backgrounds, headers, footer
- **Primary Light:** #1A2D42 — card backgrounds, secondary surfaces
- **Accent (Amber/Gold):** #D4A843 — CTAs, highlights, active states
- **Accent Hover:** #E8BE5A — hover on amber elements
- **Steel Gray:** #8B95A5 — body text on dark backgrounds
- **Light Surface:** #F4F6F9 — light section backgrounds
- **White:** #FFFFFF — cards, text on dark backgrounds
- **Charcoal Text:** #1E293B — headings on light backgrounds
- **Muted Text:** #64748B — secondary text on light backgrounds

## Typography
- **Headings:** Inter, 700/800 weight, tight letter-spacing (-0.02em)
- **Body:** Inter, 400/500 weight
- **h1:** text-4xl md:text-5xl lg:text-6xl
- **h2:** text-3xl md:text-4xl lg:text-5xl
- **h3:** text-xl md:text-2xl
- **Body:** text-base md:text-lg

## Layout
- Max content width: max-w-7xl (1280px)
- Section padding: py-16 md:py-24
- Card border radius: rounded-xl
- Button border radius: rounded-lg

## Spacing
- Section vertical gap: gap-16 md:gap-24
- Card internal padding: p-6 md:p-8
- Grid gaps: gap-6 md:gap-8

## Visual Style
- Subtle box shadows on cards (shadow-lg)
- No heavy borders; use shadow or bg contrast
- Hero section uses full-bleed background
- Product cards have hover lift effect (transform transition)
- Clean horizontal separators between sections

## Do's
- Use generous whitespace
- Keep contrast ratios high (WCAG AA minimum)
- Use amber/gold sparingly for emphasis only
- Maintain consistent spacing rhythm
- Use Tailwind CSS classes only (no inline styles)

## Don't
- Don't use bright or neon colors
- Don't use playful/rounded cartoonish styles
- Don't overcrowd sections
- Don't mix more than 2 accent colors
- Don't use inline styles
