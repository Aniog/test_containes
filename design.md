# Visual Identity for SSourcing China

## Core Principles
- **Professional & Trustworthy**: Focus on reliability and transparency.
- **International B2B**: Clean, corporate feel that appeals to global buyers.
- **Efficiency**: Visuals and layout should suggest a streamlined sourcing process.

## Typography
- **Primary Font**: Inter (Sans-serif)
  - Headings: Semi-bold to Extra-bold (`font-semibold` to `font-extrabold`)
  - Body: Normal to Medium (`font-normal` to `font-medium`)

## Color Palette
- **Primary (Corporate Blue)**: `#1a3d66` (Tailwind: `bg-[#1a3d66]`, `text-[#1a3d66]`)
- **Secondary (Trust Silver/Gray)**: `#f8fafc` (Tailwind: `bg-slate-50`)
- **Accent (Call to Action)**: `#ff6b00` (Tailwind: `bg-[#ff6b00]`, `text-[#ff6b00]`)
- **Text (Dark Slate)**: `#1e293b` (Tailwind: `text-slate-800`)
- **Muted Text**: `#64748b` (Tailwind: `text-slate-500`)

## UI Elements
- **Buttons**: Rounded-md (`rounded-md`), consistent padding, hover effects with slight darkening.
- **Cards**: Subtle shadows (`shadow-sm` to `shadow-md`), white background, thin borders (`border-slate-100`).
- **Input Fields**: Clean, professional borders, focus state using primary blue.
- **Navigation**: Simple, top-aligned, clear active states.

## Spacing & Layout
- Generous whitespace to ensure clarity.
- Responsive grids (1 column mobile, 3-4 columns desktop).
- Max-width containers for content (e.g., `max-w-7xl mx-auto`).

## Imagery
- Realistic stock photos of:
  - Modern China factories and production lines.
  - Quality control inspectors at work.
  - Shipping containers and logistics hubs.
  - Business meetings and team collaborations.
- Use the `data-strk-img` system for all stock images.
