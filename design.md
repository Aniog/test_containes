# Velmora Fine Jewelry — Design System

## Visual Identity
- Mood: Quiet luxury, warm, editorial. Premium demi-fine jewelry.
- NOT loud, NOT discount-looking, NOT generic e-commerce.

## Color Palette
- **Background**: `#FAF7F2` (brand-cream) — warm off-white, primary page background
- **Surface**: `#F5EDE3` (brand-warm) — slightly warmer, for cards and sections
- **Border/Divider**: `#E8DFD1` (brand-sand) — hairline dividers, subtle borders
- **Accent**: `#C9A96E` (brand-gold) — primary accent, buttons, links, highlights
- **Accent Hover**: `#A8863F` (brand-gold-dark) — button hover states
- **Text Primary**: `#2C2C2C` (brand-charcoal) — body text, headings
- **Text Muted**: `#8A8278` (brand-muted) — secondary text, captions
- **Text Light Muted**: `#B5AFA7` (brand-muted-light) — placeholders, disabled
- **Dark Section BG**: `#1A1612` (brand-espresso) — hero overlays, dark sections
- **Dark Section Surface**: `#2A2420` (brand-espresso-light) — cards on dark sections

## Typography
- **Headings**: Cormorant Garamond (serif), weights 400-600
- **Body/UI**: Inter (sans-serif), weights 300-500
- **Product names**: UPPERCASE, letter-spacing 0.2em (tracking-super-wide), serif
- **Section headings**: UPPERCASE, letter-spacing 0.15em, serif
- **Nav links**: UPPERCASE, letter-spacing 0.15em, sans-serif, text-xs

## Spacing & Layout
- Generous whitespace. Sections: py-20 md:py-28
- Container max-width: max-w-7xl (80rem), centered
- Hairline dividers: h-px bg-brand-sand
- Card padding: p-6 md:p-8

## Buttons
- **Primary (accent)**: bg-brand-gold text-white, hover:bg-brand-gold-dark, tracking-wide, uppercase, text-xs, px-8 py-3, transition-all duration-300
- **Secondary (outlined)**: border border-brand-gold text-brand-gold, hover:bg-brand-gold hover:text-white, same sizing
- **Subtle**: text-brand-charcoal underline-offset-4 hover:underline

## Images
- Large editorial imagery, warm gold jewelry on dark/neutral backgrounds
- Product cards: aspect-[3/4], object-cover
- Category tiles: aspect-[4/5], object-cover
- Hero: full-bleed, aspect-[16/9] on desktop, aspect-[4/5] on mobile

## Animations
- Subtle hover transitions: duration-300
- Fade-in on scroll: animate-fade-in, animate-slide-up
- Image hover: scale-105 transition-transform duration-700
- No jarring or bouncy animations

## Do's
- Use warm, muted tones throughout
- Keep whitespace generous
- Use serif for emotional/editorial moments
- Ensure strong contrast (charcoal on cream, white on espresso)
- Use thin hairline dividers between sections

## Don'ts
- Don't use bright or saturated colors
- Don't crowd elements — give them room to breathe
- Don't use sans-serif for product names or hero headlines
- Don't use heavy shadows — keep them soft and subtle
- Don't use rounded buttons — keep them refined with minimal radius
