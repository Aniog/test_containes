# Velmora Fine Jewelry - Design System

## Brand Identity
- **Brand Name**: Velmora Fine Jewelry
- **Type**: Direct-to-consumer (B2C) demi-fine jewelry
- **Target Audience**: Women 25-45, gifting and self-purchase
- **Price Range**: $30–$120 (premium-but-accessible)

## Visual Identity

### Mood
- Quiet luxury, warm, editorial
- Premium demi-fine jewelry
- NOT loud, NOT discount-looking, NOT generic e-commerce

### Color Palette
**Primary Colors:**
- `velmora-cream` - #FAF7F2 (warm off-white background)
- `velmora-charcoal` - #1A1A1A (primary text, dark)
- `velmora-gold` - #C9A962 (accent, CTAs, highlights)
- `velmora-gold-light` - #E8D5A8 (hover states, secondary accents)

**Neutral Colors:**
- `velmora-warm-gray` - #8B8680 (secondary text)
- `velmora-light-gray` - #F5F3EF (card backgrounds, dividers)
- `velmora-border` - #E5E0D8 (borders, hairline dividers)

**Semantic Colors:**
- `velmora-success` - #4A7C59 (success states)
- `velmora-error` - #A65D57 (error states)

### Typography
**Headings & Product Names:**
- Font Family: "Cormorant Garamond", serif
- Weight: 500-600
- Letter-spacing: 0.05em (wide for product names)
- Product names: UPPERCASE

**Body & UI:**
- Font Family: "Manrope", sans-serif
- Weight: 400-500
- Clean, readable

### Spacing System
- Base unit: 4px
- Section padding: 80px vertical (desktop), 48px (mobile)
- Card padding: 24px
- Generous whitespace throughout

### Visual Effects
- Hairline dividers: 1px solid velmora-border
- Soft shadows: 0 4px 20px rgba(26, 26, 26, 0.08)
- Hover transitions: 300ms ease
- Subtle scale on hover: scale(1.02)
- Buttons: solid or outlined accent style

## Components

### Navigation
- Sticky, transparent over hero, turns solid on scroll
- Left: serif logo "VELMORA"
- Center: Shop, Collections, About, Journal
- Right: search + cart icons

### Buttons
- Primary: solid velmora-gold background, velmora-charcoal text
- Secondary: outlined velmora-gold border, transparent background
- Hover: velmora-gold-light background
- Padding: 14px 32px
- Border-radius: 0 (sharp edges for premium feel)

### Product Cards
- Image with hover reveal second image
- Quick "Add to Cart" on hover
- Product name: serif, uppercase, wide spacing
- Price below name

### Dividers
- Hairline style: 1px solid velmora-border
- Used between sections

## Responsive Breakpoints
- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px
- Mobile-first approach

## Animations
- Fade-in on scroll: opacity 0 to 1, translateY 20px to 0
- Hover transitions: 300ms ease
- Image hover: subtle scale
- Button hover: background color shift
