# Strikingly Generators Hub Visual Style

## Brand voice
Polished, direct, and helpful. The page should feel like a useful Strikingly product directory, not a search-engine landing page.

## Typography
- Headings and buttons use Brandon Grotesque where available, with Josefin Sans as the Google Fonts fallback. Use uppercase, weight 700, and tight line-height.
- Body copy uses Open Sans, 14px base, line-height 1.5.

## Colors
- Headings: #4B5056; hero primary line: #2E2E2F.
- Body text: #636972.
- Brand purple: #8159BB.
- AI gradient: #7671FF to #CB0C9F. Use only for primary CTAs and the second H1 line.
- Card border: #C6C9CD; divider: #E2E4E7; light background: #F4F6F8; white: #FFFFFF.

## Layout and spacing
- Max content width 1200px, centered.
- Use spacing in 10px increments, with 5px only for tight pairs.
- Sections use 40px vertical rhythm; hero uses 60px to 80px vertical padding.
- Desktop grids should use three columns where requested, tablet two columns, mobile one column.

## Components
- Buttons: 36px height by default, 44px for primary large CTAs, 4px radius, uppercase heading font. Filled/gradient buttons must use white text.
- Cards: white background, 1px #C6C9CD border, 3px radius, 20px padding. Hover lifts subtly and changes border to #8159BB.
- Tags: 11px uppercase heading font, 2px 6px padding, 3px radius, transparent with #8159BB border and text.

## Do
- Keep all text clearly readable.
- Use semantic sections and real links/buttons.
- Use inline SVG or CSS illustrations only.

## Don't
- Do not use invented social proof.
- Do not use gradient backgrounds for sections.
- Do not embed the AI builder form on the hub page.
