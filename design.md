# Velmora Fine Jewelry - Design System

## Visual Identity

### Mood
- Quiet luxury, warm, editorial
- Premium demi-fine jewelry aesthetic
- NOT loud, NOT discount-looking, NOT generic e-commerce

### Color Palette

**Primary Colors:**
- `velmora-cream` - #FAF7F2 (warm off-white background)
- `velmora-charcoal` - #1A1A1A (primary text, dark backgrounds)
- `velmora-gold` - #C9A962 (accent color - warm gold)
- `velmora-gold-light` - #E8D5A8 (light gold for hover states)

**Secondary Colors:**
- `velmora-sand` - #F0EBE3 (secondary backgrounds, cards)
- `velmora-warm-gray` - #6B6560 (secondary text)
- `velmora-border` - #E5E0D8 (borders, dividers)

**Semantic Colors:**
- `velmora-success` - #4A7C59 (success states)
- `velmora-error` - #9B3D3D (error states)

### Typography

**Headings & Product Names:**
- Font: "Cormorant Garamond" (serif)
- Weight: 500, 600
- Letter-spacing: 0.05em (wide for uppercase)
- Product names: UPPERCASE with 0.1em letter-spacing

**Body & UI:**
- Font: "Manrope" (sans-serif)
- Weight: 400, 500, 600
- Clean, readable, modern

**Font Sizes:**
- Hero headline: 3.5rem (56px) desktop, 2.5rem mobile
- Section titles: 2rem (32px) desktop, 1.5rem mobile
- Product names: 1rem (16px)
- Body: 0.9375rem (15px)
- Small/caption: 0.8125rem (13px)

### Spacing System
- Base unit: 4px
- Section padding: 80px vertical (desktop), 48px (mobile)
- Container max-width: 1280px
- Grid gap: 24px (desktop), 16px (mobile)

### Visual Elements
- Generous whitespace
- Large editorial imagery
- Thin hairline dividers (1px, velmora-border)
- Restrained accent color usage
- Subtle hover transitions (200-300ms ease)
- Soft shadows: 0 4px 20px rgba(0,0,0,0.08)

### Buttons

**Primary (Accent):**
- Background: velmora-gold (#C9A962)
- Text: velmora-charcoal (#1A1A1A)
- Padding: 14px 32px
- Border-radius: 0 (sharp edges for premium feel)
- Hover: velmora-gold-light background

**Secondary (Outlined):**
- Border: 1px solid velmora-charcoal
- Background: transparent
- Text: velmora-charcoal
- Hover: velmora-charcoal background, cream text

### Components

**Product Cards:**
- Image container with aspect-ratio 4:5
- Hover: reveal second image with fade transition
- Quick "Add to Cart" button appears on hover
- Product name in uppercase serif
- Price below name

**Navigation:**
- Transparent over hero, solid on scroll
- Logo: serif "VELMORA" 
- Links: clean sans-serif
- Icons: minimal line style

**Accordions:**
- Thin top border
- Plus/minus icon indicator
- Smooth expand/collapse animation

**Form Inputs:**
- Border-bottom style (underline)
- Focus: gold accent color
- Clean, minimal appearance

## Responsive Breakpoints
- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px

## Do's and Don'ts

**DO:**
- Use generous whitespace between sections
- Keep images large and editorial
- Use gold accent sparingly (buttons, links, stars)
- Maintain consistent letter-spacing on uppercase text
- Use thin dividers to separate content

**DON'T:**
- Use bright or neon accent colors
- Create cluttered layouts
- Use generic sans-serif for headings
- Add too many visual elements
- Make it look like a discount store
