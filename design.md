# Velmora Fine Jewelry — Design System

## Visual Identity
- Mood: Quiet luxury, warm, editorial. Premium demi-fine jewelry.
- NOT loud, NOT discount-looking, NOT generic e-commerce.

## Color Palette
- **Background**: `#FAF7F2` (warm cream) — primary page background
- **Surface**: `#F5F0E8` (ivory) — cards, sections
- **Border/Divider**: `#E8DFD0` (sand) — hairline dividers, subtle borders
- **Accent**: `#C9A96E` (gold) — CTAs, highlights, hover states
- **Accent Hover**: `#A8863F` (gold-dark) — button hover
- **Text Primary**: `#2C2C2C` (charcoal) — headings, body
- **Text Muted**: `#9C9486` (muted) — secondary text, captions
- **Dark Surface**: `#1A1A1A` (onyx) — hero overlays, dark sections

## Typography
- **Headings**: Cormorant Garamond (serif), weights 400-600
- **Body/UI**: Inter (sans-serif), weights 300-500
- **Product names**: Uppercase, letter-spacing 0.2em, serif
- **Section labels**: Uppercase, letter-spacing 0.15em, sans, small

## Spacing
- Generous whitespace throughout
- Section padding: 80-120px vertical on desktop, 48-64px on mobile
- Content max-width: 1280px
- Grid gaps: 24-32px

## Components
- **Buttons**: Solid charcoal or gold, uppercase, wide letter-spacing, smooth hover transitions
- **Cards**: Minimal, no visible border, subtle shadow on hover
- **Dividers**: 1px hairline in sand color
- **Images**: Editorial, warm-toned, generous sizing

## Do's
- Use warm, cream backgrounds
- Use serif for all headings and product names
- Keep generous whitespace
- Use subtle, smooth transitions (300ms ease)
- Use gold as accent sparingly

## Don'ts
- Don't use bright or saturated colors
- Don't use heavy borders or shadows
- Don't use sans-serif for headings
- Don't crowd elements — let them breathe
- Don't use discount/sale visual patterns
