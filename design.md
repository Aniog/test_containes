# SSourcing China - Visual Style Guide

## Brand Identity
- **Name**: SSourcing China
- **Tagline**: China Sourcing Agent for Global Buyers
- **Tone**: Professional, clear, practical, trustworthy. No exaggerated claims.

## Color Palette
- **Primary**: Deep Navy `#0F2B46` — trust, authority, B2B professionalism
- **Primary Accent**: Steel Blue `#2563EB` — CTAs, links, interactive elements
- **Secondary**: Warm Gold `#D4A853` — premium feel, highlights, badges
- **Background**: White `#FFFFFF` and Light Gray `#F8FAFC`
- **Text Primary**: Dark Slate `#0F172A`
- **Text Secondary**: Slate `#475569`
- **Text Muted**: Slate `#94A3B8`
- **Border**: `#E2E8F0`
- **Success**: `#16A34A`
- **Warning**: `#D4A853`

## Typography
- **Font Family**: Inter (Google Fonts)
- **Headings**: Inter, font-weight 700-800, tracking tight
- **Body**: Inter, font-weight 400, line-height 1.6-1.75
- **H1**: text-4xl md:text-5xl lg:text-6xl, font-extrabold
- **H2**: text-3xl md:text-4xl, font-bold
- **H3**: text-xl md:text-2xl, font-semibold
- **Body Large**: text-lg
- **Body**: text-base
- **Small**: text-sm

## Spacing
- Section padding: py-16 md:py-24
- Container max-width: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8
- Card padding: p-6 md:p-8
- Gap between grid items: gap-6 md:gap-8

## Borders & Shadows
- Card border: border border-slate-200
- Card hover shadow: shadow-lg
- Subtle shadow: shadow-sm
- Border radius: rounded-lg (cards), rounded-xl (hero), rounded-full (buttons/badges)

## Buttons
- **Primary CTA**: bg-blue-600 hover:bg-blue-700 text-white px-8 py-3 rounded-full font-semibold text-base transition
- **Secondary CTA**: border-2 border-blue-600 text-blue-600 hover:bg-blue-50 px-8 py-3 rounded-full font-semibold text-base transition
- **Ghost**: text-slate-600 hover:text-blue-600 font-medium transition

## Cards
- White background, subtle border, rounded-lg
- Hover: slight shadow elevation
- Icon + title + description pattern

## Images
- Use data-strk-img for content images (factory, QC, shipping visuals)
- Use data-strk-bg for hero backgrounds
- Ratios: 16x9 for heroes, 4x3 for cards, 3x2 for features
- Placeholder SVG for all content images

## Do's
- Use generous whitespace
- Keep layouts clean and uncluttered
- Use icons from Lucide React for service/process steps
- Ensure high contrast for all text
- Use responsive grid layouts (1 col mobile, 2-3 cols desktop)
- Add subtle hover transitions on interactive elements

## Don'ts
- Don't use dark backgrounds with dark text
- Don't use more than 2-3 colors per section
- Don't use exaggerated marketing language
- Don't use low-contrast text
- Don't use hardcoded pixel values for font sizes
