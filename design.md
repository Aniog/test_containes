# Velmora Fine Jewelry - Design System

## Visual Identity

### Color Palette
- **Primary/Background**: Warm Ivory `#FAF8F5` - Soft, warm base that flatters gold jewelry
- **Deep Accent**: Rich Charcoal `#1A1A1A` - Premium contrast for text and buttons
- **Warm Gold**: `#C9A962` - Signature accent for CTAs, highlights, and decorative elements
- **Muted Gold**: `#D4B978` - Secondary gold tone for subtle accents
- **Text Primary**: `#2C2C2C` - Readable dark text
- **Text Secondary**: `#6B6B6B` - Subdued descriptions
- **Border/Divider**: `#E8E4DF` - Hairline dividers, soft separation
- **Cream Accent**: `#F5F0E8` - Subtle section backgrounds

### Typography
- **Headings**: Cormorant Garamond (serif) - Elegant, editorial feel
  - Product names: UPPERCASE with letter-spacing: 0.15em
- **Body/UI**: Inter (sans-serif) - Clean, modern readability
- **Hierarchy**:
  - H1: 3.5rem-4rem, weight 500
  - H2: 2rem-2.5rem, weight 500
  - H3: 1.5rem, weight 500
  - Body: 1rem, weight 400, line-height 1.7

### Spacing & Layout
- Container max-width: 1400px
- Generous vertical padding: 80px-120px between sections
- Horizontal padding: 24px mobile, 48px tablet, 80px desktop
- Grid gaps: 24px-32px

### Visual Elements
- Hairline dividers: 1px solid #E8E4DF
- Soft shadows: 0 4px 20px rgba(0,0,0,0.06)
- Hover shadows: 0 8px 30px rgba(0,0,0,0.1)
- Border radius: 4px (subtle, not rounded)
- Transitions: 0.3s ease for all interactions

### Buttons
- **Primary**: Solid #1A1A1A with white text, full-width or auto
- **Accent**: Outlined #C9A962 with gold border and text
- Height: 52px
- Letter-spacing: 0.1em
- Font-weight: 500

### Do's
- Use generous whitespace to create premium feel
- Large editorial imagery
- Subtle hover transitions on cards
- Clean, minimal interface
- Warm, inviting atmosphere

### Don'ts
- No loud colors or discount-looking elements
- No generic e-commerce styling
- No harsh borders or aggressive CTAs
- No cramped layouts
- No bright/neon accent colors
