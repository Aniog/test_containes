# Design System

## Color Palette
- **Primary**: Indigo — `bg-indigo-600`, `text-indigo-600`, `hover:bg-indigo-700`
- **Primary Light**: `bg-indigo-50`, `text-indigo-700`
- **Accent**: `bg-indigo-100`
- **Background**: White — `bg-white`
- **Surface**: Light gray — `bg-gray-50`, `bg-slate-50`
- **Border**: `border-gray-200`, `border-gray-300`
- **Text Primary**: `text-gray-900`
- **Text Secondary**: `text-gray-600`
- **Text Muted**: `text-gray-400`
- **Success**: `text-green-600`, `bg-green-50`
- **Error**: `text-red-600`, `bg-red-50`

## Typography
- **Font**: Inter (Google Fonts)
- **Hero Heading**: `text-5xl font-bold text-gray-900` (desktop), `text-3xl` (mobile)
- **Section Heading**: `text-3xl font-bold text-gray-900`
- **Card Heading**: `text-xl font-semibold text-gray-900`
- **Body**: `text-base text-gray-600`
- **Small/Muted**: `text-sm text-gray-500`
- **Label**: `text-sm font-medium text-gray-700`

## Spacing
- **Section padding**: `py-20 px-4` (desktop), `py-12 px-4` (mobile)
- **Card padding**: `p-6` or `p-8`
- **Form gap**: `gap-4` or `gap-6`
- **Button padding**: `px-6 py-3`

## Borders & Shadows
- **Card**: `rounded-2xl shadow-sm border border-gray-100`
- **Input**: `rounded-lg border border-gray-300 focus:ring-2 focus:ring-indigo-500`
- **Button**: `rounded-lg`
- **Hero card**: `rounded-3xl shadow-xl`

## Components
- **Primary Button**: `bg-indigo-600 text-white hover:bg-indigo-700 rounded-lg px-6 py-3 font-medium transition-colors`
- **Secondary Button**: `border border-gray-300 text-gray-700 hover:bg-gray-50 rounded-lg px-6 py-3 font-medium transition-colors`
- **Badge**: `bg-indigo-100 text-indigo-700 text-xs font-medium px-3 py-1 rounded-full`
- **Input**: `w-full border border-gray-300 rounded-lg px-4 py-2.5 text-gray-900 focus:outline-none focus:ring-2 focus:ring-indigo-500`

## Do's
- Use white cards on light gray backgrounds for depth
- Use indigo as the single accent color throughout
- Keep generous whitespace between sections
- Use rounded-2xl or rounded-3xl for large containers
- Always pair dark text on light backgrounds

## Don'ts
- Don't use dark backgrounds for the main page
- Don't mix multiple accent colors
- Don't use text smaller than text-sm for body content
- Don't use low-contrast text (e.g. gray-300 on white)
