# Design System

## Visual Style
Clean, modern SaaS landing page aesthetic. Light background with strong contrast. Professional and trustworthy feel.

## Colors
- Primary: Indigo (`bg-indigo-600`, `text-indigo-600`) — used for CTAs, links, accents
- Primary hover: `bg-indigo-700`
- Background: White (`bg-white`) for cards/sections, Light gray (`bg-gray-50`) for page background
- Text primary: `text-gray-900`
- Text secondary: `text-gray-600`
- Text muted: `text-gray-400`
- Border: `border-gray-200`
- Success: `text-green-600`, `bg-green-50`
- Error: `text-red-600`, `bg-red-50`

## Typography
- Font: Inter (Google Fonts)
- Hero heading: `text-4xl md:text-5xl lg:text-6xl font-bold text-gray-900`
- Section heading: `text-3xl font-bold text-gray-900`
- Subheading: `text-xl font-semibold text-gray-900`
- Body: `text-base text-gray-600`
- Small/caption: `text-sm text-gray-500`

## Spacing
- Section padding: `py-16 md:py-24`
- Container: `max-w-6xl mx-auto px-4 sm:px-6 lg:px-8`
- Card padding: `p-6 md:p-8`
- Gap between items: `gap-6` or `gap-8`

## Borders & Shadows
- Card: `rounded-2xl border border-gray-200 shadow-sm`
- Input: `rounded-lg border border-gray-300 focus:ring-2 focus:ring-indigo-500 focus:border-indigo-500`
- Button primary: `rounded-lg bg-indigo-600 hover:bg-indigo-700 text-white font-semibold`
- Button secondary: `rounded-lg border border-gray-300 text-gray-700 hover:bg-gray-50`

## Do's
- Use consistent indigo as the single accent color
- Keep generous whitespace between sections
- Use subtle shadows on cards, not heavy drop shadows
- Ensure all text has strong contrast against its background

## Don'ts
- Don't use dark backgrounds for the main page
- Don't mix multiple accent colors
- Don't use tiny font sizes below `text-sm`
- Don't use low-contrast text (e.g. gray on gray)
