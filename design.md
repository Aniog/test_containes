# SSourcing China - Design System

## Brand Identity
A professional B2B sourcing agent website. The visual identity conveys trust, reliability, and international expertise.

## Color Palette
- **Primary (Navy)**: `#0c2d6b` - Headers, nav, primary text accents
- **Primary Dark**: `#091f4a` - Dark sections backgrounds
- **Accent (Coral/Orange)**: `#e85d26` - CTA buttons, highlights
- **Accent Hover**: `#cf4d1b` - Button hover states
- **Background**: `#ffffff` - Page background
- **Surface Light**: `#f8fafc` - Alternating section backgrounds
- **Surface Gray**: `#f1f5f9` - Card backgrounds, subtle sections
- **Border**: `#e2e8f0` - Card borders, dividers
- **Text Primary**: `#0f172a` - Main body text
- **Text Secondary**: `#475569` - Descriptions, subtitles
- **Text Muted**: `#94a3b8` - Labels, meta text
- **Success Green**: `#16a34a` - Success indicators
- **Warning**: `#f59e0b` - Trust badges, highlights

## Typography
- **Font Family**: Inter (Google Fonts)
- **Hero Headline**: 48px / 3rem, font-weight 800, line-height 1.1
- **Section Headline**: 36px / 2.25rem, font-weight 700, line-height 1.2
- **Sub-headline**: 24px / 1.5rem, font-weight 600, line-height 1.3
- **Body**: 16px / 1rem, font-weight 400, line-height 1.75
- **Small/Label**: 14px / 0.875rem, font-weight 500, line-height 1.5
- **Tiny**: 12px / 0.75rem, font-weight 500, uppercase, letter-spacing 0.05em

## Spacing
- Section padding: py-20 (80px) on desktop, py-12 on mobile
- Container max-width: max-w-7xl (1280px)
- Card padding: p-8 on desktop, p-6 on mobile
- Gap between sections: 0 (sections self-contained with padding)
- Grid gaps: gap-8 (32px)

## Layout Patterns
- Full-width sections with contained inner content (max-w-7xl mx-auto px-6)
- Alternating section backgrounds (white / light gray)
- 3-column grid for feature cards on desktop, 1-column on mobile
- 2-column layouts for content + image sections

## Buttons
- **Primary CTA**: bg-accent (#e85d26), text-white, rounded-lg, px-8 py-4, font-semibold, hover:bg-accent-hover
- **Secondary**: bg-white, border-2 border-primary, text-primary, rounded-lg, hover:bg-primary hover:text-white
- **Ghost**: text-primary, underline, no background

## Cards
- White background, rounded-xl, border border-border, shadow-sm
- Hover: shadow-md transition
- Icon in accent color at top
- Title in font-semibold text-lg
- Description in text-secondary

## Shadows
- sm: 0 1px 2px rgba(0,0,0,0.05)
- md: 0 4px 6px rgba(0,0,0,0.07)
- lg: 0 10px 15px rgba(0,0,0,0.1)

## Do's
- Use clean white space generously
- Keep text highly readable (good contrast)
- Use navy + white as the primary visual language
- Reserve accent color (coral) for CTAs and key highlights only
- Use Inter font throughout
- Maintain consistent section widths (max-w-7xl)

## Don'ts
- Don't use dark mode (B2B sites should be clean and light)
- Don't use decorative or script fonts
- Don't overuse the accent color
- Don't use low-contrast text combinations
- Don't use rounded-full buttons (keep rounded-lg for professional look)
