# Velmora Fine Jewelry — Design System

## Visual Identity
- **Mood**: Quiet luxury, warm, editorial. Premium demi-fine jewelry.
- **Audience**: Women 25–45, gifting & self-purchase, $30–$120 price point.

## Color Palette
- **Primary Background**: `#FAF7F2` (warm cream) — main page background
- **Card/Surface**: `#FFFFFF` (white) — cards, modals, drawers
- **Deep Base**: `#1A1612` (rich espresso black) — navbar, footer, deep sections
- **Gold Accent**: `#C9A96E` (warm 18k gold) — CTAs, accents, highlights
- **Gold Light**: `#E8D5A8` (soft gold) — subtle accents, borders
- **Gold Dark**: `#A8874F` (deep gold) — hover states
- **Text Primary**: `#1A1612` (rich dark) — headings, body on light bg
- **Text Secondary**: `#6B6358` (warm gray) — secondary text, descriptions
- **Text Light**: `#FAF7F2` (cream) — text on dark backgrounds
- **Border**: `#E8E0D4` (warm border) — dividers, card borders

## Typography
- **Heading Font**: Cormorant Garamond (serif, elegant)
- **Body Font**: Inter (clean sans-serif)
- **Product Names**: UPPERCASE, `tracking-[0.2em]` letter-spacing
- **Hero Headline**: Cormorant Garamond, 500 weight, large scale
- **Nav Links**: Inter, 400 weight, uppercase, tracking-wide

## Spacing & Layout
- Generous whitespace throughout
- Section padding: `py-20 md:py-28`
- Container max-width: `max-w-7xl mx-auto px-4 sm:px-6 lg:px-8`
- Card gaps: `gap-6 md:gap-8`

## Components
- **Buttons**: 
  - Primary: bg-gold-accent text-deep-base, uppercase tracking-wider
  - Secondary: outlined with gold-accent border
  - Hover: subtle scale + darker shade
- **Cards**: White bg, subtle shadow, rounded-lg, hover lift effect
- **Dividers**: Thin 1px hairline, warm border color
- **Input fields**: Clean, minimal, warm border, focus ring gold

## Animations
- Subtle hover transitions: `transition-all duration-300`
- Card hover: slight lift `translateY(-4px)`
- Button hover: gentle opacity change
- Page transitions: fade in
- Scroll reveals: fade up on viewport entry

## Do's
- Use generous whitespace
- Keep gold accents restrained but present
- Ensure high contrast for readability
- Use thin hairline dividers
- Maintain consistent warm tones

## Don'ts
- Don't use bright/neon colors
- Don't use heavy borders or boxy designs
- Don't use discount-looking elements
- Don't overcrowd sections
- Don't use cold/blue tones
