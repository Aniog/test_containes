# Velmora Fine Jewelry — Design System

## Direction
Quiet luxury, warm, editorial. Premium demi-fine jewelry. No loud discount vibes. No generic e-commerce templates. Warm gold on dark/neutral backgrounds.

## Color Palette
- **Background**: `#F8F5F1` warm off-white
- **Surface / cards**: `#FFFFFF`
- **Primary text**: `#1A1A1A` near-black
- **Secondary text**: `#6B6B6B` warm gray
- **Accent**: `#B57F50` warm antique gold
- **Accent hover**: `#9A6A40`
- **Dark base**: `#1C1C1C` for hero backgrounds / dark sections
- **Muted**: `#EDE8E1` warm beige for subtle backgrounds
- **Border**: `#E0DCD6` warm hairline
- **Star / review**: `#D4A843` gold star

## Typography
- **Serif headings**: Cormorant Garamond, weight 400–600
  - H1: 48–64px, line-height 1.1, tracking -0.01em
  - H2: 32–40px, line-height 1.2
  - Product names: uppercase, tracking 0.14em, weight 500, 13–14px
- **Sans-serif body**: Inter, weight 300–500
  - Body: 14–16px, line-height 1.6, color secondary-text
  - UI labels: 12–13px, uppercase, tracking 0.08em

## Spacing
- Generous whitespace. Sections: 80–120px vertical padding.
- Container max-width: 1280px, centered with auto margins.
- Page horizontal padding: 16px mobile, 24px tablet, 48px desktop.

## Components
- **Buttons**:
  - Primary: solid accent `#B57F50`, white text, no radius (square), hover darken.
  - Outline: 1px border `#1A1A1A`, transparent bg, hover fill black text white.
  - Ghost: underline on hover.
- **Cards**:
  - No heavy shadows. Subtle `0 1px 3px rgba(0,0,0,0.04)`.
  - Image aspect 4:3 or 3:4 for products.
- **Dividers**: 1px hairline `#E0DCD6`.
- **Inputs**: bottom-border only or subtle outline, accent focus ring.

## Animation
- Hover transitions: 300ms ease.
- Image scale on hover: 1.03.
- Cart drawer slide: 300ms ease-out.
- Navbar background fade: 200ms.

## Imagery
- Warm gold jewelry on dark charcoal or soft beige backgrounds.
- Editorial, close-up, soft directional light.
- Use `data-strk-img` for all product/hero images.
