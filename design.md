# MicroCosmos Design System

## Theme
Dark, immersive scientific aesthetic inspired by microscopy and the hidden world of tiny things.
Deep navy/black backgrounds with vibrant accent colors (teal, cyan, violet) to evoke bioluminescence and scientific imagery.

## Colors
- Background primary: `bg-gray-950` (#030712)
- Background secondary: `bg-gray-900` (#111827)
- Background card: `bg-gray-800` (#1f2937)
- Accent primary (teal/cyan): `text-teal-400` / `bg-teal-400` (#2dd4bf)
- Accent secondary (violet): `text-violet-400` / `bg-violet-400` (#a78bfa)
- Accent tertiary (cyan): `text-cyan-400` (#22d3ee)
- Text primary: `text-white`
- Text secondary: `text-gray-300`
- Text muted: `text-gray-400`
- Border: `border-gray-700`

## Typography
- Font: Inter (Google Fonts)
- Hero heading: `text-6xl md:text-8xl font-black tracking-tight`
- Section heading: `text-4xl md:text-5xl font-bold`
- Card heading: `text-xl font-semibold`
- Body: `text-base text-gray-300`
- Caption: `text-sm text-gray-400`
- Label/tag: `text-xs font-medium uppercase tracking-widest`

## Spacing
- Section padding: `py-20 md:py-28 px-6 md:px-12`
- Card padding: `p-6`
- Gap between grid items: `gap-4 md:gap-6`

## Borders & Shadows
- Card border: `border border-gray-700`
- Card radius: `rounded-2xl`
- Image radius: `rounded-xl`
- Glow effect: `shadow-lg shadow-teal-500/10`
- Hover glow: `hover:shadow-teal-400/20`

## Components
- Cards: dark bg-gray-800/900, border-gray-700, rounded-2xl, overflow-hidden
- Buttons: teal gradient or outlined, rounded-full, px-8 py-3
- Tags/badges: bg-teal-400/10 text-teal-400 border border-teal-400/20 rounded-full text-xs uppercase tracking-widest
- Section dividers: subtle gradient lines or spacing

## Do's
- Use dark backgrounds throughout for immersive feel
- Use teal/cyan/violet accents sparingly for highlights
- Use large, full-bleed images in hero and gallery
- Use grid layouts for image-heavy sections
- Ensure all text is clearly readable (white/gray-300 on dark backgrounds)
- Use subtle hover transitions on cards and images

## Don'ts
- Don't use light backgrounds (breaks the dark scientific theme)
- Don't use too many colors — stick to the teal/violet/cyan palette
- Don't use small images — microscopy is about detail, use large images
- Don't use harsh borders — prefer subtle gray-700 borders
- Don't use black text on dark backgrounds
