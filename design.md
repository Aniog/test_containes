# ARTITECT MACHINERY Design System

## Visual Style
- **Overall Aesthetic**: Elegant industrial with modern sophistication
- **Color Palette**:
  - Primary: Deep charcoal (#1a1f2e) - professional and authoritative
  - Secondary: Warm steel blue (#4a5568) - industrial yet approachable
  - Accent: Burnished gold (#c5a46e) - premium quality indicator
  - Background: Soft off-white (#f8f7f4) - clean and inviting
  - Text: Dark slate (#2d3748) - highly readable

## Typography
- **Headings**: Inter font, bold weights (600-700), generous letter-spacing for elegance
- **Body**: Inter font, regular weight (400), comfortable line-height (1.7)
- **Hierarchy**: Clear size progression from 48px hero to 14px captions

## Layout Principles
- Generous whitespace for breathing room
- Consistent 80px vertical spacing between major sections
- Maximum content width: 1280px with responsive padding
- Grid-based layouts with 24px gaps

## Component Guidelines
- **Buttons**: Rounded corners (8px), subtle hover lift, clear primary/secondary distinction
- **Cards**: Soft shadows, subtle borders, generous internal padding (32px)
- **Navigation**: Fixed header, clean logo placement, minimal menu items
- **Forms**: Clean inputs with subtle borders, clear focus states

## Do's
- Use ample padding around content blocks
- Maintain consistent icon sizing (24px for inline, 48px for feature icons)
- Ensure all text has excellent contrast ratios
- Use subtle transitions (200-300ms) for interactive elements

## Don'ts
- Avoid cluttered layouts or cramped spacing
- Don't use overly bright or neon colors
- Avoid decorative elements that don't serve a purpose
- Don't use small touch targets (minimum 44px)
