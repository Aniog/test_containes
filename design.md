# Velmora Fine Jewelry — Design System

## Visual Identity
- **Mood**: Quiet luxury, warm, editorial. Premium demi-fine jewelry — NOT loud, NOT discount-looking.
- **Audience**: Women 25–45, gifting and self-purchase, premium-but-accessible ($30–$120).

## Color Palette
- **Base (Deep Charcoal)**: `#1C1917` — primary background for hero/dark sections, nav solid state
- **Warm Cream**: `#FAF7F2` — page background, light sections
- **Soft Ivory**: `#F5F0E8` — card backgrounds, subtle surfaces
- **Gold Accent**: `#C9A96E` — primary accent, buttons, highlights, links
- **Gold Light**: `#D4B87A` — hover state for gold accent
- **Gold Dark**: `#A88B4A` — active/pressed state for gold accent
- **Text Primary**: `#1C1917` — headings, body on light
- **Text Secondary**: `#78716C` — muted text, descriptions
- **Text On Dark**: `#FAF7F2` — text on dark backgrounds
- **Divider**: `#E7E0D5` — hairline dividers
- **White**: `#FFFFFF` — pure white for contrast elements

## Typography
- **Headings**: Cormorant Garamond (serif), weights 400/500/600
- **Body/UI**: Inter (sans-serif), weights 300/400/500/600
- **Product Names**: UPPERCASE, letter-spacing 0.15em, Cormorant Garamond
- **Section Headings**: UPPERCASE, letter-spacing 0.2em, smaller size
- **Body Text**: 16px base, line-height 1.6

## Spacing & Layout
- Generous whitespace throughout
- Max content width: 1280px
- Section padding: 80px vertical (desktop), 48px (mobile)
- Card padding: 24px
- Grid gaps: 24px (product cards), 16px (tight grids)

## Components
- **Buttons**: Solid gold accent with cream text, or outlined with gold border. Rounded-none (sharp corners) for premium feel. Min height 48px.
- **Cards**: Soft shadow on hover, no border. Cream background.
- **Dividers**: 1px hairline in divider color
- **Inputs**: Minimal, border-bottom style or thin border, gold focus ring
- **Accordions**: Thin divider between items, plus/minus toggle

## Images
- Editorial, warm-lit photography
- Dark/neutral backgrounds for product shots
- Large hero imagery, full-bleed where possible
- Hover reveals second product image

## Animations
- Subtle, smooth transitions (200-300ms ease)
- Hover: gentle scale (1.02) on cards, opacity shifts
- Nav: transparent to solid on scroll
- No bouncy or playful animations

## Do's
- Use generous whitespace
- Use serif for elegance
- Keep gold accent restrained
- Use thin hairline dividers
- Large editorial imagery

## Don'ts
- Don't use bright/saturated colors
- Don't use rounded buttons (keep sharp/minimal radius)
- Don't crowd content
- Don't use playful/bouncy animations
- Don't use generic e-commerce patterns (badges, countdown timers, etc.)
