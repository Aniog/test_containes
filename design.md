# MicroCosmos Design System

## Brand Identity
MicroCosmos is a science/nature brand focused on the microscopic world — cells, organisms, patterns, and textures found at the smallest scales. The visual style is dark, immersive, and scientific with a sense of wonder.

## Color Palette
- Background (deep dark): `bg-gray-950` (#030712)
- Surface / Card: `bg-gray-900` (#111827)
- Surface elevated: `bg-gray-800` (#1f2937)
- Primary accent (teal/cyan): `text-teal-400`, `bg-teal-500` (#14b8a6 / #06b6d4)
- Secondary accent (emerald): `text-emerald-400` (#34d399)
- Highlight (amber): `text-amber-400` (#fbbf24)
- Text primary: `text-white`
- Text secondary: `text-gray-300`
- Text muted: `text-gray-500`
- Border: `border-gray-700`

## Typography
- Font family: Inter (Google Fonts)
- Hero heading: `text-5xl md:text-7xl font-black tracking-tight text-white`
- Section heading: `text-3xl md:text-4xl font-bold text-white`
- Subheading: `text-xl font-semibold text-teal-400`
- Body: `text-base text-gray-300 leading-relaxed`
- Caption: `text-sm text-gray-500`
- Label: `text-xs font-semibold uppercase tracking-widest text-teal-400`

## Spacing
- Section padding: `py-20 md:py-28`
- Container: `max-w-7xl mx-auto px-4 md:px-8`
- Card padding: `p-6 md:p-8`
- Gap between grid items: `gap-6 md:gap-8`

## Borders & Radius
- Card radius: `rounded-2xl`
- Button radius: `rounded-full`
- Image radius: `rounded-xl`
- Border: `border border-gray-700`

## Shadows & Effects
- Card shadow: `shadow-xl shadow-black/40`
- Glow effect on accent: `drop-shadow-[0_0_12px_rgba(20,184,166,0.5)]`
- Hover scale: `hover:scale-105 transition-transform duration-300`
- Backdrop blur for overlays: `backdrop-blur-md`

## Components

### Buttons
- Primary: `bg-teal-500 hover:bg-teal-400 text-white font-semibold px-8 py-3 rounded-full transition-colors`
- Secondary/outline: `border border-teal-500 text-teal-400 hover:bg-teal-500/10 px-8 py-3 rounded-full transition-colors`

### Cards
- Base: `bg-gray-900 border border-gray-700 rounded-2xl overflow-hidden shadow-xl`
- Hover: `hover:border-teal-500/50 transition-colors duration-300`

### Navigation
- Sticky top nav: `fixed top-0 w-full bg-gray-950/80 backdrop-blur-md border-b border-gray-800 z-50`
- Nav links: `text-gray-300 hover:text-teal-400 transition-colors text-sm font-medium`

### Forms
- Input: `bg-gray-800 border border-gray-700 text-white placeholder-gray-500 rounded-xl px-4 py-3 focus:outline-none focus:border-teal-500 w-full`
- Label: `text-sm font-medium text-gray-300 mb-1 block`

### Section Dividers
- Use subtle gradient dividers or `border-t border-gray-800`

## Do's
- Use dark backgrounds throughout for immersive feel
- Use teal/cyan as the primary accent color consistently
- Use large, full-width images for visual impact
- Add subtle glow effects on key elements
- Use grid layouts for image galleries (3-4 columns on desktop)
- Ensure all text has strong contrast against dark backgrounds

## Don'ts
- Don't use white or light backgrounds
- Don't use more than 3 accent colors
- Don't use small images — prefer large, high-quality visuals
- Don't use serif fonts
- Don't use low-contrast text (e.g. gray-600 on gray-900)
