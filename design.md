# Design Guidelines: ARTITECT MACHINERY

## Visual Style
Elegant, professional, industrial yet sophisticated. High contrast between industrial textures and clean, user-friendly UI.

## Typography
- Main Headings: 'Playfair Display', serif (for an elegant look)
- Body: 'Inter', sans-serif (for readability and modern feel)

## Colors
- Primary: #1a1a1a (Deep Obsidian)
- Secondary: #d4af37 (Metallic Gold - for accents/elegant feel)
- Accent: #3b82f6 (Industrial Blue - for functional elements)
- Background: #f8f9fa (Off-white / Light Grey)
- Text: #2d3436 (Dark Grey)

## Spacing & Layout
- Generous whitespace to convey premium quality.
- Grid-based layouts for machinery showcase.
- Rounded corners (md to lg) for a friendly touch on industrial elements.

## UI Elements (Tailwind Classes)
- **Buttons**: `bg-slate-900 text-white hover:bg-slate-800 px-6 py-2 transition-all rounded-md font-medium`
- **Cards**: `bg-white border border-slate-100 shadow-sm hover:shadow-md transition-shadow rounded-xl overflow-hidden`
- **Labels**: `text-xs font-semibold uppercase tracking-wider text-slate-500`
