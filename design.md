# SSourcing China - Visual Style Guide

## Brand Colors
- Primary: Deep navy blue (#1A365D) - trust, professionalism
- Primary light: (#2B4F8E) - interactive elements
- Secondary: Warm amber (#E67E22) - CTAs, highlights
- Accent: Cool teal (#0EA5A0) - secondary accents
- Background: Off-white (#F8FAFC) - page backgrounds
- Card background: White (#FFFFFF)
- Text primary: Dark slate (#1E293B)
- Text secondary: Medium gray (#64748B)
- Border: Light gray (#E2E8F0)
- Success: Green (#10B981)
- Error: Red (#EF4444)

## Typography
- Font family: Inter (from Google Fonts)
- Headings: font-weight 700 or 800, tracking -0.02em
- Body: font-weight 400, line-height 1.6
- Small/UI: text-sm, font-medium

## Spacing
- Section padding: py-16 md:py-24
- Card padding: p-6 md:p-8
- Grid gap: gap-6 md:gap-8
- Container max-width: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8

## Component Styles
- Cards: bg-white rounded-xl shadow-sm border border-gray-100
- Buttons (primary): bg-amber-500 hover:bg-amber-600 text-white font-semibold rounded-lg px-6 py-3
- Buttons (outline): border-2 border-navy-700 text-navy-700 hover:bg-navy-700 hover:text-white
- Section headings: text-3xl md:text-4xl font-bold text-navy-900 text-center mb-4
- Section subheadings: text-lg text-gray-500 text-center max-w-2xl mx-auto mb-12

## Do's
- Use clean, generous whitespace
- Use real-looking B2B visuals (factory, QC, shipping images)
- Use consistent navy + amber accent scheme
- Keep typography professional and readable
- Use subtle shadows and borders for depth

## Don'ts
- Don't use overly bright or playful colors
- Don't use casual/colloquial language
- Don't use exaggerated claims like "best" or "#1"
- Don't overcrowd sections
- Don't use dark backgrounds for text-heavy sections
