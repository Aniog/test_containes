# Pizza App Design System

## Color Palette
- Primary: `bg-red-600` (Pizza red)
- Primary hover: `bg-red-700`
- Secondary: `bg-orange-500` (Cheese orange)
- Secondary hover: `bg-orange-600`
- Accent: `bg-yellow-400` (Cheese yellow)
- Background: `bg-gray-50`
- Card background: `bg-white`
- Text primary: `text-gray-900`
- Text secondary: `text-gray-600`
- Text muted: `text-gray-500`

## Typography
- Headings: `font-bold text-gray-900`
- Body text: `text-gray-700`
- Small text: `text-sm text-gray-500`
- Price text: `font-semibold text-red-600`

## Components
- Cards: `bg-white rounded-lg shadow-md border border-gray-200`
- Buttons: `px-4 py-2 rounded-lg font-medium transition-colors`
- Primary buttons: `bg-red-600 text-white hover:bg-red-700`
- Secondary buttons: `bg-gray-200 text-gray-800 hover:bg-gray-300`
- Input fields: `border border-gray-300 rounded-lg px-3 py-2 focus:ring-2 focus:ring-red-500 focus:border-red-500`

## Spacing
- Container padding: `p-4 md:p-6 lg:p-8`
- Card padding: `p-6`
- Section spacing: `space-y-6`
- Grid gaps: `gap-4 md:gap-6`

## Layout
- Max width: `max-w-6xl mx-auto`
- Responsive breakpoints: Use `md:` and `lg:` prefixes
- Grid layouts: `grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3`

## Visual Style
- Rounded corners: `rounded-lg` for cards, `rounded-full` for buttons
- Shadows: `shadow-md` for cards, `shadow-lg` for modals
- Borders: `border border-gray-200` for subtle separation
- Hover effects: Always include `transition-colors` or `transition-all`

## Do's and Don'ts
✅ DO: Use warm colors (reds, oranges, yellows) for food-related elements
✅ DO: Ensure high contrast for text readability
✅ DO: Use consistent spacing and sizing
✅ DO: Include hover states for interactive elements

❌ DON'T: Use cold colors (blues, purples) for primary elements
❌ DON'T: Make text too small or low contrast
❌ DON'T: Mix different border radius styles
❌ DON'T: Forget responsive design considerations
