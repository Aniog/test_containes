# Visual Style Guide - SSourcing China

## Brand Persona
- **Professional**: Business-to-business focus, clear communication.
- **Trustworthy**: Reliable, expert, international standards.
- **Practical**: Focused on results, problem-solving, transparency.

## Colors
- **Primary**: Deep Navy Blue (`#0f172a` - Slate 900) - For headers, trust, authority.
- **Secondary**: Professional Blue (`#2563eb` - Blue 600) - For CTAs, links, accents.
- **Accent**: Soft Gray (`#f8fafc` - Slate 50) - For background sections.
- **Success**: Emerald (`#10b981` - Emerald 500) - For status, verification.
- **Foreground**: Slate 900 (`#0f172a`) on Light, White (`#ffffff`) on Dark.

## Typography
- **Font Family**: 'Inter', sans-serif (Google Fonts).
- **Headings**: Semi-bold to Bold, slate-900.
- **Body**: Regular, slate-600 for readability.

## Spacing & Layout
- **Container**: Max width 1280px, centered.
- **Padding**: Large vertical padding for sections (py-20).
- **Grid**: Responsive 1-column on mobile, up to 4-columns on desktop.

## Visual Elements
- **Borders**: Subtle slate-200 borders for cards.
- **Shadows**: Small to medium shadows for depth (shadow-sm, shadow-md).
- **Icons**: Lucide React for consistent, clean iconography.
- **Imagery**: Realistic industrial/factory/shipping stock images with overlays.

## Do's and Don'ts
### Do's
- Use clear, descriptive headings.
- Maintain high contrast for readability.
- Use whitespace to separate concerns.
- Ensure all CTAs are prominent (blue-600).
- Explicitly set readable text colors on all background changes.

### Don'ts
- Use neon or flashy colors.
- Use overwhelming animations.
- Use low-contrast text (e.g., light gray on white).
- Use pixelated or non-professional images.
