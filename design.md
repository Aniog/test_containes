# Design System — The Starry Night & Astronomy

## Philosophy
Japanese Minimalism meets deep-space wonder. Every element earns its place.
Generous whitespace, deliberate typography, and high-contrast imagery.

## Color Palette
- Background Deep: `#0B0F19` (bg-[#0B0F19]) — primary page background
- Background Mid: `#111827` (bg-gray-900) — card/section backgrounds
- Background Surface: `#1a2035` (bg-[#1a2035]) — elevated surfaces, nav
- Accent Amber: `#F59E0B` (text-amber-400 / border-amber-400) — primary accent
- Accent Indigo: `#6366F1` (text-indigo-400) — secondary accent
- Text Primary: `#F9FAFB` (text-gray-50) — headings
- Text Secondary: `#D1D5DB` (text-gray-300) — body text
- Text Muted: `#6B7280` (text-gray-500) — captions, labels
- Border: `#1F2937` (border-gray-800) — subtle dividers

## Typography
- Font Family: "Inter" (Google Fonts) — clean, modern, legible
- Display Heading: `text-5xl md:text-7xl font-light tracking-tight text-gray-50`
- Section Heading: `text-3xl md:text-4xl font-light tracking-tight text-gray-50`
- Sub-heading: `text-xl font-medium text-amber-400`
- Body: `text-base text-gray-300 leading-relaxed`
- Caption: `text-sm text-gray-500 tracking-wide uppercase`

## Spacing
- Section padding: `py-24 px-6 md:px-12 lg:px-24`
- Card padding: `p-6 md:p-8`
- Gap between elements: `gap-8 md:gap-12`

## Borders & Radius
- Cards: `rounded-2xl border border-gray-800`
- Buttons: `rounded-full`
- Inputs: `rounded-xl border border-gray-700`

## Shadows & Effects
- Card hover: `hover:border-amber-400/30 transition-all duration-300`
- Glow effect: `shadow-[0_0_30px_rgba(245,158,11,0.1)]`
- Image overlay: `bg-gradient-to-t from-[#0B0F19] via-transparent to-transparent`

## Do's
- Use `bg-[#0B0F19]` as the base background everywhere
- Use amber-400 for interactive highlights and key labels
- Use generous padding — never crowd elements
- Use `font-light` for large headings (Japanese minimalism)
- Pair every image with explanatory text in a side-by-side layout

## Don'ts
- Never use white backgrounds
- Never use bright saturated colors except amber/indigo accents
- Never stack too many elements without whitespace breathing room
- Never use bold fonts for large display text
