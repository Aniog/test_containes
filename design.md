# Design System - ARTITECT MACHINERY

## Typography
- Primary Font: 'Inter', system-ui, sans-serif
- Headings: Bold, clean, industrial yet elegant (Inter)
- Body: Regular, readable (Inter)

## Colors
- Primary: #1a1a1a (Deep Charcoal/Black) - represents strength and machinery
- Secondary: #f5f5f5 (Light Gray) - clean background
- Accent: #d4af37 (Metallic Gold) - represents elegance and precision
- Text: #333333 (Dark Gray)
- Muted: #666666 (Gray)

## Spacing
- Standard padding: 1rem (4)
- Section spacing: 4rem (16)
- Grid gap: 2rem (8)

## Components Style
- Buttons: Rounded edges (md), solid primary color with gold hover
- Cards: Subtle shadow, white background, clean borders
- Navigation: Minimalist, clear legible text
- Images: High quality, slightly desaturated for professional look

## Tailwind Classes
- Heading: `text-4xl font-bold tracking-tight text-foreground`
- Section: `py-16 px-4 max-w-7xl mx-auto`
- Card: `bg-card text-card-foreground rounded-lg border shadow-sm p-6`
- Accent Text: `text-[#d4af37]`
- Primary Button: `bg-[#1a1a1a] text-white hover:bg-[#d4af37] transition-colors px-6 py-2 rounded-md`
