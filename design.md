# ARTITECT MACHINERY - Design System

## Overview
Elegant, professional industrial machinery brand with a user-friendly approach. Clean, modern aesthetic that conveys precision engineering and trust.

## Typography
- **Primary Font**: Inter (Google Fonts)
- **Headings**: Bold weights (700-800), tight tracking
- **Body**: Regular weight (400-500), comfortable line height (1.5-1.6)
- **Scale**: Large, confident headings with generous spacing

## Color Palette
- **Primary**: Slate-900 (#0f172a) - Main brand color, headings, CTAs
- **Secondary**: Slate-600 (#475569) - Body text, secondary elements
- **Accent**: Slate-700 (#334155) - Hover states, emphasis
- **Background**: White (#ffffff) - Clean, professional base
- **Surface**: Slate-50 (#f8fafc) - Section backgrounds, cards
- **Border**: Slate-200 (#e2e8f0) - Subtle dividers, card borders

## Spacing
- **Section padding**: py-20 to py-28 (80-112px)
- **Container max-width**: max-w-7xl (1280px)
- **Grid gaps**: gap-8 to gap-16 (32-64px)
- **Element spacing**: Consistent use of mt-4, mt-6, mt-10

## Components

### Buttons
- **Primary**: Slate-900 background, white text, rounded-lg, shadow-lg
- **Secondary**: White background, slate-900 text, border slate-300
- **Hover**: Darken background, increase shadow
- **Padding**: px-8 py-4 for prominent CTAs

### Cards
- White background with slate-200 border
- Rounded-xl corners
- Hover: shadow-xl, border color change
- Subtle transitions (duration-300)

### Navigation
- Sticky header with backdrop-blur
- White/95 opacity background
- Clean, minimal link styling
- Mobile: hamburger menu with smooth transitions

## Visual Style
- **Minimalist**: Clean lines, ample white space
- **Professional**: Industrial yet approachable
- **Modern**: Subtle shadows, smooth transitions
- **Trustworthy**: Consistent, reliable aesthetic

## Responsive Behavior
- Mobile-first approach
- Breakpoints: sm (640px), md (768px), lg (1024px)
- Grid layouts adapt from 1 to 3 columns
- Typography scales appropriately

## Do's
- Use slate color family consistently
- Maintain generous whitespace
- Apply subtle shadows for depth
- Use smooth transitions (200-300ms)
- Keep typography hierarchy clear

## Don'ts
- Avoid harsh colors or gradients
- Don't overcrowd layouts
- Avoid inconsistent spacing
- Don't use decorative elements excessively
- Avoid low contrast text
