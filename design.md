## Do's
- Use navy blue for headers, footers, and primary UI elements
- Use orange for CTAs, highlights, and interactive elements
- Maintain generous whitespace between sections
- Use card-based layouts for products
- Keep forms clean with clear labels above inputs

## Don'ts
- Don't use dark text on dark backgrounds
- Don't use orange for large background areas
- Don't crowd elements вЂ” maintain breathing room
- Don't use more than 2 font weights in a single component
