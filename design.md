# ARTITECT MACHINERY - Design System

## Typography
- Primary font: Inter (weights 300, 400, 500, 600, 700, 800)
- Headings: font-bold (700) or font-extrabold (800)
- Body text: font-normal (400)
- Labels/captions: font-medium (500)

## Colors
- Primary: slate-900 (dark backgrounds, headings)
- Secondary: slate-700 (subheadings, secondary text)
- Accent: blue-600 (CTAs, links, highlights)
- Accent hover: blue-700
- Background: white (main), slate-50 (alternating sections)
- Text: slate-900 (primary), slate-600 (secondary)
- Borders: slate-200

## Layout
- Max container width: max-w-7xl
- Section padding: py-16 md:py-24
- Grid gaps: gap-6 md:gap-8
- Card padding: p-6 md:p-8

## Components
- Cards: rounded-2xl, shadow-lg, hover:shadow-xl transition
- Buttons: rounded-lg, px-6 py-3, font-medium
- Navigation: sticky top-0, backdrop-blur, bg-white/90

## Do's
- Use generous whitespace
- Keep text readable with proper contrast
- Use subtle shadows for depth
- Maintain consistent spacing scale
- Use hover transitions for interactive elements

## Don'ts
- Don't use more than 3 colors in one section
- Don't overcrowd content
- Don't use harsh shadows
- Don't mix too many font weights
- Don't use low contrast text
