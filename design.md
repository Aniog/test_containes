# SSourcing China - Design System

## Typography
- Font family: Inter, system-ui, sans-serif
- Headings: font-weight 700-800, tight letter-spacing
- Body: font-weight 400-500, normal letter-spacing
- Small text: font-weight 400, slightly muted color

## Colors
- Primary: blue-700 (#1d4ed8) - trust, professionalism
- Primary hover: blue-800 (#1e40af)
- Primary light: blue-50 (#eff6ff)
- Secondary: slate-900 (#0f172a) - headings, strong text
- Body text: slate-600 (#475569)
- Muted text: slate-500 (#64748b)
- Background: white (#ffffff)
- Background alt: slate-50 (#f8fafc)
- Background dark: slate-900 (#0f172a)
- Border: slate-200 (#e2e8f0)
- Success: green-600 (#16a34a)
- Warning: amber-500 (#f59e0b)

## Spacing
- Section padding: py-16 md:py-24
- Container: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8
- Card padding: p-6
- Grid gap: gap-6 md:gap-8
- Element spacing: space-y-4, space-y-6

## Borders & Shadows
- Cards: rounded-xl border border-slate-200 bg-white shadow-sm
- Buttons: rounded-lg
- Inputs: rounded-lg border border-slate-300
- Hover shadows: shadow-md

## Layout
- Max width: max-w-7xl for content containers
- Grid: grid-cols-1 md:grid-cols-2 lg:grid-cols-3 for cards
- Two-column layouts: grid-cols-1 lg:grid-cols-2

## Do's
- Use plenty of white space
- Keep text concise and scannable
- Use icons to support text
- Maintain consistent section spacing
- Use blue for primary actions and links
- Use slate colors for text hierarchy

## Don'ts
- Don't use bright or neon colors
- Don't overcrowd sections with text
- Don't use decorative fonts
- Don't use heavy shadows or gradients
- Don't use exaggerated claims or hype language
