# SSourcing China — Design System

## Overview
Professional, clean, trustworthy B2B website for a China sourcing agent. The design conveys reliability, international expertise, and industrial competence without being flashy.

## Colors
- Primary: `#1e3a5f` (deep navy — trust, professionalism)
- Primary Accent: `#2563eb` (bright blue — CTAs, links, highlights)
- Secondary: `#0f766e` (teal — supporting accents, success states)
- Background: `#f8fafc` (cool light gray — clean, modern)
- Surface: `#ffffff` (white — cards, sections)
- Text Primary: `#0f172a` (near-black)
- Text Secondary: `#475569` (slate-600)
- Text Muted: `#94a3b8` (slate-400)
- Border: `#e2e8f0` (slate-200)
- Danger: `#dc2626` (red for errors)
- Success: `#16a34a` (green for success)

## Typography
- Font: Inter (Google Fonts)
- Headings: font-weight 700, tight line-height (1.1–1.2)
- Body: font-weight 400, line-height 1.6
- Small / Labels: font-weight 500, uppercase, letter-spacing wide

## Spacing
- Section padding: py-20 (80px) on desktop, py-12 on mobile
- Container: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8
- Card padding: p-6 to p-8
- Gap between grid items: gap-6 to gap-8

## Components
- Buttons: rounded-md, px-6 py-3, font-semibold
  - Primary: bg-primary-accent text-white hover:bg-blue-700
  - Secondary: bg-white border border-slate-300 text-slate-800 hover:bg-slate-50
- Cards: bg-white rounded-xl shadow-sm border border-slate-100 p-6
- Badges: rounded-full px-3 py-1 text-xs font-semibold
- Inputs: rounded-md border-slate-300 focus:ring-2 focus:ring-blue-500

## Visual Rules
- Do not use gradients for backgrounds; stick to solid colors.
- Use subtle shadows (shadow-sm) for cards, not heavy drop shadows.
- Icons should be Lucide, 24px default, slate-500 or primary-accent color.
- Images should be realistic factory, QC, shipping, product visuals.
- Maintain generous whitespace between sections.

## Do's
- Use high-contrast text on all backgrounds.
- Keep layouts aligned with consistent grid gutters.
- Use uppercase tracking-wide labels for section subtitles.
- Make CTAs prominent and clearly actionable.

## Don'ts
- No neon or overly saturated colors.
- No exaggerated marketing language.
- No clutter — every element must earn its place.
- No low-contrast text (e.g., light gray on white).
