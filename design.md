# Velmora Fine Jewelry — Design System

## Color Palette
- **Cream** `#FAF7F2` — primary page background
- **Ivory** `#F5F0E8` — cards, sections, hover states
- **Sand** `#E8DFD0` — borders, dividers, muted backgrounds
- **Linen** `#D4C9B8` — subtle accents, disabled states
- **Bronze** `#B8956A` — primary accent CTA buttons
- **Gold** `#C9A96E` — decorative accents, hover states, highlights
- **Deep** `#1C1814` — primary text, dark sections, nav solid state
- **Charcoal** `#2A2520` — secondary dark, nav hover
- **Warm Gray** `#6B6358` — body text, descriptions
- **Light Gray** `#9B9488` — captions, placeholder text
- **Off White** `#FDFCFA` — brightest highlight

## Typography
- **Serif**: Cormorant Garamond — all headings, product names, display text
  - H1: 48–64px, weight 300, line-height 1.1
  - H2: 32–40px, weight 400
  - H3: 24–28px, weight 400
  - Product names: 14–16px, weight 500, uppercase, letter-spacing 0.2em
- **Sans**: Inter — body, UI, buttons, nav links
  - Body: 14–16px, weight 400, line-height 1.6
  - Nav links: 13px, weight 400, uppercase, letter-spacing 0.08em
  - Buttons: 13px, weight 500, uppercase, letter-spacing 0.1em

## Spacing & Layout
- Generous whitespace: section padding 80–120px vertical
- Container max-width: 1280px
- Thin hairline dividers: 1px solid #E8DFD0
- Card border-radius: 0px (sharp editorial edges) or 4px for subtle rounding
- Button padding: 14px 32px

## Components
- **Primary Button**: solid Bronze `#B8956A` background, white text, no border-radius
- **Secondary Button**: 1px solid Deep border, transparent bg, Deep text
- **Pills**: rounded-full, 1px border, small padding
- **Cards**: minimal shadow on hover, subtle transition
- **Dividers**: 1px hairline in Sand

## Animation
- Hover transitions: 300ms ease
- Nav scroll transition: 200ms ease
- Image hover: scale 1.03, 400ms ease
- Page load: subtle fade-in, 500ms
