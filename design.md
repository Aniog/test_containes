# SSourcing China - Design System

## Brand Identity
Professional, trustworthy, international B2B sourcing agent. Clean and practical.

## Colors
- Primary (Navy): `#0f172a` (slate-900) - Headers, nav, trust sections
- Primary Accent (Blue): `#1d4ed8` (blue-700) - CTAs, links, highlights
- Secondary Accent (Teal): `#0d9488` (teal-600) - Sub-highlights, icons
- Background Light: `#f8fafc` (slate-50)
- Background White: `#ffffff`
- Text Primary: `#1e293b` (slate-800)
- Text Secondary: `#64748b` (slate-500)
- Text Muted: `#94a3b8` (slate-400)
- Border: `#e2e8f0` (slate-200)
- Success: `#16a34a` (green-600)
- Warning: `#d97706` (amber-600)

## Typography
- Font: Inter, system-ui, sans-serif
- H1: 48px / 800 weight / line-height 1.1
- H2: 36px / 700 weight / line-height 1.2
- H3: 24px / 600 weight / line-height 1.3
- Body: 16px / 400 weight / line-height 1.6
- Small: 14px / 400 weight
- Caption: 12px / 500 weight uppercase

## Spacing
- Section padding: py-20 (80px) desktop, py-12 (48px) mobile
- Container max-width: max-w-7xl (1280px), centered
- Card padding: p-6 (24px)
- Gap between grid items: gap-6 to gap-8
- Component margin: space-y-4 to space-y-6

## Buttons
- Primary: bg-blue-700 text-white px-6 py-3 rounded-md font-semibold hover:bg-blue-800
- Secondary: border border-slate-300 text-slate-700 px-6 py-3 rounded-md hover:bg-slate-50
- Ghost: text-blue-700 hover:text-blue-800 font-medium

## Cards
- Background: white
- Border: 1px solid slate-200
- Border radius: rounded-lg
- Shadow: shadow-sm, hover:shadow-md transition
- Padding: p-6

## Layout
- Full-width sections with centered containers
- Responsive grid: 1 col mobile, 2 col tablet, 3-4 col desktop
- Hero: full-width with dark overlay on background image

## Do's
- Use navy/blue for trust and professionalism
- Use generous whitespace
- Use clear hierarchy with font weights
- Include realistic factory/QC/shipping imagery
- Keep CTAs prominent and repeated

## Don'ts
- Don't use bright flashy colors
- Don't clutter sections with too much text
- Don't use casual or playful fonts
- Don't make exaggerated claims
