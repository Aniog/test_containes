# Velmora Fine Jewelry - Design System

## Visual Identity

### Brand Essence
- **Mood**: Quiet luxury, warm, editorial. Premium demi-fine jewelry — not loud, not discount-looking, not generic e-commerce.
- **Target**: Women 25–45, gifting and self-purchase, premium-but-accessible price point ($30–$120)

### Color Palette

| Name | Hex | Usage |
|------|-----|-------|
| Cream | `#FAF7F2` | Primary background |
| Warm | `#F5EFE6` | Secondary background, cards |
| Sand | `#E8E0D5` | Borders, dividers |
| Taupe | `#C9BFB0` | Disabled states, subtle elements |
| Gold | `#C5A572` | Primary accent, CTAs, highlights |
| Gold Light | `#D4B88A` | Hover states, gradients |
| Gold Dark | `#8B7355` | Active states, emphasis |
| Charcoal | `#2C2824` | Primary text, headings |
| Charcoal Light | `#3D3833` | Secondary text |
| Espresso | `#1A1816` | Dark sections, footer |

### Typography

**Headings & Product Names**
- Font: Cormorant Garamond
- Weight: 400-600
- Transform: Uppercase with letter-spacing 0.25em
- Usage: Hero headlines, product names, section titles

**Body & UI**
- Font: Inter
- Weight: 300-500
- Usage: Navigation, descriptions, buttons, form elements

### Spacing & Layout
- Generous whitespace throughout
- Large editorial imagery
- Thin hairline dividers (1px, sand color)
- Consistent padding: 16px base unit

### Motion & Interaction
- Subtle hover transitions (0.3s ease)
- Soft shadows on cards
- Image zoom on hover (scale 1.05)
- Fade-in animations on scroll
- Cart drawer slides from right

### Component Styles

**Buttons**
- Primary: Solid gold background, white text
- Outline: Gold border, charcoal text, fills on hover
- Font: Inter, medium weight, tracking-wide
- Border radius: subtle (2px)

**Cards**
- Background: Warm (#F5EFE6) or white
- Shadow: Soft, warm-toned
- Hover: Subtle lift effect

**Inputs**
- Border: Sand color
- Focus: Gold border
- Background: White
- Padding: 12px 16px

## Accessibility
- Minimum contrast ratio 4.5:1 for text
- Focus states visible
- Touch targets minimum 44px
- Alt text for all images

## Do's and Don'ts

### Do
- Use generous whitespace
- Keep CTAs consistent (gold accent)
- Use editorial photography
- Maintain typographic hierarchy
- Add subtle animations

### Don't
- Use bright/discount colors
- Add loud graphics or patterns
- Overcrowd sections
- Use generic stock photos
- Mix multiple accent colors
