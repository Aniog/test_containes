# SSourcing China - Design System

## Brand Identity
- Professional B2B sourcing company based in China
- Trustworthy, clean, international feel
- Target audience: overseas buyers looking for China sourcing support

## Colors
- Primary: `#1e40af` (blue-800) — trust, professionalism
- Primary hover: `#1d4ed8` (blue-700)
- Primary light: `#dbeafe` (blue-100) — backgrounds
- Accent: `#f97316` (orange-500) — CTAs, highlights
- Accent hover: `#ea580c` (orange-600)
- Dark: `#0f172a` (slate-900) — headings, nav
- Body text: `#334155` (slate-700)
- Muted text: `#64748b` (slate-500)
- Light background: `#f8fafc` (slate-50)
- White: `#ffffff`
- Border: `#e2e8f0` (slate-200)

## Typography
- Font family: Inter (Google Fonts)
- Headings: font-bold, tracking-tight
  - H1: text-4xl md:text-5xl lg:text-6xl
  - H2: text-3xl md:text-4xl
  - H3: text-xl md:text-2xl
  - H4: text-lg md:text-xl
- Body: text-base (16px), text-slate-700
- Small/muted: text-sm, text-slate-500

## Spacing
- Section padding: py-16 md:py-24
- Container: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8
- Card padding: p-6 md:p-8
- Gap between cards: gap-6 md:gap-8

## Components
- Buttons:
  - Primary: bg-orange-500 hover:bg-orange-600 text-white font-semibold px-6 py-3 rounded-lg
  - Secondary: bg-blue-800 hover:bg-blue-700 text-white font-semibold px-6 py-3 rounded-lg
  - Outline: border-2 border-blue-800 text-blue-800 hover:bg-blue-800 hover:text-white px-6 py-3 rounded-lg
- Cards: bg-white rounded-xl shadow-sm border border-slate-200 p-6
- Badges: bg-blue-100 text-blue-800 text-sm font-medium px-3 py-1 rounded-full

## Layout
- Navigation: sticky top-0, white background, shadow-sm
- Hero sections: large padding, centered text or split layout
- Grid layouts: grid-cols-1 md:grid-cols-2 lg:grid-cols-3
- Footer: bg-slate-900 text-white

## Do's
- Use consistent spacing between sections
- Keep text readable with proper contrast
- Use icons (Lucide) to support text content
- Use stock images for factory, QC, shipping contexts
- Keep CTAs prominent with orange accent color

## Don'ts
- Don't use dark text on dark backgrounds
- Don't use more than 2-3 colors per section
- Don't overcrowd sections with too much content
- Don't use exaggerated marketing language
- Don't use rounded-full on large containers
