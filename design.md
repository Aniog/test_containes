# SSourcing China - B2B Website Design Guidelines

## Project Overview
- **Project Name**: SSourcing China Website
- **Type**: B2B Corporate Website
- **Purpose**: Generate qualified sourcing inquiries for China sourcing agent services
- **Target Audience**: Overseas buyers, procurement managers, business owners looking to source products from China

## Visual Style

### Color Palette
- **Primary**: `#1E3A5F` (Deep Navy Blue - trust, professionalism)
- **Primary Light**: `#2C5282` (Medium Blue)
- **Secondary**: `#C9A227` (Gold/Amber - premium, quality)
- **Accent**: `#38A169` (Green - success, reliability)
- **Background**: `#F8FAFC` (Off-white)
- **Surface**: `#FFFFFF` (White)
- **Text Primary**: `#1A202C` (Near Black)
- **Text Secondary**: `#4A5568` (Dark Gray)
- **Text Muted**: `#718096` (Medium Gray)
- **Border**: `#E2E8F0` (Light Gray)
- **Error**: `#E53E3E` (Red)

### Typography
- **Font Family**: "Inter", system-ui, sans-serif
- **Headings**: 
  - H1: 48px/56px, font-weight 700
  - H2: 36px/44px, font-weight 600
  - H3: 24px/32px, font-weight 600
  - H4: 20px/28px, font-weight 500
- **Body**: 16px/24px, font-weight 400
- **Small**: 14px/20px, font-weight 400

### Spacing System
- Base unit: 4px
- Section padding: 80px vertical (desktop), 48px (mobile)
- Container max-width: 1200px
- Grid gap: 24px (desktop), 16px (mobile)

### Visual Effects
- Card shadows: `0 4px 6px -1px rgba(0, 0, 0, 0.1), 0 2px 4px -1px rgba(0, 0, 0, 0.06)`
- Hover shadows: `0 10px 15px -3px rgba(0, 0, 0, 0.1), 0 4px 6px -2px rgba(0, 0, 0, 0.05)`
- Border radius: 8px (cards), 4px (buttons), 50% (avatars)
- Transitions: 200ms ease-in-out

## Layout Guidelines

### Header
- Fixed position, white background with subtle shadow on scroll
- Logo on left, navigation center, CTA button right
- Mobile: hamburger menu

### Footer
- Dark background (#1E3A5F)
- 4 columns: About, Services, Resources, Contact
- Copyright and social links

### Responsive Breakpoints
- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px

## Component Guidelines

### Buttons
- **Primary**: Navy background, white text, gold hover
- **Secondary**: White background, navy border, navy text
- **CTA**: Gold background, navy text, prominent size

### Cards
- White background, subtle shadow
- 24px padding
- Icon or image at top
- Title, description, optional CTA

### Form Inputs
- White background, light border
- Focus: navy border, subtle shadow
- Error: red border, error message below

### Icons
- Lucide React icons
- Consistent sizing: 24px (default), 32px (featured)

## Page Structure

### Homepage Sections
1. **Hero**: Headline, subheadline, CTA, background image
2. **Trust Bar**: Client logos, statistics
3. **Services Overview**: 4-6 service cards
4. **Sourcing Process**: 5-step process visualization
5. **Products We Source**: Category grid
6. **Problems We Solve**: Pain points and solutions
7. **Case Studies**: Featured success stories
8. **FAQ**: Common questions
9. **CTA Section**: Final call to action
10. **Footer**: Links and contact info

### Page Templates
- Standard header/footer on all pages
- Breadcrumb navigation where appropriate
- Related content suggestions

## Image Guidelines
- Use professional B2B imagery
- Factory, warehouse, QC inspection, shipping containers
- People: business professionals, factory workers
- Avoid: overly stock-looking photos, generic office shots
- Aspect ratios: 16:9 (hero), 4:3 (cards), 1:1 (avatars)

## Tone & Voice
- Professional and confident
- Practical, not exaggerated
- Clear and direct
- Use "we" for company, "you" for visitor
- Avoid: buzzwords, empty promises, overly casual language
