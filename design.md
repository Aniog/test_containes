# Velmora Fine Jewelry — Design System

## Mood
Quiet luxury, warm, editorial. Premium demi-fine jewelry — NOT loud, NOT discount-looking, NOT generic e-commerce.

## Color Palette
- **Background (cream):** `cream` → `#FAF7F2` — warm off-white page base
- **Surface (ivory):** `ivory` → `#F5F0E8` — card/section backgrounds
- **Foreground (charcoal):** `charcoal` → `#1C1917` — primary text
- **Muted text:** `stone` → `#78716C` — secondary/body text
- **Accent (gold):** `gold` → `#B8860B` — CTAs, highlights, hover states
- **Accent hover:** `gold-dark` → `#996F09` — button hover
- **Accent light:** `gold-light` → `#D4A843` — subtle gold tints
- **Border:** `border` → `#E7E0D6` — hairline dividers

## Typography
- **Headings / Product names:** `Cormorant Garamond` (serif), weights 400/500/600
- **Body / UI:** `Inter` (sans-serif), weights 300/400/500/600
- Product names: UPPERCASE, letter-spacing `0.15em`
- Section headings: Title case or uppercase, generous letter-spacing `0.05em`

## Spacing & Layout
- Generous whitespace: sections use `py-16 md:py-24`
- Container max-width: `max-w-7xl mx-auto px-4 md:px-8`
- Cards: subtle shadow `shadow-sm`, rounded `rounded-sm`
- Hairline dividers: `border-t border-border`

## Buttons
- Primary: `bg-gold text-cream` with `hover:bg-gold-dark`, `px-8 py-3`, uppercase tracking-wider, font-medium
- Outlined: `border border-gold text-gold` with `hover:bg-gold hover:text-cream`
- Transitions: `transition-all duration-300`

## Do's
- Use generous whitespace
- Large editorial imagery
- Thin hairline dividers between sections
- Subtle hover transitions (opacity, translate-y)
- Soft shadows on cards
- Restrained use of gold accent

## Don'ts
- No bright/neon colors
- No heavy drop shadows
- No rounded-full buttons (use rounded-sm or rounded)
- No cluttered layouts
- No discount/sale-style badges
- No generic stock photo feel
