# Design System: ARTITECT MACHINERY

## Typography
- **Primary Font:** 'Inter', system-ui, sans-serif. Used for body text and general interface. `font-sans`
- **Headings:** Bold, elegant, authoritative. `font-bold tracking-tight text-slate-900`
- **Body:** Clean, legible. `text-slate-600 leading-relaxed`

## Colors
- **Primary:** `bg-slate-900` #0f172a (Deep Slate/Black for authority)
- **Secondary:** `bg-blue-600` #2563eb (Professional blue for accents)
- **Accent:** `bg-blue-50` #eff6ff (Subtle hover states and light backgrounds)
- **Background:** `bg-white` #ffffff (Clean, user-friendly)
- **Text:** `text-slate-900` (Headings), `text-slate-600` (Body)

## Spacing & Layout
- **Container:** `max-w-7xl mx-auto px-4 sm:px-6 lg:px-8`
- **Sections:** `py-16 md:py-24`
- **Cards:** White background, subtle border, gentle shadow. `bg-white border border-slate-200 rounded-xl overflow-hidden shadow-sm hover:shadow-md transition-shadow`

## UI Elements
- **Buttons:**
  - Primary: `bg-slate-900 text-white px-6 py-3 rounded-lg font-medium hover:bg-slate-800 transition-colors`
  - Secondary: `bg-blue-600 text-white px-6 py-3 rounded-lg font-medium hover:bg-blue-700 transition-colors`
- **Inputs:** `border border-slate-300 rounded-lg px-4 py-2 focus:ring-2 focus:ring-blue-500 focus:border-transparent outline-none transition-all`

## Do's and Don'ts
### Do's
- Use high-quality imagery of machinery.
- Maintain generous white space.
- Use clear, professional language.
- Ensure all technical terms are correct.

### Don'ts
- Use overly bright or neon colors.
- Overcrowd the interface with technical specs on the first view.
- Use generic placeholders for machine names.
