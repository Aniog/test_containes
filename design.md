# SSourcing China - Visual Style Guide

## Brand Colors
- Primary: #1A365D (dark navy blue) - trustworthy, professional
- Primary Light: #2B4C7E (medium navy)
- Accent: #D97706 (amber/gold) - CTA buttons, highlights
- Accent Light: #F59E0B (lighter amber)
- Background: #FFFFFF (white)
- Surface: #F8FAFC (light gray)
- Border: #E2E8F0 (light border)
- Muted Text: #64748B (slate gray)
- Body Text: #1E293B (dark slate)
- Success: #16A34A (green for trust badges)

## Typography
- Font Family: Inter (headings and body)
- Headings: font-bold, tracking-tight
- Body: font-normal, leading-relaxed
- Sizes: text-4xl/3xl (headings), text-lg (body), text-sm (meta)

## Spacing
- Section padding: py-16 lg:py-24
- Container max-width: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8
- Card padding: p-6 lg:p-8
- Grid gap: gap-8 lg:gap-12

## Borders & Shadows
- Cards: rounded-xl border border-gray-100 shadow-sm
- Hover: shadow-md transition-shadow
- Inputs: rounded-lg border border-gray-300

## Style Rules
- DO use clean, white backgrounds with plenty of whitespace
- DO use the amber accent sparingly for CTAs and key highlights
- DO use factory/QC/shipping imagery via stock image system
- DON'T use dark modes or heavy backgrounds
- DON'T use exaggerated claims or hype language
- DON'T use complex animations that distract
- Keep the tone professional, factual, and B2B-oriented

## Component Styles
- Buttons: rounded-lg font-semibold px-6 py-3 transition-all
- Primary CTA: bg-amber-500 hover:bg-amber-600 text-white
- Secondary: bg-navy-700 hover:bg-navy-800 text-white
- Outline: border-2 border-navy-700 text-navy-700 hover:bg-navy-50
- Section titles: text-3xl lg:text-4xl font-bold text-navy-900 text-center mb-4
- Section subtitles: text-lg text-slate-600 text-center max-w-2xl mx-auto mb-12
