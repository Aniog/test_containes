# Velmora Fine Jewelry — Design System

## Visual Identity
- Mood: Quiet luxury, warm, editorial. Premium demi-fine jewelry.
- NOT loud, NOT discount-looking, NOT generic e-commerce.

## Color Palette
- Background: `#FAF7F2` (warm cream) — primary page background
- Surface: `#F5F0E8` (ivory) — cards, alternate sections
- Accent Surface: `#E8DFD0` (sand) — subtle backgrounds
- Gold Accent: `#C9A96E` — primary accent, CTAs, highlights
- Gold Hover: `#A88B52` — darker gold for hover states
- Text Primary: `#2C2C2C` (charcoal) — headings, body text
- Text Secondary: `#6B6560` (warm gray) — descriptions, captions
- Text Muted: `#9B9590` — placeholders, disabled
- Border: `#E5DFD6` — hairline dividers, card borders
- Dark Block: `#1A1714` (warm black) — hero overlays, footer

## Typography
- Headings: Cormorant Garamond (serif), weight 400-600
- Body/UI: Inter (sans-serif), weight 300-500
- Product names: UPPERCASE, letter-spacing 0.15em, Cormorant Garamond 500
- Nav links: Inter 500, uppercase, tracking-widest (0.1em), text-xs
- Section headings: Cormorant Garamond, text-display-sm to text-display-lg

## Spacing
- Section vertical padding: py-16 md:py-24
- Content max-width: max-w-7xl (80rem)
- Card gaps: gap-6 md:gap-8
- Generous whitespace throughout

## Components
- Buttons: Solid charcoal or gold, uppercase tracking-widest text-xs, no border-radius (sharp), hover darkens
- Cards: No visible border by default, subtle shadow on hover, image zoom on hover
- Dividers: 1px solid #E5DFD6 (hairline)
- Inputs: Border #E5DFD6, focus ring gold

## Do's
- Use warm cream backgrounds
- Use generous whitespace
- Use serif for all headings and product names
- Use hairline dividers between sections
- Use subtle hover transitions (300ms ease)
- Use large editorial imagery

## Don'ts
- Don't use bright/saturated colors
- Don't use rounded buttons (keep sharp/0 radius)
- Don't use heavy shadows
- Don't use sans-serif for headings
- Don't use discount/sale visual language
- Don't use generic e-commerce patterns
