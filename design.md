# ARTITECT MACHINERY Design System

## Brand Direction
ARTITECT MACHINERY should feel elegant, precise, and dependable. The design should combine industrial confidence with a premium presentation that is still easy for buyers to scan quickly.

## Visual Style
- Use a refined industrial palette built from Tailwind slate, zinc, stone, and amber tones.
- Prefer deep backgrounds like `bg-slate-950`, `bg-slate-900`, and `bg-zinc-950` for premium sections.
- Use warm metallic accents like `bg-amber-500`, `text-amber-300`, `border-amber-400/30` sparingly for highlights and calls to action.
- Main readable surfaces should use `bg-white`, `bg-stone-50`, or `bg-slate-900/70` with explicit foregrounds like `text-slate-900`, `text-slate-100`, and `text-slate-300`.

## Typography
- Use Inter for the main font.
- Headlines should be bold, spacious, and confident with classes such as `text-4xl md:text-6xl font-semibold tracking-tight`.
- Supporting text should stay readable with classes such as `text-base md:text-lg leading-7 text-slate-300` or `text-slate-600`.
- Small labels should use uppercase tracking, e.g. `text-xs font-semibold uppercase tracking-[0.3em]`.

## Layout
- Use wide section containers like `max-w-7xl mx-auto px-6 md:px-10`.
- Keep generous vertical rhythm with `py-16 md:py-24`.
- Desktop layouts should feel structured and balanced, often with 2-column or 3-column grids rather than stacking everything in one column.
- Mobile layouts should collapse cleanly with responsive grid classes.

## Components
- Cards: `rounded-3xl border border-slate-200/70 bg-white shadow-sm` for light sections, or `rounded-3xl border border-white/10 bg-slate-900/70` for dark sections.
- Buttons: primary buttons should use dark backgrounds with warm accents or amber-filled CTAs; secondary buttons should use bordered neutral styles.
- Statistic blocks and badges should have explicit text colors and subtle borders.

## Imagery
- Use high-quality machinery and manufacturing stock images where they support the page.
- Hero and product visuals should depict precise industrial sheet metal folding equipment and fabrication environments.
- Use the Strikingly image attributes rather than hardcoded image URLs.

## Do
- Keep every text element high-contrast and clearly readable.
- Make the page easy to scan for product buyers and factory decision makers.
- Maintain a polished premium feel through spacing, hierarchy, and restrained accent color use.

## Don’t
- Do not use playful colors or cartoon-like styling.
- Do not create cramped mobile layouts.
- Do not rely on low-contrast gray text on light metallic backgrounds.
- Do not hardcode arbitrary hex colors into class names.
