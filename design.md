# Baseball Website Design System

## Theme
Sports-focused, energetic, bold. Dark navy and red color palette inspired by classic baseball aesthetics.

## Colors
- Primary background: `#0f172a` (slate-900) — deep navy
- Secondary background: `#1e293b` (slate-800) — dark navy card
- Accent red: `#dc2626` (red-600) — baseball red
- Accent red hover: `#b91c1c` (red-700)
- Gold/amber: `#f59e0b` (amber-500) — highlights and stars
- Text primary: `#f8fafc` (slate-50) — white text on dark
- Text secondary: `#94a3b8` (slate-400) — muted text
- Text muted: `#64748b` (slate-500)
- Border: `#334155` (slate-700)
- Card surface: `#1e293b` (slate-800)

## Typography
- Font family: "Oswald" for headings (bold, sporty), "Inter" for body text
- Heading sizes: text-5xl / text-4xl / text-3xl / text-2xl / text-xl
- Body: text-base / text-sm
- Font weights: font-black (900) for hero, font-bold (700) for headings, font-medium (500) for labels

## Spacing
- Section padding: py-20 px-4 or px-6
- Card padding: p-6 or p-8
- Gap between cards: gap-6 or gap-8
- Max content width: max-w-6xl mx-auto

## Borders & Shadows
- Card border: border border-slate-700
- Card radius: rounded-2xl
- Hover shadow: shadow-lg shadow-red-900/20
- Divider: border-t border-slate-700

## Components
- Buttons: bg-red-600 hover:bg-red-700 text-white font-bold px-8 py-3 rounded-full
- Badges: bg-red-600/20 text-red-400 border border-red-600/30 rounded-full px-3 py-1 text-sm
- Stat cards: bg-slate-800 border border-slate-700 rounded-2xl p-6 text-center
- Section headers: uppercase tracking-widest text-red-500 text-sm font-bold mb-2

## Do's
- Use bold, uppercase section labels with letter-spacing
- Use large numbers for stats to create visual impact
- Use gradient text for hero headings: from-white to-slate-400
- Use subtle red glow effects on key elements
- Keep cards dark with light text for contrast

## Don'ts
- Don't use light backgrounds for main sections
- Don't use small font sizes for stats
- Don't use low-contrast text combinations
- Don't use more than 3 accent colors
