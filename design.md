# Velmora Fine Jewelry — Design System

## Visual Identity
- Mood: Quiet luxury, warm, editorial. Premium demi-fine jewelry — NOT loud, NOT discount-looking.
- Target: Women 25–45, gifting and self-purchase, $30–$120 price point.

## Color Palette
- **Background / Base**: `#F5F0EB` (velmora-cream) — warm off-white, the primary page background
- **Secondary Surface**: `#E8DDD3` (velmora-sand) — slightly deeper warm neutral for cards, sections
- **Warm Accent Surface**: `#D4C5B5` (velmora-warm) — for dividers, borders, subtle backgrounds
- **Gold Accent**: `#C9A96E` (velmora-gold) — primary accent for buttons, links, highlights, CTAs
- **Gold Hover**: `#B08D4F` (velmora-gold-dark) — darker gold for hover states
- **Primary Text**: `#2C2C2C` (velmora-charcoal) — headings and body text
- **Secondary Text**: `#6B6B6B` (velmora-text-secondary) — descriptions, captions
- **Muted Text**: `#9A9A9A` (velmora-text-muted) — placeholders, disabled states
- **Dark Surface**: `#1A1A1A` (velmora-espresso) — hero overlays, dark sections, footer

## Typography
- **Headings / Product Names**: Cormorant Garamond (serif), uppercase, letter-spacing 0.2em
- **Body / UI**: Inter (sans-serif), normal case, standard spacing
- **Product names**: ALWAYS uppercase with wide letter-spacing (0.2em)
- **CTA buttons**: Inter, uppercase, letter-spacing 0.15em, weight 500

## Spacing & Layout
- Generous whitespace — sections separated by 80–120px vertical padding
- Hairline dividers (0.5px, velmora-warm color) between sections
- Max content width: 1280px, centered
- Product grid: 4–5 columns on desktop, 2 on mobile
- Mobile-first responsive design

## Component Styles
- **Buttons**: Solid gold accent (#C9A96E) or outlined gold border; rounded-none or slight rounded; uppercase text; hover transitions 0.3s ease
- **Cards**: Minimal, clean; soft shadow on hover (shadow-md); no heavy borders
- **Images**: Large, editorial; warm-toned; aspect ratios preserved
- **Navigation**: Transparent over hero, solid cream on scroll; serif logo, sans-serif links
- **Footer**: Dark background (espresso), cream/gold text

## Do's
- Use warm, gold-flattering color combinations
- Keep typography elegant and restrained
- Use generous whitespace
- Ensure strong contrast for accessibility (gold on dark, charcoal on cream)
- Use subtle hover transitions (0.3s ease)
- Product names in uppercase serif with wide spacing

## Don'ts
- Never use bright/saturated colors
- Never use generic e-commerce styling (badges, loud banners)
- Never use light text on light backgrounds without explicit contrast
- Never hardcode arbitrary hex codes — use the velmora palette
- Never use discount/sale visual language
