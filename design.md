# ARTITECT MACHINERY - Design System

## Overview
An elegant, professional website for an industrial machinery company. The design balances industrial precision with refined aesthetics.

## Color Palette
- **Primary**: #0F172A (deep navy/slate - conveys trust, precision, industry)
- **Secondary**: #1E293B (slightly lighter navy for cards/surfaces)
- **Accent**: #D4AF37 (gold/brass - evokes quality machinery and craftsmanship)
- **Background**: #F8FAFC (clean, light gray-white)
- **Surface**: #FFFFFF (white cards)
- **Text Primary**: #0F172A (dark navy on light backgrounds)
- **Text Secondary**: #64748B (muted slate for body text)
- **Text Light**: #FFFFFF (on dark backgrounds)
- **Border**: #E2E8F0 (subtle dividers)

## Typography
- **Headings**: Inter, weight 700-800, tight letter-spacing
- **Body**: Inter, weight 400-500, line-height 1.6
- **Labels/Nav**: Inter, weight 500, uppercase with wide letter-spacing for nav items

## Spacing & Layout
- Max content width: 1280px
- Section padding: py-20 (80px vertical)
- Card padding: p-6 to p-8
- Border radius: rounded-lg (8px) for cards, rounded-full for buttons
- Grid gap: gap-6 to gap-8

## Component Styles
- **Buttons**: 
  - Primary: bg-[#D4AF37] text-[#0F172A] font-semibold, hover:bg-[#B8960C]
  - Secondary: bg-transparent border-2 border-[#0F172A] text-[#0F172A], hover:bg-[#0F172A] hover:text-white
- **Cards**: bg-white rounded-lg shadow-md border border-[#E2E8F0], hover:shadow-lg transition
- **Navigation**: Fixed top, bg-[#0F172A] text-white, smooth scroll behavior

## Do's
- Use generous whitespace
- Maintain strong contrast (navy on white, white on navy)
- Use accent gold sparingly for CTAs and highlights
- Use responsive grids (1 col mobile, 2-3 col tablet, 3-4 col desktop)
- Include clear hierarchy with font sizes

## Don'ts
- Don't use bright or neon colors
- Don't crowd elements together
- Don't use gradients except very subtle ones
- Don't use more than 3 font sizes on a single section
