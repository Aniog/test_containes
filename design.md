# Velmora Fine Jewelry Design System

## Brand Personality
- **Mood**: Quiet luxury, warm, editorial, sophisticated, timeless.
- **Target**: Woman 25-45, self-purchasing and gifting.
- **Price Point**: $30-$120 (Demi-fine).

## Typography
- **Headings**: `Playfair Display`, serif. Letter spacing widened for product names.
- **Body**: `Inter`, sans-serif. Clean and legible.
- **UI Elements**: `Inter`, 12px-14px, uppercase with tracking for a premium feel.

## Color Palette
- **Background**: `#FDFCFB` (Off-white / Warm Cream)
- **Primary Text**: `#1A1A1A` (Near Black)
- **Secondary Text**: `#6B6B6B` (Muted Grey)
- **Accent/Jewelry**: `#D4AF37` (Soft Gold - use sparingly)
- **Surface**: `#FAF9F6` (Very light warm beige for cards)
- **Border**: `#E5E5E5` (Thin hairlines)

## Spacing & Layout
- Generous whitespace (`py-20`, `px-6` on mobile, `px-20` on desktop).
- Thin hairline dividers (`border-b border-stone-200`).
- Large editorial imagery with soft shadows or no shadows for a flat editorial look.

## Components
- **Buttons**: Square edges or slightly rounded. Solid black or outlined black for primary actions.
- **Transitions**: Ease-in-out, 300ms.
- **Icons**: Lucide React - set to `strokeWidth={1.2}` for a delicate look.

## Do's & Don'ts
- **Do**: Use high-quality editorial images.
- **Do**: Keep the UI minimal.
- **Don't**: Use rounded "bubbly" buttons.
- **Don't**: Use loud colors or heavy drop shadows.
