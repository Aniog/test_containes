# Velmora Fine Jewelry — Design System

## Visual Direction
Quiet luxury, warm, editorial. Premium demi-fine jewelry — NOT loud, NOT discount-looking, NOT generic e-commerce.

## Color Palette
- **Base (deep warm charcoal):** #1C1917 (stone-950) — primary background for hero/dark sections
- **Cream (warm off-white):** #FAF7F2 — primary light background
- **Gold Accent:** #C9A96E — primary accent, buttons, highlights
- **Gold Hover:** #B8944F — darker gold for hover states
- **Text Primary:** #1C1917 — on light backgrounds
- **Text Secondary:** #78716C (stone-500) — muted text on light
- **Text Light:** #FAF7F2 — on dark backgrounds
- **Text Muted Light:** #D6D3D1 (stone-300) — muted text on dark
- **Divider:** #E7E5E4 (stone-200) — hairline dividers
- **Card Background:** #FFFFFF
- **Surface Warm:** #F5F0EB — subtle warm gray for alternating sections

## Typography
- **Headings:** Cormorant Garamond (serif) — elegant, editorial
- **Body/UI:** Inter (sans-serif) — clean, readable
- **Product names:** UPPERCASE, letter-spacing 0.15em, Cormorant Garamond
- **Section headings:** UPPERCASE, letter-spacing 0.2em, Cormorant Garamond
- **Body text:** 16px base, line-height 1.6

## Spacing & Layout
- Generous whitespace — sections have py-20 to py-28
- Max content width: 1280px (max-w-7xl)
- Product grid: 2 cols mobile, 3 cols tablet, 5 cols desktop (bestsellers)
- Hairline dividers (1px, stone-200)

## Buttons
- Primary: bg-gold text-white, hover:bg-gold-hover, px-8 py-3, tracking-widest uppercase text-xs, font-sans
- Secondary: border-gold text-gold, hover:bg-gold hover:text-white, same sizing
- Transitions: all 300ms ease

## Cards
- Minimal, no visible border
- Subtle shadow on hover (shadow-md)
- Image aspect ratio: 4x5 for products
- Hover: second image reveal, slight scale

## Animations
- Subtle, smooth transitions (300ms ease)
- No bouncy or playful animations
- Fade-in on scroll where appropriate
- Cart drawer slides from right

## Do's
- Use generous whitespace
- Use serif for all headings and product names
- Use warm, muted tones
- Keep layouts clean and editorial
- Ensure strong contrast for accessibility

## Don'ts
- Don't use bright/saturated colors
- Don't use playful or bouncy animations
- Don't crowd sections with too many elements
- Don't use generic e-commerce patterns (badges, aggressive sales text)
- Don't use light text on light backgrounds
