# Visual Style Design

## Typography
- **Primary Font**: Inter, sans-serif (Google Fonts)
- **Headings**: Semi-bold to Bold (font-semibold, font-bold)
- **Body**: Regular (font-normal)

## Colors
- **Primary**: Blue-600 (#2563eb) - Tailwind `blue-600`
- **Secondary**: Slate-600 (#475569) - Tailwind `slate-600`
- **Background**: Slate-50 (#f8fafc) - Tailwind `slate-50`
- **Surface**: White (#ffffff) - Tailwind `white`
- **Text**: Slate-900 (#0f172a) - Tailwind `slate-900`

## Components
- **Buttons**: Rounded-lg (border-radius: 0.5rem), blue-600 background, white text.
- **Cards**: Rounded-2xl (border-radius: 1rem), shadow-xl, white background.
- **Containers**: Responsive max-widths with centered alignment.

## Spacing
- Standard Tailwind spacing scale applies.
- Preferred page padding: p-4 for mobile, p-8 for desktop.

## Do's
- Use consistent rounding for elements (rounded-lg or rounded-2xl).
- Ensure high contrast for readability.
- Use Lucide icons for visual cues.

## Don'ts
- Don't use pure black for text; use Slate-900 for a softer look.
- Don't overcrowd the layout; use generous whitespace.
