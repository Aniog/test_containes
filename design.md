# Design Guidelines - MicroCosmos

## Typography
- Primary Font: **Inter** (Sans-serif) for body and UI elements. Standard Tailwind classes: `font-sans`.
- Accent Font: **Playfair Display** (Serif) for headings and elegant touches. Tailwind classes: `font-serif`.

## Colors
- Background: Deep Slate/Navy (`#0f172a`). Tailwind class: `bg-slate-900`.
- Text: 
  - Primary: Off-white/Silver (`#f8fafc`). Tailwind class: `text-slate-50`.
  - Secondary: Muted Slate (`#94a3b8`). Tailwind class: `text-slate-400`.
- Accent: Emerald/Teal for "life" vibe (`#10b981`). Tailwind class: `text-emerald-500`.

## Spacing
- Use generous padding for sections: `py-20` on desktop, `py-12` on mobile.
- Container width: `max-w-7xl mx-auto px-4`.

## Visual Style
- Cards: Dark backgrounds with subtle borders and shadows. `bg-slate-800/50 border border-slate-700 rounded-xl shadow-lg backdrop-blur-sm`.
- Images: Rounded corners (`rounded-lg`), subtle hover transformations.
- Sections: Alternating backgrounds or purely dark with visual separators.

## Components
- Navbar: Sticky, transparent with backdrop blur.
- Hero: Large background image (stock), bold serif heading.
- Gallery: Mosaic or grid layout for microscopic life.
- Sections: Educational snippets with side-by-side text and images.
