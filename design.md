# MicroCosmos Design System

## Theme
Dark, mysterious, scientific — evoking the hidden world of microscopic life.
Deep navy/black backgrounds with vibrant teal, cyan, and green accents.

## Colors
- Background primary: `bg-[#050d1a]` (near-black deep navy)
- Background secondary: `bg-[#0a1628]` (dark navy)
- Background card: `bg-[#0f1f35]` (dark blue-navy)
- Accent primary: `text-teal-400`, `bg-teal-400` (#2dd4bf)
- Accent secondary: `text-cyan-400`, `bg-cyan-400` (#22d3ee)
- Accent glow: `text-emerald-400` (#34d399)
- Text primary: `text-white`
- Text secondary: `text-slate-300`
- Text muted: `text-slate-500`
- Border: `border-teal-900/40`, `border-slate-700/50`

## Typography
- Font: Inter (Google Fonts)
- Hero heading: `text-5xl md:text-7xl font-black tracking-tight`
- Section heading: `text-3xl md:text-4xl font-bold`
- Card heading: `text-xl font-semibold`
- Body: `text-base text-slate-300 leading-relaxed`
- Caption: `text-sm text-slate-500`

## Spacing
- Section padding: `py-20 px-4 md:px-8`
- Card padding: `p-6`
- Gap between cards: `gap-6`

## Borders & Shadows
- Card border: `border border-teal-900/40 rounded-2xl`
- Glow effect: `shadow-[0_0_30px_rgba(45,212,191,0.15)]`
- Image overlay: gradient from transparent to `#050d1a`

## Components
- Navbar: fixed top, dark glass effect `bg-[#050d1a]/80 backdrop-blur-md`
- Hero: full-screen with background image and gradient overlay
- Cards: dark navy with teal border, hover scale effect
- Gallery: masonry-style grid with hover zoom
- Badges: `bg-teal-900/50 text-teal-300 rounded-full px-3 py-1 text-xs`

## Do's
- Use dark backgrounds throughout
- Use teal/cyan as accent colors for interactive elements
- Add subtle glow effects on hover
- Use large, full-bleed images in hero and gallery sections
- Maintain high contrast: white/light text on dark backgrounds

## Don'ts
- No light backgrounds (white, light gray)
- No warm color accents (orange, red, yellow) as primary
- No small, cramped image layouts — images should breathe
- No flat, colorless designs — use subtle gradients
