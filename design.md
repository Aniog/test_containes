# Velmora Fine Jewelry - Design System

## Brand Identity
- **Brand Name**: Velmora Fine Jewelry
- **Tagline**: Crafted to be Treasured
- **Target Audience**: Women 25-45, gifting and self-purchase
- **Price Point**: Premium demi-fine ($30–$120)

## Visual Identity

### Mood
- Quiet luxury, warm, editorial
- Premium demi-fine jewelry
- NOT loud, NOT discount-looking, NOT generic e-commerce

### Color Palette
- **Primary Background**: `#0D0D0D` (rich black)
- **Secondary Background**: `#1A1A1A` (soft black)
- **Tertiary/Cards**: `#242424` (charcoal)
- **Primary Text**: `#F5F5F0` (warm white)
- **Secondary Text**: `#A8A8A0` (muted warm gray)
- **Accent/Gold**: `#C9A962` (warm gold)
- **Accent Hover**: `#D4B978` (lighter gold)
- **Border/Divider**: `#333333` (subtle gray)
- **Success**: `#4A7C59` (muted green)
- **Error**: `#9C4146` (muted red)

### Typography
- **Headings/Product Names**: Cormorant Garamond (serif)
  - Font weights: 400, 500, 600
  - Product names: UPPERCASE with letter-spacing: 0.15em
- **Body/UI**: Manrope (sans-serif)
  - Font weights: 400, 500, 600
  - Body: 400 weight, line-height 1.6

### Spacing System
- Base unit: 4px
- Section padding: 80px vertical (desktop), 48px (mobile)
- Container max-width: 1400px
- Grid gap: 24px (desktop), 16px (mobile)

### Visual Effects
- **Shadows**: Soft, subtle shadows
  - Card: `0 4px 20px rgba(0,0,0,0.3)`
  - Hover: `0 8px 30px rgba(0,0,0,0.4)`
- **Borders**: 1px hairline dividers, color `#333333`
- **Transitions**: 300ms ease for all interactions
- **Border Radius**: 0px for buttons (sharp, premium feel), 4px for cards

## Component Guidelines

### Buttons
- **Primary**: Solid gold background, dark text
  - Background: `#C9A962`
  - Text: `#0D0D0D`
  - Padding: 16px 32px
  - Uppercase, letter-spacing: 0.1em
  - Hover: `#D4B978`
- **Secondary**: Outlined, gold border
  - Border: 1px solid `#C9A962`
  - Text: `#C9A962`
  - Hover: fill with gold, dark text

### Cards
- Background: `#242424`
- Padding: 24px
- Sharp corners (0px radius)
- Subtle shadow on hover

### Dividers
- 1px height
- Color: `#333333`
- Used between sections and within content

### Form Inputs
- Background: transparent
- Border: 1px solid `#333333`
- Focus: border-color `#C9A962`
- Padding: 12px 16px

## Responsive Breakpoints
- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px

## Do's and Don'ts
- DO: Use generous whitespace
- DO: Use large editorial imagery
- DO: Use thin hairline dividers
- DO: Use restrained accent color
- DO: Use subtle hover transitions
- DO: Use soft shadows
- DON'T: Use loud colors or gradients
- DON'T: Use discount-style layouts
- DON'T: Use generic e-commerce patterns
- DON'T: Use rounded corners on buttons
- DON'T: Use more than 2-3 accent colors
