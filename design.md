# Design System — Animal World

## Visual Theme
Nature-inspired, warm and earthy tones with lush greens and deep forest colors.
Clean, modern layout with generous whitespace and organic shapes.

## Colors
- Primary green: `#2d6a4f` (deep forest green) — `text-[#2d6a4f]`, `bg-[#2d6a4f]`
- Light green: `#52b788` — `text-[#52b788]`
- Accent amber: `#e9c46a` — `text-[#e9c46a]`, `bg-[#e9c46a]`
- Warm orange: `#f4a261` — `text-[#f4a261]`
- Dark background: `#1b2e22` — `bg-[#1b2e22]`
- Light surface: `#f0f7f4` — `bg-[#f0f7f4]`
- Card background: `#ffffff` — `bg-white`
- Text primary: `#1a2e1e` — `text-[#1a2e1e]`
- Text muted: `#4a6741` — `text-[#4a6741]`

## Typography
- Font: Inter (Google Fonts)
- Hero heading: `text-5xl md:text-7xl font-extrabold`
- Section heading: `text-3xl md:text-4xl font-bold`
- Card title: `text-xl font-semibold`
- Body: `text-base font-normal`
- Caption/label: `text-sm font-medium`

## Spacing
- Section padding: `py-16 md:py-24 px-4 md:px-8`
- Card padding: `p-6`
- Gap between grid items: `gap-6 md:gap-8`

## Borders & Shadows
- Card border radius: `rounded-2xl`
- Image border radius: `rounded-xl`
- Card shadow: `shadow-md hover:shadow-xl`
- Transition: `transition-all duration-300`

## Do's
- Use earthy, nature-inspired color palette
- Use generous whitespace between sections
- Use rounded corners on cards and images
- Use hover effects on interactive cards
- Ensure all text is readable against its background

## Don'ts
- Don't use harsh neon colors
- Don't use dark text on dark backgrounds
- Don't use light text on light backgrounds
- Don't use too many font sizes — stick to the scale above
