# Velmora Fine Jewelry - Design System

## Brand Identity
- **Brand Name**: Velmora Fine Jewelry
- **Tagline**: Crafted to be Treasured
- **Target Audience**: Women 25-45, gifting and self-purchase
- **Price Range**: $30–$120 (premium-but-accessible)

## Visual Identity

### Mood
- Quiet luxury, warm, editorial
- Premium demi-fine jewelry
- NOT loud, NOT discount-looking, NOT generic e-commerce

### Color Palette
- **Primary Background**: `#FAF9F7` (warm off-white/cream)
- **Secondary Background**: `#1A1814` (deep warm charcoal/brown-black)
- **Text Primary**: `#1A1814` (deep warm charcoal)
- **Text Secondary**: `#6B635A` (warm taupe/gray-brown)
- **Text on Dark**: `#FAF9F7` (cream)
- **Accent/Gold**: `#C9A962` (warm muted gold)
- **Accent Hover**: `#B8954F` (deeper gold)
- **Border/Divider**: `#E8E5E0` (warm light gray)
- **Card Background**: `#FFFFFF` (pure white)
- **Success**: `#4A7C59` (muted forest green)
- **Error**: `#A65D57` (muted terracotta)

### Typography
- **Headings/Product Names**: 
  - Font: `Cormorant Garamond` (Google Fonts)
  - Weight: 500-600
  - Letter-spacing: `0.15em` (wide for product names - uppercase)
- **Body/UI**:
  - Font: `Manrope` (Google Fonts) 
  - Weight: 400-500
  - Letter-spacing: `0.02em` (normal)

### Spacing System
- Base unit: 4px
- Section padding: 80px vertical (desktop), 48px (mobile)
- Container max-width: 1280px
- Grid gap: 24px (desktop), 16px (mobile)

### Visual Effects
- **Shadows**: 
  - Soft: `0 4px 20px rgba(26, 24, 20, 0.06)`
  - Hover: `0 8px 30px rgba(26, 24, 20, 0.1)`
- **Border Radius**: 0px (sharp, editorial) or 4px (subtle)
- **Dividers**: 1px hairline, color `#E8E5E0`
- **Transitions**: 300ms ease-out (subtle, premium feel)

## Component Guidelines

### Buttons
- **Primary**: 
  - Background: `#1A1814`
  - Text: `#FAF9F7`
  - Padding: 16px 32px
  - Hover: Background `#C9A962`
- **Secondary/Outline**:
  - Border: 1px solid `#1A1814`
  - Text: `#1A1814`
  - Hover: Background `#1A1814`, Text `#FAF9F7`

### Product Cards
- White background
- Subtle shadow on hover
- Image aspect ratio: 4:5 (portrait)
- Product name: Uppercase, wide letter-spacing
- Price: Regular weight
- Quick Add to Cart on hover

### Form Inputs
- Border: 1px solid `#E8E5E0`
- Focus: Border `#1A1814`
- Padding: 12px 16px
- Border-radius: 0px (sharp)

## Responsive Breakpoints
- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px

## Do's and Don'ts
- DO: Use generous whitespace
- DO: Use large editorial imagery
- DO: Use thin hairline dividers
- DO: Use restrained accent color
- DO: Use subtle hover transitions
- DO: Use soft shadows
- DON'T: Use loud colors or gradients
- DON'T: Use discount-style badges
- DON'T: Use generic e-commerce layouts
- DON'T: Use rounded corners > 4px
