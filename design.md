# SSourcing China - Design Guide

## Brand Identity
SSourcing China is a professional B2B sourcing agent. Design must communicate: trust, reliability, expertise, global reach.

## Colors
- Primary: #0f2b4a (deep navy) - Trust, stability, professionalism
- Primary Light: #1a4a7a (medium blue) - Interactive elements, links
- Accent: #0d9488 (teal) - CTAs, highlights, trust signals
- Accent Light: #14b8a6 (light teal) - Hover states
- Background: #ffffff (white) - Clean page backgrounds
- Surface: #f8fafc (ice gray) - Section backgrounds
- Border: #e2e8f0 (light gray) - Cards, dividers
- Text Primary: #0f172a (near-black) - Body text
- Text Secondary: #475569 (slate) - Supporting text
- Text Muted: #94a3b8 (gray) - Captions, metadata

## Typography
- Headings: Inter (weights 600, 700, 800)
- Body: Inter (weight 400, 500)
- Use font-sans Tailwind class

## Spacing
- Section padding: py-16 md:py-24
- Container max-width: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8
- Card padding: p-6 md:p-8
- Gap between sections: gap-12 md:gap-16

## Component Styles
- Cards: bg-white rounded-xl border border-border shadow-sm
- Buttons (primary): bg-accent text-white px-6 py-3 rounded-lg font-medium hover:bg-accent/90 transition
- Buttons (outline): border-2 border-primary text-primary px-6 py-3 rounded-lg font-medium hover:bg-primary hover:text-white transition
- Section titles: text-3xl md:text-4xl font-bold text-primary text-center
- Section subtitles: text-lg text-secondary max-w-2xl mx-auto text-center mt-4

## Do's
- Use clean, generous whitespace
- Use consistent border radius (rounded-xl for cards, rounded-lg for buttons)
- Use subtle shadows for elevation
- Keep CTAs prominent with teal color
- Use grid layouts for service cards, product categories
- Show trust badges and certifications

## Don'ts
- Don't use overly bright or playful colors
- Don't use casual language - maintain professional B2B tone
- Don't overcrowd sections - keep breathing room
- Don't use carousels or auto-rotating content
- Don't use exaggerated claims or superlatives
