# ARTITECT MACHINERY Design System

## Visual Style
Elegant industrial design that conveys precision, reliability, and professional craftsmanship.

## Typography
- Primary font: Inter (Google Fonts)
- Headings: font-weight 700-800, tracking-tight
- Body: font-weight 400-500, leading-relaxed
- Hero text: text-5xl to text-7xl on desktop

## Colors
- Primary: slate-900 (#0f172a) - deep navy for headers, footers
- Secondary: slate-800 (#1e293b) - slightly lighter navy
- Accent: amber-500 (#f59e0b) - industrial gold/amber for CTAs and highlights
- Accent hover: amber-600 (#d97706)
- Background: white (#ffffff) and slate-50 (#f8fafc)
- Text: slate-900 for headings, slate-600 for body
- Borders: slate-200 (#e2e8f0)

## Layout
- Max container width: max-w-7xl
- Section padding: py-20 to py-24
- Grid gaps: gap-6 to gap-8
- Card border-radius: rounded-xl
- Button border-radius: rounded-lg

## Components
- Cards: white background, subtle shadow, rounded-xl, hover:shadow-lg transition
- Buttons: amber-500 bg, white text, rounded-lg, hover:bg-amber-600
- Navigation: sticky, white/transparent bg, backdrop-blur
- Hero: full-width with overlay gradient

## Spacing
- Section spacing: py-20 (80px)
- Component gaps: gap-6, gap-8
- Padding: px-6, px-8 on containers

## Do's
- Use ample white space
- Keep text concise and readable
- Use subtle shadows for depth
- Maintain consistent spacing
- Use high-quality industrial imagery

## Don'ts
- Don't use bright/neon colors
- Don't overcrowd layouts
- Don't use more than 2-3 font weights
- Don't use harsh borders
- Don't use low-contrast text
