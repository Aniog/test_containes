# Velmora Fine Jewelry — Design System

## Mood
Quiet luxury, warm, editorial. Premium demi-fine jewelry. NOT loud, NOT discount-looking.

## Color Palette
- **Background (cream):** `cream` → `#FAF7F2` — warm off-white page base
- **Foreground (charcoal):** `charcoal` → `#1A1A1A` — primary text
- **Accent (gold):** `gold` → `#B8860B` — warm dark gold for CTAs, accents
- **Gold light:** `gold-light` → `#D4A853` — hover states, highlights
- **Muted:** `muted` → `#F0EBE3` — card backgrounds, subtle sections
- **Muted foreground:** `muted-fg` → `#6B6358` — secondary text, captions
- **Border:** `border` → `#E8E2D9` — hairline dividers

## Typography
- **Headings / Product names:** Cormorant Garamond (serif), weight 300–600
- **Body / UI:** Inter (sans-serif), weight 300–500
- **Product names:** UPPERCASE, letter-spacing 0.15em
- **Section headings:** Cormorant Garamond, font-size 2.5rem+, weight 300

## Spacing & Layout
- Generous whitespace: sections py-20 to py-28
- Max content width: 1280px (max-w-7xl)
- Grid gaps: gap-6 to gap-8
- Card padding: p-0 (image bleeds) or p-6 for text areas

## Components
- **Buttons:** Rounded-none (square), uppercase tracking-widest text-xs, py-3 px-8. Primary: bg-gold text-white. Outlined: border-gold text-gold.
- **Cards:** No border-radius, subtle shadow-sm on hover, transition-all duration-300
- **Dividers:** 1px border-border (hairline)
- **Hover effects:** scale-[1.02] on cards, opacity transitions on images

## Do's
- Use Cormorant Garamond for all headings and product names
- Use Inter for body text and UI elements
- Keep generous whitespace between sections
- Use gold accent sparingly — CTAs, stars, small highlights
- Thin hairline dividers between sections
- Large editorial imagery

## Don'ts
- No rounded corners on buttons or cards
- No bright/saturated colors
- No heavy shadows
- No cluttered layouts
- No generic e-commerce feel
- No dark mode (this is a light, warm brand)
