# Velmora Fine Jewelry — Design System

## Visual Identity
- **Mood**: Quiet luxury, warm, editorial. Premium demi-fine jewelry — NOT loud, NOT discount-looking.
- **Direction**: Warm neutral base with gold metallic accents. Deep espresso/charcoal for contrast. Cream backgrounds.

## Color Palette
| Token | Hex | Usage |
|-------|-----|-------|
| brand-cream | #FAF7F2 | Page background |
| brand-warm | #F5EDE3 | Section backgrounds, cards |
| brand-sand | #E8DDD0 | Borders, hairlines, dividers |
| brand-gold | #C9A96E | Primary accent, CTAs, links |
| brand-gold-light | #D4BA8A | Hover states, subtle accents |
| brand-gold-dark | #A8863F | Active states, emphasis |
| brand-charcoal | #2C2C2C | Primary text |
| brand-espresso | #1A1612 | Dark sections, hero overlays |
| brand-muted | #8A8279 | Secondary text, captions |
| brand-muted-light | #A69E95 | Placeholder text |

## Typography
- **Headings**: Cormorant Garamond (serif) — weights 400, 500, 600
- **Body/UI**: Inter (sans-serif) — weights 300, 400, 500, 600
- **Product names**: UPPERCASE, letter-spacing 0.15em, Cormorant Garamond
- **CTA buttons**: UPPERCASE, letter-spacing 0.1em, Inter 500, text-sm

## Spacing
- Generous whitespace. Sections: py-20 md:py-28 lg:py-32
- Container max-width: max-w-7xl (80rem)
- Section padding: px-5 md:px-8 lg:px-12

## Borders & Dividers
- Hairline dividers: border-brand-sand, border-t
- Cards: no border, subtle shadow on hover
- Buttons: solid gold or outlined gold

## Shadows
- Cards: shadow-sm default, shadow-md on hover
- Soft, warm shadows only

## Transitions
- All hover transitions: duration-300
- Subtle scale on button press: active:scale-[0.98]
- Image hover: scale-105 over duration-700

## Do's
- Use serif for all headings and product names
- Use generous whitespace between sections
- Use warm, muted tones for backgrounds
- Ensure high contrast between text and backgrounds
- Use thin hairline dividers between sections

## Don'ts
- Never use bright/saturated colors
- Never use rounded buttons (use sharp or very slight rounding)
- Never crowd sections with too much content
- Never use dark text on dark backgrounds
- Never use more than 2 typefaces
