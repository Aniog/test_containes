# Strikingly Generators Hub Visual Style

Use a clean Strikingly marketing-page style: white page background, faint hero-only purple/pink wash, generous spacing, thin dividers, and compact professional cards.

## Typography
- Headings and buttons use Josefin Sans as the free fallback for Brandon Grotesque, weight 700, uppercase, line-height 1.2.
- Body text uses Open Sans, 14px base, line-height 1.5.
- Hero H1 is 40-48px on desktop and 28-32px on mobile. H2 is 26-32px on desktop.

## Colors
- Brand purple: #8159BB.
- AI gradient: linear-gradient(90deg, #7671FF, #CB0C9F), only for primary CTAs and the second H1 line.
- Headings: #4B5056, with hero first line #2E2E2F.
- Body text: #636972.
- Card borders: #C6C9CD.
- Dividers: #E2E4E7.
- Light background: #F4F6F8.
- Default surface: #FFFFFF.

## Components
- Cards: white background, 1px #C6C9CD border, 3px radius, 20px padding. Hover lifts subtly with a light shadow and brand-purple border.
- Buttons: 36px height, 4px radius, uppercase heading font. Large buttons are 44px high. Filled buttons always use white text.
- Ghost buttons and tags: transparent background, brand-purple border and text.

## Spacing
- Use 10px spacing rhythm, with 5px only for tight pairs.
- Sections are separated by 40px. Hero uses 60-80px vertical padding.
- Main content max width is 1200px and centered.

## Do / Don't
- Do keep all text clearly readable against its background.
- Do use inline SVG or CSS illustrations only.
- Do keep directory cards useful and descriptive.
- Don't add fake reviews, star ratings, or placeholder links.
- Don't use the AI gradient as a generic background treatment outside the hero H1 and primary CTA buttons.
