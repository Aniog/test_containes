# ARTITECT MACHINERY Website Design Guide

## Brand Overview
- **Brand Name**: ARTITECT MACHINERY
- **Products**: Double folding machines, double folders, sheet metal folders, metal folding machines
- **Style**: Elegant but user-friendly

## Visual Style

### Color Palette
- **Primary Dark**: #1a1a2e (Deep navy/black)
- **Primary Accent**: #c9a227 (Golden bronze - industrial elegance)
- **Secondary**: #f5f5f5 (Light gray background)
- **Text Primary**: #1a1a2e (Dark navy)
- **Text Secondary**: #4a4a5a (Muted gray)
- **White**: #ffffff
- **Accent Hover**: #e0b82d (Lighter gold for hover states)

### Typography
- **Headings**: 'Inter', sans-serif - Bold, clean, modern
- **Body**: 'Inter', sans-serif - Regular weight for readability
- **Logo**: 'Inter', 800 weight - Strong and authoritative

### Design Principles
1. **Elegant Simplicity**: Clean layouts with generous whitespace
2. **Industrial Sophistication**: Subtle geometric accents reflecting precision machinery
3. **User-Friendly**: Clear navigation, intuitive layout, readable content
4. **Professional Trust**: Trust-building elements like certifications, features, testimonials

## Layout Structure

### Header
- Fixed navigation with logo on left
- Clean navigation links (Products, About, Contact)
- CTA button "Get Quote"
- Subtle border-bottom separation

### Hero Section
- Large headline with brand messaging
- Supporting subtext about quality and precision
- Dual CTAs: "Explore Products" and "Contact Us"
- Optional background with subtle industrial pattern

### Products Section
- Grid layout showcasing 3-4 main product categories
- Each card: Image, product name, brief description
- Hover effects with subtle shadow and scale
- Clean, modern card design

### Features Section
- Icon + text layout highlighting key benefits
- 4-6 features (Precision, Durability, Efficiency, Support, etc.)
- Two-column or three-column grid

### About Section
- Company story with elegant typography
- Trust indicators (years experience, machines sold, etc.)

### Contact Section
- Simple contact form
- Company contact information
- Clear call-to-action

### Footer
- Company logo and tagline
- Quick links
- Social media icons
- Copyright

## Component Styles

### Buttons
- Primary: Golden bronze background (#c9a227), dark text
- Primary Hover: Lighter gold (#e0b82d)
- Secondary: Transparent with golden border
- Rounded corners (rounded-lg)

### Cards
- White background with subtle shadow
- Rounded corners (rounded-xl)
- Hover: Elevated shadow, subtle scale

### Form Inputs
- Clean border with rounded corners
- Focus: Golden border accent
- Placeholder text in muted color

### Navigation
- Clean links with hover underline effect
- Active state with golden accent
- Mobile: Hamburger menu

## Do's and Don'ts

### Do
- Use generous whitespace
- Keep typography hierarchy clear
- Use golden accent sparingly for emphasis
- Include professional imagery
- Maintain consistent spacing

### Don't
- Use too many colors
- Overcrowd content
- Use low-contrast text
- Include unnecessary decorative elements
- Make the design feel cluttered

## Spacing System
- Section padding: 80px (py-20)
- Container max-width: 1200px
- Card gap: 24px (gap-6)
- Component spacing: 16px-32px
