# Velmora Fine Jewelry — Design System

## Color Palette
- **Ivory** (background): `#FAF8F5` — warm off-white, primary page background
- **Charcoal** (text): `#1C1C1C` — primary text, headings
- **Graphite** (secondary text): `#6B6560` — secondary text, descriptions
- **Gold** (accent): `#B8956A` — primary accent, CTAs, gold jewelry highlights
- **Gold Dark** (accent hover): `#9A7B52` — hover state for gold elements
- **Gold Light**: `#E8DDD0` — subtle gold backgrounds, borders
- **Warm Gray** (cards): `#F5F2EF` — card backgrounds, subtle section fills
- **Sand** (divider): `#E4DFD8` — hairline dividers, borders
- **Pure White**: `#FFFFFF` — cards, overlays, nav
- **Deep Black**: `#0A0A0A` — footer background, hero overlays

## Typography
- **Heading Serif**: Cormorant Garamond — for headings, product names, hero text
- **Body Sans**: Inter — for body text, UI elements, descriptions
- **Product names**: UPPERCASE, letter-spacing: 0.15em
- **Hero headings**: Cormorant Garamond, light weight (300-400), large sizes

## Spacing & Layout
- **Section padding**: py-16 md:py-24 lg:py-32 (vertical), px-4 md:px-8 lg:px-16 (horizontal)
- **Max content width**: max-w-7xl (1280px)
- **Card border-radius**: rounded-lg (8px)
- **Button border-radius**: rounded-none (sharp, editorial feel)

## Components
- **Buttons**: Solid gold bg with white text, or outlined gold border with gold text. Sharp corners (rounded-none). Font: Inter, uppercase, tracking-widest, text-sm.
- **Dividers**: 1px solid #E4DFD8, no heavy borders
- **Cards**: White bg, subtle shadow on hover, clean spacing
- **Input fields**: Transparent bottom border, minimal, elegant

## Do's
- Use generous whitespace
- Maintain warm, intimate tone
- Use subtle hover transitions (300ms ease)
- Keep contrast ratios above 4.5:1 for text

## Don'ts
- Don't use bright or neon colors
- Don't use rounded pill buttons
- Don't add heavy box shadows
- Don't use more than 2 font families
- Don't use discount/coupon styling
