# ARTITECT MACHINERY - Website Design System

## Visual Style
- **Theme**: Industrial elegance - clean, professional, sophisticated
- **Feel**: Trustworthy, modern, precise (like the machines themselves)

## Color Palette
- **Primary**: `#1a365d` (Deep Navy Blue) - Authority, trust
- **Secondary**: `#2d3748` (Charcoal Gray) - Industrial strength
- **Accent**: `#c9a227` (Golden Bronze) - Premium quality, highlights
- **Background**: `#f7fafc` (Off White) - Clean, professional
- **Surface**: `#ffffff` (White) - Cards and content areas
- **Text Primary**: `#1a202c` (Near Black)
- **Text Secondary**: `#718096` (Muted Gray)
- **Success**: `#38a169` (Green)

## Typography
- **Headings**: Inter (700 weight) - Bold, clean, modern
- **Body**: Inter (400, 500 weight) - Readable, professional
- **Accent Text**: Inter (600 weight) - Labels and CTAs

## Spacing System
- Base unit: 4px
- Section padding: 80px vertical (desktop), 48px (mobile)
- Container max-width: 1280px
- Card padding: 24px-32px

## Component Styles
- **Cards**: White background, subtle shadow (`0 4px 6px rgba(0,0,0,0.05)`), rounded corners (8px)
- **Buttons**: Primary: Deep navy with white text, hover: lighter navy; Secondary: White with navy border
- **Icons**: Lucide React, 24px default size
- **Images**: Rounded corners, subtle shadow on hover

## Design Principles
- Generous white space
- Clear visual hierarchy
- Professional imagery feel
- Smooth transitions (200-300ms)
- Mobile-first responsive design
