# Velmora Fine Jewelry - Design System

## Visual Identity

### Mood
- Quiet luxury, warm, editorial
- Premium demi-fine jewelry aesthetic
- NOT loud, NOT discount-looking, NOT generic e-commerce

### Color Palette
**Primary Colors:**
- `cream` - #FAF7F2 (warm off-white background)
- `charcoal` - #1A1A1A (primary text, dark backgrounds)
- `warm-black` - #0D0D0D (deepest backgrounds)

**Accent Colors:**
- `gold` - #C9A962 (primary accent, CTAs, highlights)
- `gold-light` - #E5D4A1 (hover states, secondary accents)
- `gold-dark` - #9A7B3F (active states)

**Neutral Colors:**
- `stone-50` - #FAFAF9
- `stone-100` - #F5F5F4
- `stone-200` - #E7E5E4
- `stone-300` - #D6D3D1
- `stone-400` - #A8A29E
- `stone-500` - #78716C
- `stone-600` - #57534E

### Typography
**Headings & Product Names:**
- Font: Cormorant Garamond (serif)
- Weights: 400, 500, 600
- Product names: UPPERCASE with letter-spacing: 0.15em

**Body & UI:**
- Font: Manrope (sans-serif)
- Weights: 400, 500, 600

### Spacing System
- Base unit: 4px
- Section padding: 80px (desktop), 48px (mobile)
- Component gaps: 16px, 24px, 32px

### Visual Effects
- Hairline dividers: 1px solid rgba(0,0,0,0.08)
- Soft shadows: 0 4px 20px rgba(0,0,0,0.06)
- Hover transitions: 300ms ease
- Image hover: subtle scale (1.02) + opacity change

## Components

### Buttons
- Primary: Gold background (#C9A962), charcoal text, solid
- Secondary: Transparent with gold border, gold text
- Hover: Slight darken, subtle lift

### Cards
- White background with soft shadow
- Hairline border on hover
- Image zoom on hover

### Inputs
- Clean border, subtle focus ring in gold
- Placeholder in stone-400

## Responsive Breakpoints
- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px
