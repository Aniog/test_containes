# Travel Itinerary – Design System

## Colors
- Background: `bg-slate-100` (#f1f5f9)
- Surface/Card: `bg-white`
- Primary accent: `bg-sky-500`, `text-sky-600`, `border-sky-400`
- Danger: `text-red-400`, hover `text-red-600`
- Muted text: `text-slate-400`
- Body text: `text-slate-700`
- Headings: `text-slate-800`

## Typography
- Font: Inter (Google Fonts)
- Page title: `text-3xl font-bold text-slate-800`
- Day title: `text-lg font-semibold text-slate-800`
- Activity text: `text-sm text-slate-700`
- Labels/meta: `text-xs text-slate-400 uppercase tracking-wide`

## Spacing & Layout
- Page padding: `px-4 py-8 max-w-2xl mx-auto`
- Card: `rounded-2xl shadow-sm border border-slate-200 p-5`
- Gap between days: `space-y-4`
- Gap between activities: `space-y-2`

## Buttons
- Primary: `bg-sky-500 hover:bg-sky-600 text-white rounded-lg px-4 py-2 text-sm font-medium`
- Ghost/outline: `border border-slate-300 hover:border-sky-400 text-slate-600 rounded-lg px-3 py-1.5 text-sm`
- Icon button: `p-1 rounded hover:bg-slate-100 text-slate-400 hover:text-red-500`

## Do's
- Use rounded-2xl for day cards, rounded-lg for activity rows
- Keep the layout single-column, centered, max-w-2xl
- Use sky-500 as the primary brand color throughout
- Subtle shadows (shadow-sm) only

## Don'ts
- No dark backgrounds on the main surface
- No complex multi-column layouts
- No heavy borders or harsh colors
