# Velmora Fine Jewelry — Visual Style Guide

## Brand Essence
Quiet luxury, warm, editorial. Premium demi-fine jewelry. Never loud, never discount-looking, never generic.

## Color Palette
- **Background**: #FAF7F2 (warm ivory/cream) — sitewide page background
- **Surface / Cards**: #FFFFFF (white) — cards, product tiles, modals
- **Primary Text**: #1A1A1A (near-black) — headings, body text
- **Secondary Text**: #6B6258 (warm taupe) — secondary labels, meta
- **Accent / Gold**: #B8860B (dark goldenrod) — CTAs, accent borders, hover states, links
- **Muted Gold**: #D4A853 (warm gold) — decorative accents, stars, badges
- **Light Border**: #E8E3DC (warm beige) — dividers, hairline borders
- **Footer Background**: #1A1A1A (dark)
- **Footer Text**: #C4BEB6 (warm light gray)

## Typography
- **Headings (serif)**: Cormorant Garamond — hero titles, section headings, product names
- **Body (sans-serif)**: Inter — UI text, descriptions, nav links, buttons
- **Product names**: UPPERCASE, tracking-widest (letter-spacing 0.15em+), serif
- **Section headings**: uppercase, tracking-wider, serif, text-sm or text-base

## Spacing & Layout
- Max content width: 1280px, centered
- Generous whitespace — min section padding: py-16 md:py-24
- Thin hr dividers: border-t border-[#E8E3DC] my-16
- Grid gaps: gap-6 to gap-8

## Visual Elements
- **Buttons**: 
  - Primary: solid bg-[#B8860B] text-white, hover:bg-[#A0750A], rounded-sm, px-8 py-3
  - Outline: border border-[#B8860B] text-[#B8860B] hover:bg-[#B8860B] hover:text-white, rounded-sm
- **Images**: Large editorial style, warm lighting, gold jewelry on dark/neutral backgrounds
- **Hover transitions**: duration-300 ease-out on all interactive elements
- **Shadows**: subtle — shadow-sm hover:shadow-md
- **Card borders**: thin border border-[#E8E3DC]/60
- **Accent borders**: top border accent color on feature sections

## Icons
- Lucide React for all icons
- Search icon, ShoppingCart icon, Menu (hamburger), X (close), ChevronDown, Star, ArrowRight, Check

## Do's
- Use generous whitespace
- Keep text fully readable with strong contrast
- Use warm gold accents sparingly for elegance
- UPPERCASE with wide tracking for product names
- Thin hairline dividers between sections

## Don'ts
- No bright/neon colors
- No heavy shadows or overdone effects
- No cluttered layouts
- No low-contrast text
- No generic e-commerce look
