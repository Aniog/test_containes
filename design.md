# SSourcing China — Design Guidelines

## Brand Identity
- **Company**: SSourcing China
- **Industry**: B2B sourcing / supply chain services
- **Audience**: Overseas buyers, importers, Amazon sellers, procurement managers
- **Tone**: Professional, trustworthy, practical, no exaggerated claims

## Color Palette

### Primary Colors
- **Navy Blue** (primary): `#1B3A5C` — trust, authority, B2B professionalism
- **Sky Blue** (accent): `#2E86DE` — energy, action, CTA buttons
- **White**: `#FFFFFF` — clean backgrounds
- **Light Gray**: `#F7F9FC` — section backgrounds
- **Dark Gray**: `#2D3748` — body text

### Secondary Colors
- **Green** (success/trust): `#27AE60` — verification badges, trust indicators
- **Orange** (warm accent): `#F39C12` — highlights, warnings
- **Border Gray**: `#E2E8F0` — card borders, dividers

## Typography
- **Font Family**: Inter (Google Fonts)
- **Headings**: font-weight 700-800, color #1B3A5C
- **Body**: font-weight 400, color #2D3748, line-height 1.7
- **Small/Caption**: font-weight 500, color #718096

### Scale
- H1: text-4xl md:text-5xl lg:text-6xl
- H2: text-3xl md:text-4xl
- H3: text-xl md:text-2xl
- Body: text-base (16px)
- Small: text-sm (14px)

## Spacing & Layout
- Section padding: py-16 md:py-24
- Container max-width: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8
- Card padding: p-6 md:p-8
- Grid gaps: gap-6 md:gap-8

## Components

### Buttons
- **Primary CTA**: bg-sky-blue text-white px-8 py-3 rounded-lg font-semibold hover:bg-opacity-90
- **Secondary**: border-2 border-navy text-navy px-8 py-3 rounded-lg font-semibold hover:bg-navy hover:text-white
- **Ghost**: text-sky-blue font-semibold hover:underline

### Cards
- White background, rounded-xl, shadow-sm, border border-border-gray
- Hover: shadow-md, translate-y-[-2px] transition

### Section Headers
- Centered, with a small accent line (w-16 h-1 bg-sky-blue mx-auto mb-4)

## Visual Style
- Clean, spacious layouts with generous whitespace
- Professional photography: factories, quality inspection, shipping containers, warehouses
- Icons: Lucide React, stroke-based, consistent sizing
- No gradients on backgrounds; use solid colors
- Subtle shadows, no heavy drop shadows

## Do's
- Use navy blue for trust and authority
- Keep layouts clean and scannable
- Use real industry terminology
- Show process steps clearly
- Include trust indicators (certifications, years, client count)
- Use whitespace generously

## Don'ts
- Don't use bright/saturated colors as backgrounds
- Don't use cartoon illustrations
- Don't make exaggerated claims ("best in the world")
- Don't clutter sections with too much text
- Don't use dark backgrounds for content sections (only hero)
- Don't use more than 2 font families
