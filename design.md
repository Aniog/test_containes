# ARTITECT MACHINERY — Design System

## Overview
An elegant, user-friendly industrial machinery brand site. Dark, refined palette with warm gold accents. Clean typography, generous whitespace, and professional imagery.

## Colors
- Primary background: `#0B0F19` (near-black, deep charcoal)
- Surface/card: `#141B2D` (dark navy)
- Accent: `#C9A45C` (warm gold)
- Accent hover: `#D4B76A` (lighter gold)
- Text primary: `#F5F5F5` (soft white)
- Text secondary: `#9BA3AF` (muted gray-blue)
- Border: `#1E293B` (subtle dark border)
- Light section bg: `#F8F9FC` (off-white for alternating sections)
- Light text: `#1A2332` (dark text on light bg)

## Typography
- Headings: Inter, weight 700-800, tight letter-spacing (`tracking-tight`)
- Body: Inter, weight 400-500, size 16px, line-height 1.65
- Accent text: uppercase, wide tracking (`tracking-widest`), gold color
- Hero title: 56px (desktop), 36px (mobile)
- Section titles: 36px (desktop), 28px (mobile)

## Spacing
- Section padding: 96px vertical (desktop), 64px (mobile)
- Container max-width: 1280px, centered
- Card padding: 32px
- Grid gap: 32px

## Border & Shadow
- Card border-radius: 12px
- Button border-radius: 8px
- Cards: subtle border `border border-[#1E293B]` on dark bg
- No heavy shadows; rely on borders and subtle backgrounds

## Buttons
- Primary: gold background (`#C9A45C`), dark text, hover `#D4B76A`
- Secondary: transparent with gold border, gold text, hover fills gold
- Padding: 14px 28px
- Font: 600 weight, uppercase, tracking-wide

## Images
- Use `data-strk-img` for product and hero imagery
- Preferred ratios: 16x9 for hero, 4x3 for cards
- Overlay gradients on hero for text readability

## Do's
- Use generous whitespace between sections
- Keep typography clean and readable
- Use gold sparingly as an accent
- Ensure high contrast for all text
- Make navigation sticky and subtle

## Don'ts
- Don't use bright saturated colors
- Don't crowd elements together
- Don't use default browser focus styles without customizing
- Don't make text too small on mobile
