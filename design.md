# SSourcing China Visual Design Guide

## Brand position
SSourcing China is a professional China-based sourcing agent for overseas buyers. The design should feel trustworthy, practical, international, and B2B-focused. Avoid hype, exaggerated claims, and consumer-style decorations.

## Color system
- Primary navy: use `bg-slate-950`, `text-slate-950`, and `border-slate-900` for authority and trust.
- Secondary blue: use `bg-blue-700`, `text-blue-700`, and hover `bg-blue-800` for calls to action and key accents.
- Supporting blue surfaces: use `bg-blue-50`, `text-blue-950`, and `border-blue-100` for light information blocks.
- Neutral surfaces: use `bg-white`, `bg-slate-50`, `bg-slate-100`, `border-slate-200`, and `text-slate-600`.
- Success/check accents: use `text-emerald-600`, `bg-emerald-50`, and `border-emerald-100` sparingly for verified or completed steps.
- Warm logistics accent: use `text-amber-700`, `bg-amber-50`, and `border-amber-100` sparingly for shipping and production notes.

## Typography
- Use Inter as the primary font through the Google Fonts embed in `index.html`.
- Headings should be clear and confident: `font-semibold` or `font-bold`, tight tracking, and strong slate colors.
- Body copy should be practical and readable: `text-slate-600` or `text-slate-700` with relaxed line height.
- Avoid overly large display text on mobile. Use responsive sizes such as `text-4xl md:text-6xl`.

## Layout and spacing
- Use spacious section padding: `py-16 md:py-24`.
- Use a centered max-width container: `mx-auto max-w-7xl px-4 sm:px-6 lg:px-8`.
- Desktop layouts should use balanced multi-column grids. Mobile layouts should stack cleanly using Tailwind responsive classes.
- Cards should use `rounded-2xl`, `border border-slate-200`, `bg-white`, and subtle shadows such as `shadow-sm`.

## Components
- Primary CTA: `rounded-full bg-blue-700 px-6 py-3 text-sm font-semibold text-white shadow-sm hover:bg-blue-800`.
- Secondary CTA: `rounded-full border border-slate-300 bg-white px-6 py-3 text-sm font-semibold text-slate-900 hover:bg-slate-50`.
- Badges: `rounded-full border border-blue-100 bg-blue-50 px-3 py-1 text-sm font-medium text-blue-800`.
- Inputs: `rounded-xl border border-slate-300 bg-white px-4 py-3 text-slate-950 placeholder:text-slate-400 focus:border-blue-700 focus:outline-none focus:ring-2 focus:ring-blue-100`.

## Imagery
- Use realistic factory, quality control, production, warehouse, and shipping visuals via Strikingly stock image attributes.
- Images should support credibility and operations, not feel abstract or decorative.
- Prefer `rounded-3xl`, `object-cover`, and subtle borders/shadows for visual panels.

## Do's
- Be clear, practical, and specific.
- Show sourcing process, inspection, supplier verification, and shipping coordination.
- Make contact and quote CTAs easy to find.
- Keep all text high-contrast and readable.

## Don'ts
- Do not use neon colors, playful gradients, or unrealistic claims.
- Do not use low-contrast gray text on colored backgrounds.
- Do not overcrowd the homepage or use mobile-only layouts on desktop.
