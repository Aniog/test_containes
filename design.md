# Visual Style Guide - Strikingly Hub Page

## Typography
- **Headings**: Josefin Sans (fallback for Brandon Grotesque), weight 700, ALL UPPERCASE, line-height 1.2.
  - H1: 40-48px (desktop), 28-32px (mobile)
  - H2: 26-32px (desktop)
  - H3: Subsection headings
- **Body**: Open Sans, weight 400, 14px base, line-height 1.5. `#636972`
- **Buttons**: Same as headings, weight 700, uppercase.

## Colors
- **Brand Purple**: `#8159BB`
- **AI Gradient**: `linear-gradient(to right, #7671FF, #CB0C9F)`
- **Body Text**: `#636972`
- **Headings**: 
  - Section Headings: `#4B5056`
  - Hero H1 Line 1: `#2E2E2F`
- **Card Border**: `#C6C9CD`
- **Subtle Divider**: `#E2E4E7`
- **Light Background**: `#F4F6F8`
- **White**: `#FFFFFF` (default background)

## Components
- **Buttons**:
  - Height: 36px (standard), 44px (big)
  - Border-radius: 4px
  - Text: 14px, uppercase, white on gradient/solid.
  - Ghost: Transparent bg, 1px brand-purple border, brand-purple text.
- **Cards**:
  - Bg: White, border: 1px `#C6C9CD`, 3px border-radius, 20px padding.
  - Hover: Elevation lift (small box-shadow), 1px brand-purple border.
- **Tags**:
  - 11px uppercase, 2x6px padding, 3px border-radius.
  - Ghost: brand-purple text, 1px brand-purple border.

## Spacing
- Multiples of 10px (5px for tight pairs).
- 20px between blocks.
- 40px between sections.
- Hero: 60-80px top and bottom.
- Max content width: 1200px.
