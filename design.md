# SSourcing China - Design System

## Brand Identity
- Professional B2B sourcing agent website
- Clean, trustworthy, international feel
- Target: overseas buyers looking for China sourcing support

## Colors
- Primary: Deep navy blue `#1e3a5f` (trust, professionalism)
- Primary light: `#2d5a8e`
- Accent: Warm orange `#e86c2b` (CTA, energy)
- Accent hover: `#d45a1a`
- Background: White `#ffffff`
- Background alt: Light gray `#f7f9fc`
- Text primary: Dark charcoal `#1a2332`
- Text secondary: Medium gray `#5a6b7d`
- Text muted: `#8a9bb0`
- Border: `#e2e8f0`
- Success: `#16a34a`

## Typography
- Font: Inter (Google Fonts)
- Headings: font-bold, tracking-tight
  - H1: text-4xl md:text-5xl lg:text-6xl
  - H2: text-3xl md:text-4xl
  - H3: text-xl md:text-2xl
  - H4: text-lg md:text-xl
- Body: text-base (16px), text-secondary color
- Small: text-sm

## Spacing
- Section padding: py-16 md:py-24
- Container: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8
- Card padding: p-6 md:p-8
- Gap between cards: gap-6 md:gap-8

## Components
- Buttons:
  - Primary: bg-[#e86c2b] hover:bg-[#d45a1a] text-white font-semibold px-6 py-3 rounded-lg
  - Secondary: border-2 border-[#1e3a5f] text-[#1e3a5f] hover:bg-[#1e3a5f] hover:text-white px-6 py-3 rounded-lg
- Cards: bg-white rounded-xl shadow-sm border border-[#e2e8f0] hover:shadow-md transition-shadow
- Badges: bg-[#f7f9fc] text-[#1e3a5f] text-sm font-medium px-3 py-1 rounded-full

## Do's
- Use consistent spacing between sections
- Keep text readable with proper contrast
- Use stock images for factory, QC, shipping visuals
- Maintain professional B2B tone
- Use icons from Lucide React

## Don'ts
- No dark mode (B2B professional site)
- No exaggerated claims or flashy animations
- No magic hex values outside this system
- No cramped layouts on mobile
