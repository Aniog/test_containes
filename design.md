# SSourcing China Design System

## Brand Identity
- **Industry**: B2B Sourcing / Supply Chain / International Trade
- **Tone**: Professional, trustworthy, international, practical
- **Primary audience**: Global buyers, procurement managers, e-commerce sellers

## Color Palette
- **Primary**: `#1E3A5F` (Deep navy blue) - Trust, professionalism, international business
- **Primary Light**: `#2B5B8A` - Hover states, accents
- **Secondary**: `#E8A838` (Amber/Gold) - CTAs, highlights, Chinese gold accent
- **Secondary Light**: `#F0C060` - Hover state for CTAs
- **Neutral 900**: `#0F172A` - Headlines, primary text
- **Neutral 700**: `#334155` - Body text
- **Neutral 500**: `#64748B` - Secondary text, captions
- **Neutral 200**: `#E2E8F0` - Borders, dividers
- **Neutral 50**: `#F8FAFC` - Light backgrounds, card surfaces
- **White**: `#FFFFFF` - Primary background
- **Success**: `#16A34A` - Verified, passed quality checks
- **Danger**: `#DC2626` - Alerts, failed checks

## Typography
- **Font Family**: Inter (Google Fonts)
- **H1**: `text-4xl md:text-5xl lg:text-6xl font-bold` (tracking-tight)
- **H2**: `text-3xl md:text-4xl font-bold` (tracking-tight)
- **H3**: `text-xl md:text-2xl font-semibold`
- **H4**: `text-lg md:text-xl font-semibold`
- **Body**: `text-base md:text-lg leading-relaxed`
- **Small/Caption**: `text-sm leading-normal`

## Spacing
- **Section padding**: `py-16 md:py-24`
- **Container**: `max-w-7xl mx-auto px-4 sm:px-6 lg:px-8`
- **Card padding**: `p-6 md:p-8`
- **Grid gap**: `gap-6 md:gap-8`

## Components
- **Buttons**: Rounded-md, font-semibold, transition-all
  - Primary: `bg-secondary text-neutral-900 hover:bg-secondary-light`
  - Secondary: `bg-primary text-white hover:bg-primary-light`
  - Outline: `border-2 border-primary text-primary hover:bg-primary hover:text-white`
- **Cards**: White background, subtle shadow, rounded-xl, hover lift effect
- **Sections**: Alternating white/neutral-50 backgrounds
- **Badge**: Small rounded pills for categories/tags

## Visual Style
- Clean, whitespace-heavy layouts
- Subtle shadows and borders for depth
- Professional stock imagery (factories, containers, inspections, shipping)
- Icons from Lucide React
- Grid-based layouts with clear hierarchy
- Trust badges and verification indicators prominently displayed

## Do's
- Use whitespace generously for breathing room
- Keep text readable with high contrast
- Use consistent section headings with decorative underlines
- Include trust signals (years of experience, number of suppliers, countries served)
- Use professional icons from Lucide

## Don'ts
- Don't use excessive colors or gradients
- Don't use decorative fonts
- Don't use exaggerated claims or marketing language
- Don't crowd sections with too much information
- Don't use low contrast text combinations
