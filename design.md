# ARTITECT MACHINERY Design System

## Visual Style

**Overall Aesthetic**: Elegant industrial design with a focus on precision, reliability, and professionalism. The design balances sophisticated machinery aesthetics with approachable, user-friendly interfaces.

### Color Palette
- **Primary**: Deep charcoal (#1a252f) - Professional, industrial
- **Secondary**: Warm steel blue (#4a6fa5) - Trust, precision
- **Accent**: Copper bronze (#b87333) - Industrial heritage, quality
- **Background**: Off-white (#f8f7f4) - Clean, approachable
- **Text Primary**: Dark charcoal (#1a252f)
- **Text Secondary**: Warm gray (#5c6b7a)
- **Light Accent**: Soft cream (#f5f3ed)

### Typography
- **Headings**: Inter, 600-700 weight, clean and modern
- **Body**: Inter, 400 weight, highly readable
- **Scale**: 
  - H1: 3rem (48px)
  - H2: 2.25rem (36px)
  - H3: 1.5rem (24px)
  - Body: 1rem (16px)

### Spacing
- Use generous whitespace for elegance
- Consistent 8px grid system
- Section padding: 4rem vertical on desktop, 2.5rem on mobile
- Card gaps: 1.5rem

### Borders & Shadows
- Subtle borders: 1px solid rgba(26, 37, 47, 0.1)
- Soft shadows: 0 4px 6px -1px rgba(0, 0, 0, 0.1)
- Rounded corners: 8px for cards, 4px for buttons

### Buttons
- Primary: Deep charcoal background, white text, hover to steel blue
- Secondary: Transparent with charcoal border, hover fills with charcoal
- Padding: 0.75rem 1.75rem
- Transition: 200ms ease

### Cards
- White/off-white background
- Subtle shadow for depth
- Clean borders
- Generous internal padding (1.5rem)

## Layout Principles

**Desktop First**: Design for desktop viewing with responsive breakpoints
- Desktop: Full-width layouts with generous spacing
- Tablet (768px): Two-column grids where appropriate
- Mobile (480px): Single column, stacked elements

**Navigation**: Fixed header with logo left, nav center, CTA right
**Hero**: Full viewport height with centered content
**Products**: Grid layout with 3 columns desktop, 2 tablet, 1 mobile
**Footer**: Dark background with organized link sections

## User Experience Guidelines

- Smooth scroll behavior for anchor links
- Clear visual hierarchy
- Product cards with hover lift effect
- Contact form with clear validation states
- Loading states for form submissions
- Accessible focus states on all interactive elements

## Do's
- Use ample whitespace between sections
- Maintain consistent icon sizing (24px)
- Ensure text contrast meets accessibility standards
- Use subtle animations (200-300ms)
- Keep CTAs prominent but not overwhelming

## Don'ts
- Avoid overly bright or saturated colors
- Don't use decorative elements that distract from content
- Avoid cramped layouts
- Don't use more than 2 font weights per section
- Avoid complex shadows or gradients
