# Velmora Fine Jewelry - Design System

## Visual Identity

### Mood
- Quiet luxury, warm, editorial
- Premium demi-fine jewelry aesthetic
- NOT loud, NOT discount-looking, NOT generic e-commerce

### Color Palette
- **Primary Background**: `#0D0D0D` (rich black)
- **Secondary Background**: `#1A1A1A` (soft black)
- **Tertiary Background**: `#262626` (card backgrounds)
- **Primary Text**: `#F5F5F0` (warm white)
- **Secondary Text**: `#A3A3A3` (muted gray)
- **Accent Gold**: `#C9A962` (warm gold)
- **Accent Gold Hover**: `#D4B978` (lighter gold)
- **Border Color**: `#333333` (subtle borders)
- **Success**: `#4A7C59` (muted green)
- **Error**: `#9B4A4A` (muted red)

### Typography
- **Headings**: Cormorant Garamond (serif)
  - H1: 56px / 64px line-height / font-weight 500
  - H2: 40px / 48px line-height / font-weight 500
  - H3: 28px / 36px line-height / font-weight 500
  - H4: 20px / 28px line-height / font-weight 500
  
- **Body**: Manrope (sans-serif)
  - Large: 18px / 28px line-height
  - Regular: 16px / 24px line-height
  - Small: 14px / 20px line-height
  - Caption: 12px / 16px line-height

- **Product Names**: UPPERCASE with letter-spacing: 0.15em

### Spacing System
- Base unit: 4px
- Spacing scale: 4, 8, 12, 16, 24, 32, 48, 64, 96, 128

### Visual Effects
- **Shadows**: 
  - Card: `0 4px 20px rgba(0,0,0,0.3)`
  - Elevated: `0 8px 40px rgba(0,0,0,0.4)`
- **Border Radius**: 0px (sharp edges for editorial feel) or 2px (subtle)
- **Dividers**: 1px solid #333333 (hairline dividers)
- **Transitions**: 300ms ease-out (subtle, premium feel)

## Components

### Buttons
- **Primary**: Gold background (#C9A962), black text, full-width or auto
- **Secondary**: Transparent with gold border, gold text
- **States**: Hover (lighter gold), Active (darker), Disabled (50% opacity)

### Cards
- Dark background (#1A1A1A)
- Subtle border on hover
- Image with second image on hover reveal

### Inputs
- Dark background (#262626)
- Gold border on focus
- Placeholder in muted gray

### Badges/Tags
- Small, uppercase, letter-spacing
- Gold accent for highlights

## Layout

### Container
- Max-width: 1400px
- Padding: 24px (mobile), 48px (desktop)

### Grid
- Mobile: 1-2 columns
- Tablet: 3 columns
- Desktop: 4-5 columns
- Gap: 24px

## Responsive Breakpoints
- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px

## Do's and Don'ts
- DO: Use generous whitespace, large imagery, thin dividers
- DO: Keep accent color restrained, use for CTAs only
- DO: Use serif fonts for headings, sans for body
- DON'T: Use bright/discount colors
- DON'T: Clutter the interface
- DON'T: Use rounded corners (keep sharp for editorial feel)
