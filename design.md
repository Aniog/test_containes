# ARTITECT MACHINERY - Website Design Specification

## Brand Identity
- **Brand Name**: ARTITECT MACHINERY
- **Industry**: Industrial metalworking machinery - specifically sheet metal folding machines
- **Products**: Double folding machine, double folder, sheet metal folder, metal folder machine
- **Core Values**: Precision engineering, reliability, industrial strength with refined aesthetics

## Design Philosophy
Elegant but user-friendly - sophisticated industrial design with clean lines, professional typography, and intuitive navigation. The design should feel premium and trustworthy while remaining approachable for B2B buyers.

## Color Palette
- **Primary**: #1a1a2e (Deep navy/charcoal - authority, professionalism)
- **Secondary**: #16213e (Darker blue - depth, stability)
- **Accent**: #e94560 (Coral red - energy, action, CTAs)
- **Background Light**: #f8f9fa (Off-white - clean, spacious)
- **Background Dark**: #0f0f1a (Near black - premium sections)
- **Text Primary**: #1a1a2e (On light) / #ffffff (On dark)
- **Text Secondary**: #6c757d (Muted gray)
- **Metal/Industrial**: #c0c0c0 to #808080 gradient for machinery accents

## Typography
- **Headings**: "Inter" - clean, modern, industrial feel (700 weight)
- **Body**: "Inter" - highly readable (400, 500 weights)
- **Accent Text**: "Space Grotesk" for brand name and special elements

## Layout Principles
- Full-width hero with dramatic imagery
- Generous whitespace for elegant breathing room
- Grid-based product cards with hover effects
- Smooth scroll navigation
- Responsive from mobile to desktop

## Section Structure

### 1. Navigation
- Fixed top nav, transparent over hero, solid on scroll
- Logo left, nav links center, CTA button right
- Smooth scroll to sections on click

### 2. Hero Section
- Full viewport height
- Large headline with animated text reveal
- Subheading with value proposition
- Primary CTA button
- Subtle geometric background pattern
- Scroll indicator at bottom

### 3. Products Section
- Section title with accent underline
- 4 product categories in card grid
- Each card: image, title, brief description
- Hover effect: scale up, shadow increase, accent border
- Clear call-to-action on each card

### 4. Features Section
- Dark background for contrast
- 3-4 key differentiators
- Icon + heading + description format
- Grid layout (2x2 on desktop, stacked mobile)

### 5. About Section
- Company story and values
- Split layout: text left, image right
- Statistics/achievements with counters

### 6. Contact Section
- Clean form: name, email, message, product interest
- Company contact information sidebar
- Map placeholder or location info

### 7. Footer
- Brand logo and tagline
- Quick links
- Social links
- Copyright

## Interactions & Animations
- Smooth scroll behavior
- Fade-in animations on scroll (Intersection Observer)
- Button hover: scale + color shift
- Card hover: lift + shadow
- Form field focus states
- Loading states for form submission

## Responsive Breakpoints
- Mobile: < 640px (single column)
- Tablet: 640px - 1024px (2 columns)
- Desktop: > 1024px (full layout)
