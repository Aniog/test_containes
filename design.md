# Velmora Fine Jewelry - Design System

## Visual Identity

### Mood
- **Quiet luxury, warm, editorial**
- Premium demi-fine jewelry
- NOT loud, NOT discount-looking, NOT generic e-commerce

### Color Palette

**Primary Colors:**
- `velmora-cream` - #FAF7F2 (warm off-white background)
- `velmora-charcoal` - #1A1A1A (deep refined base for text)
- `velmora-gold` - #C9A962 (warm metallic accent)
- `velmora-gold-light` - #E8D5A8 (lighter gold for hover states)
- `velmora-gold-dark` - #9A7B3F (darker gold for emphasis)

**Neutral Colors:**
- `velmora-warm-gray` - #6B6560 (secondary text)
- `velmora-light-gray` - #F0EDE8 (subtle backgrounds, dividers)
- `velmora-white` - #FFFFFF (cards, modals)

**Accent:**
- `velmora-gold-accent` - #C9A962 (buttons, links, highlights)

### Typography

**Headings & Product Names:**
- Font: **Cormorant Garamond** (elegant serif)
- Weight: 500, 600
- Product names: UPPERCASE with letter-spacing: 0.15em

**Body & UI:**
- Font: **Inter** (clean sans-serif)
- Weight: 400, 500, 600

**Font Sizes:**
- Hero headline: 56px (desktop), 36px (mobile)
- Section titles: 40px (desktop), 28px (mobile)
- Product names: 14px uppercase
- Body: 16px
- Small/UI: 14px
- Caption: 12px

### Spacing System
- Base unit: 4px
- Section padding: 80px vertical (desktop), 48px (mobile)
- Card padding: 24px
- Grid gap: 24px (desktop), 16px (mobile)

### Visual Effects
- Thin hairline dividers: 1px solid #F0EDE8
- Subtle shadows: 0 4px 20px rgba(0,0,0,0.06)
- Hover transitions: 300ms ease
- Button hover: slight lift + shadow

### Components

**Buttons:**
- Primary: solid gold background, white text
- Secondary: outlined gold border, gold text
- Border radius: 0 (sharp, editorial) or 2px
- Padding: 16px 32px

**Cards:**
- White background
- Subtle shadow on hover
- Sharp corners or minimal radius (2px)

**Inputs:**
- Border: 1px solid #F0EDE8
- Focus: gold border
- Padding: 12px 16px

### Responsive Breakpoints
- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px
- Container max-width: 1280px

## Do's and Don'ts

**DO:**
- Use generous whitespace
- Keep imagery large and editorial
- Use thin dividers
- Maintain gold accent consistently
- Use uppercase with wide spacing for product names

**DON'T:**
- Use bright/neon colors
- Use rounded corners > 4px
- Use generic stock photos
- Clutter with too many elements
- Use discount-style callouts
