# Velmora Fine Jewelry — Design System

## Color Palette
- Base Background: `#faf7f2` (warm ivory)
- Surface / Cards: `#ffffff` (white)
- Text Primary: `#1a1a1a` (near-black)
- Text Secondary: `#6b6359` (warm taupe)
- Accent / Gold: `#c9a96e` (warm gold)
- Accent Hover: `#b8944f` (deeper gold)
- Border / Hairline: `#e8e3da` (warm grey)
- Hero Overlay: `#1a1a1a` at 30% opacity
- Newsletter BG: `#2c2825` (deep charcoal)
- Footer BG: `#1a1a1a` (black)

## Typography
- Headings: 'Playfair Display', Georgia, serif
- Body: 'Inter', -apple-system, sans-serif
- Product names: uppercase, tracking-wide (tracking-[0.15em])
- Nav links: uppercase, tracking-wider, 11px, Inter, font-medium

## Spacing & Layout
- Section padding: py-20 md:py-28
- Max content width: max-w-7xl mx-auto px-6
- Section dividers: thin 1px border-t border-[#e8e3da]
- Card gaps: gap-6 md:gap-8

## Visual Style
- Buttons: solid accent (bg-[#c9a96e] text-white) or outlined (border-[#c9a96e] text-[#c9a96e])
- Button hover: bg-[#b8944f] or bg-[#c9a96e]/10
- Button radius: none (0px) — sharp/square for editorial feel
- Image radius: none
- Box shadows: none — rely on contrast, spacing, and typography
- Hover transitions: duration-300 ease-out
- Thin hairline dividers between sections
- Product cards: clean white bg with subtle lift on hover (translate-y-[-2px])
- Nav: transparent over hero, bg-white shadow-sm on scroll

## Do's
- Use generous whitespace
- Keep text high-contrast (dark text on light backgrounds, light text on dark)
- Use uppercase with wide letter-spacing for product names
- Use thin (1px) dividers
- Use warm gold accents sparingly

## Don'ts
- Don't use loud/bright colors
- Don't use discount-style badges or tags
- Don't use heavy borders or large radiuses
- Don't pile elements close together — let them breathe
