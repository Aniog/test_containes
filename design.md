# ARTITECT MACHINERY - Design Guidelines

## Typography
- Primary font: Inter (Google Fonts)
- Headings: font-weight 700-800, uppercase for main titles
- Body text: font-weight 400-500
- Use tracking-wide for headings, tracking-normal for body

## Colors
- Primary: slate-900 (near black) for text and dark backgrounds
- Secondary: slate-700 for secondary text
- Accent: amber-600 for CTAs and highlights
- Background: white for main content, slate-50 for alternating sections
- Borders: slate-200 for subtle dividers
- Hover states: amber-700 for accent elements

## Layout
- Max container width: max-w-7xl
- Section padding: py-16 md:py-24
- Grid gaps: gap-6 md:gap-8
- Card padding: p-6 md:p-8

## Spacing
- Use Tailwind spacing scale consistently
- Section margins: mb-12 md:mb-16
- Element gaps: gap-4 md:gap-6

## Visual Style
- Clean, industrial elegance
- Sharp corners (no rounded-xl or rounded-2xl)
- Subtle borders and shadows
- High contrast for readability
- Professional photography placeholders
- Minimal decorative elements

## Components
- Buttons: uppercase text, tracking-wider, px-6 py-3
- Cards: border border-slate-200, hover:shadow-lg transition
- Navigation: sticky top-0, backdrop-blur on scroll
- Footer: dark background (slate-900), light text

## Do's
- Use high contrast text on all backgrounds
- Keep navigation simple and clear
- Use consistent spacing throughout
- Maintain industrial/professional aesthetic
- Ensure all CTAs are clearly visible

## Don'ts
- Don't use bright/neon colors
- Don't use rounded corners excessively
- Don't use decorative fonts
- Don't clutter the layout
- Don't use low contrast text
