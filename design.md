# MicroCosmos Design System

## Theme
Dark, immersive scientific aesthetic inspired by microscopy and the hidden world of microorganisms. Deep blacks and dark navy backgrounds contrast with vivid teal, cyan, and emerald accent colors to evoke bioluminescence and scientific discovery.

## Colors

### Backgrounds
- Page background: `bg-gray-950` (#030712)
- Section alternate: `bg-gray-900` (#111827)
- Card background: `bg-gray-800/60` with backdrop blur
- Overlay: `bg-black/50` or `bg-black/60`

### Accent Colors
- Primary teal: `text-teal-400` / `bg-teal-400` (#2dd4bf)
- Cyan highlight: `text-cyan-300` / `bg-cyan-300` (#67e8f9)
- Emerald accent: `text-emerald-400` (#34d399)
- Amber warm: `text-amber-400` (#fbbf24)

### Text
- Headings: `text-white` on dark backgrounds
- Body: `text-gray-300`
- Muted: `text-gray-400`
- Accent labels: `text-teal-400`

## Typography

### Fonts
- Heading font: "Playfair Display" (serif, elegant scientific feel) — loaded via Google Fonts
- Body font: "Inter" (clean, readable)

### Scale
- Hero title: `text-6xl md:text-8xl font-bold` (Playfair Display)
- Section title: `text-4xl md:text-5xl font-bold`
- Card title: `text-xl font-semibold`
- Body: `text-base` or `text-lg`
- Label/caption: `text-sm text-teal-400 uppercase tracking-widest`

## Spacing
- Section padding: `py-20 md:py-28 px-6`
- Container max width: `max-w-7xl mx-auto`
- Card gap: `gap-6` or `gap-8`
- Section gap between elements: `mb-4`, `mb-6`, `mb-12`

## Borders & Shadows
- Card border: `border border-gray-700/50`
- Glow effect: `shadow-lg shadow-teal-500/20`
- Rounded corners: `rounded-2xl` for cards, `rounded-full` for badges
- Divider: `border-t border-gray-800`

## Images
- Hero: full-screen background, 16x9 ratio, 1600px wide
- Gallery grid: 4x3 ratio, 600px wide
- Organism cards: 3x2 ratio, 500px wide
- World cards: 1x1 ratio, 400px wide

## Do's
- Use dark overlays on background images for text legibility
- Use teal/cyan for interactive elements and highlights
- Use generous whitespace between sections
- Use uppercase tracking-widest for section labels
- Animate subtle hover effects on cards (scale, glow)

## Don'ts
- Never use white backgrounds
- Never use light text on light backgrounds
- Don't use more than 3 accent colors per section
- Don't use tight spacing on mobile
