# Velmora Fine Jewelry - Design System

## Brand Overview
- **Brand Name**: Velmora Fine Jewelry
- **Type**: Direct-to-consumer (B2C) demi-fine jewelry e-commerce
- **Target Audience**: Women 25-45, gifting and self-purchase
- **Price Point**: Premium-but-accessible ($30–$120)
- **Product Categories**: Earrings, Necklaces, Huggies

## Visual Identity

### Mood
- Quiet luxury, warm, editorial
- Premium demi-fine jewelry
- NOT loud, NOT discount-looking, NOT generic e-commerce

### Color Palette
**Primary Colors**:
- `velmora-cream`: #FAF7F2 (warm cream background)
- `velmora-charcoal`: #1A1A1A (rich dark for text)
- `velmora-gold`: #C9A962 (warm metallic gold accent)
- `velmora-gold-light`: #E8D5A8 (lighter gold for hover states)

**Secondary Colors**:
- `velmora-sand`: #F0EBE3 (subtle backgrounds)
- `velmora-muted`: #8B8680 (secondary text)
- `velmora-white`: #FFFFFF (cards, modals)

**Accent**:
- `velmora-gold` as primary accent for CTAs and highlights

### Typography
**Headings & Product Names**:
- Font: Cormorant Garamond (elegant serif)
- Weight: 500-600
- Product names: UPPERCASE with letter-spacing: 0.15em

**Body & UI**:
- Font: Manrope (clean sans-serif)
- Weight: 400-500
- Base size: 16px
- Line height: 1.6

### Spacing System
- Base unit: 4px
- Section padding: 80px vertical (desktop), 48px (mobile)
- Card padding: 24px
- Grid gap: 24px (desktop), 16px (mobile)

### Visual Effects
- Hairline dividers: 1px solid rgba(26, 26, 26, 0.1)
- Card shadows: 0 4px 20px rgba(0, 0, 0, 0.06)
- Hover transitions: 300ms ease
- Button hover: subtle lift + shadow

## Component Guidelines

### Buttons
- Primary: Solid velmora-gold background, velmora-charcoal text
- Secondary: Outlined with velmora-gold border
- Padding: 14px 32px
- Border radius: 0 (sharp, editorial) or 2px
- Font: Manrope, uppercase, letter-spacing: 0.1em

### Cards
- Background: velmora-white
- Border: none
- Shadow: subtle, soft
- Hover: reveal second image + quick add to cart

### Forms
- Input border: 1px solid rgba(26, 26, 26, 0.2)
- Focus: velmora-gold border
- Placeholder: velmora-muted

## Responsive Breakpoints
- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px

## Do's and Don'ts
- DO: Use generous whitespace
- DO: Large editorial imagery
- DO: Thin hairline dividers
- DO: Restrained accent color
- DO: Subtle hover transitions
- DO: Soft shadows
- DON'T: Use bright/discount colors
- DON'T: Clutter with too many elements
- DON'T: Use generic e-commerce patterns
- DON'T: Overuse gold - keep it as accent only
