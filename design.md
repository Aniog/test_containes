# ARTITECT MACHINERY Design System

## Typography
- **Font Family**: 'Inter', sans-serif (Google Fonts)
- **Headings**: Semi-bold to Bold (600-700), tracking tight
- **Body**: Regular (400), line-height 1.6
- **Logo**: Uppercase, tracked out slightly for an architectural feel

## Color Palette
- **Primary**: Midnight Blue/Slate (`#1e293b`) - represents reliability and machinery
- **Accent**: Metallic Gold/Copper (`#b45309`) - represents quality and precision
- **Background**: Soft Gray (`#f8fafc`) or Clean White (`#ffffff`)
- **Text**: Deep Gray (`#334155`)
- **Muted**: Light Gray (`#94a3b8`)

## Components Styles
- **Buttons**: Rounded-md, smooth transitions, subtle shadows
- **Cards**: Minimal border, slight elevation, clean spacing
- **Sections**: Max-width 1200px, ample whitespace (padding 64px - 80px)

## Visual Style: Elegant & User Friendly
- Use high-quality imagery of metal folding machines
- Clean typography with professional spacing
- Interaction feedback using subtle hover states
- Responsive design from mobile up
