# ARTITECT MACHINERY - Design Guidelines

## Brand Overview
- **Brand Name**: ARTITECT MACHINERY
- **Industry**: Industrial machinery manufacturing (sheet metal folding machines)
- **Tone**: Elegant but user-friendly, professional, trustworthy

## Color Palette
- **Primary**: `#1a1a2e` (Deep navy/charcoal - sophistication)
- **Secondary**: `#16213e` (Dark blue - industrial reliability)
- **Accent**: `#e94560` (Coral red - energy, action)
- **Accent Secondary**: `#0f3460` (Medium blue - depth)
- **Light**: `#f8f9fa` (Off-white - clean, professional)
- **Text Primary**: `#1a1a2e` (Dark for readability)
- **Text Secondary**: `#6c757d` (Muted gray)
- **Text Light**: `#ffffff` (White for dark backgrounds)
- **Border**: `#e9ecef` (Light gray)

## Typography
- **Primary Font**: "Outfit" (Google Fonts) - Modern, geometric, professional
- **Secondary Font**: "DM Sans" (Google Fonts) - Clean, readable body text
- **Headings**: Outfit, weights 600-700
- **Body**: DM Sans, weight 400-500
- **Logo/Brand**: Outfit, weight 700, uppercase, letter-spacing 0.15em

## Visual Style
- Clean, minimalist design with generous whitespace
- Subtle shadows for depth: `0 4px 20px rgba(0,0,0,0.08)`
- Rounded corners: 8px for cards, 4px for buttons
- Smooth transitions: 0.3s ease for hover effects
- Professional imagery with industrial aesthetic

## Layout
- Max content width: 1200px
- Section padding: 80px vertical (desktop), 48px (mobile)
- Grid gap: 32px
- Card padding: 24px

## Components

### Header
- Fixed position, white background with subtle shadow on scroll
- Logo on left, navigation centered, CTA button on right
- Mobile: hamburger menu

### Hero Section
- Full-width, dark gradient background
- Large headline with accent color highlight
- Subheadline in lighter color
- Two CTA buttons: primary (accent) and secondary (outline)
- Background: subtle geometric pattern or industrial image

### Product Cards
- White background with subtle shadow
- Image at top (aspect ratio 4:3)
- Product name in bold
- Brief description
- "Learn More" link with arrow
- Hover: lift effect with enhanced shadow

### Features Section
- Icon + title + description layout
- 3-4 columns on desktop, 1-2 on mobile
- Accent color for icons

### Contact Section
- Dark background for contrast
- Contact form with name, email, message
- Company info sidebar

### Footer
- Dark background matching brand
- Logo, navigation links, social icons
- Copyright notice

## Do's and Don'ts
- DO: Use ample whitespace, clean typography, consistent spacing
- DO: Use accent color sparingly for CTAs and highlights
- DON'T: Use too many colors (stick to palette)
- DON'T: Clutter the design with too many elements
- DON'T: Use generic stock photos - prefer clean, industrial imagery
