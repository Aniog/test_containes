# Velmora Fine Jewelry — Design System

## Brand Voice
Quiet luxury. Warm, editorial, premium demi-fine jewelry. NOT loud, NOT discount, NOT generic.

## Color Palette
- **Charcoal (base):** `#1a1714` — primary text, dark backgrounds
- **Ivory (light bg):** `#faf7f2` — page background, light sections
- **Gold (accent):** `#c9a96e` — CTAs, accents, highlights
- **Gold Dark:** `#b8923f` — hover states, pressed buttons
- **Cream:** `#f5f0e8` — card backgrounds, subtle surfaces
- **Rose Gold:** `#d4a574` — secondary accent
- **Warm Gray:** `#8a8279` — muted text, secondary labels
- **Divider:** `#e8e2d8` — hairline dividers, borders

## Typography
- **Headings / Brand:** Cormorant Garamond (serif, elegant) — weights 400, 500, 600, 700
- **Body / UI:** Inter (sans-serif, clean) — weights 300, 400, 500, 600
- **Product names:** UPPERCASE, tracking-wide (letter-spacing: 0.15em)
- **Heading sizes:** h1: 4rem, h2: 2.5rem, h3: 1.5rem (responsive)

## Spacing
- Section padding: 5rem vertical (responsive: 3rem mobile)
- Container max-width: 1280px, centered with horizontal padding
- Generous whitespace between elements

## Components
- **Buttons:** Solid gold accent fill with charcoal text, or outlined gold border with gold text. Rounded-sm, uppercase tracking-wide text, font-weight 500.
- **Cards:** Cream background (#f5f0e8), no harsh borders, subtle shadow on hover
- **Dividers:** 1px solid #e8e2d8 (hairline)
- **Hover transitions:** 0.3s ease, subtle lifts/shadows

## Do's
- Use generous whitespace
- Maintain warm tones throughout
- Use thin hairline dividers
- Apply soft shadows for depth
- Keep consistent letter-spacing on uppercase text

## Don'ts
- Don't use cold/blues or harsh neon colors
- Don't use heavy borders or boxy designs
- Don't use too many font weights
- Don't use discount-looking red/orange for CTAs
