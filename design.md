# Strikingly Generators Hub Design System

## Overview
This project follows a clean Strikingly-style marketing layout with strong hierarchy, light surfaces, restrained purple accents, and limited AI gradient usage.

## Typography
- Headings: Brandon Grotesque, fallback to Josefin Sans, then Poppins, sans-serif.
- Headings are uppercase, bold, and tight with a 1.2 line-height.
- Body: Open Sans, 14px base, 1.5 line-height.
- Buttons use the same heading family as headings.

## Colors
- Brand purple: `#8159BB`
- AI blue: `#7671FF`
- AI pink: `#CB0C9F`
- Hero heading dark: `#2E2E2F`
- Section heading gray: `#4B5056`
- Body gray: `#636972`
- Card border: `#C6C9CD`
- Divider: `#E2E4E7`
- Light background: `#F4F6F8`
- White: `#FFFFFF`

## Components
- Buttons: 4px radius, uppercase, medium-tight letter spacing, gradient fill only for primary actions.
- Cards: white surface, 1px border, 3px radius, 20px padding, subtle hover lift.
- Tags: ghost style with purple text and border.
- Section containers: centered, max width 1200px.

## Spacing
- Use multiples of 10px for layout spacing.
- 20px between related blocks.
- 40px between sections.
- Hero sections get 60px to 80px vertical padding.

## Do
- Keep text contrast high and explicit.
- Use semantic sections and readable card layouts.
- Keep the gradient limited to the H1 highlight and primary CTAs.
- Build responsive grids that collapse cleanly to one column on mobile.

## Do Not
- Do not use dark page backgrounds.
- Do not use gradient section backgrounds outside the hero wash.
- Do not use fake social proof.
- Do not rely on JavaScript for essential directory visibility.
