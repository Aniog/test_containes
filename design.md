# SSourcing China - Design System

## Brand Identity
Professional, trustworthy B2B sourcing agent based in China. Target audience: overseas buyers, procurement managers, e-commerce sellers.

## Color Palette
- **Primary**: `#0F172A` (Slate 900 / Navy) - headers, nav, footer, primary text
- **Primary Accent**: `#1E40AF` (Blue 800) - CTAs, links, highlights
- **Primary Accent Hover**: `#1D4ED8` (Blue 700) - button hover states
- **Secondary Accent**: `#10B981` (Emerald 500) - trust badges, success states
- **Background**: `#FFFFFF` (White) - main content background
- **Background Alt**: `#F8FAFC` (Slate 50) - alternating sections
- **Background Dark**: `#0F172A` (Slate 900) - footer, dark sections
- **Text Primary**: `#1E293B` (Slate 800) - body text
- **Text Secondary**: `#64748B` (Slate 500) - captions, meta text
- **Text Muted**: `#94A3B8` (Slate 400) - placeholders, disabled
- **Text on Dark**: `#F1F5F9` (Slate 100) - text on dark backgrounds
- **Border**: `#E2E8F0` (Slate 200) - borders, dividers
- **Border Light**: `#F1F5F9` (Slate 100) - subtle borders

## Typography
- **Font Family**: Inter, system-ui, sans-serif
- **H1**: 48px / 1.1 line-height, font-weight 700, letter-spacing -0.02em
- **H2**: 36px / 1.2 line-height, font-weight 700, letter-spacing -0.01em
- **H3**: 24px / 1.3 line-height, font-weight 600
- **H4**: 18px / 1.4 line-height, font-weight 600
- **Body Large**: 18px / 1.6 line-height, font-weight 400
- **Body**: 16px / 1.6 line-height, font-weight 400
- **Caption**: 14px / 1.5 line-height, font-weight 400
- **Small**: 12px / 1.4 line-height, font-weight 500, letter-spacing 0.02em

## Spacing Scale
- Section padding: py-20 (80px vertical)
- Section padding mobile: py-12 (48px vertical)
- Container max-width: max-w-7xl (1280px)
- Content max-width: max-w-3xl (768px) for text-heavy sections
- Grid gap: gap-8 (32px)
- Card padding: p-6 (24px)
- Component spacing: gap-4 (16px)

## Border Radius
- Cards: rounded-xl (12px)
- Buttons: rounded-lg (8px)
- Small elements: rounded-md (6px)
- Pills/tags: rounded-full

## Shadows
- Card shadow: shadow-sm (subtle)
- Card hover: shadow-md
- Elevated: shadow-lg

## Button Styles
- **Primary**: bg-blue-800 text-white px-6 py-3 rounded-lg font-medium hover:bg-blue-700 transition
- **Secondary**: bg-white text-slate-900 border border-slate-200 px-6 py-3 rounded-lg font-medium hover:bg-slate-50 transition
- **Ghost on dark**: text-white border border-white/30 px-6 py-3 rounded-lg hover:bg-white/10 transition

## Component Patterns
- Cards: white bg, rounded-xl, p-6, border border-slate-100, hover:shadow-md
- Section headers: centered, max-w-2xl, mb-12
- Icon circles: 48px or 56px, rounded-full, bg-blue-50 or bg-emerald-50
- Stats: large number + small label, flex row or grid
- FAQ: accordion style with plus/minus toggle
- Forms: rounded-lg inputs, border-slate-200, focus:ring-2 focus:ring-blue-500

## Layout Rules
- Single column for mobile, 2-col for tablet, 3-4 col for desktop
- Hero: full-width, dark bg with subtle gradient or image overlay
- Navigation: sticky top-0, white bg with shadow-sm, z-50
- Footer: dark bg, multi-column links, contact info at bottom
- Max content width always centered with mx-auto px-4 sm:px-6 lg:px-8

## Do's and Don'ts
- DO use plenty of whitespace
- DO use the blue accent sparingly for CTAs only
- DO use realistic factory/supply chain imagery
- DO keep text concise and scannable
- DON'T use flashy animations or gradients
- DON'T use more than 2 accent colors
- DON'T crowd elements - B2B buyers need clarity
- DON'T use casual or slang language
