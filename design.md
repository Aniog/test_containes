# Velmora Fine Jewelry — Design System

## Color Palette
- `velmora-black`: `#1A1A1A` — Headings, primary text
- `velmora-charcoal`: `#2D2D2D` — Body text
- `velmora-cream`: `#FAF7F2` — Page background
- `velmora-pearl`: `#F5F0EB` — Card/section backgrounds
- `velmora-sand`: `#E8E2D9` — Borders, dividers
- `velmora-stone`: `#8A8580` — Muted/secondary text
- `velmora-gold`: `#C9A96E` — Primary accent, buttons, links
- `velmora-gold-muted`: `#B8956A` — Hover state for gold
- `velmora-gold-light`: `#D4B87C` — Light gold accent

## Typography
- **Headings**: Cormorant Garamond (serif), weight 400, uppercase with 0.15em tracking for product names
- **Body**: Inter (sans-serif), weight 400–500
- **Product names**: Uppercase, letter-spacing 0.15em

## Spacing
- Generous whitespace, section padding 48px mobile / 80px desktop
- Container max-width 1280px, padding 24px mobile / 48px desktop

## Components
- Buttons: Solid gold `bg-velmora-gold` or outlined `border-velmora-gold`
- Dividers: 1px `bg-velmora-sand`
- Cards: White/pearl background, subtle shadow on hover
- Transitions: 250ms ease for hover effects

## Do's
- Use warm, editorial imagery with gold tones
- Maintain generous whitespace
- Use thin hairline dividers between sections
- Product names always UPPERCASE with wide letter-spacing

## Don'ts
- No loud/discount-looking elements
- No bright accent colors other than gold
- No dense layouts
- No generic e-commerce styling
