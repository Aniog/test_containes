# ARTITECT MACHINERY - Design System

## Brand Identity
ARTITECT MACHINERY specializes in precision sheet metal folding equipment. The brand conveys industrial strength, precision engineering, and modern elegance.

## Color Palette

### Primary Colors
- **Primary (Navy)**: `#1e3a5f` - Deep navy blue representing trust, stability, and industrial precision
- **Primary Light**: `#2d5a8e` - Lighter navy for hover states and secondary elements
- **Primary Dark**: `#0f1f33` - Dark navy for headers and strong contrast

### Accent Colors
- **Accent (Copper)**: `#c87941` - Warm copper/bronze representing metalwork and craftsmanship
- **Accent Light**: `#e8956a` - Lighter copper for highlights
- **Accent Dark**: `#a05d2e` - Darker copper for depth

### Neutral Colors
- **White**: `#ffffff` - Clean backgrounds
- **Light Gray**: `#f5f7fa` - Section backgrounds
- **Medium Gray**: `#6b7280` - Secondary text
- **Dark Gray**: `#374151` - Body text
- **Charcoal**: `#1f2937` - Headings

### Status Colors
- **Success**: `#059669`
- **Warning**: `#d97706`
- **Error**: `#dc2626`

## Typography

### Font Family
- **Primary**: Inter (Google Fonts) - Clean, modern, highly legible
- **Fallback**: system-ui, -apple-system, sans-serif

### Font Weights
- **Light**: 300 - Subtle descriptions
- **Regular**: 400 - Body text
- **Medium**: 500 - Navigation, labels
- **SemiBold**: 600 - Subheadings, buttons
- **Bold**: 700 - Headings, emphasis
- **ExtraBold**: 800 - Hero headings, brand name

### Font Sizes
- **Hero**: 3.5rem - 4.5rem (56px - 72px)
- **H1**: 2.5rem (40px)
- **H2**: 2rem (32px)
- **H3**: 1.5rem (24px)
- **H4**: 1.25rem (20px)
- **Body Large**: 1.125rem (18px)
- **Body**: 1rem (16px)
- **Small**: 0.875rem (14px)
- **Caption**: 0.75rem (12px)

## Spacing System
Based on 4px grid:
- **xs**: 0.25rem (4px)
- **sm**: 0.5rem (8px)
- **md**: 1rem (16px)
- **lg**: 1.5rem (24px)
- **xl**: 2rem (32px)
- **2xl**: 3rem (48px)
- **3xl**: 4rem (64px)
- **4xl**: 6rem (96px)

## Layout
- **Container Max Width**: 1280px
- **Section Padding**: 4rem - 6rem vertical
- **Grid Gap**: 1.5rem - 2rem
- **Card Padding**: 1.5rem - 2rem

## Components

### Buttons
- **Primary**: Navy background, white text, rounded-lg, padding 0.75rem 1.5rem
- **Secondary**: Transparent with copper border, copper text
- **Hover**: Slight scale transform, shadow increase

### Cards
- **Background**: White
- **Border**: Subtle gray border or shadow
- **Shadow**: `0 4px 6px -1px rgba(0, 0, 0, 0.1)`
- **Hover**: Shadow increase, slight lift effect
- **Border Radius**: 0.5rem (8px)

### Navigation
- **Height**: 4rem (64px)
- **Background**: White with subtle shadow
- **Active Link**: Copper underline or background

## Do's
- Use plenty of white space for breathing room
- Maintain high contrast for readability
- Use copper accents sparingly for emphasis
- Keep layouts clean and uncluttered
- Use consistent spacing throughout

## Don'ts
- Don't use more than 3 colors in one section
- Don't overcrowd content
- Don't use decorative fonts
- Don't ignore mobile responsiveness
- Don't use low-contrast text
