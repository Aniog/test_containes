# ARTITECT MACHINERY - Design System

## Brand Identity
Elegant industrial machinery brand. Premium sheet metal folding machines. Clean, sophisticated, trustworthy.

## Typography
- Headings: "Playfair Display", serif — elegant, refined
- Body: "Inter", sans-serif — clean, readable
- Do: Use Playfair Display for h1, h2, h3
- Don't: Use Playfair Display for body text or UI labels

## Colors
- Primary (Navy): #1a1f2e — headers, footer, dark sections
- Accent (Gold/Brass): #c8a45c — buttons, highlights, links
- Accent Light: #e8d5a3 — subtle highlights, hover states
- Background (Warm White): #faf8f5 — main background
- Surface (White): #ffffff — cards, sections
- Text Primary: #1a1f2e — main text
- Text Secondary: #5a5f6e — muted text
- Border: #e5e0d8 — subtle borders

## Spacing
- Section padding: py-20 md:py-28
- Container max-width: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8
- Card padding: p-6 md:p-8

## Visual Style
- Clean, generous whitespace
- Subtle shadows on cards (shadow-sm, shadow-md)
- Rounded corners: rounded-lg for cards, rounded-md for buttons
- Thin gold accent lines/dividers
- Do: Use gold sparingly as accent
- Don't: Overuse gold — it should feel premium, not gaudy
- Do: Keep layouts airy with breathing room
- Don't: Crowd elements together

## Buttons
- Primary: bg-[#c8a45c] text-white hover:bg-[#b8944c] rounded-md px-6 py-3
- Secondary: border border-[#c8a45c] text-[#c8a45c] hover:bg-[#c8a45c] hover:text-white rounded-md px-6 py-3
