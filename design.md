# MicroCosmos Design System

## Concept
A visually immersive, science-meets-wonder aesthetic. Dark backgrounds with vivid bioluminescent accent colors evoke the feeling of peering through a microscope into an alien world.

## Color Palette
- Background (deep): `bg-slate-950` (#020617)
- Background (surface): `bg-slate-900` (#0f172a)
- Background (card): `bg-slate-800/60` (semi-transparent)
- Primary accent: `text-cyan-400` / `bg-cyan-400` (#22d3ee)
- Secondary accent: `text-violet-400` / `bg-violet-400` (#a78bfa)
- Tertiary accent: `text-emerald-400` / `bg-emerald-400` (#34d399)
- Warm accent: `text-amber-400` / `bg-amber-400` (#fbbf24)
- Text primary: `text-white`
- Text secondary: `text-slate-300`
- Text muted: `text-slate-400`
- Border: `border-slate-700`

## Typography
- Font: Inter (Google Fonts)
- Hero heading: `text-5xl md:text-7xl font-black tracking-tight`
- Section heading: `text-3xl md:text-4xl font-bold`
- Subheading: `text-xl font-semibold`
- Body: `text-base text-slate-300 leading-relaxed`
- Caption: `text-sm text-slate-400`
- Label/tag: `text-xs font-semibold uppercase tracking-widest`

## Spacing
- Section padding: `py-20 md:py-28`
- Container: `max-w-7xl mx-auto px-4 md:px-8`
- Card padding: `p-6`
- Gap between grid items: `gap-6 md:gap-8`

## Borders & Shadows
- Card border: `border border-slate-700/50`
- Card radius: `rounded-2xl`
- Image radius: `rounded-xl`
- Glow effect (cyan): `shadow-[0_0_30px_rgba(34,211,238,0.15)]`
- Glow effect (violet): `shadow-[0_0_30px_rgba(167,139,250,0.15)]`

## Components
- Buttons: `bg-cyan-400 text-slate-950 font-bold px-6 py-3 rounded-full hover:bg-cyan-300 transition`
- Outline button: `border border-cyan-400 text-cyan-400 px-6 py-3 rounded-full hover:bg-cyan-400/10 transition`
- Tags/badges: `bg-slate-800 text-cyan-400 text-xs font-semibold px-3 py-1 rounded-full border border-slate-700`
- Section divider: subtle gradient line using `bg-gradient-to-r from-transparent via-slate-600 to-transparent`

## Image Style
- Images should be full-bleed, high-quality, and science/nature themed
- Use aspect ratios: 16x9 for hero/wide, 4x3 for cards, 1x1 for circular/square
- Overlay images with subtle dark gradient for text legibility
- Images should feel microscopic, biological, cosmic, or natural

## Do's
- Use dark backgrounds throughout
- Use glowing accent colors sparingly for highlights
- Use large, immersive images
- Use grid layouts for image galleries
- Maintain generous whitespace

## Don'ts
- No white or light backgrounds
- No low-contrast text on dark surfaces
- No cluttered layouts
- No more than 3 accent colors per section
