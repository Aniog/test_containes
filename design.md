# SSourcing China - Design System

## Brand Identity
- **Company**: SSourcing China - China-based sourcing agent for global buyers
- **Tone**: Professional, trustworthy, practical, international B2B
- **Goal**: Generate qualified sourcing inquiries

## Color Palette
- **Primary**: `#0F4C81` (deep navy blue - trust, professionalism)
- **Primary Light**: `#1A6FB5` (medium blue for hover/accents)
- **Primary Dark**: `#0A3459` (darkest blue for text on light backgrounds)
- **Accent**: `#E8910C` (warm amber/gold - CTAs, highlights)
- **Accent Light**: `#F5A623` (lighter gold for hover states)
- **Success**: `#16A34A` (green for trust badges, checkmarks)
- **Background**: `#FFFFFF` (white - clean B2B)
- **Surface**: `#F8FAFC` (light gray surface)
- **Surface Alt**: `#EFF3F8` (alternate section backgrounds)
- **Text Primary**: `#1E293B` (near-black for headings)
- **Text Secondary**: `#475569` (medium gray for body text)
- **Text Muted**: `#94A3B8` (light gray for captions)
- **Border**: `#E2E8F0` (subtle borders)
- **Border Dark**: `#CBD5E1` (more visible borders)

## Typography
- **Font Family**: Inter (Google Fonts)
- **Heading 1**: 3.5rem/4rem, weight 800, tracking -0.02em
- **Heading 2**: 2.25rem/2.5rem, weight 700, tracking -0.01em
- **Heading 3**: 1.5rem, weight 600
- **Body**: 1rem/1.125rem, weight 400, line-height 1.7
- **Small**: 0.875rem, weight 400
- **Caption**: 0.75rem, weight 500, uppercase, tracking 0.1em (for labels/badges)

## Spacing
- Section padding: 5rem (80px) vertical, responsive
- Container max-width: 1280px
- Card padding: 1.5rem - 2rem
- Grid gaps: 1.5rem - 2rem

## Buttons
- **Primary CTA**: bg accent gold, white text, weight 600, rounded-lg, px-8 py-4, hover darken
- **Secondary**: border primary blue, primary blue text, hover bg primary, white text
- **Ghost**: transparent, underline on hover

## Cards
- White background, rounded-xl, shadow-sm, border subtle
- Hover: shadow-md, slight lift transition

## Shadows
- sm: `0 1px 2px rgba(0,0,0,0.05)`
- md: `0 4px 6px rgba(0,0,0,0.07)`
- lg: `0 10px 15px rgba(0,0,0,0.1)`

## Layout Principles
- Clean whitespace, professional feel
- Alternating section backgrounds (white / surface-alt)
- Full-width sections with centered content container
- Mobile-first responsive design
- Desktop: multi-column layouts, side-by-side content
- Mobile: single column stacking

## Do's
- Use navy blue for trust signals and headers
- Use gold/amber only for primary CTAs and important highlights
- Keep backgrounds clean and light
- Use icons from Lucide for consistency
- Use stock images for factory/QC/shipping visuals
- Maintain professional spacing

## Don'ts
- Don't use bright or flashy colors
- Don't overcrowd sections
- Don't use decorative fonts
- Don't use low contrast text
- Don't use exaggerated claims or marketing fluff
