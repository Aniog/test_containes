# ARTITECT MACHINERY — Design System

## Brand Essence
- Precision engineering, industrial strength, elegant simplicity.
- Tone: confident, professional, approachable.

## Color Palette
- Background: `#0B0F17` (near-black navy)
- Surface: `#141A25` (charcoal card / section)
- Primary Accent: `#D4A853` (brushed gold — machinery highlights)
- Secondary Accent: `#60A5FA` (steel blue — technical, trustworthy)
- Text Primary: `#F4F6F9` (off-white)
- Text Secondary: `#9CA3AF` (cool gray)
- Border: `rgba(255, 255, 255, 0.08)`
- Success: `#22C55E`

## Typography
- Headings: `Inter`, font-weight 700-800, tight line-height (1.0-1.1), uppercase for small labels.
- Body: `Inter`, font-weight 400, line-height 1.6.
- Size scale: use Tailwind `text-sm`, `text-base`, `text-lg`, `text-2xl`, `text-4xl`, `text-5xl`, `text-6xl` with responsive `md:`/`lg:` prefixes.

## Spacing
- Page padding: `px-4 sm:px-6 lg:px-8 xl:px-16`
- Section vertical: `py-16 md:py-24 lg:py-32`
- Grid gaps: `gap-6 md:gap-8 lg:gap-10`
- Card padding: `p-6 md:p-8`

## Components
- Buttons:
  - Primary: gold background `#D4A853`, dark text `#0B0F17`, rounded-full, px-8 py-3, hover scale & shadow.
  - Secondary: transparent, gold border, gold text, rounded-full, hover gold/10 background.
- Cards: dark surface `#141A25`, subtle border, rounded-2xl, hover lift with shadow-xl.
- Inputs: dark surface, light border focus ring gold, rounded-xl.

## Visual Effects
- Gradients: use subtle radial gradients from gold/blue on dark backgrounds.
- Hero: large headline, supporting subhead, CTA, background image with dark overlay.
- Sections alternate between near-black and slightly lighter charcoal to create rhythm.

## Do's
- Use generous whitespace.
- Ensure text contrast is high on every background.
- Use rounded corners for a friendly feel.
- Include clear CTAs and contact hooks.
- Use the stock-image tagging system for product/factory imagery.

## Don'ts
- No harsh neon colors.
- No cluttered layouts or tiny text.
- No all-caps body copy.
- No low-contrast gray-on-gray text.
