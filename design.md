# Velmora Fine Jewelry — Design System

## Color Palette
- **Cream** `#FAF8F5` — Primary background, warm neutral base
- **Ivory** `#F5F0EA` — Secondary background, section alternation
- **Charcoal** `#1A1611` — Primary text, headings, buttons
- **Gold** `#C9A96E` — Accent, highlights, hover states
- **Gold Dark** `#A8873F` — Hover accent, active states
- **Gold Light** `#D4BA85` — Subtle accents, borders
- **Burgundy** `#722F37` — Sale badges, limited accent
- **Warm Gray** `#787878` — Secondary text, muted labels
- **Light Gray** `#E8E4DF` — Borders, dividers, subtle backgrounds

## Typography
- **Headings**: Cormorant Garamond (serif), weight 400–500
- **Body/UI**: Inter (sans-serif), weight 300–500
- **Product Names**: Uppercase, letter-spacing 0.25em
- **Nav Links**: Uppercase, letter-spacing 0.15em, font-size 11–12px

## Spacing & Layout
- Section padding: 4rem vertical, responsive horizontal (1rem mobile → 6rem desktop)
- Max content width: 1280px, centered
- Generous whitespace between sections (80–120px)
- Grid gaps: 24px mobile, 32px desktop

## Components
- Buttons: Rounded-none (sharp corners), solid fill or outlined
- Cards: Clean white background, subtle shadow on hover, no visible border
- Dividers: 1px solid #E8E4DF, used between sections and content
- Hero: Full-bleed, dark overlay, centered serif text

## Motion
- Hover transitions: 0.3s ease
- Page load: fade-in 0.6s ease-out
- Scroll reveal: slide-up 0.6s ease-out
- Cart drawer: slide-in-right 0.4s ease-out

## Rules
- No rounded corners on primary buttons (keep sharp, editorial feel)
- No bright colors — everything in warm gold/neutral family
- Always ensure WCAG AA contrast for text
- Use data-strk-img for product imagery
- Mobile-first responsive design
