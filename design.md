# SSourcing China - Design System

## Brand Identity
- Professional B2B sourcing agent website
- Clean, trustworthy, international feel
- Target audience: overseas buyers looking for China suppliers

## Color Palette
- Primary: Blue (#1E40AF - tailwind blue-800) - trust, professionalism
- Primary Light: (#3B82F6 - tailwind blue-500) - accents, links
- Secondary: Slate (#0F172A - tailwind slate-900) - text, headings
- Accent: Amber (#F59E0B - tailwind amber-500) - CTAs, highlights
- Background: White (#FFFFFF) and Light Gray (#F8FAFC - slate-50)
- Success: Green (#10B981 - emerald-500)
- Error: Red (#EF4444 - red-500)

## Typography
- Font: Inter (Google Fonts)
- Headings: font-bold, tracking-tight
- Body: font-normal, leading-relaxed
- Small text: text-sm, text-muted-foreground

## Layout
- Max content width: 1280px (max-w-7xl)
- Section padding: py-16 md:py-24
- Container padding: px-4 sm:px-6 lg:px-8
- Grid gaps: gap-6, gap-8

## Components
- Buttons: rounded-lg, px-6 py-3, font-medium
- Cards: rounded-xl, shadow-sm, border border-gray-200
- Forms: rounded-lg, border-gray-300, focus:ring-2 focus:ring-blue-500
- Navigation: sticky top-0, backdrop-blur, bg-white/90

## Visual Style
- Clean, minimal, professional
- Subtle shadows and borders
- Consistent spacing scale
- Realistic factory/QC/shipping imagery
- Trust indicators (certifications, years of experience, client count)

## Do's
- Use blue as primary brand color
- Keep layouts clean with plenty of white space
- Use professional imagery
- Show trust signals prominently
- Clear CTAs with amber accent color

## Don'ts
- Don't use bright/neon colors
- Don't clutter layouts
- Don't use exaggerated claims
- Don't use casual/informal language
- Don't use dark themes (B2B prefers light)
