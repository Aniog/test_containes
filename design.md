# ARTITECT MACHINERY — Design System

## Brand Identity
- **Brand Name:** ARTITECT MACHINERY
- **Tagline:** Precision Engineered. Perfectly Formed.
- **Industry:** Industrial sheet metal folding machinery
- **Tone:** Elegant, professional, trustworthy, user-friendly

## Color Palette

### Primary Colors
- `brand-navy`: `#0D1B2A` — Deep navy, primary background and text
- `brand-steel`: `#1C3A5E` — Steel blue, secondary backgrounds and accents
- `brand-gold`: `#C9A84C` — Warm gold, primary accent / CTA color
- `brand-gold-light`: `#E8C97A` — Light gold, hover states

### Neutral Colors
- `brand-white`: `#FFFFFF`
- `brand-offwhite`: `#F5F5F0` — Warm off-white for section backgrounds
- `brand-gray`: `#8A9BB0` — Muted text, secondary info
- `brand-light`: `#E8EDF3` — Light borders and dividers
- `brand-dark`: `#060E18` — Near-black for deep sections

## Typography

### Font Family
- **Primary:** Inter (Google Fonts) — clean, modern, readable
- **Display:** Inter with tight tracking for headings

### Scale
- Display: `text-5xl` to `text-7xl`, `font-bold`, `tracking-tight`
- H1: `text-4xl`, `font-bold`
- H2: `text-3xl`, `font-semibold`
- H3: `text-xl`, `font-semibold`
- Body: `text-base`, `font-normal`
- Small: `text-sm`, `text-brand-gray`
- Label/Tag: `text-xs`, `font-semibold`, `uppercase`, `tracking-widest`

## Spacing & Layout
- Max content width: `max-w-7xl mx-auto`
- Section padding: `py-20 px-6` (desktop), `py-12 px-4` (mobile)
- Card padding: `p-8`
- Border radius: `rounded-2xl` for cards, `rounded-full` for pills/badges

## Component Styles

### Buttons
- **Primary CTA:** `bg-brand-gold text-brand-navy font-semibold px-8 py-3 rounded-full hover:bg-brand-gold-light transition-all`
- **Secondary:** `border border-brand-gold text-brand-gold px-8 py-3 rounded-full hover:bg-brand-gold hover:text-brand-navy transition-all`
- **Ghost:** `text-brand-gray hover:text-brand-white transition-colors`

### Cards
- Background: `bg-white` or `bg-brand-offwhite`
- Border: `border border-brand-light`
- Shadow: `shadow-lg hover:shadow-xl transition-shadow`
- Radius: `rounded-2xl`

### Navigation
- Background: `bg-brand-navy/95 backdrop-blur-sm`
- Logo: Gold accent color
- Links: `text-brand-gray hover:text-brand-white`
- Active: `text-brand-gold`

### Section Dividers
- Use thin gold lines: `border-t border-brand-gold/20`
- Alternating section backgrounds: `bg-white` / `bg-brand-offwhite` / `bg-brand-navy`

## Do's
- Use gold sparingly as an accent to maintain elegance
- Maintain generous whitespace for a premium feel
- Use uppercase tracking for labels and section tags
- Pair dark navy backgrounds with gold accents for hero sections
- Use subtle gradients on hero backgrounds

## Don'ts
- Don't use bright/neon colors
- Don't crowd elements — keep breathing room
- Don't use more than 2 font weights in a single section
- Don't use pure black (#000) — use brand-navy instead
- Don't use low-contrast text combinations
