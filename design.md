# ARTITECT MACHINERY Design System

## Brand Identity
- Brand: ARTITECT MACHINERY
- Style: Elegant, professional, industrial yet user-friendly
- Target: B2B industrial machinery buyers

## Color Palette
- Primary: Deep navy blue (#1B2A4A) - main brand color for headers/nav
- Primary Light: (#2C3E6B)
- Accent: Warm gold (#C8A44E) - CTAs, highlights, hover states
- Accent Light: (#E8C86A)
- Dark: #0F1923 - hero backgrounds, footer
- Surface: #F8F9FB - page backgrounds
- Card: #FFFFFF - cards, sections
- Text Primary: #1A1A2E - main body text
- Text Secondary: #5A6270 - muted text
- Text Light: #FFFFFF - text on dark backgrounds
- Border: #E2E6ED - subtle borders

## Typography
- Headings: 'Inter', sans-serif (already loaded)
- Body: 'Inter', sans-serif
- Font weights: 300 (light), 400 (regular), 500 (medium), 600 (semibold), 700 (bold), 800 (extrabold)
- Hero title: 800 weight
- Section headings: 700 weight
- Body text: 400 weight

## Spacing
- Section padding: py-20 md:py-28
- Container max width: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8
- Card padding: p-6 md:p-8
- Gap between cards: gap-6 md:gap-8

## Components

### Buttons
- Primary: bg-[#C8A44E] hover:bg-[#B8922E] text-white, rounded-md, font-medium
- Secondary: border-2 border-[#C8A44E] text-[#C8A44E] hover:bg-[#C8A44E] hover:text-white, rounded-md, font-medium

### Cards
- Background: white
- Border: subtle border border-gray-100
- Shadow: shadow-sm hover:shadow-md transition-shadow
- Border radius: rounded-lg

### Navigation
- Sticky header with glass effect (bg-white/90 backdrop-blur)
- Brand logo/text on left, links on right
- Mobile: hamburger menu

### Sections
- Alternating: light section (bg-[#F8F9FB]) and white section
- Hero: full-width with dark background overlay
