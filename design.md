# SSourcing China Visual Style

## Overview
Clean, trustworthy, international B2B style. Prioritize readability, whitespace, and restrained color usage.

## Colors
- Primary background: white
- Secondary background: slate-50
- Text primary: slate-900
- Text secondary: slate-600
- Text muted: slate-500
- Accent/CTA: slate-900 with white text
- Borders: slate-200
- Success: green-700
- Error: red-600

## Typography
- Font family: Inter
- Headings: semibold, tight tracking
- Body: regular, comfortable line height
- Small text: text-sm minimum for readability

## Layout
- Max width container: max-w-7xl
- Section padding: py-16
- Card radius: rounded-xl
- Card shadow: shadow-sm
- Grid gaps: gap-6 to gap-8

## Components
- Buttons: rounded-md, clear hover states
- Inputs: rounded-md, border slate-200, focus ring slate-400
- Cards: white background, slate-200 border, subtle shadow

## Imagery
- Use stock images sparingly for hero and section backgrounds
- Prefer clean iconography over decorative imagery
- Ensure all text remains readable over backgrounds

## Do
- Use consistent spacing and alignment
- Keep copy concise and scannable
- Use semantic HTML and accessible labels

## Don't
- Use low-contrast text
- Overcrowd sections
- Use exaggerated marketing language
- Rely on color alone to convey meaning
