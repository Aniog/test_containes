# SSourcing China Design System

## Brand Overview
A professional, trustworthy B2B sourcing agent website for global buyers looking for reliable China suppliers.

## Color Palette
- Primary Navy: `#0f172a` (slate-900) - headers, footer, trust
- Primary Blue: `#1e40af` (blue-800) - CTAs, links, accents
- Accent Teal: `#0d9488` (teal-600) - highlights, success states
- Background: `#ffffff` (white) - main surfaces
- Background Alt: `#f8fafc` (slate-50) - section alternation
- Text Primary: `#1e293b` (slate-800)
- Text Secondary: `#475569` (slate-600)
- Text Muted: `#64748b` (slate-500)
- Border: `#e2e8f0` (slate-200)

## Typography
- Headings: Inter, 700-800 weight, tight line-height (1.1-1.2)
- Body: Inter, 400 weight, 1.6 line-height
- Small/Labels: Inter, 500 weight, uppercase tracking wide for labels

## Spacing
- Section padding: py-20 md:py-28
- Container: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8
- Card gap: gap-6 md:gap-8
- Component internal spacing: p-6 md:p-8

## Components
- Buttons: rounded-lg, px-6 py-3, font-medium
  - Primary: bg-blue-800 text-white hover:bg-blue-900
  - Secondary: white bg, slate border, hover:bg-slate-50
- Cards: rounded-xl border border-slate-200 bg-white shadow-sm
- Icons: Lucide icons, slate/blue/teal colors
- Forms: rounded-lg border-slate-300 focus:ring-blue-800

## Visual Style
- Clean, spacious, corporate B2B feel
- Use real factory/QC/shipping visuals via data-strk-img
- Trust signals: numbers, certifications, process steps
- No exaggerated claims, professional tone

## Do's and Don'ts
- DO use generous whitespace
- DO keep text readable (slate-800 on white)
- DO use consistent section rhythm
- DON'T use bright neon colors
- DON'T use casual or playful fonts
- DON'T clutter layouts
