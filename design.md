# Design System — Green Nature Theme

## Color Palette
- Primary green: `#16a34a` (green-600)
- Dark green: `#14532d` (green-900)
- Medium green: `#15803d` (green-700)
- Light green: `#bbf7d0` (green-200)
- Accent green: `#4ade80` (green-400)
- Background light: `#f0fdf4` (green-50)
- Background white: `#ffffff`
- Text dark: `#14532d` (green-900)
- Text muted: `#166534` (green-800)
- Text on dark: `#ffffff`

## Typography
- Font: Inter (Google Fonts)
- Headings: font-bold, tracking-tight
- Body: font-normal, leading-relaxed
- Hero H1: text-5xl md:text-7xl font-extrabold
- Section H2: text-3xl md:text-4xl font-bold
- Card H3: text-xl font-semibold

## Borders & Radius
- Cards: rounded-2xl
- Buttons: rounded-full
- Inputs: rounded-xl
- Dividers: border-green-200

## Spacing
- Section padding: py-20 px-6
- Card padding: p-8
- Gap between cards: gap-8

## Shadows
- Cards: shadow-lg hover:shadow-xl transition
- Buttons: shadow-md hover:shadow-lg

## Component Styles
- Primary button: bg-green-600 text-white rounded-full px-8 py-3 font-semibold hover:bg-green-700
- Secondary button: border-2 border-green-600 text-green-700 rounded-full px-8 py-3 font-semibold hover:bg-green-50
- Card: bg-white rounded-2xl shadow-lg p-8 border border-green-100
- Nav: bg-white/90 backdrop-blur border-b border-green-100

## Do's
- Use green shades consistently throughout
- Use white cards on light green backgrounds
- Use dark green text on light backgrounds
- Use white text on dark green backgrounds
- Add subtle green gradients to hero sections

## Don'ts
- Don't use blue or purple accents
- Don't use dark/black backgrounds
- Don't mix too many different green shades in one section
