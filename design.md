# Strikingly Landing Page — Design System

## Brand Colors
- Primary: #7C3AED (purple-700 / violet)
- Primary Light: #8B5CF6 (violet-500)
- Primary Dark: #5B21B6 (violet-800)
- Accent: #A78BFA (violet-400)
- Dark BG: #0F0F0F
- Dark Surface: #1A1A2E
- Light BG: #FFFFFF
- Off-white BG: #F5F3FF (violet-50)

## Typography
- Font: Inter (Google Fonts)
- Hero heading: text-5xl md:text-7xl font-extrabold tracking-tight
- Section heading: text-3xl md:text-4xl font-bold
- Body: text-base text-gray-600
- Label: text-xs font-semibold uppercase tracking-widest text-violet-600

## Buttons
- Primary: bg-violet-600 hover:bg-violet-700 text-white rounded-full px-6 py-3 font-semibold
- Secondary: border border-white/30 text-white hover:bg-white/10 rounded-full px-6 py-3 font-semibold
- Outline: border border-violet-600 text-violet-600 hover:bg-violet-50 rounded-full px-6 py-3 font-semibold

## Gradients
- Hero gradient blobs: from-violet-600/30 to-purple-800/20
- CTA section: bg-gradient-to-br from-violet-900 via-purple-900 to-indigo-900

## Cards
- Light: bg-white rounded-2xl shadow-sm border border-gray-100 hover:shadow-md
- Dark: bg-white/5 rounded-2xl border border-white/10

## Spacing
- Section padding: py-20 md:py-28
- Container: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8

## Do's
- Use violet/purple consistently for all interactive elements
- Use dark (#0F0F0F) for Hero, CTA, Footer sections
- Use white or violet-50 for light sections
- Ensure all text has strong contrast

## Don'ts
- No orange or red accent colors
- No low-contrast text on colored backgrounds
