# SSourcing China - Design System

## Brand Colors
- Primary: #1E3A5F (deep navy blue - trust, professionalism)
- Primary Light: #2B5A8C (lighter navy for hover states)
- Accent: #C41E3A (Chinese red - for CTAs and highlights)
- Accent Hover: #A0182E
- Background: #F8F9FA (light gray)
- Card BG: #FFFFFF
- Text Primary: #1A1A2E
- Text Secondary: #6B7280
- Border: #E5E7EB
- Success: #059669

## Typography
- Headings: font-bold, tracking-tight
- Body: font-normal, leading-relaxed
- Font: Inter (loaded from Google Fonts)

## Spacing
- Section padding: py-16 md:py-24
- Container max-width: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8
- Card padding: p-6 md:p-8

## Visual Style
- Cards: bg-white rounded-xl shadow-sm border border-gray-100
- Buttons (Primary): bg-[#C41E3A] hover:bg-[#A0182E] text-white font-semibold px-6 py-3 rounded-lg transition-colors
- Buttons (Outline): border-2 border-[#1E3A5F] text-[#1E3A5F] hover:bg-[#1E3A5F] hover:text-white font-semibold px-6 py-3 rounded-lg transition-colors
- Section titles: text-3xl md:text-4xl font-bold text-[#1E3A5F] text-center mb-4
- Section subtitles: text-lg text-gray-600 text-center max-w-2xl mx-auto mb-12

## DO's
- Use clean, generous whitespace
- Use professional photography style imagery (factory, QC, shipping, products)
- Keep layout symmetrical and well-aligned
- Use consistent card-based layouts
- Use icons for service/feature lists
- Use subtle shadows and borders

## DON'Ts
- No overly bright/flashy colors
- No playful fonts
- No cluttered layouts
- No exaggerated marketing claims
- No low-quality stock photos
