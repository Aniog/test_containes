# ARTITECT MACHINERY — Design System

## Brand
- Name: ARTITECT MACHINERY
- Personality: precision engineering, industrial elegance, trustworthy, modern

## Color Palette
- Primary dark: `#0f172a` (slate-900) — headers, footer, primary text
- Primary accent: `#b45309` (amber-700) — CTAs, highlights, borders
- Accent hover: `#92400e` (amber-800)
- Surface light: `#f8fafc` (slate-50) — page background
- Surface card: `#ffffff` — cards
- Muted text: `#475569` (slate-600)
- Subtle border: `#e2e8f0` (slate-200)

## Typography
- Font family: Inter (Google Fonts)
- Headings: font-weight 700/800, tight letter-spacing
- Body: font-weight 400/500, line-height 1.6

## Spacing & Layout
- Max container width: 1280px (`max-w-7xl`)
- Section vertical padding: `py-16 md:py-24`
- Card radius: `rounded-xl`
- Button radius: `rounded-lg`

## Components
- Buttons: solid amber background with white text, rounded-lg, hover darker amber
- Cards: white background, subtle border, rounded-xl, soft shadow on hover
- Navigation: sticky top, white background, subtle bottom border

## Do's
- Use warm amber sparingly for emphasis and CTAs
- Keep plenty of whitespace for an elegant feel
- Use high-contrast text on all backgrounds
- Use responsive grids (1 col mobile, 2-3 col desktop)

## Don'ts
- Don't use neon or overly bright colors
- Don't clutter sections with too many elements
- Don't use low-contrast text on dark surfaces
