# MicroCosmos Design System

## Visual Identity
MicroCosmos is a science-focused website exploring the microscopic world. The design evokes the feeling of peering through a microscope — dark backgrounds, glowing teal/cyan accents, and clean scientific typography.

## Color Palette
- **Background (deep dark):** `bg-gray-950` (#030712)
- **Surface (card/section):** `bg-gray-900` (#111827)
- **Surface elevated:** `bg-gray-800` (#1f2937)
- **Border:** `border-gray-700` (#374151)
- **Primary accent (teal):** `text-teal-400` / `bg-teal-400` (#2dd4bf)
- **Secondary accent (cyan):** `text-cyan-400` / `bg-cyan-400` (#22d3ee)
- **Highlight (emerald):** `text-emerald-400` (#34d399)
- **Text primary:** `text-white` (#ffffff)
- **Text secondary:** `text-gray-300` (#d1d5db)
- **Text muted:** `text-gray-400` (#9ca3af)

## Typography
- **Font:** Inter (Google Fonts)
- **Display headings:** `text-5xl font-bold tracking-tight text-white`
- **Section headings:** `text-3xl font-bold text-white`
- **Card headings:** `text-xl font-semibold text-white`
- **Body text:** `text-base text-gray-300 leading-relaxed`
- **Labels/captions:** `text-sm text-gray-400`
- **Accent text:** `text-teal-400 font-medium`

## Spacing
- **Section padding:** `py-20 px-6` or `py-16 px-6`
- **Container max width:** `max-w-7xl mx-auto`
- **Card padding:** `p-6`
- **Grid gaps:** `gap-6` or `gap-8`

## Components

### Navbar
- Dark background: `bg-gray-950/90 backdrop-blur-md`
- Fixed at top: `fixed top-0 w-full z-50`
- Logo: teal accent color
- Nav links: `text-gray-300 hover:text-teal-400 transition-colors`
- Active link: `text-teal-400`

### Hero Section
- Full viewport height: `min-h-screen`
- Dark overlay on background image
- Large display heading with teal gradient
- Subtle animated glow effects

### Cards
- Background: `bg-gray-900 border border-gray-700`
- Rounded: `rounded-2xl`
- Hover: `hover:border-teal-500 transition-all`
- Image: rounded top corners, aspect ratio maintained

### Buttons
- Primary: `bg-teal-500 hover:bg-teal-400 text-gray-950 font-semibold px-6 py-3 rounded-full`
- Secondary: `border border-teal-500 text-teal-400 hover:bg-teal-500/10 px-6 py-3 rounded-full`

### Badges/Tags
- `bg-teal-500/20 text-teal-300 text-xs font-medium px-3 py-1 rounded-full`

## Do's
- Use dark backgrounds throughout for a consistent microscopic/scientific feel
- Use teal/cyan as the primary accent color
- Include plenty of images — this is a visual-first website
- Use subtle gradients and glows to add depth
- Keep text readable with high contrast against dark backgrounds

## Don'ts
- Don't use light backgrounds (white/light gray) as primary surfaces
- Don't use low-contrast text (e.g., gray-600 on gray-900)
- Don't use more than 2-3 accent colors
- Don't use overly decorative fonts — keep it clean and scientific
