# Velmora Fine Jewelry — Design System

## Visual Identity
- **Mood**: Quiet luxury, warm, editorial. Premium demi-fine jewelry — NOT loud, NOT discount-looking.
- **Direction**: Warm neutral/editorial scheme with deep refined base + warm metallic accents.

## Color Palette
| Token | Hex | Usage |
|-------|-----|-------|
| brand-cream | #FAF7F2 | Page background |
| brand-ivory | #F5F0E8 | Card backgrounds, alternate sections |
| brand-sand | #E8DFD0 | Borders, dividers, subtle backgrounds |
| brand-gold | #C9A96E | Primary accent, CTAs, highlights |
| brand-gold-light | #D4BA8A | Hover states, secondary gold |
| brand-gold-dark | #A88B4A | Active/pressed gold states |
| brand-charcoal | #2C2C2C | Primary text |
| brand-charcoal-light | #3D3D3D | Secondary text |
| brand-onyx | #1A1A1A | Dark sections, hero overlays |
| brand-warm | #8B7355 | Muted text, captions |
| brand-muted | #9C9488 | Placeholder text, disabled states |
| brand-warm-gray | #6B6560 | Tertiary text |

## Typography
- **Headings**: Cormorant Garamond (serif), weights 400-600
- **Body/UI**: Inter (sans-serif), weights 300-500
- **Product names**: UPPERCASE, letter-spacing 0.2em-0.25em, serif
- **Section headings**: serif, tracking wide, often uppercase on mobile

## Spacing & Layout
- Generous whitespace throughout
- Max content width: 1280px
- Section padding: 80px-120px vertical (desktop), 48px-64px (mobile)
- Hairline dividers: 1px brand-sand

## Buttons
- Primary: bg-brand-gold text-white, hover:bg-brand-gold-dark, rounded-none or subtle rounded
- Secondary: border-brand-gold text-brand-gold, hover:bg-brand-gold hover:text-white
- All buttons: uppercase, tracking-wider, font-sans, font-medium, smooth transitions

## Cards
- Minimal: no border or shadow by default
- Hover: subtle shadow or image swap
- Product cards: image top, name + price below, serif uppercase name

## Images
- Large, editorial, warm-lit
- Jewelry on dark/neutral backgrounds
- Hover reveals second image on product cards

## Animations
- Subtle, smooth transitions (200-400ms ease)
- Fade-in on scroll
- No bouncy or playful animations

## Do's
- Use serif for all headings and product names
- Keep generous whitespace
- Use gold accents sparingly
- Ensure strong text contrast on all backgrounds

## Don'ts
- Don't use bright/saturated colors
- Don't use rounded/bubbly UI elements
- Don't crowd elements — let them breathe
- Don't use generic e-commerce patterns (badges, loud sales banners)
