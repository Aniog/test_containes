# ARTITECT MACHINERY - Design Guidelines

## Brand Identity
- Brand: ARTITECT MACHINERY
- Industry: Industrial machinery (sheet metal folding machines)
- Tone: Elegant, professional, trustworthy, precision

## Color Palette
- Primary: `#1a1a2e` (deep navy) - headers, primary backgrounds
- Secondary: `#16213e` (dark blue) - section backgrounds
- Accent: `#c9954a` (warm gold) - CTAs, highlights, decorative elements
- Light: `#f8f6f0` (warm off-white) - light backgrounds
- Text primary: `#1e1e1e` (near black) - body text
- Text muted: `#6b7280` (gray) - secondary text
- Text on dark: `#f3f4f6` (light gray) - text on dark backgrounds
- White: `#ffffff` - cards, overlays

## Typography
- Font family: Inter (system-ui, sans-serif fallback)
- Headings: Bold (700) or Extra Bold (800)
- Body: Regular (400)
- Navigation: Medium (500)
- Scale: responsive, with large hero headings

## Spacing
- Section padding: py-20 md:py-28
- Container max-width: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8
- Element gaps: 6-8 spacing between major elements

## Visual Style
- Rounded corners: rounded-lg (8px) for cards, rounded-full for buttons
- Shadows: subtle shadows on cards (shadow-md, shadow-lg on hover)
- Borders: thin, subtle borders (border border-gray-100)
- Transitions: smooth hover transitions (transition-all duration-300)
- Gradients: subtle gradient overlays on hero images

## Navigation
- Fixed/sticky top nav
- Transparent or semi-transparent background
- Smooth scroll to sections
- Gold accent on hover/active

## Components
- Cards: white background, subtle shadow, rounded corners, hover elevation
- Buttons: filled (gold accent) or outlined, rounded-full
- Section headings: centered, with ornamental accent line below
- Product cards: image + title + brief description

## Do's
- Use warm gold sparingly for accent
- Keep layouts clean and uncluttered
- Use high-quality imagery of machinery
- Maintain consistent spacing
- Use responsive grid layouts

## Don'ts
- Don't use overly bright or neon colors
- Don't overcrowd sections
- Don't use overly casual language
- Don't use too many different font sizes
