# Velmora Fine Jewelry - Design System

## Visual Identity

### Color Palette
- **Primary Background**: `#FAF9F7` (warm off-white/cream)
- **Secondary Background**: `#F5F3EF` (warm light gray)
- **Primary Text**: `#1A1814` (warm black)
- **Secondary Text**: `#6B635A` (warm gray)
- **Accent Gold**: `#C9A962` (warm gold)
- **Accent Gold Hover**: `#B8954F` (darker gold)
- **Border Color**: `#E8E4DE` (warm light border)
- **Card Background**: `#FFFFFF` (white)

### Typography
- **Headings**: Cormorant Garamond (serif) - elegant, editorial
  - H1: 56px / 64px line-height / font-weight 500
  - H2: 40px / 48px line-height / font-weight 500
  - H3: 28px / 36px line-height / font-weight 500
  - Product names: 18px / uppercase / 0.1em letter-spacing / font-weight 500
  
- **Body**: Manrope (sans-serif) - clean, modern
  - Body: 16px / 26px line-height / font-weight 400
  - Small: 14px / 22px line-height
  - Button: 14px / uppercase / 0.05em letter-spacing / font-weight 500

### Spacing System
- Section padding: 80px vertical (desktop), 48px (mobile)
- Container max-width: 1280px
- Grid gap: 24px (desktop), 16px (mobile)
- Component padding: 24px

### Visual Effects
- **Shadows**: 
  - Card hover: `0 20px 40px rgba(26, 24, 20, 0.08)`
  - Button: `0 4px 12px rgba(201, 169, 98, 0.25)`
- **Borders**: 1px solid, hairline dividers
- **Transitions**: 0.3s ease for all interactive elements

### Components

#### Buttons
- **Primary**: Gold background, white text, full rounded corners (4px)
- **Secondary**: Transparent with gold border, gold text
- **States**: Hover darkens gold, subtle shadow appears

#### Product Cards
- White background
- Image with hover zoom effect
- Second image reveals on hover
- Quick "Add to Cart" button appears on hover
- Product name in uppercase serif with letter-spacing
- Price below in regular weight

#### Navigation
- Transparent over hero, solid on scroll
- Logo in serif font
- Clean icon buttons for search/cart
- Mobile hamburger menu

#### Dividers
- Hairline style: 1px solid #E8E4DE
- Used between sections and content blocks
