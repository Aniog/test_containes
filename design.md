# MicroCosmos Design System

## Visual Style
A nature-inspired design celebrating the microscopic world of trees and their ecosystems.

## Colors
- Primary: Deep Forest Green (#1a4d3a) - for headers and primary elements
- Secondary: Moss Green (#4a7c59) - for secondary elements and accents
- Accent: Golden Amber (#d4a574) - for highlights and call-to-action elements
- Background: Soft Cream (#faf8f5) - main background
- Text Primary: Dark Charcoal (#2d3748) - main text
- Text Secondary: Medium Gray (#718096) - secondary text
- Light Green: (#e6f3e6) - for subtle backgrounds and cards

## Typography
- Headings: Use font-bold and font-semibold with generous spacing
- Body: Use font-normal with good line-height for readability
- Sizes: text-4xl/5xl for main headings, text-2xl/3xl for section headings, text-lg for body

## Layout & Spacing
- Use generous padding: p-8, p-12, p-16 for sections
- Consistent margins: mb-8, mb-12 for vertical spacing
- Grid layouts: grid-cols-1 md:grid-cols-2 lg:grid-cols-3 for responsive galleries
- Container max-width: max-w-7xl mx-auto for content centering

## Components
- Cards: Rounded corners (rounded-xl), subtle shadows (shadow-lg), light backgrounds
- Buttons: Rounded (rounded-lg), bold text, hover effects with scale transforms
- Images: Rounded corners, aspect-ratio containers, hover effects with gentle scaling
- Sections: Full-width backgrounds with contained content

## Visual Effects
- Subtle hover animations: hover:scale-105 transition-transform duration-300
- Gentle shadows: shadow-md, shadow-lg for depth
- Gradient overlays for hero sections
- Smooth transitions for interactive elements

## Do's
- Use nature-inspired imagery extensively
- Maintain consistent spacing and typography
- Ensure high contrast for accessibility
- Use rounded corners for a modern, organic feel
- Implement smooth hover effects

## Don'ts
- Avoid harsh, artificial colors
- Don't use small, cramped spacing
- Avoid low-contrast text combinations
- Don't overuse animations or effects
- Avoid cluttered layouts
