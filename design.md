# SSourcing China - B2B Website Design System

## Brand Colors
- Primary Blue: #1E3A5F (dark navy - trust, professionalism)
- Primary Light: #2563EB (bright blue - CTAs, links, accents)
- Accent Red: #DC2626 (urgency, quality alerts - use sparingly)
- Success Green: #16A34A (positive indicators, checkmarks)
- Background: #FFFFFF (white)
- Section Alt BG: #F8FAFC (light gray sections)
- Footer BG: #0F172A (very dark navy)
- Text Primary: #1E293B (dark slate)
- Text Secondary: #64748B (medium slate)
- Border: #E2E8F0 (light borders)

## Typography
- Font: Inter (Google Font)
- Headlines: font-bold (700), tracking-tight
- Body: font-normal (400), leading-relaxed
- Hero Headline: text-4xl md:text-5xl lg:text-6xl
- Section Title: text-3xl md:text-4xl font-bold
- Subtitle: text-lg md:text-xl text-secondary

## Spacing
- Section padding: py-16 md:py-24
- Container max-width: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8
- Card padding: p-6 md:p-8
- Gap between cards: gap-6 md:gap-8

## Component Styles
- Cards: bg-white rounded-xl border border-border shadow-sm hover:shadow-md transition-shadow
- Buttons Primary: bg-primary-light hover:bg-primary-dark text-white font-semibold rounded-lg px-6 py-3
- Buttons Secondary: border-2 border-primary-light text-primary-light hover:bg-primary-light hover:text-white font-semibold rounded-lg px-6 py-3
- Form Inputs: border border-border rounded-lg px-4 py-3 focus:ring-2 focus:ring-primary-light focus:border-transparent
- Badges/Tags: bg-primary/10 text-primary text-xs font-medium px-3 py-1 rounded-full

## Image Style
- Hero background: dark overlay (bg-gradient-to-r from-primary/90 to-primary/70), factory/warehouse imagery
- Service icons: clean line icons (from Lucide)
- Case study images: 16:9 ratio
- Team/office photos: clean, professional

## Layout Rules
- DO use generous whitespace
- DO use consistent padding
- DO keep CTAs prominent with the bright blue
- DON'T use playful or casual styles
- DON'T use gradients except for hero overlay
- DON'T use rounded corners larger than 12px
- DON'T use shadows that are too heavy

## Tone
- Professional, trustworthy, international
- Clear benefits-oriented language
- Data points and numbers to build credibility
- No exaggerated claims or hype
