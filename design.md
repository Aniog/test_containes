# ARTITECT MACHINERY Website Design Guidelines

## Brand Identity
- **Brand Name**: ARTITECT MACHINERY
- **Industry**: Industrial metal folding/folder machines
- **Products**: Double folding machine, double folder, sheet metal folder, sheet metal folding machine, metal folder, metal folder machine, metal folding machine
- **Style Direction**: Elegant, professional, modern industrial - clean lines that reflect precision engineering

## Color Palette
- **Primary**: #1A1A2E (Deep Navy) - Trust, professionalism, industrial strength
- **Secondary**: #16213E (Dark Blue) - Depth, reliability
- **Accent**: #E94560 (Coral Red) - Energy, call-to-action, machinery highlights
- **Light Accent**: #0F3460 (Steel Blue) - Industrial feel
- **Background Light**: #F8F9FA (Off-White)
- **Background Dark**: #0D0D1A (Near Black)
- **Text Primary**: #1A1A2E
- **Text Secondary**: #6B7280
- **Text Light**: #FFFFFF

## Typography
- **Headings**: Inter, 700-800 weight, tracking tight
- **Body**: Inter, 400-500 weight, relaxed line height
- **Accent Text**: Inter, 600 weight, uppercase with letter-spacing

## Visual Style
- Clean, geometric layouts reflecting precision engineering
- Generous whitespace for elegant breathing room
- Subtle gradients and shadows for depth
- Strong horizontal/vertical lines (like machine guides)
- Professional imagery with industrial equipment
- Smooth animations with industrial precision

## Layout Principles
- Hero sections with bold typography and subtle background elements
- Card-based product displays with clean borders
- Grid layouts for product categorization
- Clear visual hierarchy with section numbering
- Footer with comprehensive navigation

## UI Components
- **Buttons**: Rounded corners (8px), solid fills, hover animations
- **Cards**: Clean white backgrounds, subtle shadows, hover lift effects
- **Icons**: Lucide React icons, consistent sizing
- **Section Dividers**: Subtle lines or geometric patterns
- **Navigation**: Sticky header with blur backdrop

## Animation Guidelines
- Fade-in on scroll for sections
- Hover scale/lift effects on cards
- Smooth transitions (300-400ms ease)
- Subtle parallax on hero backgrounds

## Do's and Don'ts
- DO: Use bold typography for headings
- DO: Keep layouts clean with generous spacing
- DO: Use accent colors sparingly for CTAs
- DON'T: Use too many colors
- DON'T: Clutter the design
- DON'T: Use overly decorative elements
- DO: Maintain industrial, professional aesthetic
