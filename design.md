# Design System — World Cup Website

## Theme
Dark, energetic, football-inspired. Deep navy/black backgrounds with vibrant gold and red accents. Clean, modern typography with bold headings.

## Colors
- Background primary: `bg-[#0a0e1a]` (deep navy black)
- Background secondary: `bg-[#111827]` (dark navy)
- Background card: `bg-[#1a2235]` (card surface)
- Accent gold: `text-[#f5c518]` / `bg-[#f5c518]` (FIFA gold)
- Accent red: `text-[#e63946]` / `bg-[#e63946]` (energy red)
- Accent green: `text-[#2dc653]` (pitch green)
- Text primary: `text-white`
- Text secondary: `text-gray-300`
- Text muted: `text-gray-500`
- Border: `border-gray-700` / `border-[#f5c518]/30`

## Typography
- Font: Inter (Google Fonts)
- Hero heading: `text-5xl md:text-7xl font-black tracking-tight`
- Section heading: `text-3xl md:text-4xl font-bold`
- Card title: `text-xl font-semibold`
- Body: `text-base text-gray-300`
- Caption: `text-sm text-gray-500`

## Spacing
- Section padding: `py-16 md:py-24`
- Container: `max-w-7xl mx-auto px-4 md:px-8`
- Card padding: `p-6`
- Gap between cards: `gap-6`

## Borders & Shadows
- Card border: `border border-gray-700/50`
- Card hover: `hover:border-[#f5c518]/50`
- Card shadow: `shadow-lg shadow-black/40`
- Glow effect: `shadow-[0_0_30px_rgba(245,197,24,0.15)]`

## Components
- Buttons: rounded-full, gold background, dark text, hover scale
- Cards: rounded-2xl, dark bg, subtle border, hover lift
- Badges: rounded-full, small, colored backgrounds
- Nav: sticky top, blur backdrop, semi-transparent

## Do's
- Use bold, high-contrast text on dark backgrounds
- Use gold accents for highlights and CTAs
- Use gradient text for hero headings
- Use grid layouts for desktop, single column for mobile

## Don'ts
- No light backgrounds in main sections
- No low-contrast text (e.g. gray on gray)
- No overly complex animations
- No serif fonts
